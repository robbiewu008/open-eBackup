Obtaining the CA Certificate of a Storage Device

To ensure the security of interaction between the OceanProtect and a storage device, you can import the storage device certificate to the system. This operation is optional.

Obtain the CA certificate of a storage device as follows:

  • Method 1: Obtain the certificate from the storage device administrator.
  • Method 2: Download the certificate of Huawei storage devices by yourself. For other storage devices, obtain the certificate from the storage device administrator. The following gives an example of the procedure for obtaining the CA certificate of a Huawei OceanStor storage device.
    1. Enter the DeviceManager login address of a storage device whose certificate is to be obtained in the address box of a browser. The browser displays a message indicating that the login is insecure (Microsoft Edge displays a message indicating that the certificate is incorrect).

      The displayed information varies depending on the browser. The following uses Google Chrome 84 as an example.

    2. Click Not secure on the left of the URL address box.

    3. In the dialog box that is displayed, click Certificate is not valid.

    4. In the displayed dialog box, verify that the issuer is Huawei on the General tab page.

    5. Click the Details tab and click Export.

    6. Change the file name extension to .pem, set Save as type to Base64-encoded ASCII, Single certificate, and click Save. Save the exported certificate file to the local PC.
