步骤1：创建卷

执行卷保护前请先创建卷。

操作步骤

  1. 选择“保护 > 文件系统 > 卷”。
  2. 单击“创建”。
  3. 在弹出的“创建”对话框中为单个主机创建卷。

    相关参数说明如表1所示。

    表1 创建卷参数说明

    参数

    说明

    名称

    自定义卷名称。

    客户端

    选择要进行卷保护的客户端。

    操作系统备份

    勾选后，支持在创建/修改卷保护操作时开启/关闭操作系统备份开关，若需要备份所有系统卷的数据，则必须选择所有系统卷。

    备份类型

    说明：

    仅Windows OS存在此参数。

    选择备份类型：卷、裸机。

    说明：

    目前只支持备份NTFS文件系统的卷

    卷

    选择要保护的卷。

  4. 单击“确定”。
