Modifying a Resource Set

You can modify a resource set if you need to change its name or description or reselect resources for it.

Precautions

The default resource sets and default public resource set cannot be modified.

Procedure

  1. Choose System > Security > RBAC > Resource Sets.
  2. (Optional) Click next to Resource Set to search for a resource set.
  3. In the row of the resource set to be modified, click Modify on the right.
  4. On the Modify Resource Set page, you can modify the resource set name and description, and reselect resources.
  5. Click OK.
