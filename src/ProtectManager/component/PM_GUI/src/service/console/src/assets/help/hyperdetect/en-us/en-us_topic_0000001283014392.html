Performing Ransomware Detection Snapshot-based Backup Manually

If you want to perform a backup immediately, you can manually perform the backup.

Context

  • During manual backup, parameters (except Automatic Retry) defined in the SLA are applied to the manual backup job.
  • Copies generated by manual backup are retained for the duration defined in the SLA.

Procedure

  1. Choose Protection > Storage > Local File System.
  2. In the local file system list, select the file system that you want to protect and choose More > Manual Backup.

    You can also select multiple local file systems for batch manual backup.

  3. Optional: Enter a snapshot name and click OK.
