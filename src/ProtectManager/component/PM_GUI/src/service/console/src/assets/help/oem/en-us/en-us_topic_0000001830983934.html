Viewing Reports

Procedure

  1. Choose Insight > Reports.
  2. Click Create. Table 1 describes the parameters.

    Table 1 Report parameters

    Parameter

    Description

    Name

    Name of the report.

    Report Status

    Status of the report.

    [Value range]

    Generating, Generation Failed, and Generated

    Report Type

    Type of the report.

    • Storage Space Report
    • Backup Job Report
    • Restoration Job Report

    Data Source

    Local cluster node or external cluster node that has been connected to the product.

    Statistical Range

    Time range within which data is collected. If the report type is Storage Space Report, -- is displayed.

    Created At

    Time when the report is created.
