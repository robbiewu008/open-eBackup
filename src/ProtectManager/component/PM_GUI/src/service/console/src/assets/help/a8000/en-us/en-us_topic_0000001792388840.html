Step 2: Creating an HBase Backup Set

Before performing HBase backup set protection, create a backup set. An HBase backup set is a collection of its namespaces and tables.

Procedure

  1. Choose Protection > Big Data > HBase.
  2. Click Backup Sets.
  3. Click Create to create an HBase backup set.

    Table 1 describes the related parameters.
    Table 1 HBase backup set parameters

    Parameter

    Description

    Name

    Customize an HBase backup set name.

    Cluster

    Select the HBase cluster to be protected.

    Before selecting an HBase cluster, ensure that the cluster to be protected has been registered with the OceanProtect.

    Namespace

    Select the namespace to be protected in the HBase cluster.

    You can select one or more namespaces to create a backup set.

    Table

    Select the table(s) to be protected in the namespace.

  4. Click OK.
