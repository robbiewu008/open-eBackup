Managing Logs

By viewing debug logs, you can understand the error information and prompt information that are triggered when you operate or design a template.

Procedure

  1. Choose System > Log Management.
  2. In the System Debug Log Settings area, select the level of logs to be exported from the drop-down list on the right of Log Level.

    You can export the debug logs at the selected level (and above).

  3. Click OK.
  4. In the Log Export area, select the node and software module whose logs are to be exported from the Node Name and Software Module Name drop-down lists, respectively.

    The exported logs include system debug logs and system running parameters.

  5. Click Export to export debug logs.
