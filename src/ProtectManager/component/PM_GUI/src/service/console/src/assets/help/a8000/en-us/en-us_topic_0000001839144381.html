Exporting Records

The OceanProtect allows you to export system logs, agent logs, detection reports, and copy files.

Procedure

  1. Choose System > Log > Export Records .
  2. On the Export Records page, view, download, or delete export records.

    • View export records. Table 1 describes the parameters.
      Table 1 Export records parameters

      Parameter

      Description

      Node

      Local cluster node where the exported object resides.

      Name

      Name of the exported object.

      Type

      Type of the exported object.

      The value can be System Log, Config Files, Copy File, Detection Report, or Agent Log.

      Size

      Size of the exported object.

      Status

      Export status. The value can be Exporting, Succeeded, Failed, or Partial Success.

      Created At

      Time when an object is exported.

      Expired At

      Expiration time of the exported object.
    • Download export records.

      Locate the row that contains the export record to be downloaded, and choose More > Download.

    • Delete export records.
      1. Locate the row that contains the export record to be deleted, and choose More > Delete.
      2. In the Danger dialog box that is displayed, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
