Enabling the SFTP Service

The OceanProtect allows you to enable the SFTP service to provide storage space for the SFTP client.

Precautions

  • If an IP address already exists on the port, ensure that the entered IP address is in the same network segment as the IP address.
  • You can set WORM parameters when enabling the SFTP service for the first time. If the SFTP service has been enabled, WORM parameters cannot be set when you enable the SFTP service again.

Procedure

  1. In the navigation pane, choose System > Settings > SFTP Service.
  2. On the SFTP Service tab page, click on the right to enable the SFTP service.

    In the displayed Enable SFTP dialog box, set related parameters by referring to Table 1.
    Table 1 SFTP parameters

    Parameter

    Description

    IP Address Type

    This parameter can be set to IPV4 or IPV6.

    IPv4 Address

    This parameter is displayed when IP Address Type is set to IPV4.

    IP address of the SFTP service.

    The IP address and subnet mask are used by the SFTP client to connect to the SFTP server of the OceanProtect. Ensure that the IP address can communicate with the SFTP client and does not conflict with IP addresses in the network segment of the SFTP client.

    Subnet Mask

    Subnet mask of the IPv4 address.

    IPv6 Address

    This parameter is displayed when IP Address Type is set to IPV6.

    IP address of the SFTP service.

    The IP address and prefix length are used by the SFTP client to connect to the SFTP server of the OceanProtect. Ensure that the IP address can communicate with the SFTP client and does not conflict with IP addresses in the network segment of the SFTP client.

    Prefix Length

    Prefix length of the IPv6 address.

    Configure Service Port

    Enable VLAN

    This option is deselected by default. Before selecting this option, ensure that the VLAN has been configured for the service. After this option is selected, the configured service ports will apply to the VLAN.

    Port Type

    Available options include Ethernet port and Bond port.

    Port Selection

    This parameter is displayed after Enable VLAN is selected or Port Type is set to Bond port. Available options include Creating a port and Reusing the configured port.

    Available Ports

    Available front-end ports.

    Selected Ports

    Selected front-end ports.

    VLAN ID

    This parameter is available after you select Creating a port. Enter the used VLAN ID.

    WORM

    Mode

    This parameter is displayed after WORM is enabled.

    • Compliance

      For files within the protection period, super administrators, common administrators, and common users cannot modify or delete the files. For files whose protection period expires, super administrators, common administrators, and common users cannot modify the files but can delete them.

    • Enterprise

      For files within the protection period, common administrators and users cannot modify or delete the files, but privileged users can delete them. For files whose protection period expires, super administrators, common administrators, and common users cannot modify the files but can delete them.

    Protection Period

    This parameter is displayed after WORM is enabled.

    Set file protection time period. The value ranges from 0 to 70 years.

    Automatic Lockout

    This parameter is displayed after WORM is enabled.

    After this function is enabled, a file automatically enters the locked state if not being modified within Lockout Wait Time (hours). The file in the locked state is protected. You can only read the file, but cannot modify, rename, or delete it.

    Lockout Wait Time

    This parameter is displayed after WORM is enabled.

    If a file is not modified within the lockout wait time, it automatically enters the locked state. A locked file can only be read, but cannot be modified or deleted. Value range: 1 minute to 10 years.

    Automatic Deletion

    This parameter is displayed after WORM is enabled.

    After this function is enabled, the system automatically deletes files whose protection period expires.

  3. Click OK.

Follow-up Procedure

On the SFTP Service tab page, click on the right. In the dialog box that is displayed, click OK to disable the SFTP service.

Before disabling the SFTP service, ensure that no SFTP user uses the SFTP service to transfer files and that all SFTP users do not need to use the SFTP service.
