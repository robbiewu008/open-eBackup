Creating a Backup Storage Unit

This section describes how to create a backup storage unit.

Procedure

  1. Choose System > Infrastructure.
  2. Click Backup Storage Unit Groups.
  3. Click Backup Storage Units.
  4. Click Create to create a backup storage unit.

    Table 1 describes the related parameters.
    Table 1 Parameters for adding a backup storage unit

    Parameter

    Description

    Name

    User-defined name of the backup storage unit.

    Device Type

    Select OceanProtect at present.

    Storage Device

    Select the created backup storage device.

    Storage pool

    Select the storage pool of the backup storage device.

  5. Click OK.
