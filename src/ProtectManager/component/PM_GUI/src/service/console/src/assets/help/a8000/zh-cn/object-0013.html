备份对象存储

您可以根据本章节指引完成备份配置。也支持通过智能助手提供伴随式教学和流程引导，在用户操作过程中及时提供指引，降低操作难度。您登录OceanProtect管理界面后，单击界面上方的，在打开的“智能助手”页面，单击“了解如何进行备份操作”，选择待保护的资源类型后单击“下一步”，即可根据备份配置引导逐步完成本章节的备份配置。

  • 步骤1：注册对象存储
  • 步骤2：创建对象集合
  • 步骤3：（可选）创建限速策略
  • 步骤4：（可选）开启备份链路加密开关（适用于OceanProtect X系列备份一体机和OceanProtect E1000（备份存储为OceanProtect））
  • 步骤5：创建备份SLA
  • 步骤6：开启NFSv4.1服务（适用于OceanProtect X系列备份一体机和OceanProtect E1000（备份存储为OceanProtect））
  • 步骤7：执行备份
