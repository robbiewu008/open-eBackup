Step 4: Enabling the sysadmin Permission

Before performing backup and restoration for SQL Server, you must enable the sysadmin permission of the database user. Otherwise, the backup and restoration will fail.

Procedure

  1. Log in to the SQL Server database host.
  2. Enable the sysadmin permission of the NT AUTHORITY\SYSTEM user.
