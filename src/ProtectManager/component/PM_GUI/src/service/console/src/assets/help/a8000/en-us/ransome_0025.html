Viewing Detection Details of All Resources

This section describes how to view the detection details of all resources.

Procedure

  1. Choose Explore > Ransomware Protection.
  2. In the Overview area, view the number and ratios of copies in each detection status in a pie chart.

    View the number of copies of different types of resources in each detection status in a bar chart.
