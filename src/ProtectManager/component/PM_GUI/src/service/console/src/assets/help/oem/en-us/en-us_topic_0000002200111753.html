Step 3: Registering a Nutanix Virtualization Environment

Before performing Nutanix backup and restoration, you must register the Nutanix virtualization environment with the product.

Context

After registering the Nutanix virtualization environment, you can view information about Nutanix VMs on the management page and protect them.

Prerequisites

  • A storage pool has been added to the Nutanix platform to be protected, and the target host to be protected and the host where the client is installed use the same storage pool.
  • You have registered a user on the Nutanix platform and added user permissions. Otherwise, the backup fails due to insufficient permissions.

Procedure

  1. Choose Protection > Virtualization > Nutanix.
  2. Click Register and register the Nutanix virtualization platform.

    Table 1 lists the Nutanix virtualization platform registration information.
    Table 1 Nutanix virtualization platform registration information

    Parameter

    Description

    Name

    User-defined name of the Nutanix virtualization platform.

    IP Address/Domain Name

    Management IP address or domain name of the Nutanix virtualization platform.

    Port

    Port number used by the product to communicate with the Nutanix virtualization platform.

    The default port number is 9440.

    Username

    Register a username and password, and add user permissions on the Nutanix platform. For details, see Step 1: Registering a User with the Nutanix Virtualization Platform and Adding User Permissions.

    Password

    Clients

    Select the client used for backup and restoration.

    Resource Scanning Interval

    Interval for the system to automatically scan for resources.

    The default interval is 1 hour, and the value ranges from 1 to 72 hours.

    Certificate Verification

    This function is enabled by default. When you register a Nutanix environment to be protected, the system verifies the information about the devices to be accessed. After this function is disabled, security risks may exist in the communication between the system and Nutanix. Exercise caution when performing this operation.

    Certificate

    After Certificate Verification is enabled, import the certificate obtained in Step 2: Obtaining the Nutanix Certificate.

    Revocation List

    After the certificate revocation list (CRL) is imported, the system checks whether the client certificate has been revoked. If the certificate has been revoked, cluster registration fails.

  3. Click OK.

    After the registration is successful, information about the registered Nutanix virtualization environment is displayed on the left of the page.
