Adding a Report

This section describes how to generate a report if you have performed real-time detection or intelligent detection on the file system of a storage device and want to view the detection result within a specified time range.

Procedure

  1. On the top menu bar, choose Data Security > Report.
  2. Click Add and set parameters for adding a report. Table 1 lists the parameters.

    Table 1 Parameters for adding a report

    Parameter

    Description

    Name

    Customize a custom report name.

    Statistics Range

    Select the statistical range of the report.

    Storage Device

    Select a storage device that has been added to the system.

    Tenant

    Select the vStore and file system of the storage device.

    File Systems

  3. Click OK.
