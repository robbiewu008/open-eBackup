步骤2：添加存储设备

执行NAS文件系统备份恢复前，需要将待保护的NAS文件系统所在的存储设备添加至OceanProtect。

背景信息

OceanProtect支持对以下存储设备中的NAS文件系统进行保护。

  • OceanStor Dorado V7
  • OceanStor V7
  • OceanStor Dorado 6.x
  • OceanStor 6.x
  • OceanProtect 1.3.0及之后版本

注意事项

  • 当OceanStor Dorado存储设备版本低于6.1.3时，不支持通过添加存储设备的方式发现NAS文件系统并对文件系统进行保护，需要通过注册NAS共享的方式保护NAS文件系统。注册NAS共享的详细操作，请参见步骤5：注册NAS共享（适用于除OceanStor V5/OceanStor Pacific/NetApp ONTAP以外的存储设备）。
  • 当存储设备为OceanStor Dorado 6.1.3及以上版本或者OceanStor 6.1.3及以上版本，请确保存储设备已激活的License包含“远程复制（HyperReplication）”。

操作步骤

  1. 选择“保护 > 文件系统 > 存储设备”。

    对于1.5.0版本，请选择“保护 > 文件服务 > 存储设备”。

  2. 单击“添加设备”，添加存储设备。
  3. 选择设备类型。

    • OceanStor Dorado V7
    • OceanStor V7
    • OceanStor Dorado 6.x
    • OceanStor 6.x
    • OceanProtect 1.3.0及之后版本

  4. 按界面要求配置参数。参数说明如表1所示。

    表1 添加设备参数说明

    参数

    说明

    设备名称

    用户自定义设备名称。

    IP地址

    存储设备的管理IP地址。

    端口

    OceanProtect与存储设备的管理平面通信的端口号。

    默认端口号为8088。

    用户名

    存储设备管理员角色的用户名及密码。请确保该用户具有Restful的登录方式，且已完成密码初始化。

    说明：

    请使用管理员角色的用户，否则可能导致备份该设备下的资源失败。

    密码

    校验证书

    默认开启，请导入步骤1：获取存储设备CA证书章节中获取的存储设备证书以确保OceanProtect与存储设备交互的安全性。关闭后，系统与存储设备通信可能存在安全风险，请谨慎操作。

    指定复制端口

    默认为关，当开关打开时，用户可以自行选择用于NAS文件系统备份的复制逻辑端口。当存储设备配置了双活时，此开关必须打开，且不允许选择双活正在使用的逻辑端口，否则会导致NAS文件系统备份失败。

  5. 单击“确定”。

    添加成功后，已添加的存储设备信息将在“存储设备”页面显示。

    OceanProtect将自动扫描存储设备中的NAS文件系统，并在“保护 > 文件系统 > NAS文件系统”页面展示。对于1.5.0版本，则在“保护 > 文件服务 > NAS文件系统”页面展示。当注册的设备新增了文件系统时，可在“存储设备”页面中，设备所在行选择“更多 > 资源扫描”更新文件系统资源。
