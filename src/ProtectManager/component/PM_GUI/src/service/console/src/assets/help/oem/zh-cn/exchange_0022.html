配置邮箱备份与恢复账户

操作步骤

  1. 新建一个邮箱备份恢复账户，并为该用户创建一个可用邮箱，创建完成后该账户会自动成为域用户的新账户。

    • Exchange 2010：登录Exchange代理端主机，打开Exchange管理控制台，展开“收件⼈配置”，右键单击“邮箱”，选择“新建邮箱”，根据提示创建⼀个新的Exchange邮箱，用户名如“MailBackupUser”。
    • Exchange 2013/2016/2019：登录Exchange Web管理中心,在收件⼈界⾯单击新建邮箱,根据提示创建⼀个新的Exchange邮箱，用户名如MailBackupUser。

  2. 将新创建的用户添加为域管理员的成员。

    在所属域控制器上，打开“Active Directory 用户和计算机”，双击刚创建的账⼾，选择“隶属于”选项卡，单击“添加”，并将此用户添加到域管理员组“Domain Admins”，单击“确定”。

  3. 将MailBackupUser加入到Discovery Management、Organization Management、Public Folder Management安全组中,在Members选项中单击"+"新增成员。

  4. 给用户MailBackupUser赋予邮箱访问权限，执行如下命令进行设置。

    Get-Mailbox | Add-MailboxPermission –User "MailBackupUser" -AccessRights FullAccess
