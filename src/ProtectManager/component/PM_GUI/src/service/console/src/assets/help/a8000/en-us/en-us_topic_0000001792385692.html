Single Resource

This section uses VMware as an example to describe how to view the detection details of a single resource type.

Procedure

  1. Choose Explore > Ransomware Protection.
  2. Click VMware.
  3. View the detection status of all VMware resources associated with ransomware protection policies or of a single VMware resource.

    • Viewing the detection status of all VMware resources associated with the ransomware protection policy
      In the VMware resource list, view the detection status of all resources. Table 1 describes the parameters.
      Table 1 Resource detection details

      Parameter

      Description

      Name

      Resource name.

      Location

      Resource location.

      Associated Policies

      Name of the ransomware protection policy associated with the resource.

      Copy

      Number of backup copies and replica copies of the resource associated with the ransomware protection policy.

      NOTE:

      Click Copy. In the displayed dialog box, click More in the row of a copy and then perform recovery, instant recovery, live mount, retention policy modification, or deletion for the copy. For details, see Managing VMware Copies in the OceanProtect Appliance 1.5.0-1.6.0 Data Backup Feature Guide (for VMware).

      Preparing

      Number of copies whose detection status is Preparing. After the ransomware protection policy is scheduled or manual detection is performed, the detection status of the copy changes to Preparing.

      Detecting

      Number of copies whose detection status is Detecting. After ransomware detection starts, the detection status of the copy changes to Detecting.

      Uninfected

      Number of copies whose detection status is Uninfected.

      Infected

      Number of copies whose detection status is Infected.

      Exception

      Number of copies whose detection status is Exception. A detection exception is caused by a software error.

      Undetected

      Number of copies whose detection status is Undetected.
    • Viewing the detection status of a single VMware resource

      In the VMware resource list, click Copy in the row that contains the target resource to view the detection status of all copies of the resource. Table 2 describes the parameters.

      Table 2 Single resource detection details

      Parameter

      Description

      Copy Time

      Time when a copy is generated.

      Location

      Copy location.

      • local
      • external

      Model Version

      Version number of the detection model.

      Ransomware Detection Status

      Detection status of a copy. Values include Undetected, Preparing, Detecting, Uninfected, Infected, and Exception.

      Detection Time

      Time when a copy starts to be detected, that is, time when the detection status changes to Detecting.
