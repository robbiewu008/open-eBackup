Updating a File Name Extension Filtering Rule

This section describes how to update the File Name Extension Filtering Rule of a vStore.

Prerequisites

If you need to add file name extensions to be intercepted for a vStore, create a File Name Extension Filtering Rule by referring to Creating a File Name Extension Filtering Rule.

Procedure

  1. On the navigation pane, choose Explore > Ransomware Protection > Ransomware Detection.
  2. On the Detection Settings tab, locate the row that contains the target vStore and choose More > Update File Name Extension Filtering Rule.
  3. The Update File Name Extension Filtering Rule dialog box is displayed.

    Select or deselect any File Name Extension Filtering Rule in the list as needed. All selected File Name Extension Filtering Rules are automatically associated with the current vStore.

    By default, all file name extension filtering rules that have been associated with the current vStore are selected.

  4. Click OK.
