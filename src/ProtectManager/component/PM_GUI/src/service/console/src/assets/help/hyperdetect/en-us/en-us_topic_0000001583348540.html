Viewing Local Storage Authentication Information

This section describes operations to view local storage authentication information in the Authentication Information area.

Procedure

  1. Choose System > Infrastructure > Local Storage.
  2. In the Authentication Information area on the Local Storage tab page, view the local storage authentication information. Authentication information includes login authentication and service authentication. Login authentication is used for login to DeviceManager. Service authentication is used by the backup system and storage system to perform data protection operations.

    • Login authentication
      • Username: .

        By default, the maximum number of sessions per user is not limited. Do not change the value unless necessary.

      • Password: Login authentication user password.
      • Status: User authentication status. If the authentication status is Abnormal, modify authentication information.
    • Service authentication
      • Username: .

        By default, the maximum number of sessions per user is not limited. Do not change the value unless necessary.

      • Password: Service authentication user password.
      • Status: User authentication status. If the authentication status is Abnormal, modify authentication information.
