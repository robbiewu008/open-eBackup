Preparing for the Backup

Before backing up an HBase cluster, prepare related information as follows:

  • Table 1 lists the information required.
  • Table 2 lists the information required when the authentication mode for communication between the HBase cluster and OceanProtect is Simple Authentication.
  • Table 3 lists the information required when the authentication mode for communication between the HBase cluster and OceanProtect is Kerberos Authentication.
