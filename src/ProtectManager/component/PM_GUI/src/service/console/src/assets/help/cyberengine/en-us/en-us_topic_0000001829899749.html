Associating a File Name Extension Filtering Rule with a File System

The preset file name extension filtering rules are associated with all file systems in the system by default. A new file name extension filtering rule takes effect only after it is associated with the file system.

Procedure

  1. In the navigation pane, choose Data Security > File Interception.
  2. Click the File Name Extension Filtering Rule tab.
  3. In the row that contains a File Name Extension Filtering Rule, choose More > Associated File System.

    You can also select multiple File Name Extension Filtering Rules and choose More > Associated File System in the upper part.

  4. Select the file system to be associated and click OK.
