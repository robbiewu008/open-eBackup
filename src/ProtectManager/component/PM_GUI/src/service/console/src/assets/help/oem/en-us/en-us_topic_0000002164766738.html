Backing Up an NDMP NAS File System

This section describes how to complete backup configuration. IntelliMate (smart assistant) provides step-by-step help to streamline user operations. Specifically, log in to the product WebUI and click in the upper part of the page. On the IntelliMate page that is displayed, click Learn How to Perform Backup Operations, select the type of the resource to be protected, and click Next to complete the backup configuration according to the backup configuration guide.

  • Step 2: (Optional) Creating a File Directory
