查看SAP HANA数据库环境信息

当SAP HANA数据库注册至OceanProtect后，您可以在管理界面查看SAP HANA数据库的详细信息。

部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

操作步骤

  1. 选择“保护 > 应用 > SAP HANA”。

    • 在“实例”页签，已注册至OceanProtect的数据库详细信息如表1所示。
      表1 实例信息

      参数

      说明

      名称

      实例名称。

      状态

      实例状态。

      当数据库处于运行中时，该状态为“在线”，否则为“离线”。

      版本

      数据库版本。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
    • 在“数据库”页签，已注册至OceanProtect的数据库详细信息如表2所示。
      表2 数据库信息

      参数

      说明

      名称

      数据库名称。

      状态

      数据库状态。

      当数据库处于运行中时，该状态为“在线”，否则为“离线”。

      数据库类型

      数据库类型，包括“系统数据库”和“租户数据库”。

      数据库部署类型

      数据库的部署方式，包括“单机模式”、“互为主模式”、“主备模式”、“共享模式”、“分布式模式”。

      所属实例

      数据库所属的实例名。

      节点IP地址

      数据库所在的节点IP地址。

      SLA

      当数据库已关联SLA时，则此处显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      数据库当前是否被保护。

      • 未保护：数据库未关联SLA，或者数据库已关联SLA但已禁用保护。
      • 已保护：数据库已关联SLA，且已激活保护。
      • 创建中：数据库已关联SLA，正在创建保护。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。

  2. 查看实例或数据库的基本信息。

    • 在“实例”页签，单击实例名称，浏览实例的基本信息和实例下的数据库基本信息。
    • 在“数据库”页签，单击数据库名称，浏览数据库总览信息、副本信息以及相关任务。
      • 数据库总览

        显示数据库的名称、SLA信息、任务执行时间、副本总数等信息。

      • 数据库副本数据

        您可以依次按照年、月、天查找相应时间的副本。

        时间下方显示，表示该时间存在副本。

      • 数据库任务

        数据库相关的所有任务展示，包括备份、恢复、归档等任务。可选择展示执行中的任务或历史任务。
