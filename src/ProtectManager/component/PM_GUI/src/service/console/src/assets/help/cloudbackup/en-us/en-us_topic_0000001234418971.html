Deleting a Backup Storage

This section describes how to delete a backup storage on ProtectManager. After a backup storage is deleted, it cannot be used to store backup data.

Prerequisites

  • A backup storage has been added.
  • The backup storage is not added to the SLA.

Precautions

A backup storage cannot be restored after being deleted.

Procedure

  1. Choose System > Infrastructure > Backup Storage.
  2. Click More next to the backup storage to be deleted and choose Delete.
  3. In the dialog box that is displayed, click OK.
