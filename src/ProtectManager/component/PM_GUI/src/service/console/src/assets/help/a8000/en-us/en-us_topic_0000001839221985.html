Viewing a Backup Storage Unit Group

This section describes how to view information about a backup storage unit group.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups to view information about the backup storage unit group in the list. Table 1 describes the related parameters.

    Table 1 Parameters of a backup storage unit group

    Parameter

    Description

    Name

    Name of a backup storage unit group. You can click the name of a backup storage unit group to view its basic information and backup storage unit information.

    Description

    Description of a backup storage unit group.

    Backup Storage Units

    Number of backup storage units in a backup storage unit group. You can click the number to view information about backup storage units.
