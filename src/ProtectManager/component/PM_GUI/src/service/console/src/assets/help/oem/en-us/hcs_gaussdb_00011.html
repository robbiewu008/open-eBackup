Step 1: Registering a Huawei Cloud Stack GaussDB Project

Before Huawei Cloud Stack GaussDB backup and restoration, you must register Huawei Cloud Stack GaussDB with the product.

Context

Identity and Access Management (IAM) provides permission management for secure access to user's cloud services and resources.

An account is created upon user's successful registration with Huawei Cloud, is the entity that owns, uses, and pays for resources, has full control over these resources, and can access all cloud services of Huawei Cloud. An IAM user is created by an account to use cloud services. Each IAM user has its own identity credential (password and access key) and uses resources based on assigned permissions.

Prerequisites

  • You have registered with Huawei Cloud and created an IAM user on Huawei Cloud. This user is used to connect Huawei Cloud Stack GaussDB to the product. For details about how to create an IAM user, see Create User Groups and Assign Permissions and Create IAM Users and Log In.
  • A private line has been established for the Huawei Cloud Stack GaussDB project and the agent host. For details, contact technical support engineers of the service provider.
  • If you need to enable XBSA SSL Certificate Management, ensure that the SSL certificate has been obtained during ProtectAgent installation. For details, see "Configuring Certificate Files for Mutual SSL Authentication" in the ProtectAgent Installation Guide.

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack GaussDB.
  2. Click Register to register Huawei Cloud Stack GaussDB.

    Table 1 describes Huawei Cloud Stack GaussDB registration information.
    Table 1 Huawei Cloud Stack GaussDB registration information

    Parameter

    Description

    Name

    User-defined name of Huawei Cloud Stack GaussDB.

    Management Plane URL

    To obtain the management plane URL, perform the following steps:

    1. On the Basic_Parameters sheet of xxx_export_all_EN exported by the Huawei Cloud Stack deployment tool, search for region_id and external_global_domain_name to obtain their values.
    2. On the Portal sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, obtain the URL, username, and password for logging in to Service OM.
    3. On Service OM, click VM.
    4. Obtain any IP address of DBS-GaussCore on the VM page.
    5. Log in to the target host using the IP address through PuTTY.
    6. Run the ping iam-apigateway-proxy.{region_id}.{external_global_domain_name} command to obtain the management plane IP address.

    7. Check the IAM version.
      1. On the Portal sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, obtain the COP website, and the username and password for logging in to COP.
      2. Choose CloudCMDB > Service Mgmt, and search for IAM in the service tree. If IAM exists, IAM 2.0 is used. If IAM does not exist, IAM 1.0 is used.
    8. Obtain the management plane address.
      • IAM 1.0: https://XX.XX.XX.XX:26335 indicates the management plane address.
      • IAM 2.0: https://XX.XX.XX.XX:443 indicates the management plane address.

    Service Plane URL

    On the Tool_generated_IP_Params sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, search for AGW-LB-Float-IP to obtain the corresponding parameter value XX.XX.XX.XX. https://XX.XX.XX.XX indicates the service plane address.

    IAM Username

    IAM username created by the user on Huawei Cloud. To obtain the username:

    1. Log in to ManageOne Operation Portal and choose Organization > Resource Spaces.
    2. Search for the corresponding resource set based on the resource set ID obtained in Preparing for the Backup. VDC specifies the name of the account to which the IAM user belongs.
    3. Click Users. The user management page is displayed. Search for a user based on the value of VDC and select a user to obtain the username.

    IAM User Password

    Password of the IAM user created on Huawei Cloud. Use the IAM user password created in Preparing for the Backup.

    Owning Account of IAM User

    Name of the account to which the IAM user belongs. To obtain the name:

    1. Log in to ManageOne Operation Portal and choose Organization > Resource Spaces.
    2. Search for the corresponding resource set based on the resource set ID obtained in Preparing for the Backup. VDC specifies the name of the account to which the IAM user belongs.

    Resource Set Name

    Name of a resource set of a tenant. Obtain the value from the VDC administrator.

    Resource Set ID

    ID of a resource set of a tenant. Obtain the value from the VDC administrator.

    Agent Host

    Agent host registered with the product.

    XBSA Configuration Path

    Path to the xbsa.conf file configured during backUpAgent installation. You can obtain the path by referring to the XBSA configuration path in Table 1.

    XBSA SSL Certificate Management

    This function is enabled by default. After this function is enabled, the certificate of Huawei Cloud Stack GaussDB will be verified during registration to ensure security of interactions between the product and external devices. After this function is disabled, security risks may exist in the communication between the system and the cloud platform. Exercise caution when performing this operation.

    Root Certificate

    Root certificate file with the file name extension .pem. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    User Certificate

    Client user certificate file with the file name extension .crt. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    User Private Key

    Client user private key file with the file name extension .key. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    Private Key Password

    Client private key password character string set when the certificate file is created. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

  3. Click OK.
