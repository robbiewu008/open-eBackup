What Can I Do If a VMDK File Is Locked?

A message is displayed on the VMware vSphere Client, indicating that a VMDK file is locked. The backup and restoration jobs for the VMDK or VM will fail.

To ensure that the jobs run properly, you need to unlock the corresponding VMDK file.

For details, see the VMware official website.
