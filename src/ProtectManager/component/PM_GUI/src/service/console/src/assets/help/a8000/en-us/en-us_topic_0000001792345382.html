Viewing the SFTP Service

On the SFTP Service page, you can view basic information about the SFTP service.

Prerequisites

The SFTP service has been enabled.

Procedure

  1. Choose System > Settings.
  2. In the navigation tree, click SFTP Service.
  3. On the SFTP Service tab page, view the basic information about the SFTP service.

    • IP address: IP address of the SFTP server.
    • Authorized user: Click Name of an authorized user. In the dialog box that is displayed on the right, view the information about the authorized user.
      • Quota: quota allocated to the current user.

        The quota for a user is limited by the available capacity of the storage system. If the capacity is less than the allocated quota, the actual quota is the available capacity of the storage system.

      • Usage: capacity used by the current user as a percentage to its quota.
      • Used Capacity: capacity used by the current user.
      • Unused Capacity: unused capacity of the current user.
