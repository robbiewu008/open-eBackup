Step 1 Registering the GBase 8a Database

Before the GBase 8a database backup and restoration, you must register the GBase 8a database with the product.

Prerequisites

You need to ensure that the host where the database is located has been registered on the Protection > Clients page.

Procedure

  1. Choose Protection > Databases > General Databases.
  2. Click Register to register the GBase 8a database.

    Table 1 lists the registration information.
    Table 1 GBase 8a database registration information

    Parameter

    Description

    Type

    Database type. Currently, GBase 8a supports only Sharding cluster.

    Database Name

    Name of the database to be protected.

    The name must be the same as the database name in the production environment.

    Hosts

    Host where the database resides.

    Database Type

    Select GBase 8a.

    Authentication Mode

    Select Database authentication.

    Database Username

    Database username. The default username is gbase.

    Database Password

    Database password. The default system password is obtained from the product documentation downloaded from the GBase official website.

    Other Authentication Information

    DBA username and password.

    The format is as follows: 
    os_username=DBA username,os_password=DBA user password

    Customized Parameters

    Set parameters in some database scenarios.

  3. Click OK.
