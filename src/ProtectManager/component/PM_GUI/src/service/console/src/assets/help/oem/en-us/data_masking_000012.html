Adding Identification Rules

Common identification rules have been preset in the product. If the preset identification rules do not meet your requirements, you can customize some rules.

Procedure

  1. In the navigation pane, choose Data Security > Data Anonymization > Anonymization Policies.

    For 1.5.0, choose Explore > Data Utilization Policies in the navigation pane and click Anonymization Policies.

  2. In the right Anonymization Policies pane, click the Identification Rules tab.
  3. Click Add, and set identification rules as prompted.

    Table 1 describes the parameters.

    Table 1 Parameters related to identification rules

    Parameter

    Description

    Name

    Name of an identification rule. It contains a maximum of 64 characters in length.

    Expression

    Standard regular expression. It contains a maximum of 2000 characters in length.

    CAUTION:

    Before entering a regular expression, check it by using SDL Regex Fuzzer to prevent denial of service (DoS) attacks.

  4. In the Select Anonymization Rule area, select the anonymization rule to be associated with.

    If the existing anonymization rules do not meet your requirements, you can click Add to create one. For details about the anonymization rule parameters, see Adding Anonymization Rules.

  5. Click OK.
