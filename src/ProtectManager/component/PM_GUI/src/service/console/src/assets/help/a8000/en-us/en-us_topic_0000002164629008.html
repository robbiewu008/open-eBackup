Configuring the Performance Statistics Switch

This section describes how to manually configure the performance statistics function so that you can monitor and know the device performance more intuitively and easily and make timely adjustments.

Precautions

  • You can view historical data only after performance monitoring is enabled.
  • After performance monitoring is disabled, you can view only the historical data that has been retained but cannot view the latest historical data.

Procedure

  1. Choose Performance.

    For 1.5.0, choose Insight > Performance.

  2. Click Settings.
  3. Click next to Performance Monitoring to enable performance monitoring. If performance monitoring is disabled, subsequent monitoring data will not be saved, and the data transmission speed during backup or restoration of some applications cannot be obtained.
  4. View the performance monitoring indicators described in Table 1.

    Table 1 Performance monitoring indicators

    Parameter

    Description

    Backup

    Backup performance monitoring indicators include IOPS and Bandwidth.

    Replication

    Replication performance monitoring indicators include Replication I/O Response Time, Replication IOPS, Replication Bandwidth (KB/s), and Replication RX Bandwidth (KB/s).
