Backing Up Dameng

This section describes how to complete backup configuration. For 1.6.0 and later versions, IntelliMate (smart assistant) provides step-by-step help to streamline user operations. Specifically, log in to the product WebUI and click in the upper part of the page. On the page that is displayed, click Learn How to Perform Backup Operations, select the type of the resource to be protected, and click Next to complete the backup configuration according to the backup configuration guide.

  • Step 1: Enabling the DmAPService Service
  • Step 2: Enabling Local Archiving for the Database
  • Step 3: Registering the Dameng Database
  • Step 4: (Optional) Creating a Rate Limiting Policy
  • Step 5: (Optional) Enabling Backup Link Encryption
  • Step 6: Creating a Backup SLA
  • Step 7: Performing Backup
