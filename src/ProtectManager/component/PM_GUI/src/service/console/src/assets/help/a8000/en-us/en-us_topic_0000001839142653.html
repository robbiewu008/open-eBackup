Manually Executing Backup

If you want to perform a backup immediately, you can manually perform the backup.

Context

Copies generated by manual backup are retained for the period defined in the SLA.

Procedure

  1. Choose Protection > Databases > MySQL/MariaDB.
  2. Choose More > Manual Backup in the row where the database to be backed up is located.
  3. Set the name of the copy generated during manual backup.

    If this parameter is left empty, the system sets the name to backup_Timestamp by default.

  4. Select a protection policy: full backup, incremental backup, differential backup, or log backup.

    If the cluster type is Active/Active Replication, only full backup, incremental backup, and differential backup are supported.

  5. Click OK.
