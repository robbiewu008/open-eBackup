Querying Redis Information

Procedure

  1. Choose Protection > Big Data > Redis.
  2. Query Redis cluster information.

    1. This page displays information about Redis clusters registered with the OceanProtect.

      Table 1 describes the related parameters.

      Table 1 Redis cluster information

      Parameter

      Description

      Name

      Redis cluster name.

      Status

      Redis cluster status.

      • Online: If connection between the agent host and Redis cluster is normal, Status is Online.
      • Offline: If connection between the agent host and Redis cluster is abnormal, Status is Offline.
      • Abnormal: If some nodes in the Redis cluster are offline, Status is Abnormal.

      Version

      Redis version number.

      Owner

      User to whom resources of a vStore are allocated for management.

      SLAs

      If an SLA is associated with a cluster, the SLA name is displayed. Otherwise, -- is displayed.

      SLA Compliance
      Whether a backup job is successfully executed based on the backup frequency or backup window set in the SLA.
      • If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
      • If the backup time does not meet the backup time window or backup frequency requirements, for example, if the backup frequency set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup frequency or backup window.

      Protection Status

      Whether a Redis cluster is protected.

      • Unprotected: The Redis cluster is not associated with an SLA, or it is associated with an SLA but protection is disabled.
      • Protected: The Redis cluster is associated with an SLA and protection is enabled.
      • Creating: The cluster is associated with an SLA and protection is being created.
    2. Click the name and check the overview, copy information, and related jobs.
      • Overview

        Display the SLA information, job execution time, and total number of copies.

      • Copy data

        You can search for copies by year, month, and day.

        If is displayed below the time, a copy exists at the time.

      • Related jobs

        Displays backup and restoration jobs.
