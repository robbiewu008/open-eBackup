Stopping Jobs

You can stop eight types of jobs on the OceanCyber WebUI, including: resource scanning, ransomware detection snapshot generation, restoration, live mount, snapshot deletion, resource protection, modification protection, and intelligent detection.

Procedure

  1. Choose Insight > Jobs.
  2. Click the Current Jobs tab page.
  3. Optional: In the upper right corner of the job list, select the start time of the task to filter jobs. Alternatively, click Task Type > Object Type > Job Status > to filter jobs. Or you can click Object > Job ID > to search for jobs. Or you can click next to Start Time to sort jobs.
  4. Select one or more jobs and click Stop.
  5. In the dialog box that is displayed, click OK.
