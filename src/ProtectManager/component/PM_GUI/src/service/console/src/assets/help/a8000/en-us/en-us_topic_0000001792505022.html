Adding an Archive Storage

This section describes how to connect archive storage to OceanProtect Appliance so that a logical connection is established between the archive storage and OceanProtect Appliance for data transmission.

Prerequisites

  • An object storage platform for storing archived data is ready, and the archive network of OceanProtect Appliance can normally access the object storage platform.
  • You have obtained the access address and AK/SK of the object storage platform.
  • A bucket for storing archived data has been created on the object storage platform.

    To ensure data security, it is recommended that a dedicated bucket be created for the OceanProtect Appliance. Otherwise, there will be the risk of unauthorized access to archived data.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Object Storage tab page, click Add Archive Storage.
  3. Set archive storage parameters.

    Table 1 describes the parameters.

    Table 1 Archive storage parameters

    Parameter

    Description

    Name

    Name of the archive storage, which is user-defined.

    Type

    Type of the archive storage. Currently, only object storage complying with the S3 protocol is supported.

    Object Storage Product

    Object storage product connected to OceanProtect Appliance.

    OceanStor 100D (FusionStorage OBS), OceanStor Pacific, Huawei Cloud OBS, and AWS S3 are supported.

    Endpoint

    Access address of the archive storage

    Address format: domain name or service plane IP address of the archive storage

    Protocol

    Protocol used for data transmission between OceanProtect Appliance and archive storage.

    • When HTTPS is used, import the certificate of the Object Storage type so that the OceanProtect Appliance can verify the archive storage information.

      For details about how to import a certificate, see "Importing a Certificate" in the OceanProtect Appliance 1.5.0-1.6.0 Security Configuration Guide.

      NOTE:
      • If OceanStor Pacific is used as the archive storage, obtain the certificate by referring to "Exporting a Built-In CA Certificate" in the OceanStor Pacific Series 8.1.5 Security Configuration Guide, and export the Service certificate. When exporting a certificate, change the certificate file name extension to .pem. If the system displays "The CA certificate has expired or is invalid." when you add the archive storage and other parameter settings meet requirements, rectify the fault by referring to What Should I Do If a Message Indicating that the CA Certificate Has Expired or Is Invalid Is Displayed When I Add Archive Storage?.
      • If Huawei Cloud OBS or FusionStorage OBS is used as the archive storage, contact the archive storage administrator to obtain the certificate.
      • If AWS S3 is used as the archive storage, contact the vendor to obtain the certificate.
    • When the protocol is set to HTTP, OceanProtect Appliance cannot verify the archive storage environment information, which poses security risks. You are advised to select HTTPS for security purposes.

    AK
    The object storage service supports AK/SK authentication, in which the Access Key ID/Secret Access Key (AK/SK) encryption method is used to verify the identity of a request sender.
    • AK: indicates the ID of the access key. It is a unique identifier used together with a secret access key to sign requests cryptographically.
    • SK: is a key used with an AK to sign requests cryptographically. It can identify a request sender and prevent the request from being modified.

    Contact the archive storage administrator to obtain the AK/SK.

    SK

    Data Bucket

    Name of the bucket that stores archived data

    Index bucket

    Name of the bucket that stores index data

    Use Proxy Server

    When OceanProtect Appliance uses an agent server to access an archive storage, configure the address, username, and password of the agent server.

    NOTE:
    • If an agent server is required, ensure that the maximum size of the request body allowed by the agent server is greater than 100 MB.
    • The agent server supports only the HTTP protocol. The address format is http://Agent server IP address:Port number.

    Capacity Alarm Threshold

    Upper threshold of the used capacity of a bucket. When the used capacity reaches the threshold, an alarm is reported.

  4. Click OK.
