Step 2: Creating a Rate Limiting Policy

If you need to limit the maximum bandwidth of a data protection job to ensure the performance of critical services, configure rate limiting policies by referring to this section.

Procedure

  1. Choose Protection > Protection Policies > Rate Limiting Policies.
  2. Click Create.
  3. Set rate limiting policy parameters.

    Table 1 describes the related parameters.
    Table 1 Parameters related to a rate limiting policy

    Parameter

    Description

    Name

    Name of a rate limiting policy, which can be customized.

    Maximum Bandwidth

    Average maximum amount of data that can be transmitted per second in a data protection job.

    Default value: 1024 MB/s

    Value range: 10 MB/s to 5120 MB/s

    Description

    Information about the rate limiting policy.

  4. Click OK.
