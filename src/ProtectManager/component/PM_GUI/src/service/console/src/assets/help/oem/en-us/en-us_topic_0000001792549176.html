Manually Executing Backup

You can manually execute a backup job to back up data immediately.

Context

Copies generated by manual backup are retained for the period defined in the SLA.

Procedure

  1. Choose Protection > Big Data > MongoDB.
  2. Choose More > Manual Backup in the row where the instance to be backed up is located.
  3. Set the name of the copy generated during manual backup.

    If this parameter is left empty, the system sets the name to backup_Timestamp by default.

  4. Select full backup or log backup.
  5. Click OK.
