Performing a Detection Periodically

After associating the file systems that require intelligent detection with a specified intelligent detection policy, the system performs intelligent detection on these file systems based on the intelligent detection policy.

Procedure

  1. Choose Data Security > Ransomware Protection > Intelligent Detection.
  2. Click File Systems.
  3. Choose More > Protect in the row that contains the desired file system.

    You can also select multiple file systems and choose More > Protect.

  4. Select an existing intelligent detection policy or click Create to create an intelligent detection policy.
  5. Select the Learning Mode function. Enabling this function will improve evaluation of data features of backup copies. You are advised to enable this function, which takes effect only when the backup software type is NetBackup or Commvault, the storage device type is OceanProtect, and in-depth backup copy detection is enabled. If this function is disabled, false detection results may be generated during in-depth copy detection.

    After Learning Mode is enabled, you can select By duration or By times, and set the learning duration or learning times.

    • Learning by duration xx days indicates that the snapshot data of the xx days is selected as the dataset for learning. Learning by times xx indicates that the snapshot data of the xx times is selected as the dataset for learning.
    • The set learning duration (learning times) is the minimum learning duration (minimum number of learning times). To evaluate the data features of backup copies based on a required amount of data, the actual learning duration (learning times) may be longer than the configured learning duration (greater than the configured learning times).
    • When the learning mode is switched from By duration to By times or from By times to By duration, the system performs the learning job again.

  6. Click OK.

    If the current system time is later than the start time of the ransomware detection snapshot generated for the first time based on the intelligent detection policy, you can perform the backup immediately or wait for the next scheduling based on the intelligent detection policy after the intelligent detection policy is associated.
