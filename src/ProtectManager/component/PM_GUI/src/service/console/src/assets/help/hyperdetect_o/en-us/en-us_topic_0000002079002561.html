Performing Ransomware Detection Snapshot-based Backup Manually

If you want to perform a backup job immediately, you can manually perform it.

Context

  • During manual backup, parameters (except Automatic Retry) defined in the SLA are applied to the manual backup job.
  • Copies generated by manual backup are retained for the duration defined in the SLA.

Procedure

  1. Choose Protection > Storage > Local Resources.
  2. On the Local File Systems tab page, select the file system that you want to protect and choose More > Manual Backup.

    You can also select multiple local file systems for batch backup.

  3. On the Manual Backup page that is displayed on the right, specify the backup job name and click OK.
