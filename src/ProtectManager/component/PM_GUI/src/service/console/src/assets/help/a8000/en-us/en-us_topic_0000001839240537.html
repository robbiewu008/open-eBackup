Step 4: Creating a Hive Backup Set

Create a backup set to be protected in the Hive cluster, that is, a set of Hive database tables.

Procedure

  1. Choose Protection > Big Data > Hive.
  2. On the Backup Sets tab page, click Create to create a backup set.

    Table 1 describes related parameters.
    Table 1 Backup set parameters

    Parameter

    Description

    Name

    Customize a backup set name.

    Cluster

    Select the Hive cluster to be protected.

    Databases

    Select the Hive database to be protected and choose whether to enable the Automatic Table Protection function.

    • When the Automatic Table Protection function is enabled, the system protects all tables in the database, including new tables.
    • When the Automatic Table Protection function is disabled, you need to manually select the tables to be protected.
      NOTE:

      If the tables in the database have been protected, the Automatic Table Protection function cannot be enabled.

    Metadata Storage Path

    Select a path for storing metadata.

    This path is used to temporarily store table metadata. After the backup is complete, the stored metadata is automatically deleted.

  3. Click OK.
