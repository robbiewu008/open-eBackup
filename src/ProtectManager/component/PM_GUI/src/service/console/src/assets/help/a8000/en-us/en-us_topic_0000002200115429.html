Deleting Tapes

This section describes how to delete a tape that is no longer required.

Prerequisites

  • The tape does not reside in the tape library.
  • For a physical tape library, you need to manually remove the tape from the device.

Precautions

If a tape that does not reside in the tape library is deleted, the tape is deleted from the corresponding media set and the related copies are also deleted.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Tape tab.
  4. In the row where the tape to be deleted resides, choose More > Delete.
