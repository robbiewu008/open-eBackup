Managing a Detection Model

You can view, update, activate, or delete a ransomware detection model.

On the navigation pane, choose Explore > Ransomware Protection > Ransomware Detection > Detection Model. Perform management operations by referring to Table 1.

Table 1 Detection model management operations

Operation

Description

Navigation Path

Viewing a detection model

Scenario

View the basic information and activation status of a detection model.

In the detection model list, view the version, import time, signature information, description, and activation status.

Updating a detection model

Scenario

Common detection models are preset in the system. You can import the latest detection model (if any) of the storage system, or import or reactivate a detection model of an earlier version if it works better in your scenario.

Note

  • Obtain the latest detection model from Huawei technical support engineers.
  • A detection model is automatically activated after being imported. The earlier one enters Inactivated status.
  • The import does not affect ongoing detection jobs. Ongoing detection jobs still use the earlier detection model. The new detection model will be used in the next job.

On the Detection Model page, click Import.

Activating a detection model

Scenario

To use a deactivated detection model to detect ransomware, activate it first.

Note

  • Only detection models in Inactivated status can be activated.
  • Only one detection model in Activated status exists in the system. The activated detection model is automatically deactivated after you activate or import a new detection model.
  • If a previously activated detection model is being used for ransomware detection, the newly activated or imported detection model enters in the Activated state but does not affect the ongoing detection job and will take effect in new jobs.

Choose More > Activate in the Operation column on the right of the detection model.

Deleting a detection model

Scenario

Detection models that are no longer used for ransomware detection can be deleted.

Note

Only detection models in Inactivated status can be deleted.

Choose More > Delete in the Operation column on the right of the detection model.
