主域控制器在系统状态恢复后无法创建新用户

现象描述

创建用户时界面提示Cannot create the object because directory service was unable to allocate a relative identifier.

可能原因

  • 当RID（Relative Identifier）主机做系统状态恢复时，它会尝试与其他域控制器同步。如果没有可同步的域控制器或者同步不起作用，同步过程将失败。
  • RID池已耗尽或者RID分配的对象存在问题。

处理步骤

请参见帐户标识符分配器在Windows Server中初始化失败。
