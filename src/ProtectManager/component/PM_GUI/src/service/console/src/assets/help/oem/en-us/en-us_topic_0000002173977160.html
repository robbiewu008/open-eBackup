Restoring the Tablespace of One or More Tables in the Dameng Database

This section describes how to restore the tablespace of one or more tables in a single-server Dameng database to the original location.

Prerequisites

Before the restoration, ensure that the database in the target location has been stopped. Otherwise, the restoration will fail.

Procedure

  1. Choose Explore > Copy Data > Databases > Dameng.
  2. On the Copies tab page, find the target copy based on the copy information, and choose More > Tablespace-level Restoration in the row of the target copy.
  3. On the displayed page, select one or more table names.
  4. Click OK.
