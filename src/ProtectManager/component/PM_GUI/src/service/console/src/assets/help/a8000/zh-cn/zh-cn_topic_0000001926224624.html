在生产端获取Endpoint

Endpoint是访问对象存储的地址，可填写为生产端的域名或IP地址。生产端环境可分为OceanStor Pacific、华为云和阿里私有云，不同的生产端环境获取Endpoint的方式不同。

  • OceanStor Pacific获取Endpoint。
    1. 登录OceanStor Pacific管理界面。
    2. 选择“资源 > 访问 > 业务网络”。
    3. 选择“账户”和“子网”，查看相应账户和子网下的Zone。
    4. 单击Zone的名称，选择“IP地址池”页签，查看所创建的IP地址/掩码，所创IP地址即为Endpoint。
  • 华为云获取Endpoint。
    1. 登录华为云管理界面。
    2. 选择“服务列表 > 存储 > 对象存储服务3.0”。
    3. 在左侧导航选择“对象存储”，查看桶资源。
    4. 单击某个桶名，“概览”页签的详情内容中包含Endpoint。
  • 阿里私有云获取Endpoint。
    1. 登录阿里私有云管理界面。
    2. 选择“产品”页签，选择“存储 > 对象存储OSS”。
    3. 在左侧导航选择“Bucket列表”，单击某个Bucket。
    4. 选择“概览”页签，在“访问域名”处即可查看对象存储的Endpoint。
