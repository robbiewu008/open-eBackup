Removing a User

You can remove a user that is no longer required.

Context

The system administrator can remove other administrators.

Procedure

  1. Choose System > Security > Users and Roles > Users.
  2. (Optional) Click next to Role to filter users. You can also click next to Name to search for a user.
  3. Click More on the right of a user to be removed and choose Delete.
  4. In the dialog box that is displayed, click OK.
