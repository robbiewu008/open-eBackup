Step 5: Creating a Remote Device Administrator (Applicable to 1.6.0 and Later Versions)

This section describes how to create a remote device administrator in the replication target cluster to verify the peer cluster during the replication.

Procedure

  1. Log in to the product management page of the replication target cluster.
  2. Choose System > Security > RBAC.
  3. On the Users tab page, click Create.
  4. On the Basic User Information page that is displayed, set basic information by referring to Table 1 and click Next.

    Table 1 Basic information parameters

    Parameter

    Description

    User Type

    Select Local user.

    Name

    Name of a user, which can be customized.

    Login Authentication Mode

    Select Login password.

    Password

    Password of a user.

    Confirm Password

    Confirm password.

    Role
    1. Click Add.
    2. Select Remote Device Administrator and click OK.
    3. Set Default Role to Remote Device Administrator.

    Description

    Description of a user.

    Max. Number of User Connections

    Maximum number of concurrent logins. The value ranges from 1 to 8.

  5. Click Next to skip Resource Authorization.
  6. On the Overview page that is displayed, confirm the user information and click Finish.
