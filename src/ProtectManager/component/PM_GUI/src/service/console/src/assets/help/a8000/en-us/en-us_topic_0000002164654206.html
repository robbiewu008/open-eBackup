The Interval Between "Prepare the NAS file system" and "Start backing up VM" Is Too Long in a Backup Job

Symptom

During VMware VM backup, the interval between "Prepare the NAS file system" and "Start backing up VM" is too long (exceeding the number of disks on the VMs to be backed up multiplied by 2 minutes).

Possible Causes

The production storage is Huawei OceanStor Dorado, but Huawei multipathing software is not installed and OceanStor Dorado is not added to the OceanProtect as external storage.

Troubleshooting

Install the Huawei multipathing software. For details, see "Installing Huawei UltraPath" in the OceanProtect DataBackup 1.5.0-1.6.0 Client Installation Guide.
