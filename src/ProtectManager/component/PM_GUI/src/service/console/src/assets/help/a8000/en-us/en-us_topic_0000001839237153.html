Manually Executing Backup

If you want to perform a backup immediately, you can manually perform the backup.

Context

  • Copies generated by manual backup are retained for the duration defined in the SLA.
  • If a replication or archive policy has been defined in the SLA, the OceanProtect will perform replication or archiving once based on the SLA when you perform manual backup.
  • If no log backup is defined in the SLA, log backup cannot be automatically executed.

Procedure

  1. Choose Protection > Databases > Informix.
  2. Click the Instance tab.
  3. Choose More > Manual Backup in the row where the database to be backed up is located.

    You can select multiple resources to perform manual backup in batches. Select multiple resources and choose More > Manual Backup in the upper left corner of the page.

  4. Set the name of the copy generated during manual backup.

    If this parameter is left empty, the system sets the name to backup_Timestamp by default.

  5. Select a protection policy, which can be Full backup, Incremental backup, Differential Backup, or Log Backup.
  6. Click OK.
