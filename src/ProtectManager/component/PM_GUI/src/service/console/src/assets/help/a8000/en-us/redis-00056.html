Managing Redis Cluster Protection

This section describes how to modify, remove, activate, or disable Redis cluster protection on the OceanProtect.

Related Operations

You need to log in to the management page, choose Protection > Big Data > Redis, and find the target Redis cluster.

Table 1 describes the related operations.

Table 1 Operations related to Redis cluster protection

Operation

Description

Navigation Path

Modify Protection

Scenario

If you need to reselect an SLA associated with a cluster, you can modify the protection plan. After the modification, the next backup will be executed based on the new protection plan.

Note

The modification does not affect the protection job that is being executed. The modified protection plan will take effect in the next protection period.

In the row that contains the target Redis cluster, choose More > Modify Protection.

Remove Protection

Scenario

This operation enables you to remove protection for a cluster. After removal, the system cannot execute protection jobs for the cluster. To protect the backup set again, associate an SLA with it again.

Note

Protection cannot be removed when a protection job of the cluster set is running.

In the row that contains the target Redis cluster, choose More > Remove Protection.

Activate Protection

Scenario

This operation enables you to activate the protection plan for a cluster. After the protection plan is activated, the system backs up the cluster based on the protection plan.

Note

This operation can be performed only for a cluster that has been associated with an SLA and is in Unprotected state.

In the row that contains the target Redis cluster, choose More > Activate Protection.

Disable Protection

Scenario

This operation enables you to disable the protection plan of a cluster for which periodic protection is not required. After the protection plan is disabled, the system does not execute automatic backup jobs for the cluster.

Note

Disabling protection does not affect ongoing backup jobs.

In the row that contains the target Redis cluster, choose More > Disable Protection.

Manual Backup

Scenario

This operation enables you to perform a backup immediately.

Note

Copies generated by manual backup are retained for the duration defined in the SLA.

In the row that contains the target Redis cluster, choose More > Manual Backup.

Note: The procedure for restoring Redis clusters is the same as that described in Restoring a Redis Cluster.
