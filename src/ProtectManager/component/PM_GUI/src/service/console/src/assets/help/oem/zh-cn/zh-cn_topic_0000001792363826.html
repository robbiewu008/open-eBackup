步骤2：创建DB2表空间集

如果您需要执行DB2表空间集保护，请先创建表空间集。

操作步骤

  1. 选择“保护 > 数据库 > DB2”。
  2. 选择“表空间集”。
  3. 单击“创建”，创建DB2表空间集。

    配置相关参数。参数说明如表1。
    表1 DB2表空间集参数信息

    参数

    说明

    名称

    自定义DB2表空间集名称。

    主机/集群

    表空间集所属的主机/集群。

    实例

    选择要进行保护的DB2实例。

    数据库

    选择DB2集群中需要保护的数据库。

    表空间

    数据库下创建的表空间。

  4. 单击“确定”。
