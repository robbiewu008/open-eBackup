Configuring the Real-Time Ransomware Detection Whitelist

If you want to skip some directories or files during real-time ransomware detection, perform the operations in this section to configure a real-time ransomware detection whitelist.

Procedure

  1. On the navigation pane, choose Data Security > Ransomware Protection > Real-Time Ransomware Detection.
  2. Click Real-Time Detection Whitelist.
  3. Add a real-time detection whitelist.

    1. Click Create.
    2. The whitelist type can be Directory or File Name Extension. Select a type, enter the directory name or file name extension, and click OK.

  4. Associate with vStores. A real-time ransomware detection whitelist takes effect only for associated vStores. After vStores are associated, the real-time ransomware detection whitelist takes effect when real-time ransomware detection is performed on file systems of the vStores.

    1. Choose More > Associate with vStore.
    2. Select the vStores to be associated with the real-time ransomware detection whitelist and click OK.
