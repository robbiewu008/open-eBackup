查看文件系统副本

参考本节查看文件系统副本的详细信息。

操作步骤

  1. 选择“数据利用 > 副本数据 > 本地文件系统”。
  2. 在页面右上角选择资源浏览方式浏览副本信息。

    • 选择“资源”页签：以资源维度查看文件系统的副本信息。当文件系统中的数据损坏或丢失，需要恢复文件系统时，建议从该入口进入查找相应时间点副本。参数说明如表表1所示。
      表1 副本参数说明

      参数

      说明

      资源

      名称

      文件系统名称。

      位置

      文件系统在存储上的位置。

      状态

      文件系统是否被ProtectManager发现。

      • 是，则状态为存在。
      • 否，则状态为不存在。

      副本

      数量

      基于该文件系统产生的所有副本数量。
    • 选择“副本”页签：以副本维度查看所有副本信息。当需要基于某个指定的副本进行原位置或新位置恢复时，建议从该入口进入查找相应副本。参数说明如表表2所示。
      表2 副本参数说明

      参数

      说明

      副本

      副本时间

      副本生成时间。

      单击“副本时间”，可以查看副本过期时间等信息，过期时间显示“--”表示副本永久保留。

      状态

      副本的状态。

      • 正常：副本可用。
      • 无效：副本无效。
      • 删除中：副本处于正在删除过程中。
      • 恢复中：副本处于恢复过程中。

      位置

      存储副本的备份存储的桶名称。

      生成方式

      副本的生成方式。此处为“备份”。

      副本类型

      生成副本的备份类型。

      包括：全量备份、增量备份。

      索引

      索引的状态。

      • 未索引：副本未索引。
      • 已索引：副本已索引。
      • 索引中：副本索引中。
      • 删除中：副本处于正在删除过程中。
      • 删除失败：副本删除失败。

      资源

      名称

      文件系统名称。

      位置

      文件系统在存储上的位置。

      状态

      文件系统是否被ProtectManager发现。

      • 是，则状态为存在。
      • 否，则状态为不存在。
