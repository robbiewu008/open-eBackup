步骤1：注册华为云Stack GaussDB项目

执行华为云Stack GaussDB备份恢复前，需要将华为云Stack GaussDB注册至OceanProtect。

前提条件

  • 已注册华为云，并在华为云创建云平台租户。该用户用于将华为云Stack GaussDB接入OceanProtect。
  • 已建立华为云Stack GaussDB项目与客户端所在的主机的专线。详细操作请联系服务提供商的技术支持工程师。
  • 如果需要开启XBSA SSL 证书管理，需确保安装客户端时已获取SSL证书，详细操作请参见《OceanProtect DataBackup 1.5.0-1.6.0 客户端安装指南》中的“配置SSL双向认证证书文件”章节。

操作步骤

  1. 选择“保护 > 云平台 > 华为云Stack GaussDB”。
  2. 单击“注册”，注册华为云Stack GaussDB信息。

    华为云Stack GaussDB注册信息如表1所示。
    表1 华为云Stack GaussDB注册信息

    参数

    说明

    名称

    用户自定义华为云Stack GaussDB名称。

    管理面url

    管理面地址获取方法如下：

    1. 请参考华为云Stack软件安装阶段HCC Turnkey导出的参数信息汇总文件《xxx_export_all_v2_CN.xlsx》的“基本参数”页签，搜索“region_id”和“external_global_domain_name”，获取对应的参数值。
    2. 请参考华为云Stac软件安装阶段HCC Turnkey导出的参数信息汇总文件《xxx_export_all_v2_CN.xlsx》的“工具生成的IP参数”页签，搜索“AGW-LB-Float-IP”，获取管理面IP地址。
    3. 完成域名映射。
      1. 登录客户端所在的主机。
      2. 执行以下命令打开/etc/hosts文件。
        vi /etc/hosts
      3. 在/etc/hosts文件中新增如下信息，进行域名映射,其中xx.x.x.xx为管理面IP。
        xx.x.x.xx iam-apigateway-proxy.{region_id}.{external_global_domain_name}
    4. 获取管理面地址：https://iam-apigateway-proxy.{region_id}.{external_global_domain_name}:443。

    业务面url

    请参考华为云Stack软件安装阶段HCC Turnkey导出的参数信息汇总文件《xxx_export_all_v2_CN.xlsx》的“工具生成的IP参数”页签，搜索“AGW-LB-Float-IP”，获取对应的参数值XX.XX.XX.XX。https://XX.XX.XX.XX即为业务面地址。

    云平台租户名称

    待保护的华为云Stack GaussDB实例所属租户的名称，根据备份前准备获取。

    云平台租户密码

    待保护的华为云Stack GaussDB实例所属租户的密码，根据备份前准备获取。

    云平台租户所属VDC名称

    云平台租户所属的账户名称，可参考表1的准备项“云平台租户名称”。

    资源空间名称

    租户下资源空间名称，根据备份前准备获取。

    资源空间ID

    租户下资源空间ID，根据备份前准备获取。

    客户端

    已注册到OceanProtect的客户端所在的主机。

    XBSA配置路径

    用户安装backupAgent时配置的“xbsa.conf”文件路径。

    客户端自动推送：XBSA配置路径为/opt/backupAgent/xbsa.conf。

    客户端手动安装：可参考表1的准备项“XBSA配置路径”获取该路径。

    XBSA SSL 证书管理

    默认开启。开启后，注册华为云Stack GaussDB时会对其证书进行校验，确保OceanProtect与外部设备交互的安全性。关闭后，系统与云平台通信可能存在安全风险，请谨慎操作。

    根证书

    后缀为“.pem”的根证书文件，获取方式参见表1中的准备项“证书”。当选择“XBSA SSL 证书管理”时，需要配置该参数。

    用户证书

    后缀为“.crt”的客户端用户证书文件，获取方式参见表1中的准备项“证书”。当选择“XBSA SSL 证书管理”时，需要配置该参数。

    用户私钥

    后缀为“.key”的客户端用户私钥文件，获取方式参见表1中的准备项“证书”。当选择“XBSA SSL 证书管理”时，需要配置该参数。

    私钥密码

    创建证书文件时设置客户端私钥密码字符串。当选择“XBSA SSL 证书管理”时，需要配置该参数。

  3. 单击“确定”。
