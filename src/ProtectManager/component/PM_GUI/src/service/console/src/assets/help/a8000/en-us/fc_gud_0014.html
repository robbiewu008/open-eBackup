Step 1: Create a FusionOne Compute Interconnection User

This section describes how to create a user for interconnecting FusionOne Compute to the OceanProtect.

Procedure

  1. Log in to FusionOne Compute.
  2. Create a role on the FusionOne Compute management page. This role is used to create an account for interconnecting with the OceanProtect.

    1. Click in the navigation pane and choose System > Rights Management > Role Management.
    2. Click Add Role.
    3. Enter the role information. See Figure 1.
      • Set Role Name to OceanProtect.
      • For details about how to configure permissions, see Configuring FusionOne Compute Role Permissions.
      Figure 1 Adding a role

    4. Click OK.

  3. On the FusionOne Compute management page, create an account for interconnecting with the OceanProtect.

    1. Choose System > Rights Management > User Management.
    2. Click Add User.
    3. Enter the user information. See Figure 2.
      • User Type: Select Interface interconnection user.
      • Role: Select OceanProtect.
      Figure 2 Creating an interface interconnection user

    4. Click OK.
    5. Choose System > Rights Management > Password Policy.
    6. In the left corner of the Password Policy page, click Modify.
    7. Change the value of Forcibly change passwords upon a reset or initial login. to No.
    8. Click Save.
