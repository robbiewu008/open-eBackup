Step 3: Registering a TiDB Table Set

Before performing TiDB backup resource protection, create a table set. A TiDB table set is a collection of TiDB cluster database tables.

Procedure

  1. Choose Protection > Databases > TiDB.
  2. Click Table Sets.
  3. Click Register to register a TiDB table set.

    Table 1 describes the related parameters.
    Table 1 TiDB table set parameters

    Parameter

    Description

    Name

    Customize the TiDB table set name.

    Cluster

    Select the TiDB cluster to be protected.

    Before selecting a TiDB cluster, ensure that the cluster to be protected has been registered with the product.

    Databases

    Select the database to be protected in the TiDB cluster.

    Table Sets

    Table set to be protected in the database.

  4. Click OK.
