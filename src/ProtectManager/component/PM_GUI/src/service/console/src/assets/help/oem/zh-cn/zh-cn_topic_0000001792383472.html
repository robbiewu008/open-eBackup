步骤2：创建ClickHouse表集

如果您需要执行ClickHouse表集保护，请先创建表集。

操作步骤

  1. 选择“保护 > 大数据 > ClickHouse”。
  2. 选择“表集”。
  3. 单击“创建”，创建ClickHouse表集。

    1. 配置相关参数。参数说明如表1。
      表1 ClickHouse表集参数信息

      参数

      说明

      名称

      自定义ClickHouse表集名称。

      集群

      选择要进行保护的ClickHouse集群。

      选择ClickHouse集群前，确保需要保护的集群已注册至本产品。

      数据库

      选择ClickHouse集群中需要保护的数据库。

      表

      单击“请选择”，在右侧弹出的“选择表”页面，勾选数据库中需要保护的表。
    2. 单击“确定”。

  4. 单击“确定”。
