Quick Start

This section describes the ProtectManager management functions, helping you quickly find service operation entries and improving operation efficiency.

Table 1 describes the ProtectManager functions.

Table 1 Functions of ProtectManager

Application Type

Function

Home

Helps you quickly learn the job status, managed resources, alarms, and historical performance of the system.

Protect

Storage allows you to manage and protect local file systems.

Protection Policies allows you to manage SLAs and rate limiting policies.

NOTE:

A service level agreement (SLA) can be regarded as a backup plan template created to meet different backup requirements.

Explore

Copy Data allows you to manage all copies generated by protected resources.

Insight

Alarms and Events allows you to view uncleared alarms, cleared alarms, operation logs, and running logs.

Jobs allows you to view details about running jobs and completed jobs.

Performance allows you to configure performance monitoring and view the historical performance information (I/O response time, IOPS, and bandwidth) of nodes.

System

Infrastructure allows you to manage local storage and backup storage.

Security allows you to manage users, roles, security policies, and certificates in the system.

Log Management allows you to manage debug log levels and export logs.

Export Records allows you to view system logs and copy files.

Settings Allows you to manage system data backup, email notification, alarm notification, and event dump services.

Global Search

Allows you to use in the upper right corner of the page to quickly search for copies and resources and quickly restore files in the copies.
