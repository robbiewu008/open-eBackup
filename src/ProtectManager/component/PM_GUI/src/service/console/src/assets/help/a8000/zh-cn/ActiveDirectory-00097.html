Active Directory进行对象级恢复时报错：该对象的DistinguishedName已存在

对于已被删除且未在回收站的对象，用户新创建了同名对象，此时恢复被删除的对象会违反对象的唯一性约束导致恢复报错。用户需要选择保留其中一个对象，如果选择恢复已删除的对象，需要参考本章节删除新建的同名对象后重新进行对象级恢复；如果选择保留新创建的同名对象无需进行对象级恢复。

现象描述

Active Directory进行对象级恢复时报错：该对象的DistinguishedName已存在

可能原因

该对象的DistinguishedName已存在。

处理步骤

  1. 打开Active Directory Users and Computers。

    1. 使用Administrator用户登录远程主机。
    2. 按Win + R。
    3. 输入dsa.msc后按Enter键。

  2. 找到需要删除的对象。
  3. 右键单击对象后选择“删除”。
  4. 单击“是”。
