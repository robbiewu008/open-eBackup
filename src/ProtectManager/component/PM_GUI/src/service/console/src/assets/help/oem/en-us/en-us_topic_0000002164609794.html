Viewing Information of Active Directory Domain Controllers

After an Active Directory domain controller is registered with the , you can view details about the Active Directory domain controller on the management page.

Procedure

  1. Choose Protection > Applications > Active Directory.

    1. This page displays the details about the Active Directory domain controller.

      Some parameters are not displayed by default. You can click next to Operation and select a parameter to display it on the current page.

      Viewing Information of Active Directory Domain Controllers describes the related parameters.
      Table 1 Information of the Active Directory domain controller

      Parameter

      Description

      Name

      Name of the Active Directory domain controller.

      Host Name

      Host name of the Active Directory domain controller.

      IP Address

      IP address of the Active Directory domain controller.

      SLAs

      If an SLA has been associated with the Active Directory domain controller, the SLA name is displayed. Otherwise, -- is displayed.

      SLA Compliance
      Whether a backup job is successfully executed based on the backup interval or backup window set in the SLA.
      • Met: If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
      • Missed: If the backup time does not meet the backup time window or backup interval requirements, for example, if the backup interval set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup interval or backup window.

      Protection Status

      Whether the Active Directory domain controller is currently protected.

      • Unprotected: The Active Directory domain controller is not associated with an SLA, or the Active Directory domain controller is associated with an SLA but protection is disabled.
      • Protected: The Active Directory domain controller is associated with an SLA and protection is activated.
      • Creating: The Active Directory domain controller is associated with an SLA and protection is being created.

      Tag

      Tags help you effectively classify and search for resources. If a tag has been added to a resource, the tag name is displayed. Otherwise, -- is displayed.
    1. Click the name of an Active Directory domain controller to view the overview, copy information, and job.
      • Overview

        The SLA information, job execution time, and total number of copies are displayed.

      • Copy data

        You can search for copies by year, month, or day.

        If is displayed below the time, a copy exists at the time.

      • Job

        Jobs, such as backup and restoration jobs, are displayed.
