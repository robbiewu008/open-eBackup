Introduction

Definition

As an open source distributed relational database, TiDB is a converged distributed database product that supports online hybrid transactional/analytical processing (HTAP). The OceanProtect provides TiDB with data protection functions, such as backup, archiving, replication, and restoration.

Backup/Restoration Granularity

Backup and restoration can be performed at a granularity of the TiDB cluster, database or table set.
