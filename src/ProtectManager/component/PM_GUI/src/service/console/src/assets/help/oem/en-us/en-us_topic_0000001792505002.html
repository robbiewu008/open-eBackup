Importing Archive Storage Copies

Archive storage copies can be imported on the product. If product is faulty and archive copies cannot be used for restoration, connect the archive storage to another cluster. You can import the metadata of a copy to a new cluster using the copy import function. Then, the copy can be used for restoration.

Prerequisites

An archive storage has been added.

Precautions

  • Importing an archive copy will import all metadata of the copy to product. The copy generation mode changes from Archive to Import.
  • Currently, archive storage copy import is supported only for VMware, Oracle, HDFS, HBase, NAS file systems, and NAS shares.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Object Storage tab page, click More on the right of an archive storage system and select Import Archive Storage Copy.
  3. In the dialog box that is displayed, click OK.
