开启SFTP服务

OceanProtect支持开启SFTP服务，为SFTP客户端提供存储空间。

注意事项

  • 若端口上已存在IP地址，请确保填写的IP地址与其处于同一网段。
  • 首次开启SFTP服务可以设置WORM参数，若之前已经启用过SFTP服务，再次开启SFTP服务WORM参数也不支持配置。

操作步骤

  1. 在菜单栏上，选择“系统 > 设置”。
  2. 在导航栏中，单击“SFTP服务”。
  3. 在“SFTP服务”页签，单击右侧的，开启SFTP服务。

    系统弹出“开启SFTP”对话框。参考表1配置相关参数。
    表1 SFTP参数说明

    参数

    说明

    IP地址类型

    支持选择“IPV4”或“IPV6”。

    IPv4地址

    当IP地址类型选择为IPV4时展示该参数。

    SFTP服务IP地址。

    该IP地址和子网掩码用于SFTP客户端连接OceanProtectSFTP服务端，需确保与SFTP客户端网络连通，且不能和SFTP客户端网段中的IP冲突。

    子网掩码

    IPV4地址的子网掩码。

    IPv6地址

    当IP地址类型选择为IPV6时展示该参数。

    SFTP服务IP地址。

    该IP地址和前缀长度用于SFTP客户端连接OceanProtectSFTP服务端，需确保与SFTP客户端网络连通，且不能和SFTP客户端网段中的IP冲突。

    前缀长度

    IPV6地址的前缀长度。

    配置业务端口

    是否使用VLAN

    默认不勾选。勾选前请确认业务已配置VLAN，勾选后，配置的业务端口会应用于VLAN。

    端口类型

    支持选择“以太网端口”或“绑定端口”。

    端口选择

    勾选“是否使用VLAN”或端口类型选择“绑定端口”展示该参数，支持选择“创建新端口”或“复用当前已配置端口”。

    可选端口

    当前可选择的前端端口。

    已选端口

    当前已选择的前端端口。

    VLAN ID

    选择“创建新端口”后展示该参数。输入所使用的的VLAN ID。

    WORM

    模式

    开启WORM后显示该参数。

    • 法规遵从

      对于保护期内的文件，超级管理员、管理员和普通用户都不可以修改和删除。对于过期的文件，超级管理员、管理员和普通用户都不可以修改，但可以删除。

    • 企业遵从

      对于保护期内的文件，管理员和普通用户都不可以修改和删除，特权用户才可以删除。对于过期的文件，超级管理员、管理员和普通用户都不可以修改，但可以删除。

    保护期

    开启WORM后显示该参数。

    设置文件保护时间。取值范围：0~70年。

    自动锁定

    开启WORM后显示该参数。

    启用该功能后，文件在设置的“锁定等待时长（小时）”内未被修改，则文件自动进入锁定状态。此时文件受到保护，只能读取，不可以修改、删除和重命名。

    锁定等待时长

    开启WORM后显示该参数。

    文件在锁定等待时长内未被修改，则文件自动进入锁定状态，此时文件只能读取，不可以修改和删除。取值范围：1分钟~10年。

    自动删除

    开启WORM后显示该参数。

    启用后，系统将自动删除超过保护期的文件。

  4. 单击“确定”。

后续处理

在“SFTP服务”页签，单击右侧的，在弹出的对话框中，单击“确定”，关闭SFTP服务。

关闭SFTP服务前，请确保当前没有SFTP用户使用SFTP服务进行文件传输，且所有的SFTP用户不再需要使用SFTP服务。
