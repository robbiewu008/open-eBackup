Step 1: Registering a TiDB Cluster

Before backup and restoration of a TiDB backup resource, register the TiDB cluster with the OceanProtect.

Prerequisites

The cluster status is normal.

Procedure

  1. Choose Protection > Databases > TiDB.
  2. On the Cluster page, click Register to register a TiDB cluster.

    1. Under Name, enter the user-defined TiDB cluster name.
    2. On the Add TiUP Node tab page, configure TiUP node information. Table 1 describes the parameters for adding a TiUP node to the TiDB cluster.
      Table 1 Adding TiUP node information to the TiDB cluster

      Parameter

      Description

      Agent Host

      Host that has been registered with the OceanProtect.

      TiUP installation path

      Installation path of the TiUP node in the TiDB cluster database.
    3. Click Next and select a cluster.
    4. Click Next and add a node. Table 2 describes the parameters for adding a node.
      Table 2 Adding node to the TiDB cluster

      Parameter

      Description

      TiDB cluster username

      Username of the TiDB cluster administrator.

      Password

      Password of the TiDB cluster administrator.

      Log backup path

      Log backup path of the TiDB cluster.
    5. Select a host for each node in the TiDB cluster and click Finish.
