Logging In to the Cloud Backup Console as a VDC Administrator or VDC Operator

This section describes how to log in to the Cloud Backup Console through the DME operation portal.

Prerequisites

DME has been correctly installed.

Procedure

  1. On the maintenance terminal, open a browser.
  2. Enter https://XXX.XXX.XXX.XXX in the address box and press Enter.

    The page for logging in to the DME operation portal is displayed.

    XXX.XXX.XXX.XXX indicates the IP address for logging in to the DME operation portal.

    The web browser may prompt that the website has a security certificate error. If the IP address is correct, you can ignore the prompt and continue accessing the DME operation portal.

  3. Enter the username and password of the VDC administrator or VDC operator.

    If Verification Code is displayed, enter the correct verification code.

  4. Click Log In.

    The DME operation portal is displayed.

  5. Choose Backup > Cloud Backup to go to the cloud backup console.
