步骤2：创建GaussDB(DWS)Schema集/表集

如果您需要执行GaussDB(DWS)Schema集或表集保护，请先参考本节操作，创建Schema集或表集。

注意事项

内核版本为GaussDB A 8.0.0的GaussDB(DWS)不支持创建Schema集。

操作步骤

  1. 选择“保护 > 大数据 > GaussDB(DWS)”。
  2. 选择“Schema集”或“表集”页签。
  3. 单击“创建”，创建“Schema集”或“表集”。

    1. 配置相关参数。参数说明如表1和表2。
      表1 Schema集参数信息

      参数

      说明

      名称

      自定义Schema集名称。

      集群

      选择待保护对象的所属GaussDB(DWS)集群。

      数据库

      选择待保护对象的所属数据库。

      Schemas

      在“可选Schema”页面，勾选数据库中需要保护的Schema。
      表2 表集参数信息

      参数

      说明

      名称

      自定义表集名称。

      集群

      选择待保护对象的所属GaussDB(DWS)集群。

      数据库

      选择待保护对象的所属数据库。

      表

      在“可选表”页面，勾选数据库中需要保护的表。
    2. 单击“确定”。
