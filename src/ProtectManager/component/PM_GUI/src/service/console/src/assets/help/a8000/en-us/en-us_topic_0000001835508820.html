Viewing a Backup Storage Device

After a member node is added, the system automatically adds the backup storage corresponding to the member node as a backup storage device.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. Click Backup Storage Devices to view the backup storage device information. Table 1 describes the related parameters.

    Table 1 Backup storage device parameters

    Parameter

    Description

    Name

    Name of the backup storage device. For the backup storage device that is automatically added, the default name is the name of the member node followed by the suffix _auto_add.

    Status

    Status of the backup storage device.

    • Online: The backup storage device is properly connected to the active node.
    • Offline: The backup storage device is disconnected from the active node.
    • Upgrading: The backup storage device is being upgraded as a member node.

    Device Type

    Select OceanProtect at present.

    IP

    Management IP address of the backup storage device.

    Port

    Port number of the backup storage device.
