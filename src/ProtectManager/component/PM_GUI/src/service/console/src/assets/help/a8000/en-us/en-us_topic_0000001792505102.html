Adding an External Certificate

The OceanProtect allows you to add an external certificate for higher system security.

Procedure

  1. Choose System > Security > Certificate.
  2. Click Add External Certificate.
  3. Set certificate information.

    Table 1 lists the parameters.

    Table 1 Certificate parameters

    Parameter

    Description

    Name

    Name of the certificate to be added.

    [Value range]

    • The value contains 4 to 32 characters.
    • The value can contain only digits, letters, spaces, underscores (_), and hyphens (-), and cannot start with a digit, space, or hyphen (-).
    • The certificate name must be unique.

    Type

    Select one or multiple types from the drop-down list.

    The following types are supported: Email, Object Storage, External Storage, Backup/Multi-domain/Replication Cluster, LDAP, Huawei Cloud Stack IAM, and external systems.

    • Email: Select this type if the server encryption mode needs to be set to SSL/TLS or STARTTLS when an SMTP server is connected.
    • Object Storage: Select this type if HTTPS needs to be selected when archive storage is added.
    • External Storage: Select this type if a storage device needs to be added.
    • LDAP: Before configuring the LDAP service, you need to add an external certificate of this type.
    • Backup/Multi-domain/Replication Cluster: Before adding a backup/multi-domain/replication cluster, add an external certificate of this type first.
    • HCS IAM: Before configuring the Huawei Cloud Stack Identity and Access Management (IAM) service, you need to add an external certificate of this type.
    • External system: Before configuring the backup software management operation, you need to add the corresponding backup software certificate.

    CA Certificate

    Click and select the CA certificate corresponding to the certificate to be added.

    If the CA certificate is not the root one, add a complete certificate chain file. For details about how to create a certificate chain file, see Creating a Certificate Chain File.

  4. Click OK.
