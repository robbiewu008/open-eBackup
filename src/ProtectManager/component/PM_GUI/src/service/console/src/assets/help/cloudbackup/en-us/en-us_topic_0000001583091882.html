Modifying Local Storage Authentication Information

This section describes operations to modify local storage authentication information on Storage System. After changing the password of the interconnection user on DeviceManager, you must update the password for local storage authentication on the management interface.

Procedure

  1. Choose System > Infrastructure > Local Storage.
  2. In the Authentication Information area of the Local Storage tab page, click Modify Login Authentication or Modify Service Authentication on the right.
  3. Modify the username and password for local storage authentication with Storage System.

    • Ensure that the maximum number of sessions per user is greater than or equal to two if the system has two controllers and is greater than or equal to four if the system has four controllers.
    • Ensure that the user who logs in to DeviceManager from the current page has been logged out.

  4. Click Save.
