How Do I Install the OceanStor DataTurbo Client?

If the source deduplication function needs to be enabled after ProtectAgent is installed, you can refer to this section to install the OceanStor DataTurbo client and enable the source deduplication function.

Prerequisites

  • Ensure that the free memory of the agent host is greater than 2 GB.
  • Ensure that the OS of the agent host supports source deduplication. For details about the OSs that support source deduplication, refer to Compatibility Query Tool.

Procedure

  1. Download the ProtectAgent software package. For details, see Downloading the ProtectAgent Software Package.
  2. Open the downloaded ProtectAgent software package and obtain the OceanStor DataTurbo client installation package dataturbo.zip from the ..\DataProtect_XXX_client_general_XXX.zip\DataProtect_XXX_client_general_XXX\third_party_software directory.

    The name of the ProtectAgent software package varies depending on the version, agent type, and OS.

  3. Upload the OceanStor DataTurbo client installation package to the agent host.
  4. Install the OceanStor DataTurbo client and start the service. For details, see "Installation" and "Starting the Service" in the SourceDedupe User Guide of the corresponding version.

    • For the Linux OS, after source deduplication is enabled, add the source deduplication user to sudoers before the first backup. For details, see How Should I Add a Source Deduplication User to sudoers? (Applicable to the Linux OS).
    • For the Windows OS, after source deduplication is enabled, add the SID (S-1-5-18) of user system to the ...\oceanstor\dataturbo\conf \whitelist whitelist and run the sc stop dataturbo and sc start dataturbo commands in sequence to restart the DataTurbo service.
