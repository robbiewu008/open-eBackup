Backing Up Object Storage

This section describes how to complete backup configuration. IntelliMate (smart assistant) provides step-by-step help to streamline user operations. Specifically, log in to the OceanProtect WebUI and click in the upper part of the page. On the IntelliMate page that is displayed, click Learn How to Perform Backup Operations, select the type of the resource to be protected, and click Next to complete the backup configuration according to the backup configuration guide.

  • Step 1: Registering Object Storage
  • Step 2: Creating an Object Set
  • Step 3: (Optional) Creating a Rate Limiting Policy
  • Step 4: (Optional) Enabling Backup Link Encryption (Applicable to OceanProtect X Series Backup Appliances and OceanProtect E1000 (with the OceanProtect Used as Backup Storage))
  • Step 5: Creating a Backup SLA
  • Step 6: Enabling the NFSv4.1 Service (Applicable to OceanProtect X Series Backup Appliances and OceanProtect E1000 (with the OceanProtect Used as Backup Storage))
  • Step 7: Performing Backup
