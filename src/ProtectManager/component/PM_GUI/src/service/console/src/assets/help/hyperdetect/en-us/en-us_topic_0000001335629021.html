Enabling Real-Time Ransomware Detection

This section describes how to enable real-time ransomware detection.

Prerequisites

NFS sharing has been enabled for the file system on which real-time ransomware detection is to be performed.

Procedure

  1. On the navigation pane, choose Explore > Ransomware Protection > Ransomware Detection.
  2. On the Detection Settings tab, select the vStore for which you want to enable real-time ransomware detection and choose More > Enable Real-time Ransomware Detection.

    • You can also select multiple vStores for which you want to enable real-time ransomware detection and click Enable Real-time Ransomware Detection above the vStore list.
    • If real-time ransomware detection is not required, choose More > Disable Real-time Ransomware Detection. Alternatively, you can select multiple vStores for which you want to disable real-time ransomware detection and choose More > Disable Real-time Ransomware Detection above the vStore list.

  3. Confirm your operation as prompted.

    In the dialog box for confirmation that is displayed, you can determine whether to enable ransomware detection false alarm analysis and click OK.

    Ransomware detection false alarm analysis is enabled by default. With this function, the detection accuracy is improved, but the detection duration is prolonged.

    • If you require high detection accuracy, you are advised to enable ransomware detection false alarm analysis.
    • If you require short detection duration, you are advised to disable ransomware detection false alarm analysis.
