Step 2: Checking and Enabling the vmware-vapi-endpoint Service

Before registering the VMware virtualization environment, ensure that the vmware-vapi-endpoint service has been enabled on the vCenter Server. Otherwise, the registration will fail.

Procedure

  1. Log in to the vCenter Server through PuTTY as user root.
  2. Run the following command to enter the shell mode:

    shell

  3. Run the following command to check whether the vmware-vapi-endpoint service is enabled:

    service-control --status

    If information similar to the following is displayed, the service has been enabled. Otherwise, perform 4 to enable the service.

    root@vc161-99 [ ~ ]# service-control --status
Stopped: 
vmcam vmware-analytics vmware-eam vmware-imagebuilder vmware-mbcs vmware-netdumper vmware-rbd-watchdog vmware-statsmonitor vmware-vcha vsan-dps
Running: 
applmgmt lwsmd pschealth vmafdd vmcad vmdird vmdnsd vmonapi vmware-cis-license vmware-cm vmware-content-library vmware-perfcharts vmware-pod vmware-postgres-archiver vmware-rhttpproxy vmware-sca vmware-sps vmware-sts-idmd vmware-stsd vmware-updatemgr vmware-vapi-endpoint vmware-vmon vmware-vpostgres vmware-vpxd vmware-vpxd-svcs vmware-vsan-health vmware-vsm vsphere-client vsphere-ui

  4. Run the following command to enable the vmware-vapi-endpoint service:

    service-control --start vmware-vapi-endpoint
