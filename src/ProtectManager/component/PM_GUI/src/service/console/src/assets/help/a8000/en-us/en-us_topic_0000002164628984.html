Adding Backup Software Management

The OceanProtect supports backup software management. After adding information about backup software management, you can switch from the OceanProtect to the corresponding backup software.

Prerequisites

The certificates of eBackup and Data Protection Appliance have been imported. For details, see "Adding an External Certificate" in the OceanProtect DataBackup 1.5.0-1.6.0 Security Configuration Guide. During the operation, set Type to External System.

Procedure

  1. In the navigation pane, choose System > Backup Software Management.
  2. On the displayed Backup Software Management page, click Add.
  3. Add backup software management information. Table 1 describes the related parameters.

    Table 1 Backup software management parameters

    Parameter

    Description

    Name

    User-defined backup software name.

    Backup Software

    Managed backup software.

    IP

    Management IP address of the backup software system.

    Port

    Port of the backup software system.

    Username

    Username of the backup software system. For the eBackup system, only local users in the eBackup system can be added.

    Password

    User password of the backup software system.

    AK

    Access key of Data Protection Appliance (DPA).

    To obtain the parameter value, perform the following steps:
    1. Log in to Converged Data Management (CDM).
    2. Choose Settings > Configure > Access Key. The access key is displayed in the Access Key column on the Access Key tab page.

    SK

    Security key of DPA.

    To obtain the parameter value, perform the following steps:
    1. Log in to CDM.
    2. Choose Settings > Configure > Access Key.
    3. Select the corresponding Application Name and click Download Key. The security key is displayed in the Secret Access Key column of the downloaded file.
      NOTE:

      Each access key can only be downloaded once. Download and save it in a timely manner. After an access key is downloaded, you cannot get related information or download it again. Create an access key.

  4. Click OK.
  5. After the information is added successfully, return to the backup software management page. In the row of the backup software where you want to switch, click Access System to switch from the OceanProtect to the corresponding backup software.
