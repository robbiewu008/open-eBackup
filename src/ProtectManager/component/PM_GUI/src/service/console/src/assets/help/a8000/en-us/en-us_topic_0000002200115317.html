Exporting a Management Data Backup

You can export a management data backup to the local maintenance terminal. If some configuration data (such as user, SLA, resource, and alarm notification data) is deleted due to misoperations, you can import the management data backup to the original system to restore the management data.

Prerequisites

A management data backup whose Status is Available, Restore Failed, Creating, Invalid, Restoring, or Backup Failed exists in the system.

Precautions

The size of the exported backup data cannot exceed 4 GB.

Procedure

  1. In the navigation pane, choose System > Settings > Management Data Backup.
  2. In the row of the target management data backup, choose More > Export in the Operation column.

    The browser automatically downloads the management data backup to the local PC.
