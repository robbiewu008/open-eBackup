Managing HBase Clusters

This section describes how to modify or delete HBase clusters registered with the OceanProtect.

Related Operations

Log in to the management page, choose Protection > Big Data > HBase, click the Cluster tab, and locate the desired cluster.

Table 1 describes the related operations.

Table 1 Cluster-related operations

Operation

Description

Navigation Path

Modifying cluster information

Scenario

This operation enables you to change the authentication mode, name and configuration file of an HBase cluster.

In the row where the cluster resides, choose More > Modify.

For details about HBase cluster parameters, see Step 1: Registering an HBase Cluster.

Deleting a cluster

Scenario

This operation enables you to delete an HBase cluster if you no longer need to protect tables in it or restore data to it.

Note

  • If a job (such as backup, restoration, archiving, or replication) is in progress in the cluster, the cluster cannot be deleted. Wait until the job is complete.
  • A cluster cannot be deleted if its backup sets have been associated with an SLA.
  • Deletion of a cluster will delete all its backup sets.

In the row where the cluster resides, choose More > Delete.
