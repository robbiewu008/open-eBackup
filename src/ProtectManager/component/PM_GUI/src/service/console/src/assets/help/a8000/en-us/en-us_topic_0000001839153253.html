Step 1: Creating a Logical Port for the Replication Network (Applicable to 1.5.0)

On DeviceManager of the primary and secondary storage systems, you can create logical ports for the replication network to remotely transmit replication data.

Prerequisites

The replication network IP addresses have been planned at the primary and secondary ends.

Procedure

  1. Log in to DeviceManager.

    1. Choose System > Infrastructure > Cluster Management.
    2. In the Local Cluster Nodes area of the Backup Cluster tab page, click the node name.
    3. In the Node detail dialog box that is displayed, click Open the device management platform to go to DeviceManager.

  2. Choose Services > Network > Logical Ports.
  3. Click Create to create a logical port.

    Table 1 describes the related parameters.

    Table 1 Logical port parameters

    Parameter

    Description

    Name

    Name of a replication network logical port, which can be customized.

    Role

    Select Replication.

    IP Address Type

    IP address type of a replication network logical port, which can be IPv4 or IPv6.

    IP Address

    IP address of a replication network logical port.

    Subnet Mask

    If an IP address is of IPv4 type, set a subnet mask for the IP address.

    Prefix

    If an IP address is of IPv6 type, set a prefix for the IP address.

    Gateway

    Gateway used for the IP address of the replication network logical port.

    Port Type

    Users can create a logical port for the replication network by selecting Ethernet port, bond port, or VLAN.

    By default, the port used for replication has been bonded during initialization of the OceanProtect. Therefore, the bond port needs to be selected if there is no other operation (such as deleting the bond port).

    Home Port

    Ethernet port, bond port, or VLAN to which the created logical port belongs.

    After the system has filtered all ports used for the replication network, select one of them.

  4. Click OK.
  5. Repeat the preceding steps to configure a logical port for each home port.
