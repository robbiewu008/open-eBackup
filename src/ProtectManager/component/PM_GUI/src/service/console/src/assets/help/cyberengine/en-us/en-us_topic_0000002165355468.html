Deleting a File Name Extension Filtering Rule

This section describes how to delete a file name extension filtering rule.

Procedure

  1. Choose Data Security > File Interception.
  2. Click the File Name Extension Filtering Rules tab.
  3. In the row that contains a file name extension filtering rule, choose More > Delete.

    You can also select multiple file name extension filtering rules and choose More > Delete in the upper part.

  4. In the Danger dialog box, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
