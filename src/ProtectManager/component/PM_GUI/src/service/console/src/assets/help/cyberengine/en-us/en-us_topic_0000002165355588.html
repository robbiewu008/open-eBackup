Quick Start

This section describes the OceanCyber software functions, helping you quickly find service operation entries and improving operation efficiency.

Table 1 describes the OceanCyber software functions.

Table 1 OceanCyber software functions

Application Type

Function

Home

Allows you to quickly learn about the status of connected storage devices, alarms, intelligent detection status of file systems, job status, Air Gap status of file systems, and license capacity.

Data Security

File Interception allows you to disable write or modification for files of specific types to block the malicious encryption actions from known ransomware and protect file systems in the storage system.

Real-Time Detection provides the I/O-based real-time detection capability for NAS file systems. When an abnormal file is detected, a snapshot is created for the corresponding file system and an alarm is sent to reduce customer losses

Intelligent Detection detects copy files based on known ransomware attack signatures and snapshot data changes. After an infection is detected, uninfected snapshots can be used to restore data, effectively reducing losses.

Air Gap

Air Gap allows you to configure an Air Gap policy and associate it with a storage device.

Snapshot Management and Restoration

Snapshot Management and Restoration allows you to view the snapshots generated by intelligent detection.

Reports

Reports allows you to manage intelligent detection reports.

Jobs

Jobs allows you to view details about running and completed jobs in the system.

Alarms and Events

Alarms and Events allows you to view uncleared alarms, cleared alarms, operation logs, and run logs in the system.

Storage Devices

Storage Devices allows you to view information about the connected storage devices.

System

Security allows you to manage users and roles in the system, as well as security policies and certificates.

License allows you to manage licenses in the system.

Log Management allows you to manage debug log levels and export logs.

Export Records allows you to query the system export records.

Settings allows you to manage the system data backup, mail service, event dump, and SNMP trap.
