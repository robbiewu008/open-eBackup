Step 2: Creating an Object Set

Procedure

  1. Choose Protection > File Systems > Object Storage.
  2. Switch to the Object Set tab page and click Register.
  3. Configure the information about the object set.

    Table 1 describes the related parameters.

    Table 1 Object set registration information

    Parameter

    Description

    Object Set Name

    User-defined name of the object set.

    Object Storage

    Select the created object storage.

    Object Set

    Select buckets under the object storage.

    Filter Criteria

    Enter a prefix and press Enter to filter an object or a directory in a path in the target bucket for protection.

    NOTE:
    • The prefix is the relative path from the bucket to be backed up to an individual object or directory.

      For example, the prefix of the test/test.doc file in the bucket to be backed up is test/test.doc, and the prefix of the data in the test/test1/test2 directory in the bucket to be backed up is test/test1/test2.

    • Special characters \:*?"<>| cannot be contained in the prefix.
    • The prefix cannot start with a slash (/).
    • The prefix cannot contain two consecutive slashes (/).

  4. Click OK.
