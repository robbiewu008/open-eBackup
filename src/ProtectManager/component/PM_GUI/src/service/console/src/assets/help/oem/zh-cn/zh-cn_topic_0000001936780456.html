安装OceanStor DataTurbo客户端

ProtectAgent安装完成后，如需开启源端重删功能时，可参考本章节安装OceanStor DataTurbo客户端，开启源端重删功能。

前提条件

  • 请确保代理主机空闲内存大于2GB。
  • 请确保代理主机使用的操作系统支持源端重删功能。支持源端重删的操作系统请查看兼容性查询工具。

操作步骤

  1. 下载ProtectAgent软件包，具体操作请参考下载ProtectAgent软件包。
  2. 打开下载的ProtectAgent软件压缩包，在“..\DataProtect_XXX_client_general_XXX.zip\DataProtect_XXX_client_general_XXX\third_party_software”路径下，获取名为dataturbo.zip的OceanStor DataTurbo客户端安装包。

    不同的版本、代理类型、操作系统获取到的ProtectAgent软件包名可能会有差异，请以实际为准。

  3. 上传OceanStor DataTurbo客户端安装包至代理主机。
  4. 安装OceanStor DataTurbo客户端并启动服务，具体操作请参考对应版本的《源端重删用户指南》中“安装”和“启动服务”章节。

    • 对于Linux OS，开启源端重删功能后，请在首次备份前将源端重删用户加到sudoers中，具体操作请参见添加源端重删用户到sudoers（适用于Linux OS）。
    • 对于Windows OS，开启源端重删功能后，请将system用户的SID（S-1-5-18）添加到“...\oceanstor\dataturbo\conf \whitelist”白名单中，并依次执行sc stop dataturbo和sc start dataturbo命令重启DataTurbo服务。
