A VM or Disk Fails to Be Restored

Symptom

A VM or disk fails to be restored. The error message "Failed to restore the VM disk." is displayed in the job details, and Write failed, err:'Unknown error' is recorded in the log.

Possible Causes

In SAN transmission mode, if the copy used for restoration contains data of thick provision lazy zeroed or thin provisioned disks, write I/O errors may occur. (For details, visit VMware Knowledge Base: https://kb.vmware.com/s/article/52193.)

Troubleshooting

  1. Choose Explore > Copy Data > Virtualization > VMware.
  2. On the Copy tab page, select the copy used for restoration.

    • VM restoration: Choose Operation > Restore. On the displayed page, enable NBDSSL-based Restore.
    • Disk restoration: Select the disk to be restored and click Restore. On the Restore page, enable NBDSSL-based Restore.

  3. Click OK and start the restoration job.
