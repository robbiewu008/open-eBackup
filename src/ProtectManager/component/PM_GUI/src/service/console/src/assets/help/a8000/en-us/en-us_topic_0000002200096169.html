Setting a Windows Server to Enter the Directory Services Repair Mode (GUI)

To restore Active Directory, you need to set the Windows server where Active Directory is located to the DSRM.

  1. Restart the Windows server. Before the Windows icon is displayed, press F8 to display the Advanced Boot Options screen.
  2. Select Directory Services Repair Mode, as shown in Figure 1.

    If Directory Services Repair Mode is not displayed, select Repair Your Computer to enter the repair mode.
    Figure 1 Advanced Boot Options screen

    Once the server is in repair mode, you do not need to select this mode again every time you restart the server.

  3. On the displayed Choose an option screen, select Troubleshoot, as shown in Figure 2.

    Figure 2 Startup screen - Choose an option

  4. Select Startup Settings (as shown in Figure 3), then select Enable Safe Mode and click OK, as shown in Figure 4.

    Figure 3 Startup screen - Advanced options
    Figure 4 Startup screen - Startup Settings

  5. After the restart, press F8 before the Windows icon is displayed and select Directory Services Repair Mode to continue the startup process.
  6. On the login page that is displayed, select Other user and log in as the administrator, as shown in Figure 5.

    Figure 5 Login page

  7. After DSRM is enabled, Safe Mode is displayed in the four corners of the screen, as shown in Figure 6.

    Figure 6 Safe Mode screen
