Importing a CRL

The product allows you to import CRLs of client certificates and external certificates. After a CRL is imported, communication with the device whose certificate has been revoked is interrupted.

Context

On the product, you can import a client certificate CRL to identify whether the agent certificate is available, and you can import an external certificate CRL to identify whether the external certificate is available.

Prerequisites

You have obtained the CRL to be imported.

Precautions

  • Only one CRL file can be imported for a certificate.
  • The size of a CRL file must be less than or equal to 5 KB.
  • The CRL of the ExternalComponent certificate cannot be imported.

Procedure

  1. Choose System > Security > Certificate.
  2. Click the client certificate or external certificate whose CRL is to be imported. In the window that is displayed on the right, click the Certificate Revocation List tab.
  3. Click Import.
  4. Click Please choose file and select the CRL to be imported.
  5. The Danger dialog box is displayed. Select I have read and understand the consequences associated with performing this operation. and click OK.
