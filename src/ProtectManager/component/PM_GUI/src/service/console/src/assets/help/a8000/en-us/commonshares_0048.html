Viewing Share Information

This section describes how to view configurations of a common share resource.

Procedure

  1. Choose Explore > Copy Data > File Systems > Common Shares.
  2. You can search for copies by common share resource or copy. This section describes how to search for copies by resource.

    On the Resources tab page, find the target resource by resource name and click the resource name.

  3. Click Copy Data and choose the year, month, and day to find the copy.

    If is displayed below a month or date, copies exist in the month or on the day.

  4. Locate the row that contains the target copy, select the copy whose status is Sharing, and choose More > View Share Information.

    Table 1 describes the related parameters.

    Table 1 Parameters for viewing share information

    Parameter

    Description

    File system name

    Name of the common share file system created on DeviceManager.

    Share Path

    Path to which the file system is mounted. This parameter is displayed when Share Protocol is set to NFS.

    Share Client

    This parameter is displayed when Share Protocol is set to NFS. Enter the service IP address of a share client, service IP address segment of a share client, or an asterisk (*) to indicate service IP addresses of all share clients. The IP addresses can be IPv4 addresses, IPv6 addresses, or a combination of IPv4 and IPv6 addresses.

    Share Name

    User-defined CIFS share name. This parameter is displayed when Share Protocol is set to CIFS.

    Type

    Type of the user who accesses the CIFS share. This parameter is displayed when Share Protocol is set to CIFS.

    • Everyone
    • Windows local authentication user
    • Windows local authentication user group

    Share IP Address

    IP address for accessing the share.

  5. Click Close.
