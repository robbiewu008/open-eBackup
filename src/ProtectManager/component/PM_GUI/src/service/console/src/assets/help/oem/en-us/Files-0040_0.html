Step 2: Creating a Periodic Archive Policy for Replication Copies

This section describes how to associate a resource with an SLA if you need to periodically archive replication copies generated by the resource.

Procedure

  1. Log in to the management page of the cluster where the replication copies reside.
  2. Choose Explore > Copy Data > File Systems > Fileset.

    For 1.5.0, choose Explore > Copy Data > Bare Metal > Fileset.

  3. Locate the resource for which you want to archive replication copies and choose More > Protect.
  4. Select a created archive SLA.

    You can also click Create to create an SLA.

  5. Click OK.
