Modifying the SAML SSO Configuration

This section describes how to modify the SAML SSO configuration.

Procedure

  1. Choose System > Security > SAML SSO Configuration.
  2. Locate the row that contains the target SAML SSO configuration, and click Modify.
  3. You can modify the name, description, and metadata file of the SAML SSO configuration.
  4. Click OK.
