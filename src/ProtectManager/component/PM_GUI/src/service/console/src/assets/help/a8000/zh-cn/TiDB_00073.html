TiDB集群用户UID不一致

故障现象

TiDB子任务执行失败，任务事件中有“运行TiDB集群的用户UID不一致”的原因提示。

可能原因

运行TiDB集群的用户UID不一致。需要重新更改用户的UID。

处理步骤

建议在TiDB集群的所有节点完成以下操作：

  1. 执行以下命令查询UID需要修改用户的用户名和当前UID，“username”为待修改UID对应的用户名。

    $ id username

  2. 执行以下命令将用户UID修改为新的UID，“new_uid”为修改的新UID，“username”为待修改UID对应的用户名（确保新UID在系统中唯一）。

    $ sudo usermod -u new_uid username

  3. 再次执行1确认用户UID是否修改成功
