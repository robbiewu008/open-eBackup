Modifying Backup Software Management

After the backup software management information is added, you can modify the information by performing the following operations.

Prerequisites

Backup software management has been added.

Procedure

  1. On the menu bar, choose System > Backup Software Management.
  2. Locate the row of the backup software to be modified, and click Edit.
  3. Enter the page for modifying backup software management information. Table 1 describes the related parameters.

    Table 1 Backup software management parameters

    Parameter

    Description

    Name

    User-defined backup software name.

    Backup Software

    Managed backup software.

    IP

    Management IP address of the backup software system.

    Port

    Port of the backup software system.

    Username

    Username of the backup software system. For the eBackup system, only local users in the eBackup system can be added.

    Password

    User password of the backup software system.

    AK

    Access key of Data Protection Appliance (DPA).

    To obtain the parameter value, perform the following steps:
    1. Log in to Converged Data Management (CDM).
    2. Choose Settings > Configure > Access Key. The access key is displayed in the Access Key column on the Access Key tab page.

    SK

    Security key of DPA.

    To obtain the parameter value, perform the following steps:
    1. Log in to CDM.
    2. Choose Settings > Configure > Access Key.
    3. Select the corresponding Application Name and click Download Key. The security key is displayed in the Secret Access Key column of the downloaded file.
      NOTE:

      Each access key can only be downloaded once. Download and save it in a timely manner. After an access key is downloaded, you cannot get related information or download it again. Create an access key.

  4. Click OK.
