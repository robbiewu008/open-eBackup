(Optional) Configuring a Whitelist

If you want to skip some specific directories or files in the file system during in-depth detection of backup copies of the file system, configure a whitelist by referring to this section.

Procedure

  1. Choose Data Security > Intelligent Detection.
  2. Click Whitelist.
  3. Add a whitelist.

    1. Click Create.
    2. The type can be Directory or File Name Extension. Select a type, enter the directory name or file name extension, and click OK.

  4. Associate file systems. The whitelist takes effect only for associated file systems.

    1. Choose More > Associate File System.
    2. Select the file systems to be associated with the whitelist and click OK.

    You can also select multiple whitelists and choose More > Associate Policy to associate file systems in batches.
