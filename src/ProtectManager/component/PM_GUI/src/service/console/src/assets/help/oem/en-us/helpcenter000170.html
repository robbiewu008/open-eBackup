Importing Archive Storage Copies

The product allows you to import archive storage copies. If an archive copy cannot be used for restoration due to product faults, connect the archive storage to another set of product and use the copy import function to import the metadata of the copy to the new set of product. This way, you can use the copy for restoration.

Prerequisites

Archive storage has been added.

Precautions

  • Importing an archive storage copy will import all metadata in the copy to the product.
  • Currently, archive storage copy import is supported only for VMware, Oracle, HDFS, HBase, NAS file systems, and NAS shares.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Object Storage tab page, click More on the right of an archive storage system and select Import Archive Storage Copy.
  3. In the dialog box that is displayed, click OK.
