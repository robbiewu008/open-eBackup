步骤4：注册PDB集（适用于1.6.0及后续版本）

当要保护对象为多租用户场景（Multitenant Environment）下的CDB（容器数据库，Container Database）或PDB集（可插拔数据库，Pluggable Database）时，请参考本节注册PDB集。

背景信息

仅Oracle 12c以及后续版本支持CDB与PDB的新特性，在多租用户环境中，允许一个CDB承载多个PDB。

前提条件

  • 需提前将数据库注册至本产品，具体操作请参见步骤3：注册Oracle数据库。
  • 请确保Oracle实例配置了多租用户场景，否则注册PDB集任务中没有可选的PDB。通过执行SQL语句命令SELECT cdb FROM v$database查询，回显为“yes”则表示当前Oracle实例已配置多租用户场景。回显为“no”，则表示该实例未配置多租用户场景，不存在PDB集。

操作步骤

  1. 选择“保护 > 数据库 > Oracle”。
  2. 选择“PDB集”页签，单击“注册”。

    相关参数说明如表1所示。

    表1 注册PDB集参数说明

    参数

    说明

    名称

    要保护的PDB集资源的名称。可自定义填写。

    数据库

    待备份的PDB集所在的数据库。

    节点

    当选择的数据库为集群部署时，显示该参数。

    选择待备份的PDB集所在的Oracle节点进行注册。

  3. 在“可选PDB”区域，勾选待备份的PDB；在“已选PDB”区域，查看已选择的PDB。

    当勾选数据库实例中全部的PDB时，系统将自动识别为CDB。

  4. 单击“确定”。
