Backup

Prerequisites

  • The archive mode must be enabled before database backup.

    Modify the wal_level, archive_mode, and archive_command parameters in the postgresql.conf file.

    • Set wal_level to archive or higher (or replica or higher for PostgreSQL 9.6 or later).
    • Set archive_mode to on.
    • Set archive_command to 'cp %p /home/postgres/archivedir/%f' and ensure that the archive log path is a single path.

    For example:

    wal_level = replica

    archive_mode = on

    archive_command = 'cp %p /home/postgres/archivedir/%f'
