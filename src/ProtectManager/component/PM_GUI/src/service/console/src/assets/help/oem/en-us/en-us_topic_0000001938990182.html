Deleting Backup Software Management

After the backup software management information is added, you can delete backup software management by performing the following operations.

Prerequisites

Backup software management has been added.

Procedure

  1. On the menu bar, choose System > Backup Software Management.
  2. Locate the row of the backup software to be deleted, and click Deleted.
