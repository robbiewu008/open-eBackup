Step 2: Creating a DB2 Tablespace Set

If you want to perform DB2 tablespace set protection, create a tablespace set first.

Procedure

  1. Choose Protection > Databases > DB2.
  2. Click the Tablespace Set tab.
  3. Click Create to create a DB2 tablespace set.

    Configure related parameters as described in Table 1.
    Table 1 DB2 tablespace set parameters

    Parameter

    Description

    Name

    User-defined name of a DB2 tablespace set.

    Hosts/Clusters

    Host or cluster to which the tablespace set belongs.

    Instance

    DB2 instance to be protected.

    Databases

    Database to be protected in the DB2 cluster.

    Tablespace

    Tablespace created in the database.

  4. Click OK.
