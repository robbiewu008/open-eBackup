Modifying the AK or SK of a Backup Storage

ProtectManager allows you to modify the AK or SK of a backup storage. If the AK or SK of the backup storage connected to the storage system is changed, it must be also changed on ProtectManager.

Procedure

  1. Choose System > Infrastructure > Backup Storage.
  2. Click More next to the backup storage to be modified and choose Modify Backup Storage.
  3. Modify the backup storage parameters.

    Table 1 describes the parameters.

    Table 1 Backup storage parameters

    Parameter

    Description

    AK

    Access certificate for accessing the object storage

    SK

    Security certificate for accessing the object storage

  4. Click OK.
