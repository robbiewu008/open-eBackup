查询ClickHouse信息

操作步骤

  1. 选择“保护 > 大数据 > ClickHouse”。
  2. 在“集群”页签，查询ClickHouse集群信息。

    1. 该页面展示已注册至本产品的ClickHouse集群信息。

      相关参数说明如表1所示。

      表1 ClickHouse集群信息

      参数

      说明

      名称

      ClickHouse集群名称。

      状态

      ClickHouse集群的状态。

      • 在线：当代理主机与ClickHouse集群所有节点连接正常时，“状态”为“在线”。
      • 离线：当代理主机与ClickHouse集群所有节点连接异常时，“状态”为“离线”。
      • 异常：当代理主机与ClickHouse集群部分节点连接异常时，“状态”为“异常”。

      版本

      ClickHouse的版本号。

      所属用户

      租户下的资源分配给此用户管理。
    2. 单击集群名称，查看集群基本信息和与该集群相关的数据库或表集信息。

  3. 在“数据库”页签，查询ClickHouse数据库信息。

    1. 该页面展示ClickHouse数据库的详细信息。

      相关参数说明如表2所示。

      表2 ClickHouse数据库参数说明

      参数

      说明

      名称

      数据库名称。

      所属集群

      数据库所属的集群。

      SLA

      当ClickHouse数据库已关联SLA时，则显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      ClickHouse数据库当前是否被保护。

      • 未保护：ClickHouse数据库未关联SLA，或者已关联SLA但已禁用保护。
      • 已保护：ClickHouse数据库已关联SLA，且已激活保护。
      • 创建中：ClickHouse数据库已关联SLA，正在创建保护。
    1. 单击名称，查看总览信息、副本信息以及相关任务。
      • 总览

        显示SLA信息、任务执行时间、副本总数等。

      • 副本数据

        您可以依次按照年、月、天查找相应时间的副本。

        时间下方显示，表示该时间存在副本。

      • 相关任务

        展示备份、恢复等任务。

  4. 在“表集”页签，查看ClickHouse表集信息。

    1. 该页面展示ClickHouse表集的详细信息。

      相关参数说明如表3所示。

      表3 ClickHouse表集参数说明

      参数

      说明

      名称

      ClickHouse表集名称。

      所属集群

      ClickHouse表集所属的集群。

      所属数据库

      ClickHouse表集所属的数据库。

      SLA

      • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      • 已保护：ClickHouse表集已关联SLA，且已激活保护。

    2. 单击名称，查看总览信息、副本信息以及相关任务。
      • 总览

        显示SLA信息、任务执行时间、副本总数等。

      • 副本数据

        时间下方显示，表示该时间存在副本。

      • 相关任务

        展示备份、恢复等任务。
