Performing Periodic Backup

After the databases to be protected are associated with a specified SLA, the system periodically backs up the databases based on the SLA.

Prerequisites

  • Before the backup, you need to ensure that the pexpect package has been installed on the agent host and install pexpect 3.2. For details, see "Installing pexpect" in the OceanProtect Appliance 1.5.0-1.6.0 ProtectAgent Installation Guide.
  • Before the backup, you need to ensure that GBase 8a databases are running properly and the network connection of each node in the cluster is normal.
  • During the backup, the passwords of the GBase 8a agent host and database users cannot be changed.

Procedure

  1. Choose Protection > Databases > General Databases.
  2. Select the database to be protected and click Protect.
  3. Select an SLA. You can select a preset SLA or click Create to create one.
  4. Click OK.

    If the current system time is later than the first backup start time of the SLA, you can select to perform backup immediately after the backup SLA is associated or wait for the next scheduling based on the backup policy set in the SLA.
