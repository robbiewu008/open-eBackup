查看系统版本信息

用户可以通过OceanProtect管理界面和CLI两种途径查询并了解系统的版本信息，以便在维护过程中能根据系统版本准确判断配套软件版本。

通过管理界面获取系统当前版本信息

用户可以通过管理界面查询并了解系统当前版本信息。

  1. 选择“系统 > 基础设施 > 集群管理”。
  2. 在“备份集群”页签的“本地集群节点”区域，单击节点名称。
  3. 在弹出的“节点详情”对话框中的“基本信息”区域查看“版本”，即系统当前的版本信息，如图1所示。

    图1 系统基本信息

通过CLI获取系统当前版本信息

用户可以通过CLI查询并了解当前系统的版本信息。

  1. 使用PuTTY，以admin账户登录CLI。
  2. 执行show system general命令，查看系统版本。

    admin:/>show system general

  System Name         : Huawei.Storage               
  Health Status       : Normal                       
  Running Status      : Normal                       
  Total Capacity      : 996.471TB                    
  SN                  : 2102354PBA13D9988001         
  Location            :                              
  Product Model       : OceanProtect X8000          
  Product Version     : 1.2.1RC1                     
  High Water Level(%) : 80                           
  Low Water Level(%)  : 20                           
  WWN                 : 210030fbb8c80ded             
  Time                : 2022-06-17/10:06:16 UTC+08:00 
  Patch Version       :                              
  Description         :

    其中，“Product Version”的参数值即为当前系统的版本信息。
