Step 1: Registering a Huawei Cloud Stack GaussDB Project

Before Huawei Cloud Stack GaussDB backup and restoration, you must register Huawei Cloud Stack GaussDB with the OceanProtect.

Context

Identity and Access Management (IAM) provides permission management for secure access to user's cloud services and resources.

An account is created upon user's successful registration with Huawei Cloud, is the entity that owns, uses, and pays for resources, has full control over these resources, and can access all cloud services of Huawei Cloud. An IAM user is created by an account to use cloud services. Each IAM user has their own identity credentials (password and access key) and uses resources based on assigned permissions.

Prerequisites

  • You have registered with Huawei Cloud and created an IAM user on Huawei Cloud. This user is used to connect Huawei Cloud Stack GaussDB to the OceanProtect. For details about how to create an IAM user, see Create User Groups and Assign Permissions and Create IAM Users and Log In.
  • A private line has been established for the Huawei Cloud Stack GaussDB project and the data protection agent. For details, contact technical support engineers of the service provider.
  • If you need to enable XBSA SSL Certificate Management, ensure that the SSL certificate has been obtained during ProtectAgent installation. For details, see "Configuring Certificate Files for Mutual SSL Authentication" in the OceanProtect Appliance 1.5.0-1.6.0 ProtectAgent Installation Guide

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack GaussDB.
  2. Click Register to register Huawei Cloud Stack GaussDB.

    Table 1 describes Huawei Cloud Stack GaussDB registration information.
    Table 1 Huawei Cloud Stack GaussDB registration information

    Parameter

    Description

    Name

    User-defined name of Huawei Cloud Stack GaussDB.

    Management Plane Address

    IP address of the Huawei Cloud Stack GaussDB management plane. To obtain the IP address:

    On the Tool_generated_IP_Params sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, search for ManageOne-Tenant-Float-IP to obtain the corresponding parameter value.

    Service Plane Address

    IP address of the Huawei Cloud Stack GaussDB management plane. To obtain the IP address:

    On the Tool_generated_IP_Params sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, search for AGW-LB-Float-IP to obtain the corresponding parameter value.

    IAM Username

    IAM username created by the user on Huawei Cloud.

    IAM User Password

    IAM user password created by the user on Huawei Cloud.

    Owning Account of IAM User

    Name of the account to which the IAM user belongs.

    Resource Set Name

    Name of a resource set of a tenant. Obtain the value from the VDC administrator.

    Resource Set ID

    ID of a resource set of a tenant. Obtain the value from the VDC administrator.

    Agent Host

    Data protection agent that has been registered with the OceanProtect.

    XBSA Configuration Path

    Path of the xbsa.conf file configured during backUpAgent installation. You can obtain the path by referring to the XBSA configuration path in Table 1.

    XBSA SSL Certificate Management

    This function is enabled by default. After this function is enabled, the certificate of Huawei Cloud Stack GaussDB will be verified during registration to ensure security of interactions between the OceanProtect and external devices. After this function is disabled, security risks may exist in the communication between the system and the cloud platform. Exercise caution when performing this operation.

    Root Certificate

    Root certificate file with the file name extension .pem. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    User Certificate

    Client user certificate file with the file name extension .crt. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    User Private Key

    Client user private key file with the file name extension .key. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    Private Key Password

    Client private key password character string set when the certificate file is created. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

  3. Click OK.
