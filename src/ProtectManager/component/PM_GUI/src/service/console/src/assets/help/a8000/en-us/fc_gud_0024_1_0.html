Step 3: (Optional) Creating a FusionCompute VM Group (Applicable to 1.6.0 and Later Versions)

You can create a VM group and add VMs to the VM group to protect VM data in batches.

Precautions

A maximum of 40 VMs can be added to a VM group.

Procedure

  1. Choose Protection > Virtualization > FusionCompute.
  1. Select the FusionCompute virtualization platform on the left of the page.
  2. In the right pane, click the VM Groups tab.
  3. Click Create.

    The Create page is displayed.

  4. Set VM Group Name.
  5. (Optional) Filter VMs by Compute Location.
  6. Select the resources to be added to the VM group.

    Resources that have been protected or added to other VM groups cannot be selected.

  7. Click OK.
