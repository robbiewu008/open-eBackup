Obtaining the Endpoint at the Production End

Endpoint indicates the address for accessing the object storage. You can enter the domain name or IP address of the production end. The environment at the production end can be OceanStor Pacific, Huawei Cloud, or Alibaba private cloud. The method of obtaining the endpoint varies depending on the environment at the production end.

  • Obtaining the endpoint on OceanStor Pacific
    1. Log in to the OceanStor Pacific management page.
    2. Choose Resources > Access > Service Network.
    3. Specify Account and Subnet to view the zones for the corresponding account and subnet.
    4. Click a zone name. On the displayed page, click the IP Address Pool tab to view the created IP address/mask, which is the endpoint.
  • Obtaining the endpoint on Huawei Cloud
    1. Log in to the Huawei Cloud management page.
    2. Choose Service List > Storage > Object Storage Service 3.0.
    3. In the left navigation pane, choose Object Storage to view bucket resources.
    4. Click a bucket name to display the Overview page, where you can obtain the endpoint.
  • Obtaining the endpoint on the Alibaba private cloud
    1. Log in to the Alibaba private cloud management page.
    2. Under Products, choose Storage > Object Storage Service.
    3. In the left navigation pane, click Buckets. On the displayed page, click a bucket.
    4. You can view the endpoint of the object storage on the Overview tab page.
