Installing Windows Server Backup

Domain controllers can be backed up only after Windows Server Backup is installed. You can install the tool using the Server Manager or by running commands on Windows PowerShell.

Method 1: Installing Windows Server Backup Using the Server Manager

  1. Log in to the local host as Administrator.
  2. Choose Start > Server Manager.
  3. Click Add roles and features.
  4. In the displayed dialog box, click Next.
  5. Select Role-based or feature-based installation and click Next.
  6. Choose Select a server from the server pool and click Next.
  7. Select Windows Server Backup and click Next.
  8. Click Install.

Method 2: Installing Windows Server Backup by Running Commands on Windows PowerShell

  1. Enter Windows PowerShell in the search box.
  2. Right-click Windows PowerShell and choose Run as administrator from the shortcut menu to enable Windows PowerShell.
  3. Run the following command to perform the installation:

    Install-WindowsFeature -Name Windows-Server-Backup
