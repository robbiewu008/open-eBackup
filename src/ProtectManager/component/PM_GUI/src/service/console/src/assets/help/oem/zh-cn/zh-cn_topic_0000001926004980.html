步骤1：获取KeyStone V3地址

后续注册OpenStack时，需要使用OpenStack的KeyStone V3地址，可参考本节操作获取对应地址。

操作步骤

对于FusionSphere OpenStack，参考以下步骤获取KeyStone V3地址：

  1. 使用PuTTY，以fsp账号通过管理IP地址登录任一OpenStack节点。
  2. 执行以下命令切换到root账号。

    su - root

  3. 执行以下命令导入环境变量，选择“openstack environment variable (keystone v3)”，并输入密码。

    source set_env

  4. 执行以下命令查询KeyStone V3地址。

    openstack endpoint list | grep keystonev3
    回显类似如下，其中接口为Public的keystonev3地址即为要获取的地址，即示例中https://identity.az236.dc236.huawei.com:443/identity/v3，即为要获取的KeyStone V3地址。 
    host02:~ # openstack endpoint list | grep keystonev3
| 6894ac66481e46e493c09b76110ec474 | az236.dc236 | keystonev3    | identityv3     | True    | admin     | https://identity.az236.dc236.huawei.com:443/identity-admin/v3         |
| 70b95d31ae75419980442cfae7362d97 | az236.dc236 | keystonev3    | identityv3     | True    | internal  | https://identity.localdomain.com:8023/identity-admin/v3               |
| 25cb9711e63f420b944d048bddba0d3d | az236.dc236 | keystonev3    | identityv3     | True    | public    | https://identity.az236.dc236.huawei.com:443/identity/v3

对于OpenStack社区版，参考以下步骤获取KeyStone V3地址：

  1. 登录OpenStack社区版的管理页面。
  2. 在左侧导航栏，依次选择“管理员 > 系统 > 系统信息”。
  3. 在“系统信息”页面的“服务”页签下，“keystone ”服务地址的“Public”值即为KeyStone V3地址。
