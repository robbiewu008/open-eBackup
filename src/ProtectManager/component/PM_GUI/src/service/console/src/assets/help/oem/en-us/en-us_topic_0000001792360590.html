Replicating an Object Storage Copy

To prevent data loss in event of a disaster or fault in the local data center, remote data disaster recovery is also a key measure for enterprise data protection. The product replicates local data to a remote data center by using the copy replication technology. If a disaster occurs, the remote data center uses replication copies to restore production data and take over services, implementing remote data protection.
