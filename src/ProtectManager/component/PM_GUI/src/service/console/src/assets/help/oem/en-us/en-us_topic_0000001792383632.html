Managing Huawei Cloud Stack GaussDB Projects

After Huawei Cloud Stack GaussDB is registered with the product, you can perform operations on Huawei Cloud Stack GaussDB.

Choose Protection > Cloud Platforms > Huawei Cloud Stack GaussDB. On the Huawei Cloud Stack GaussDB page that is displayed, select the cloud platform to be operated in the left pane.

Table 1 describes the related operations.

Table 1 Operations related to Huawei Cloud Stack GaussDB projects

Operation

Description

Navigation Path

Grant permissions on Huawei Cloud Stack GaussDB projects.

Scenario

By default, only the system administrator has the permission to protect resources. If you want to assign this permission to a data protection administrator, authorize the data protection administrator as the system administrator to protect resources.

Note

The permission on a single resource can be granted to only one data protection administrator.

Choose More > Authorize Resource and select a data protection administrator to be authorized.

Reclaim Huawei Cloud Stack GaussDB projects.

Scenario

This operation enables you to reclaim resources authorized to the data protection administrator. The data protection administration is not permitted to protect the resources that have been reclaimed.

Choose More > Reclaim Resource.

Scan Huawei Cloud Stack GaussDB.

Scenario

The product automatically scans the Huawei Cloud Stack GaussDB environment every 5 minutes. If the projects in the registered Huawei Cloud Stack GaussDB environment change, the change will be automatically changed to the product. This operation enables you to immediately update the changed information to the product.

Choose More > Resource Scan.

Test Huawei Cloud Stack GaussDB connectivity.

Scenario

Manual connectivity test of the registered Huawei Cloud Stack GaussDB environment is used to check the connectivity between the Huawei Cloud Stack GaussDB environment and the agent host and between the Huawei Cloud Stack GaussDB environment and the production environment.

Choose More > Test Connectivity.

Modify Huawei Cloud Stack GaussDB projects.

Scenario

Perform this operation when you need to modify a Huawei Cloud Stack GaussDB project.

Choose More > Modify.

Delete Huawei Cloud Stack GaussDB projects.

Scenario

Perform this operation when instances in a Huawei Cloud Stack GaussDB project do not need to be protected.

Choose More > Delete.
