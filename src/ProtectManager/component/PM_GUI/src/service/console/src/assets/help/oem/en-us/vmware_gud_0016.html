Step 1: Checking and Installing VMware Tools

Before performing application-consistent backup, you must install VMware Tools on the VM to be protected or target VM. vSphere Client 8.0 is used as an example in this section.

Context

You need to install VMware Tools for VMware VMs in the following scenarios:

  • If you need to perform application-consistent backup, ensure that VMware Tools has been installed on the protected VMware VMs.

Procedure

  1. Log in to the vCenter Server or ESXi using a browser.

    Login address: https://Management IP address of the vCenter Server or ESXi

  2. Expand the navigation tree on the left to view all created VMs.
  3. In the VM list, locate the VM for which you want to perform application-consistent backup and click the VM name.
  4. On the VM details page, check whether VMware Tools 10.2 or later has been installed. If VMware Tools is not installed or the version is earlier than 10.2, install or upgrade VMware Tools by referring to the VMware Tools Documentation.

    • For login to the vCenter Server, check the installation status in the Virtual Machine Details area.
    • For login to the ESXi, check the installation status in the General Information area.
