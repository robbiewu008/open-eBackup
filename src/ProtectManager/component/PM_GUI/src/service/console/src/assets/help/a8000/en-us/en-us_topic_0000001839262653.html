Viewing Huawei Cloud Stack GaussDB Information

After Huawei Cloud Stack GaussDB is registered with the OceanProtect, you can view the project and instance information of Huawei Cloud Stack GaussDB.

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack GaussDB.
  2. On the Project tab page, view project information.

    For details about projects, see Table 1.
    Table 1 Project information

    Parameter

    Description

    Name

    Project name.

    Status

    Project status.

    • Online: Huawei Cloud Stack GaussDB is properly connected to the OceanProtect.
    • Offline: Huawei Cloud Stack GaussDB is disconnected from the OceanProtect.

    Owner

    Data protection administrator to which resources are allocated for management. If -- is displayed, the resource has not been allocated to the data protection administrator.

  3. On the Instance tab page, view the instance information.

    Table 2 describes the detailed instance information.
    Table 2 Instance information

    Parameter

    Description

    Name

    Instance name.

    Click an instance name to view the overview, copy data, and jobs of the instance.

    • Overview

      The name, SLA information, job execution time, and total number of copies are displayed.

    • Copy Data

      Copy data of an instance.

    • Jobs

      All jobs, including backup and restoration jobs, are displayed.

    Status

    Instance status.

    • Backupable: The instance is in the ACTIVE state (indicating that the instance is normal).
    • Unbackupable: The instance is in any of the following states:
      • BUILD: The instance is being created.
      • FAILED: The instance is abnormal.
      • EXPANDING: The instance is being scaled out.
      • REBOOTING: The instance is being restarted.
      • UPGRADING: The instance is being upgraded.
      • RESTORING: The instance is being restored.
      • BACKING UP: The instance is being backed up.
      • STORAGE FULL: The instance disk space is full.
    • Offline: The instance is disconnected from the production environment.

    Owning Project

    Project to which an instance belongs.

    Region

    Region to which the instance belongs.

    Version

    Huawei Cloud Stack GaussDB version.

    SLAs

    If an instance is associated with an SLA, the SLA name is displayed. Otherwise, -- is displayed.

    SLA Compliance
    Whether a backup job is successfully executed based on the backup frequency or backup window set in the SLA.
    • If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
    • If the backup time does not meet the backup time window or backup frequency requirements, for example, if the backup frequency set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup frequency or backup window.

    Protection Status

    Whether the instance is protected.

    • Unprotected: The instance is not associated with an SLA, or the instance is associated with an SLA but protection is disabled.
    • Protected: The instance is associated with an SLA and protection is activated.
    • Creating: The instance is associated with an SLA and protection is being created.
