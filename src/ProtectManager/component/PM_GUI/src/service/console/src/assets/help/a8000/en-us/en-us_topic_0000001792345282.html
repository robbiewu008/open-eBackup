Enabling a Driver

This section describes how to enable a driver.

Prerequisites

  • The running status of the driver is Disabled.
  • No library tapes are loaded by the driver.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Driver tab.
  4. Locate the row where the driver to be enabled resides, and choose More > Enable Driver.
