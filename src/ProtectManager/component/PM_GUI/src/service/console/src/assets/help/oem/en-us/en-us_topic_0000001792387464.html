Step 1: Create a FusionCompute Interconnection User

This section describes how to create an interconnection user between FusionCompute and the product.

Procedure

  1. Log in to FusionCompute.
  2. Create a role on the FusionCompute management system. This role is used to create an account for interconnecting with the product.

    1. Choose System > Rights Management > Role Management.
    2. Click Add Role.
    3. Enter the role information. Set Role Name to OceanProtect, as shown in Figure 1. For details about how to configure permissions, see Configuring FusionCompute Role Permissions.
      Figure 1 Adding a role
    4. Click OK.

  3. On the FusionCompute client, create an account for interconnecting with the product.

    1. Choose System > Rights Management > User Management.
    2. Click Add User.
    3. Enter the user information. Set Permission Type to Interface interconnection user. See Figure 2.
      Figure 2 Creating an interface interconnection user

    4. Click OK.
    5. Choose System Management > Rights Management > Rights Management Policy.
    6. In the Password Policy area, click Modify.
    7. Modify the current value of Interface interconnection user forcibly change passwords upon a reset or initial login to No.
    8. Click Save.
