Viewing a Local Cluster Node

A local cluster is an HA cluster consisting of multiple sets of OceanProtect. This section describes how to view the local cluster node information. For 1.6.0 and later versions, Internal Communication Network Plane is renamed as Internal Communication Network. This section uses 1.5.0 as an example.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Cluster.
  3. In the local cluster node list, view the local cluster node information. Table 1 describes the related parameters.

    Table 1 Local cluster node parameters

    Parameter

    Description

    Name

    Name of a local cluster node. Click the name of a cluster to view its basic information and controller information, and update the login authentication and service authentication. You can also access the device management page to perform related operations.

    Role

    Role of a node, which can be Primary Node, Standby Node, or Member Node.

    Status

    Status of a node.

    • Online: The node is properly connected to the active node. The status of the active node is Online.
    • Offline: The node is disconnected from the active node.
    • Setting: The internal communication network plane of the active node is being added, a member node is being added, or HA is being set.
    • Deleting: The member node is being deleted.

    Model

    Device model of a local cluster node.

    Capacity

    Capacity of a local cluster node.

    Management IP Address

    Management IP address of a local cluster node.

    Internal Communication Network Plane

    Name of the internal communication network plane. When you modify or delete an internal communication network, Modifying or Deleting is displayed.
