Managing NAS File Systems

You can modify, remove, enable, or disable NAS file system protection as well as perform other management operations.

Choose Protection > File Services > NAS File Systems and find the desired NAS file system.

For 1.5.0, choose Protection > File Services > NAS File Systems.

Table 1 describes the operations.

Table 1 Operations related to NAS file system protection

Operation

Description

Navigation Path

Modifying protection

Scenario

You can modify the protection plan of a NAS file system. After the modification, the next backup will be executed based on the new protection plan.

Note

  • The modification does not affect the protection job that is in progress. The new protection plan is used from the next protection period.
  • If the SLA associated with a resource changes, you are advised to modify the current SLA by choosing Protection > Protection Policies > SLAs. You are advised not to associate the resource with another SLA by modifying the protection.

In the row of the target resource, choose More > Modify Protection.

Removing protection

Scenario

You can remove the protection plan if the SLA file system no longer needs to be protected. After the protection plan is removed, the system cannot execute protection jobs for the NAS file system.

Note

  • When a protection job of the NAS file system is in progress, protection cannot be removed.
  • If the SLA associated with a resource changes, you are advised to modify the current SLA by choosing Protection > Protection Policies > SLAs. You are advised not to associate the resource with another SLA after removing the protection.

In the row of the target resource, choose More > Remove Protection.

Performing restoration

Scenario

If a file in the NAS file system is lost or damaged, you can restore the file using its backup.

In the row of the target resource, choose More > Restore.

Disabling protection

Scenario

If you do not need to perform periodic protection for the NAS file system, you can disable the protection plan of the NAS file system. After the protection plan is disabled, the system does not perform automatic backup jobs for the NAS file system.

Note

Disabling protection does not affect ongoing backup jobs.

In the row of the target resource, choose More > Disable Protection.

Activating protection

Scenario

After the protection plan of the NAS file system is disabled, you can activate it again. After the protection plan is activated, the system backs up the NAS file system based on the protection plan.

Note

This operation can be performed only for a NAS file system that has been associated with an SLA and is in the Unprotected state.

In the row of the target resource, choose More > Activate Protection.
Adding a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to quickly filter and manage resources.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.
Removing a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to remove the tag of a resource when it is no longer needed.

Choose More > Remove Tag.
