Deleting a Media Set

This section describes how to delete a media set that is no longer required.

Precautions

  • If a media set is associated with an SLA, it cannot be deleted.
  • Deleting a media set will delete all archived copies in the media set.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Media Set.
  3. In the row where the media set to be deleted resides, choose More > Delete.
  4. In the Danger dialog box that is displayed, confirm the information and click OK.
