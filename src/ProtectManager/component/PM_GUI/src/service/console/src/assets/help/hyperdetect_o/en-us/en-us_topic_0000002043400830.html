Viewing Real-Time Ransomware Detection Results

This section describes how to view real-time ransomware detection results.

Procedure

  1. On the navigation pane, choose Data Security > Ransomware Protection > Overview.
  2. In the Real-time Ransomware Detection Alarms area, view alarms.

    • No alarm information is displayed on the page.

    • An alarm is generated for the file system.

  3. If an alarm exists, perform the following steps to view the alarm information.

    1. Click the alarm information to go to the Alarms and Events page.
    2. Locate the alarm and click the alarm description. The alarm details show the vStore and file system infected by ransomware. Click Export Detection Report.

    3. Set File Name After Exporting and click OK. A message is displayed indicating that the file export command is delivered successfully.

    4. Click Export Records, select the latest detection report, and choose More > Download.
    5. You are advised to open the downloaded report using Notepad. The file column in the report displays the file system name and information about the files infected by ransomware.
