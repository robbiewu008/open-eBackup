数据库账户及密码

是

需要获取具有备份/还原操作权限的数据库账户及密码，请联系数据库管理员获取。

创建账户请参考以下方法：

  1. 使用PuTTY，登录集群任意一台数据库主机。
  2. 执行以下命令，登录postgres数据库。
    gsql -d postgres -p 5432
  3. 执行以下命令，创建账户并设置密码。
    CREATE USER 账户名 with PASSWORD '账户密码';
  4. 执行以下命令，赋予账户权限。
    GRANT ALL PRIVILEGES to 账户名;
