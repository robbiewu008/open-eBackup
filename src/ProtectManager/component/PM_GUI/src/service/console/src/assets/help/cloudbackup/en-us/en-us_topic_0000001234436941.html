Viewing a Rate Limiting Policy

Procedure

  1. Choose Protect > Protection Policies > Rate Limiting Policies.
  2. View information about a rate limiting policy.

    Table 1 describes the related parameters.

    Table 1 Parameters related to a rate limiting policy

    Parameter

    Description

    Name

    Name of a rate limiting policy.

    Maximum Bandwidth

    Maximum amount of data that can be transmitted by a data protection job per second.

    Value range: 10 MB/s to 5120 MB/s

    Description

    Information about the rate limiting policy.
