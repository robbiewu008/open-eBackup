Modifying Basic Information of an Archive Storage

This section describes how to modify basic information of archive storage on OceanProtect Appliance. If the AK or SK of the archive storage connected to the OceanProtect Appliance is changed, you need to modify basic information about the archive storage on the management interface.

Prerequisites

An archive storage has been added.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Object Storage tab page, click More on the right of the target archive storage and select Modify Archive Storage.
  3. Modify the archive storage parameters.

    Table 1 lists related parameters.

    Table 1 Archive storage parameters

    Parameter

    Description

    AK

    Access certificate for accessing the object storage

    SK

    Security certificate for accessing the object storage

  4. Click OK.
