Step 3: Registering a Cluster

This section describes how to register a database cluster when the database to be protected is deployed in a cluster.

Prerequisites

Ensure that the ProtectAgent installation user can be switched to the Oracle installation user by running the su - oracle command. If the grid component is installed, ensure that you can run the su - grid command to switch to user grid.

Procedure

  1. Choose Protection > Databases > Oracle.
  2. On the Cluster tab page, click Register.

    Table 1 describes the related parameters.

    Table 1 Parameters for registering a cluster

    Parameter

    Description

    Name

    User-defined cluster name.

    Type

    Database cluster type. Currently, only RAC is supported.

    Node

    Select all nodes in the cluster.

  3. Click OK.
