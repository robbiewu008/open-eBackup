Managing Nutanix Registration Information

After the Nutanix virtualization environment is registered with the OceanProtect, you can perform operations on the Nutanix virtualization environment on the management page.

Log in to the management page and choose Protection > Virtualization > Nutanix to go to the Nutanix page. On the left of the Nutanix page, select the desired virtualization environment.

Table 1 describes the related operations.

Table 1 Operations related to the Nutanix virtualization environment

Operation

Description

Navigation Path

Viewing Nutanix environment information

Scenario

This operation enables you to view Nutanix environment information, such as the name, port number, IP address, and username.

Choose More > Environment Info.

Performing resource scan

Scenario

The OceanProtect automatically scans the Nutanix virtualization environment every hour. If VMs are added, deleted, or modified in the registered virtualization environment, the VM information is automatically updated to the OceanProtect after the scan. This operation enables you to immediately update the changed VM information to the OceanProtect.

Note

If you do not select to apply the SLA policy to the newly created VM when creating a periodic backup job, the system will not associate the newly discovered VM with an SLA after scanning.

Choose More > Resource Scan.

Testing connectivity

Scenario

This operation tests the connectivity between the OceanProtect and the environment to be protected.

Choose More > Test Connectivity.

Modifying registration

Scenario

You can perform this operation to change the name of Nutanix registration information. If the IP address, port number, or CA certificate of the registered Nutanix virtualization environment is changed, you need to perform this operation to synchronize the change.

Note

Modify the Nutanix registration information when no backup or restoration job is running. Otherwise, the job will fail.

Choose More > Modify Registration.

Canceling registration

Scenario

If you do not need to protect the Nutanix virtualization environment or use it as a restoration target, you can perform this operation to remove the Nutanix virtualization environment.

Note

If a backup job is being executed, the unregistration will fail. Wait until the backup job is complete and try again.

Choose More > Unregister.
