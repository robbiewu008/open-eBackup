找不到系统编写器导致备份任务失败，错误详情包含“8078001D”错误码

现象描述

备份任务失败，界面提示“ActiveDirectory备份子任务由于系统原因导致执行失败，系统错误码：8078001D，错误描述：The system writer is not found in the backup.”。

可能原因

备份失败原因是找不到系统编写器。该报错可能是用户对生产主机中的%windir%\winsxs\temp\PendingRenames或%windir%\winsxs\filemaps目录中的文件的权限不正确。

%windir%为环境变量，一般默认它指向C:\Windows。

处理步骤

请参见在备份中找不到系统编写器。
