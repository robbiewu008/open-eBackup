步骤1：注册对象存储

前提条件

  • 注册对象存储前，请确保对象存储的AK/SK对应的账号具有列举桶（ListBuckets）权限。
  • 备份对象存储前，请确保对象存储的AK/SK对应的账号具有桶读权限、桶ACL读权限。

操作步骤

  1. 选择“保护 > 文件系统 > 对象存储”。
  2. 选择“对象存储”页签，单击“注册”。
  3. 配置对象存储基本信息。

    相关参数说明如表1。

    表1 对象存储注册信息

    参数

    说明

    名称

    自定义对象存储名称。

    类型

    对象存储类型，包括：OceanStor Pacific、华为云Stack对象存储，阿里云对象存储。

    Endpoint

    对象存储的访问地址。

    地址格式：对象存储的域名或业务面IP地址。推荐使用对象存储的域名。

    说明：

    请在本产品管理界面配置DNS服务，使本产品可以访问外部域名地址。配置DNS服务的详细操作，请参见配置DNS服务。

    协议

    本产品与对象存储之间的数据传输的协议。

    • 当使用“HTTPS”协议时，请导入有效的证书以便本产品验证对象存储的信息。请提前向对象存储的管理员获取证书。
      说明：

      当对象存储类型为OceanStor Pacific时，才需要导入证书，如何获取证书操作如下：

      1. 登录OceanStor Pacific管理界面。
      2. 单击“设置 > 证书 > 证书管理”。
      3. 单击“对象存储服务安全证书”，进入证书详情页面。
      4. 单击“CA证书”页签，查看CA证书，并确保CN为RootCA。
      5. 返回证书界面，单击“内置CA”页签，选择并导出Service证书。
    • 当协议设置为HTTP时，本产品将无法验证对象存储环境信息，将存在安全风险。建议您选择安全性高的HTTPS协议。

    证书

    使用“HTTPS”协议时显示该参数，请选择证书类型为对象存储的证书，用于验证对象存储环境信息。

    说明：
    • 当对象存储类型为OceanStor Pacific时，才需要导入证书。
    • 当对象存储类型为阿里云对象存储，且协议选择HTTPS协议时，不支持配置证书（ 通过HTTPS方式访问阿里云对象存储资源时仅支持单向认证）。

    AK

    访问对象存储的接入证书。

    SK

    访问对象存储的安全证书。

    客户端

    选择对象存储的客户端，若未选择，则默认使用内置客户端。

    代理服务器

    若客户端与对象存储间无法直接通信时，需要配置代理服务器。

    说明：

    当备份或恢复的单个对象不超过58GB时，请确保代理服务器最大允许请求体大小大于6MB；当备份或恢复的单个对象超过58GB时，请确保代理服务器最大允许请求体大小大于5GB。

    服务器URL

    仅开启代理服务器后显示该参数，代理服务器URL地址作为本产品与生产端的连接，当前仅支持HTTP协议。

    用户名

    仅开启代理服务器后显示该参数，代理服务器中的用户。

    密码

    仅开启代理服务器后显示该参数，代理服务器中的用户密码。

  4. 单击“确定”。
