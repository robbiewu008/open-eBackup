Backing Up Namespaces or Datasets

Namespace backup is used to back up all resources in the entire Kubernetes namespace, including the original data (YAML files used to create Kubernetes resources) and all PVCs in the Kubernetes namespace.

Dataset backup is used to back up resources (workloads and services) added with the same labels on the CCE cluster in a Kubernetes namespace.
