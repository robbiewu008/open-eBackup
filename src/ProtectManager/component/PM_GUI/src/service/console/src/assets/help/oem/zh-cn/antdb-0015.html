步骤3：注册AntDB集群实例

前提条件

注册AntDB集群实例前，请确保集群中每个主机已进行注册。

操作步骤

  1. 注册集群实例。

    1. 在AntDB页面单击“注册”。
    2. “类型”选择“集群实例”。
    3. 配置数据库实例及其认证信息。

      相关参数说明如表1所示。

      表1 AntDB集群实例注册信息

      参数

      说明

      名称

      安装数据库的用户名称。

      用户名

      运行数据库的操作系统用户名，获取方法请参见备份前准备。

      数据库用户名

      数据库管理员用户名称和密码，获取方法请参见备份前准备。

      数据库密码

      数据库流复制用户名

      数据库用于流复制的用户名称及密码，获取方法请参见备份前准备。

      数据库流复制用户密码
    4. 单击“添加”，在弹出的添加对话框中添加集群实例下的所有节点。

      相关参数说明如表2所示。

      表2 集群实例的节点信息

      参数

      说明

      主机

      选择集群下的主机节点。

      数据库安装路径

      要注册的数据库安装路径，获取方法请参见备份前准备。

      配置文件完整路径

      配置文件xxx.yaml的完整路径。示例如下：

      /etc/adbhamgr/adbhamgr_antdbcluster.yaml

      获取方法请参见备份前准备。

      业务IP地址

      数据库服务接收TCP/IP连接的主机IP地址。

      数据库端口

      数据库服务侦听TCP/IP连接的端口号，获取方法请参见备份前准备。
    5. 单击“确定”。
