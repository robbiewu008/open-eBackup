Step 2: Registering an Informix Cluster

Prerequisites

Before registering an Informix cluster instance, ensure that each host in the cluster has been registered.

Procedure

  1. On the Informix page, click the Cluster tab.
  2. Click Register.
  3. On the Register page, customize the cluster name and select a host.
  4. (Optional) Enable log backup and enter the log backup path.

    • After the log backup function is enabled, the system can restore the database based on a specified time point.
    • Ensure that the entered path exists and Informix database users have read/write permissions on this path.
    • The log path cannot be a built-in directory of the OS, such as /etc, /bin, /usr, or /dev.

  5. Click OK.
