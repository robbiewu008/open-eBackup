Upgrading a Backup Storage Device as a Member Node

The OceanProtect supports upgrading the backup storage device corresponding to the OceanProtect as a member node. This section describes related operations.

Prerequisites

  • The internal communication network has been added for the active node and the backup storage device to be upgraded, and at least two IP addresses have been reserved for the backup storage device to be upgraded.
  • At least one backup storage unit has been created for the backup storage device to be upgraded.
  • The version of the OceanProtect corresponding to the backup storage device to be upgraded is the same as that of the active node.
  • The OceanProtect corresponding to the backup storage device to be upgraded is not added to other replication clusters or multi-domain clusters.
  • The IP address of the internal communication network of the OceanProtect corresponding to the backup storage device to be upgraded must be able to communicate with the IP address of the internal communication network of the active node.
  • No job is running on the OceanProtect corresponding to the backup storage device to be upgraded.
  • You have downloaded the CA certificate of the Server Certificate type of the OceanProtect corresponding to the backup storage device and imported it to the active node. The type is set to Backup/Multi-domain/Replication Cluster. For details about how to import a CA certificate, see "Downloading a Certificate" and "Importing a Certificate" in the OceanProtect Appliance 1.5.0-1.6.0 Security Configuration Guide.

Precautions

  • Only the backup storage device corresponding to the OceanProtect can be upgraded as a member node.
  • Service configuration data will be cleared after the backup storage device corresponding to the OceanProtect, where there are jobs to be executed or data is stored, is upgraded as a member node. Exercise caution when performing this operation.
  • The backup storage device corresponding to the OceanProtect X3000 Appliance cannot be upgraded as a member node of the OceanProtect X6000/X8000/X9000 Appliance and vice versa.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. Select Backup Storage Devices.
  4. In the row that contains a backup storage device, choose More > Upgrade to Backup Member Node.
  5. Enter the member node information in the window that is displayed on the right. For details, see Table 1.
  6. Click OK.
  7. In the Danger dialog box that is displayed, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
