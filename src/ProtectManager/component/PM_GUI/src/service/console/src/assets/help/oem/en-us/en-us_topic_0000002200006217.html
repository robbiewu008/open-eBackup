Step 2: Setting Windows PowerShell Permissions

Ensure that Windows PowerShell has script execution permissions (controlled by RemoteSigned, Bypass, or Unrestricted).

Procedure

  1. Open Windows PowerShell and run the Get-ExecutionPolicy command to check whether Windows PowerShell has the script execution permission controlled by RemoteSigned, Bypass, or Unrestricted. Windows 2019 and SQL Server 2019 are used as examples.

    • If any of the preceding permissions exists, no further action is required.
    • If the preceding permissions do not exist, go to 2.

  2. Run the Set-ExecutionPolicy RemoteSigned command to grant the RemoteSigned permission.
