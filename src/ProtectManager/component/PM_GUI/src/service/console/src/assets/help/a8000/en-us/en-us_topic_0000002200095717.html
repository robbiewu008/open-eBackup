Step 1: Registering an SAP on Oracle Database

Before backing up and restoring an SAP on Oracle database, you need to register the SAP on Oracle database with the OceanProtect.

Procedure

  1. Choose Protection > Applications > SAP on Oracle.
  2. Register an SAP on Oracle instance.

    Click Register. Table 1 describes the parameters for registering an SAP on Oracle instance.

    Table 1 Registration parameters

    Parameter

    Description

    Name

    User-defined database instance name.

    -

    Hosts

    Host where the database resides.

    -

    SAP Instance ID

    SAPSID set during SAP ERP installation.

    Set them by referring to Preparations for Backup.

    Oracle Instance ID

    DBSID set during SAP ERP installation.

    Database Username

    Name of the created Oracle database user.

    Database Password

    Password of the created Oracle database user.

  3. Click OK.
