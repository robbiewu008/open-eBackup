Backing Up AntDB

This section describes how to configure backup. IntelliMate (smart assistant) provides step-by-step help to streamline user operations. Specifically, log in to the product WebUI and click in the upper part of the page. On the page that is displayed, click Learn How to Perform Backup Operations, select the type of the resource to be protected, and click Next to complete the backup configuration according to the backup configuration guide.
