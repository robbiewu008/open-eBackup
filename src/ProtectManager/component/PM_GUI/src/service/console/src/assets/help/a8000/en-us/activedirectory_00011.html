Step 2: Registering an Active Directory Domain Controller

Before performing Active Directory backup, you need to register the Active Directory domain controller with the OceanProtect.

Procedure

  1. Choose Protection > Applications > Active Directory.
  2. Click Register to register the Active Directory domain controller.
  3. Configure resource information.

    Table 1 describes the related parameters.

    Table 1 Registration information of the Active Directory domain controller

    Parameter

    Description

    Name

    Customized name of the Active Directory domain controller.

    Client

    Select the host that has been registered with the OceanProtect. Before selecting a client, complete ProtectAgent registration by referring to "Installing ProtectAgent (Automatic Push Mode)" in the OceanProtect Appliance 1.5.0-1.6.0 ProtectAgent Installation Guide.

    NOTE:

    Only one Active Directory domain controller can be registered with a host.

  4. Click OK.
