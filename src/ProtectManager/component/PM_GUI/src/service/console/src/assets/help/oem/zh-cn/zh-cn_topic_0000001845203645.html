查看卷副本信息

参考本节查看卷副本的详细信息。

操作步骤

  1. 选择“数据利用 > 副本数据 > 文件系统 > 卷”。
  2. 选择副本浏览方式浏览副本信息。

    复制副本仅支持在复制到的目标集群查看。

    • 选择“资源”页签：以资源维度查看卷副本信息。当某卷损坏，您需要恢复该卷时，建议从该入口进入查找相应时间点副本。

      参数说明如表1所示。

      表1 副本参数说明

      参数

      说明

      资源

      名称

      卷名称。

      位置

      卷所在主机的名称。

      状态

      卷是否已被删除。

      • 是，则状态为不存在。
      • 否，则状态为存在。

      副本

      数量

      卷在当前主机的副本数量。副本类型包括：备份副本、复制副本和归档副本。

      复制副本SLA

      复制副本关联的归档SLA。

      如果已为该资源产生的复制副本配置归档SLA，则此处显示SLA名称。

      复制副本保护状态
      复制副本当前是否被保护。
      • 未保护：复制副本未关联SLA，或者复制副本已关联SLA但已禁用保护。
      • 已保护：复制副本已关联SLA，且已激活保护。
      • 创建中：复制副本已关联SLA，正在创建保护。仅1.5.0及之前版本存在该状态。
    • 选择“副本”页签：以卷副本维度查看所有副本信息。当需要基于某个指定的副本进行恢复时，建议从该入口进入查找相应副本。

      参数说明如表2所示。

      表2 副本参数说明

      参数

      说明

      副本

      备份时间

      备份副本的生成时间。

      副本时间

      副本生成的时间。

      单击“副本时间”，可以查看副本过期时间等信息，过期时间显示“--”表示副本永久保留。

      状态

      副本的状态。

      • 正常：副本可用。
      • 无效：副本无效。
      • 删除中：副本处于正在删除过程中。
      • 恢复中：副本处于恢复或即时恢复过程中。
      • 挂载中：副本处于正在挂载过程中。
      • 卸载中：副本处于正在销毁过程中。
      • 已挂载：挂载完成后生成的副本状态。
      • 删除失败：副本删除失败。

      位置

      副本所在的位置。

      生成方式
      副本的生成方式。
      • 复制：由复制任务产生的副本。
      • 备份：由备份任务产生的副本。
      • 对象存储归档：由归档任务（归档存储为对象存储）产生的副本。
      • 磁带归档：由归档任务（归档存储为磁带库）产生的副本。
      • 即时挂载：由即时挂载任务产生的副本。
      • 级联复制：由级联复制任务产生的副本。
      • 反向复制：由反向复制任务产生的副本。

      副本类型

      生成副本的备份类型。

      包括：全量、永久增量（合成全量）。

      WORM
      副本是否已设置为WORM副本。
      • ：未设置。副本未设置为WORM副本，即副本为普通副本。
      • ：已设置。副本已设置为WORM副本，WORM副本不支持删除和缩短保留时间。
      • ：设置中。副本正在设置为WORM副本。处于该状态的副本不支持删除和修改保留时间。
      • ：设置失败。副本设置为WORM副本失败。设置失败的副本不支持删除和缩短保留时间。

      操作系统备份

      副本是否已开启操作系统备份。

      副本

      索引

      副本生成后，系统将为卷副本中的所有文件及目录建立索引，便于快速查找相应文件进行恢复。包括以下索引状态：

      • 已索引：系统已为副本建立索引。
      • 索引中：副本生成后，系统正在为副本建立索引。
      • 未索引：SLA中未开启索引功能。
      • 索引删除中：正在删除索引。
      • 索引失败：系统为副本创建索引失败。

      资源

      名称

      卷名称。

      位置

      卷所在主机的名称。

      状态

      卷是否已被删除。

      • 是，则状态为不存在。
      • 否，则状态为存在。
