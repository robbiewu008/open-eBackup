关于全局搜索

什么是全局搜索

本产品对接入系统的资源提供搜索功能，能够快速查找资源，以及备份副本中的文件、目录。全局副本数据搜索能力为文件级恢复和文件下载提供了基础能力。对于1.6.0及后续版本，本产品还提供了全局标签搜索能力。

副本数据全局搜索

资源全局搜索是记录资源元数据信息，对资源进行搜索查询。本节主要介绍副本数据全局搜索的业务流程。

本产品支持对FusionCompute/VMware/华为云Stack/HDFS/NAS文件系统/NAS共享/对象存储/文件集/OpenStack（仅1.6.0及后续版本）/卷/CNware/Hyper-V/阿里云/FusionOne Compute/NDMP/Nutanix的备份副本执行全局搜索，通过对搜索内容创建索引的方式实现快速查找能力，支持在线索引和离线索引两种索引创建方式。在线索引是在备份副本生成后系统立即对其创建索引。离线索引是备份完成后，手动选择副本创建索引。

以下详细介绍备份副本的索引创建流程，如图1。

图1 副本数据全局搜索业务流

  1. 备份任务完成或通过管理界面手动触发索引创建任务后，管理系统调度任务并发送至数据使能引擎。
  2. 数据使能引擎创建备份副本所在文件系统快照的克隆，并将克隆挂载至本产品。
  3. 数据使能引擎扫描克隆中虚拟磁盘文件的目录和文件列表。
  4. 根据扫描结果生成索引，插入索引至搜索引擎Elasticsearch。

    索引建立后用户即可通过界面对备份副本或副本中的文件进行搜索、展示。
