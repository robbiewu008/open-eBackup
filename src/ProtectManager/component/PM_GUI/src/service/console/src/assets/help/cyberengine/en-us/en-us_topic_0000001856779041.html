Managing a Whitelist

The OceanCyber 300 Data Security Appliance allows users to view or delete a whitelist, or disassociate a whitelist from a policy.

Log in to the OceanCyber 300 Data Security Appliance, choose Data Security > Ransomware Protection > Real-Time Detection > Whitelist, and locate the target whitelist. Table 1 describes the related operations.

Table 1 Whitelist-related operations

Job

Description

Navigation Path

View a whitelist

Scenario

This operation enables you to learn about the number and details of whitelists.

In the whitelist list, view the name, type, number of associated policies, and creation time of the whitelists.

Disassociating

Scenario

This operation enables you to disassociate a whitelist from the real-time detection policy when you do not need to use the whitelist for real-time detection.

Locate the row that contains the target whitelist, and choose More > Disassociate.

Deleting a whitelist

Scenario

This operation enables you to delete a whitelist when you do not need to use the whitelist for real-time detection.

Note

After a whitelist is deleted, it is removed from the system and cannot be restored.

Locate the row that contains the target whitelist, and choose More > Delete.
