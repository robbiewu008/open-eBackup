Importing a Management Data Backup

This section describes how to import an original management data backup stored in a local directory into the system for overwriting system management data.

Procedure

  1. Choose System > Settings.
  2. In the navigation tree, click Management Data Backup.
  3. Click Import Backup.
  4. Select a file, enter the data encryption password, and click OK.

    In the displayed dialog box, confirm the information and click OK.

Related Operations

After product reinstallation, the original management data backup file can be imported.

  1. Choose System > Settings.
  2. In the navigation tree, click Management Data Backup.
  3. Click Import Backup.
  4. Select a file, enter the data encryption password, and click OK.

    In the displayed dialog box, confirm the information and click OK.

  5. In the displayed dialog box, confirm the information and click Continue to Import.
