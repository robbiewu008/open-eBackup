Deleting a User

You can delete a user that is no longer needed.

Context

The system administrator can delete other administrators.

Procedure

  1. Choose System > Security > RBAC > Users.
  2. (Optional) Click next to Name to search for a user.
  3. In the row of the target user, choose More > Delete on the right.
  4. In the dialog box that is displayed, confirm the information and click OK.
