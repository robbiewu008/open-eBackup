Managing Logs

By viewing debug logs, you can understand the error information and prompt information that are triggered when you operate or design a template.

Procedure

  1. Choose System > Log > Log Management.
  2. In the System Debug Log Settings area, select the level of logs to be exported from the Log Level drop-down list. You can export the debug logs at the selected level and above.

    You can set log levels in batches. After selecting the levels of logs to be exported, click OK to complete the setting.

  3. Set parameters in the Log Export area.

    Table 1 describes the related parameters.

    Table 1 Parameters in the Log Export area

    Parameter

    Description

    Node Name

    Select the nodes whose information you want to export.

    Software Module Name

    Available options are Infrastructure and Data Protection Engine. You can select multiple options.

    Export Range

    Available options are Based on custom time range and All. The custom time ranges from 1 to 10,000 days.

    Log File Name

    Enter a log file name.

  4. Click Export to export debug logs.
