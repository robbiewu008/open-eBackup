Managing the Alibaba Cloud Platform

After the Alibaba Cloud platform is registered with the product, you can perform operations related to the Alibaba Cloud platform on the management page.

You need to log in to the management page, choose Protection > Cloud Platforms > Alibaba Cloud to go to the Alibaba Cloud page. On the left of the Alibaba Cloud page, select the target cloud platform and perform related operations.

Table 1 describes the related operations.

Table 1 Alibaba Cloud-related operations

Operation

Description

Navigation Path

Viewing Alibaba Cloud registration information

Scenario

This operation enables you to view Alibaba Cloud registration information.

Choose More > Environment Info.

Scanning the Alibaba Cloud platform

Scenario

The product automatically scans resources on the Alibaba Cloud platform every 24 hours. If any projects or cloud servers on the registered Alibaba Cloud platform change, the change will be automatically updated to the product after the scan. This operation enables you to immediately update changed projects or cloud servers to the product.

Note

If you do not select to apply an SLA to the newly created cloud server when creating a periodic backup job, the system will not associate the newly discovered cloud server with an SLA after the scan.

Choose More > Resource Scan.

Testing Alibaba Cloud connectivity

Scenario

Manual connectivity test of the registered Alibaba Cloud environment is used to check the connectivity between the Alibaba Cloud environment and the host where the client is installed, and between the Alibaba Cloud environment and the production environment.

Choose More > Test Connectivity.

Modifying Alibaba Cloud registration information

Scenario

This operation enables you to change Alibaba Cloud registration information. If the Keystone V3 address, username, password, or certificate of the registered Alibaba Cloud is changed, perform this operation to change the corresponding information.

Note

Modify Alibaba Cloud registration information when no backup or restoration job is running. Otherwise, the job will fail.

Choose More > Modify Registration.

Table 1 describes the parameters of Alibaba Cloud registration information.

Canceling Alibaba Cloud registration

Scenario

If a cloud server on a cloud platform no longer needs to be protected or the cloud server will not be used as the restoration target, perform this operation to delete the cloud server.

Note

If a backup job is running, the registration cannot be canceled. You can perform deregistration after the job is complete.

Choose More > Unregister.
