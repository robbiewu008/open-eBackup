Introduction

About the Application

Nutanix provides an integrated solution with the physical infrastructure (Nutanix hyper-converged server) in terms of hardware and the virtual infrastructure (Nutanix AHV virtualization) in terms of software.

Nutanix Prism is a management and monitoring tool provided by Nutanix. It provides a unified management interface to simplify data center O&M. Nutanix Prism includes the Prism Element (PE) platform and Prism Central (PC) platform, which manage a single cluster and multiple clusters, respectively. For more information, visit the Nutanix official website.

The product supports comprehensive protection for Nutanix VM data. With the deployed client, the product provides data protection functions, such as backup, replication, and restoration. The product controls the backup and restoration processes and exchanges data based on the client for data protection.

Objects to Be Protected

The product provides protection for Nutanix VM data, including VM operating systems (OSs), application files, user data, and configuration information.

Backup and Restoration Granularity

The product can back up and restore Nutanix VM data based on different granularities.

  • Backup granularity

    VM level: The entire VM is backed up.

  • Restoration granularity
    • VM level: The entire VM is restored.
