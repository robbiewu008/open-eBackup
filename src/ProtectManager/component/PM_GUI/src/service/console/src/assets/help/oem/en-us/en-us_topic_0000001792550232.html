Step2: Enabling Local Archiving for the Database

Before performing a backup job, local archiving must be enabled for the Dameng database. Otherwise, the backup fails.

Procedure

  1. Log in to the Dameng database host.
  2. Configure dm.ini and modify the ARCH_INI parameter.

    ARCH_INI = 1

  3. Configure dmarch.ini.

    [ARCHIVE_LOCAL1]
ARCH_TYPE = LOCAL 
ARCH_DEST = /dm/data/DAMENG/arch 
ARCH_FILE_SIZE = 128 
ARCH_SPACE_LIMIT = 0
