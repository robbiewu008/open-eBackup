Enabling Real-Time Ransomware Detection

This section describes how to enable real-time ransomware detection.

Prerequisites

NFS sharing has been enabled for the file system on which real-time ransomware detection is to be performed.

Procedure

  1. Click the NAS Ransomware Detection tab, select the vStore for which you want to enable real-time ransomware detection, and choose More > Enable Real-time Ransomware Detection.

    • You can also select multiple vStores for which you want to enable real-time ransomware detection and click Enable Real-time Ransomware Detection above the vStore list.
    • If real-time ransomware detection is not required, choose More > Disable Real-time Ransomware Detection. Alternatively, select multiple vStores for which you want to disable real-time ransomware detection and choose More > Disable Real-time Ransomware Detection above the vStore list.

  2. Confirm your operation as prompted.

    In the dialog box for confirmation that is displayed, you can determine whether to enable ransomware detection false alarm analysis and click OK.

    Ransomware detection false alarm analysis is enabled by default. After this function is enabled, the accuracy of ransomware detection is further improved, but the detection time is expected to increase by 30 minutes.

    • If you require high detection accuracy, you are advised to enable ransomware detection false alarm analysis.
    • If you require short detection duration, you are advised to disable ransomware detection false alarm analysis.
