How Do I Start the Windows Remote Management (WS-Management) Service?

This section describes how to start the Windows Remote Management (WS-Management) service.

Procedure

  1. Log in to the agent host as the system administrator.
  2. Right-click the task bar and choose Task Manager from the shortcut menu.
  3. In the Task Manager window, click the Services tab.
  4. Click Open Service in the lower left corner of the page.
  5. On the Services page, find the Windows Remote Management (WS-Management) service.
  6. Right-click the Windows Remote Management (WS-Management) service and choose Start from the shortcut menu.

    If the status of the service is Running, the service is started successfully.
