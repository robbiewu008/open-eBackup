Overview of Common Management Software

The software provided by the OceanProtect consists of software running on the storage system, maintenance terminal, and application. These types of software work jointly to deliver backup services in a smart, efficient, and cost-effective manner.

This document focuses on common management software. You can use OceanProtect and DeviceManager to query, configure, manage, and maintain the storage system. You can use serviceability tools, such as SmartKit and DME IQ, to improve the maintenance efficiency.

Table 1 describes software commonly used to manage the storage system.

Table 1 Common management software

Management Software

Function

OceanProtect

Data protection service management: storage resource allocation, user management, data protection feature management, device performance monitoring, and alarm management.

DeviceManager

Data protection device management: device management, system time management, and IP address management.

SmartKit

Data protection software and hardware maintenance: routine maintenance, upgrade/patching, troubleshooting, capacity expansion, and parts replacement.

DME IQ

Data protection cloud-based O&M: alarm reporting, file upload, and remote access.

Introduction to DME IQ

As a large number of Huawei IT products such as storage devices, servers, and cloud computing products are put into use, users have increasingly high requirements on the troubleshooting efficiency. In traditional service support mode, technical support personnel provide local services manually. Faults may not be detected in a timely manner and information may not be delivered correctly.

The DME IQ intelligent cloud O&M platform (DME IQ for short) integrates cloud management, remote maintenance, automatic service request (SR) creation, and proactive prediction and prevention functions to improve customer O&M capabilities and enable planned maintenance actions against potential risks.

Being authorized by customers, DME IQ monitors device alarms in 24/7 mode. Whenever an alarm is detected, the DME IQ cloud system is automatically notified, and SRs are created. Huawei service engineers will then help solve problems in a timely manner.

The eService Client works with the DME IQ cloud system.

  • The Client is deployed on the customer side.

    It collects customer device alarms and sends them to the Huawei cloud system in a timely manner to implement remote maintenance functions, including remote inspection and remote log collection.

  • The cloud system is deployed on the DME IQ cloud system.

    It receives device alarms from the Client in 24/7 mode, automatically notifies Huawei technical support personnel to handle these alarms in a timely manner, and supports remote inspection and log collection for devices on the customer side.

Intelligent O&M has the following advantages:
  • The self-service intelligent O&M system provides precise and customized information services for customers.
  • Based on Huawei Cloud, the DME IQ cloud system drives IT O&M activities through big data analysis and AI technologies to identify faults in advance, reduce O&M difficulties, and improve O&M efficiency.
  • Data is encrypted for secure transmission. DME IQ can access a customer's system only after being authorized by the customer.
  • SRs are automatically created. Services are provided in 24/7 mode. Proactive O&M is secure and reliable.
  • Customers can use any PC to access DME IQ at any time and place to check device information.

For details, see the DME IQ Intelligent Cloud O&M Platform User Guide.
