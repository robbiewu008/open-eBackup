Setting a Windows Server to Enter the Directory Services Repair Mode (CLI)

To restore Active Directory, you need to set the Windows server where Active Directory is located to the DSRM.

Procedure

  1. Log in to the Windows server.
  2. In the cmd window, run the bcdedit /set safeboot dsrepair command as the system administrator. After the command output indicating successful command execution is displayed, run the shutdown/r command to restart the Windows server.

    After the restart, the Windows server enters the DSRM. After the restoration is complete, the Windows server automatically restarts and exits the DSRM.
