Importing a Management Data Backup

This section describes how to import an original management data backup stored in the local PC into the system for restoring management data.

Precautions

The size of the imported backup data cannot exceed 4 GB.

Procedure

  1. Choose System > Settings.
  2. In the navigation tree, click Management Data Backup.
  3. Click Import Backup.
  4. Select a file and click OK.
