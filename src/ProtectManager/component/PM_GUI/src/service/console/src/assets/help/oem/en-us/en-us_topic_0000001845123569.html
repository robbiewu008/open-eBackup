Manually Executing Backup

If you want to perform a backup immediately, you can manually perform the backup.

Context

  • During a manual backup, the parameters (except Automatic Retry) defined in the SLA are applied to the manual backup job.
  • The retention period of copies generated during the manual backup is defined in the SLA.

Procedure

  1. Choose Protection > File Systems > Volume.
  2. In the row that contains the volume to be backed up, choose More > Manual Backup.

    You can select multiple resources to perform manual backup in batches. Select multiple resources and choose More > Manual Backup.

  3. Set the name of the copy generated during manual backup.

    If this parameter is left empty, the system sets the name to backup_Timestamp by default.

  4. Select a protection policy, which can be full backup or forever incremental (synthetic full) backup.
  5. Click OK.
