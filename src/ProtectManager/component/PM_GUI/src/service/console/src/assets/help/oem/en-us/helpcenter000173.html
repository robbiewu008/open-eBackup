Viewing Archive Storage Information

This section describes operations to view basic information of an archive storage.

Prerequisites

An archive storage has been added.

Procedure

  1. On the Object Storage tab page, click Name of an archive storage system. In the dialog box that is displayed on the right, view information about the archive storage.

    Table 1 lists related parameters.

    Table 1 Archive storage parameters

    Parameter

    Description

    Name

    Name of the archive storage, which is user-defined.

    Type

    Type of the archive storage.

    Object Storage Product

    Object storage products connected to the product.

    OceanStor Pacific, OceanStor 100D (FusionStorage OBS), Huawei Cloud OBS, Microsoft Azure Blob Storage (only supported in 1.6.0 and later versions), and AWS S3 are supported.

    NOTE:

    If you select OceanStor Pacific for Object Storage Product, you need to log in to DeviceManager of OceanStor Pacific by referring to Logging In to DeviceManager and perform the following operations to enable V2 authentication: choose Settings > Object Service Settings > Security Settings, and on the displayed Security Access Policy tab page, click to the right of Enable V2 Authentication.

    Endpoint

    Access address of the archive storage

    Address format: domain name or service plane IP address of the archive storage

    NOTE:
    • If the access address of archive storage is in domain name format, configure the DNS service by referring to "(Optional) Configuring DNS" in the Installation Guide to support domain name resolution.
    • If the access address of archive storage is in IP address format, the system uses port 80 to access the archive storage by default when the protocol is HTTP and uses port 443 when the protocol is HTTPS. If you do not use the default port to connect to the archive storage, enter both the IP address and the port used to connect to the archive storage, for example, 10.10.1.1:60.

    Protocol

    Protocol used for data transmission between the product and archive storage.

    • When HTTPS is used, import the certificate of the Object Storage type so that the product can verify the archive storage information.

      For details about how to import a certificate, see "Importing a Certificate" in the Security Configuration Guide.

      NOTE:
      • If OceanStor Pacific is used as the archive storage, obtain the certificate by referring to "Exporting a Built-In CA Certificate" in the OceanStor Pacific Series 8.1.5 Security Configuration Guide, and export the Service certificate. When exporting a certificate, change the certificate file name extension to .pem. If the system displays "The CA certificate has expired or is invalid." when you add the archive storage and other parameter settings meet requirements, rectify the fault by referring to What Should I Do If a Message Indicating that the CA Certificate Has Expired or Is Invalid Is Displayed When I Add Archive Storage?.
      • If Huawei Cloud OBS or FusionStorage OBS is used as the archive storage, contact the archive storage administrator to obtain the certificate.
      • If AWS S3 is used as the archive storage, contact the vendor to obtain the certificate.
    • When HTTP is used, the product cannot verify the archive storage environment information, which poses security risks. You are advised to select HTTPS for security purposes.

    AK
    The object storage service supports AK/SK authentication, in which the Access Key ID/Secret Access Key (AK/SK) encryption method is used to verify the identity of a request sender.
    • AK: indicates the ID of the access key. It is a unique identifier used together with a secret access key to sign requests cryptographically.
    • SK: is a key used with an AK to sign requests cryptographically. It can identify a request sender and prevent the request from being modified.

    Contact the archive storage administrator to obtain the AK/SK.

    SK

    Data Bucket

    Name of the bucket that stores archived data

    Index bucket

    Name of the bucket that stores index data

    Use Proxy Server

    When the product uses an agent server to access the archive storage, configure the address, username, and password of the agent server.

    NOTE:
    • If an agent server is required, ensure that the maximum size of the request body allowed by the agent server is greater than 100 MB.
    • The agent server supports only the HTTP protocol. The address format is http://Agent server IP address:Port number.

    Capacity Alarm Threshold

    Upper threshold of the used capacity of a bucket. When the used capacity reaches the threshold, an alarm is reported.
