Managing the Real-Time Ransomware Detection Whitelist

You can delete a real-time ransomware detection whitelist or disassociate vStores from the whitelist.

On the navigation pane, choose Explore > Ransomware Protection > Ransomware Detection > Real-Time Ransomware Detection Whitelist. Perform management by referring to Table 1.

Table 1 Managing a real-time ransomware detection whitelist

Operation

Description

Navigation Path

Deleting a ransomware detection whitelist

Scenario

Delete a real-time ransomware detection whitelist.

Click Delete.

Disassociating from a vStore

Scenario

Disassociate a real-time ransomware detection whitelist from a vStore.

Click Disassociate from vStore.

Associating with a vStore

Scenario

Associate a real-time ransomware detection whitelist with a vStore.

Click Associate with vStore.
