Step 3: Configuring VLAN Ports for the Replication Network

If the replication network and service network share network ports and bond networking is used, refer to this section to configure VLAN ports. Otherwise, skip this section.

Precautions

If multiple bond ports of a replication node are used as replication ports, you need to create VLAN ports separately based on the bond ports and ensure that the VLAN IDs of these VLAN ports are the same.

Procedure

  1. Log in to DeviceManager.
  2. Choose Cluster > Network > Port and Route > VLAN Port.
  3. Choose Create > Create VLAN Port.

    The Create VLAN Port page is displayed.

  4. Set the parameters listed in Table 1.

    Table 1 VLAN port parameters

    Parameter

    Description

    Port Type

    Type of the port used to create a VLAN port. Select Bond port.

    Transmission Protocol

    Transmission protocol of the VLAN port. Select TCP.

    Select Port

    Selects a port.

    VLAN Port Name

    Name of the VLAN port.

    NOTE:
    • The VLAN port name can contain a maximum of 13 characters.
    • The VLAN port name can contain only lowercase letters or digits and must end with period (.) + VLAN ID.

    VLAN ID

    ID of the VLAN that uses the VLAN port. The value ranges from 1 to 4094.

    Configure IP Address

    Sets the IP address and subnet mask/prefix of the VLAN port after enabling Configure IP Address.

    • If the IP addresses are IPv4 addresses, enter the corresponding subnet mask.
    • If the IP addresses are IPv6 addresses, enter the corresponding prefix.
    NOTE:

    Click Add to configure a second IP address for the VLAN port.

  5. Click OK.
