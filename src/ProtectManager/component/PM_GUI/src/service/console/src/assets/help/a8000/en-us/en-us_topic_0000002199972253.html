Viewing Fileset Information

After a fileset is created, you can view its details on the WebUI.

Some parameters are not displayed by default. You can click next to Operation and select a parameter to display it on the current page.

Procedure

  1. Choose Protection > File Systems > Filesets.

    For 1.5.0, choose Protection > Bare Metal > Filesets.

  2. View the fileset information.

    Table 1 describes the related parameters.
    Table 1 Fileset information

    Parameter

    Description

    Name

    Name of the fileset.

    Client Name

    Name of the host where the fileset is located.

    IP Address

    IP address of the host where the fileset is located.

    OS

    Operating system used on the host.

    OS Backup

    Whether OS backup is enabled for a copy.

    Associated Template

    Name of the template used for creating a fileset.

    SLAs

    If an SLA is associated with the fileset, the SLA name is displayed. Otherwise, -- is displayed.

    SLA Compliance
    Whether a backup job is successfully executed based on the backup interval or backup window set in the SLA.
    • Met: If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
    • Missed: If the backup time does not meet the backup time window or backup interval requirements, for example, if the backup interval set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup interval or backup window.

    Protection Status

    Indicates whether the fileset is protected.

    • Unprotected: The fileset is not associated with an SLA, or the fileset is associated with an SLA but protection is disabled.
    • Protected: The fileset is associated with an SLA and protection is activated.
    • Creating: The fileset is associated with an SLA and protection is being created.
    Tag
    NOTE:

    This parameter is available only in 1.6.0 and later versions.

    If a label has been added for the resource, the label name is displayed. Otherwise, -- is displayed.

  3. Click a fileset name to view the fileset overview, copy information, protection details, and related jobs.

    • Overview

      Displays the fileset name, SLA information, job execution time, and total number of copies.

      You can also view the files and directories that are protected in the current fileset.

    • Copy Data

      You can search for copies by year, month, and day.

      If is displayed below the time, a copy at that time exists.

    • Protection Details

      Displays the fileset protection advanced parameters that you have configured.

    • Jobs

      Displays all jobs related to the fileset, including backup and restoration jobs.
