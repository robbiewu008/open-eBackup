Deleting an External Certificate

If you no longer need an external certificate, delete it by referring to this section.

Precautions

  • A deleted external certificate will be untrusted.
  • The Huawei_IT_Product certificate cannot be deleted.

Procedure

  1. Choose System > Security > Certificates.
  2. In the row that contains the target external certificate, choose More > Delete.
  3. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
