How Do I Enable the Network Adapter of a VMware VM?

If the network adapter of a VMware VM is disconnected, you cannot remotely log in to the VM. To enable the network adapter of the VM, perform the following operations (vCenter 6.7.0 is used as an example):

  1. Access the address for logging in to the vCenter Server using a browser.

    Login address: https://vCenter Server IP address

  1. Find the VM whose network adapter needs to be enabled.
  2. Choose ACTIONS > Edit Settings.
  3. Select Connect and Connect At Power On of the network adapter.
  4. Click OK.
