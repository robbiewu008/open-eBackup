Viewing the SAP HANA Database Environment Information

After an SAP HANA database is registered with the product, you can view details about the database on the management page.

Some parameters are not displayed by default. You can click next to Operation and select a parameter to display it on the current page.

Procedure

  1. Choose Protection > Databases > General Databases.

    Table 1 lists the details about the databases that have been registered with the product.
    Table 1 Database information

    Parameter

    Description

    Name

    Database name.

    Status

    Database status.

    If a database is running, the status is Online. Otherwise, the status is Offline.

    Database Type

    Database type, for example, SAP HANA.

    Node IP Address

    IP address of the node where a database is located.

    Version

    Database version.

    SLAs

    If an SLA is associated with the database, the SLA name is displayed. Otherwise, -- is displayed.

    SLA Compliance
    Whether a backup job is successfully executed based on the backup interval or backup window set in the SLA.
    • Met: If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
    • Missed: If the backup time does not meet the backup time window or backup interval requirements, for example, if the backup interval set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup interval or backup window.

    Protection Status

    Indicates whether a database is protected.

    • Unprotected: The database is not associated with an SLA, or the database is associated with an SLA but protection is disabled.
    • Protected: The database is associated with an SLA and protection is enabled.
    • Creating: The database is associated with an SLA and protection is being created.
    Owner
    NOTE:

    This parameter is available only in 1.5.0.

    Data protection administrator of a host or cluster.

    If a host or cluster has been authorized to a data protection administrator, the name of the data protection administrator is displayed. If no data protection administrator is authorized, -- is displayed.
    Tag
    NOTE:

    This parameter is available only in 1.6.0 and later versions.

    If a tag has been added for the resource, the tag name is displayed. Otherwise, -- is displayed.

  2. Click the database name to view its overview information, copy information, and related jobs.

    • Overview

      Displays the database name, SLA information, job execution time, and total number of copies.

    • Copy Data

      You can search for copies by year, month, or day.

      If is displayed below the time, copies exist at that time.

    • Jobs

      Displays all jobs related to the database, including backup, archive, and restoration jobs. You can choose to display ongoing or historical jobs.
