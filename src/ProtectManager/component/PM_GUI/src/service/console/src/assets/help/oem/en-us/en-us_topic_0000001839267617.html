Step 1: Creating a Fileset Template

If you want to protect files in the same path on multiple hosts and the rules for filtering the files to be protected are the same on the hosts, you can create a fileset template by referring to this section. You can use a template to create filesets for multiple hosts in batches, improving the configuration efficiency.

Procedure

  1. Choose Protection > File Systems > Filesets.

    For 1.5.0, choose Protection > Bare Metal > Filesets.

  2. Click Templates.
  3. Click Create to create a template.

    Table 1 describes the related parameters.

    Table 1 Fileset template parameters

    Parameter

    Description

    Name

    User-defined template name, which contains a maximum of 32 characters.

    OS

    Select the operating system of the host where the fileset to be protected resides.

    File Path

    Click Add, enter the absolute path of the file to be protected, and click OK.

    If you want to protect files in multiple paths, click Add to add other paths. You can add a maximum of 64 paths and add paths in batches, which are separated with commas (,).

    Filtering Rules

    File

    Select the files to be included or excluded.

    The rules for filtering files are as follows:

    • Enter the full path of the file.
    • When multiple paths are configured, click Save before entering a new path.
    • The path can contain the wildcard character *.

    Directory

    Directories to be included or excluded. Click Save.

    The rules for filtering directories are as follows:

    • Enter the full path of the directory.
    • When multiple paths are configured, click Save before entering a new path.
    • The path can contain the wildcard character *.
    When multiple filtering rules are set, you can determine whether the rules take effect based on the following rules:
    • If only Contain conditions exist, the logic relationship between these conditions is AND. That is, the search result is filtered based on all the selected conditions.
    • If only Exclude conditions exist, the logic relationship between these conditions is AND. That is, the search result is filtered based on the union of all the selected conditions.
    • If both Contain and Exclude conditions exist, the Exclude condition is determined only in the Contain conditions.

  4. Click OK.
