Quick Start

This section describes functions of OceanCyber, helping you quickly find service operation entries and improving operation efficiency.

Table 1 describes OceanCyber functions.

Table 1 OceanCyber functions

Application Type

Function

Home

Allows you to quickly learn about the status of connected storage devices, alarms, intelligent detection status of file systems, job status, Air Gap status of file systems, and license capacity.

Data Security

Storage Device allows you to view information about the connected storage devices.

Ransomware Protection allows you to configure intelligent detection policies, and perform intelligent detection for file systems.

Air Gap allows you to configure an Air Gap policy and associate it with a storage device.

Snapshot Data allows you to view the snapshots generated by intelligent detection.

Report allows you to manage intelligent detection reports.

Insight

Alarms and Events allows you to view uncleared alarms, cleared alarms, operation logs, and run logs.

Jobs allows you to view details about running and completed jobs.

System

Security allows you to manage users, roles, security policies, and certificates in the system.

License allows you to manage licenses in the system.

Log Management allows you to manage debug log levels and export logs.

Export Records allows you to query the system export records.

Settings allows you to manage the system data backup, mail service, event dump, and SNMP trap.
