Deleting an SFTP User

If an SFTP user is no longer needed to transmit and store data and the user data can be deleted, you can delete the user.

Prerequisites

The SFTP service has been enabled.

Precautions

  • Before deleting an SFTP user, ensure that the user is not using the SFTP service to transfer files.
  • If a user is deleted, the user data will be deleted and the SFTP service becomes unavailable for the user.

Procedure

  1. In the navigation pane, choose System > Settings > SFTP Service.
  2. Click More on the right of a user to be deleted and choose Delete.
  3. Click OK.
