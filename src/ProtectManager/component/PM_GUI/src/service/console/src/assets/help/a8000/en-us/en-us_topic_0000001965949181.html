Backup Software Management

The OceanProtect supports eBackup management. After adding information about backup software management, you can switch from the OceanProtect to the corresponding backup software.

Prerequisites

The eBackup certificate has been imported. For details, see "Adding an External Certificate" in the OceanProtect Appliance 1.5.0-1.6.0 Security Configuration Guide. Set Type to External System.

Procedure

  1. On the menu bar, choose System > Backup Software Management.
  2. On the displayed Backup Software Management page, click Add.
  3. Add backup software management information. Table 1 describes the related parameters.

    Table 1 Backup software management parameters

    Parameter

    Description

    Name

    User-defined backup software name.

    Backup Software

    Managed backup software.

    IP

    Management IP address of the backup software system.

    Port

    Port of the backup software system.

    Username

    Username of the backup software system. For the eBackup system, only local users in the eBackup system can be added.

    Password

    User password of the backup software system.

  4. Click OK.
  5. After the information is added successfully, return to the backup software management page. In the row of the backup software where you want to switch, click Access System to switch from the OceanProtect to the corresponding backup software.
