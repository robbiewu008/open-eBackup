Step : Adding a Target Cluster

This section describes how to add a target cluster to the local cluster so that a logical connection is established between the local and remote devices for data transmission.

Prerequisites

A target cluster is ready, and the backup network between the local and the target sites is normal.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. On the Replication Clusters tab page, click Add.
  3. Set parameters related to the target cluster.

    Table 1 describes the related parameters.
    Table 1 Parameters related to the target cluster

    Parameter

    Description

    Name

    Name of the target cluster, which can be customized.

    Role

    The default value Replication Target Cluster.

    IP Address

    Backup network IP address of the target cluster.

    Log in to the management page of the target cluster, choose System > Infrastructure > Cluster Management, and click the cluster name to obtain the backup network IP address of any controller.

    Port

    Port of the target cluster.

    The default port number is 25081 if a management IP address is entered in the IP address text box and the default port number is 30068 if a backup IP address is entered.

    Username

    User name and password of the user created in Step 5: Creating a Remote Device Administrator.

    Password

  4. Click OK.
