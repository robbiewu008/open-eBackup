Managing Intelligent Detection Policies

The OceanCyber 300 Data Security Appliance allows users to view, clone, modify, or delete intelligent detection policies.

You need to log in to OceanCyber 300 Data Security Appliance, choose Data Security > Ransomware Protection > Intelligent Detection > Intelligent Detection Policy, and find the intelligent detection policy to be operated. Table 1 describes the related operations.

Table 1 Operations related to intelligent detection policies

Job

Description

Navigation Path

Viewing an intelligent detection policy

Scenario

This operation enables you to learn about the number and details of intelligent detection policies.

In the Intelligent Detection Policies list, view the name, policy (Policy01 and Policy02 are displayed by default), and associated resources of the intelligent detection policies.

Cloning an intelligent detection policy

Scenario

This operation enables you to clone an intelligent detection policy.

In the row that contains the intelligent detection policy, choose More > Clone.

Modifying an intelligent detection policy

Scenario

This operation enables you to modify an intelligent detection policy. Modifying an intelligent detection policy does not affect ongoing intelligent detection jobs. The new intelligent detection policy takes effect in the next period.

In the row that contains the desired intelligent detection policy, choose More > Modify.

Deleting an intelligent detection policy

Scenario

This operation enables you to delete an intelligent detection policy that is not required temporarily.

Note

An intelligent detection policy that has been associated with a file system cannot be deleted.

In the row that contains the intelligent detection policy, choose More > Delete.
