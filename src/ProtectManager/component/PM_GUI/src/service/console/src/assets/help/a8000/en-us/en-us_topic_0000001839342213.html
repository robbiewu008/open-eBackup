Step 1: Checking and Enabling the NFSv4.1 Service

Before performing backup, ensure that the NFSv4.1 service has been enabled on the local storage. Otherwise, the backup will fail.

  1. Log in to DeviceManager.

    1. Choose System > Infrastructure > Cluster Management.
    2. In the Local Cluster Nodes area of the Backup Cluster tab page, click the node name.
    3. In the displayed Node detail dialog box, click Open the device management platform to go to DeviceManager.

  2. Choose Settings > File Service > NFS Service.
  3. On the NFS Service page, check whether the NFSv4.1 service is enabled.

    • If yes, no further action is required.
    • If no, click Modify to enable the NFSv4.1 service.
