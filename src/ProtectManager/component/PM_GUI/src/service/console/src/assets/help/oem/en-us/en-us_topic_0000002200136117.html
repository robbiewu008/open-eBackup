Configuring a Mailbox Backup and Restoration Account

Procedure

  1. Create a mailbox backup and restoration user, and create an available mailbox for the user. After the user is created, it automatically becomes a new user of the domain user.

    • Exchange 2010: Log in to the host where the Exchange client is installed, open the Exchange Management Console, expand Recipient Configuration, right-click Mailbox, choose New Mailbox from the shortcut menu, and create an Exchange mailbox as prompted. For example, the username can be MailBackupUser.
    • Exchange 2013/2016/2019: Log in to the Exchange management center, click Mailboxes on the Recipients page, and create an Exchange mailbox as prompted. For example, the username can be MailBackupUser.

  2. Add the newly created user as a member of the domain administrator.

    On the owning domain controller, open Active Directory Users and Computers, double-click the created user, click the Member Of tab and Add, add the user to Domain Admins, and click OK.

  3. Add MailBackupUser to the Discovery Management, Organization Management, and Public Folder Management security groups, and click + in the Members area to add a member.
