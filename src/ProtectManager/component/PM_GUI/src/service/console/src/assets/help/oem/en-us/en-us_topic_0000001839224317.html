Unloading Tapes

This section describes how to unload a tape from a drive and insert it into a card slot.

Prerequisites

A tape has been loaded to a drive.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Tape tab.
  4. In the row where the tape to be unloaded resides, choose More > Uninstall.

    You can also select multiple tapes and click Uninstall to batch unload the tapes that have been loaded to the driver.

  5. In the window that is displayed on the right, select a slot.

    There are import/export slots and other slots.

    • Import/Export slots: If you want to import tapes to or export tapes from a tape library, use this type of slots.
    • Other slots: If you do not need to import tapes to or export tapes from a tape library, use this type of slots.

    If you need to import the tape to or export it from a tape library, use an import/export slot.

  6. Click OK.

    If tape library scanning is performed manually or automatically during tape unloading, the slot may be different from the slot selected in 5. Refer to the tape location displayed in the tape list.
