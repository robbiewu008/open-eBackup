下载客户端软件包

如果您希望通过手动方式为主机安装客户端，需要先参考本节下载客户端软件包。

注意事项

不同版本的界面可能存在差异，请以实际为准。

操作步骤

  1. 选择“保护 > 客户端 > 客户端”。
  2. 单击“软件包管理”，配置相关参数。

    参数说明如表1所示。

    表1 下载客户端软件包

    参数

    说明

    模式

    选择“下载”。
    代理类型
    说明：

    仅1.5.0版本支持设置该参数。
    根据受保护资源的类型选择代理类型。
    • VMware备份代理：当受保护资源为VMware虚拟机时，选择该类型。
    • 通用备份代理：除VMware外的受保护资源，选择该类型。
    • SAN Client代理：当受保护资源为DB2，且AIX或Solaris主机使用FC组网时，选择该类型。
    软件包类型
    说明：

    仅1.6.0及后续版本支持设置该参数。
    选择客户端软件包类型。
    • “客户端”：适用于所有待保护资源。
    • “SAN Client”：仅当待保护资源为DB2，且AIX生产环境的组网为FC组网时，需通过SAN Client转接来实现备份，才选择此类型。

    操作系统

    客户端所在主机的操作系统类型。
    下载包类型
    说明：

    仅1.6.0及后续版本支持设置该参数。

    当“操作系统”选择“Linux”时，请选择下载包类型，支持的类型包括“zip”和“tar”。

    应用类型

    当“代理类型”选择“通用备份代理”或“软件包类型”选择“客户端”时，请选择客户端支持保护的应用类型。

    共享

    开启后将客户端应用于所有数据保护管理员。

    当“软件包类型”选择“客户端”，“操作系统”选择“Linux”，且“应用类型”选择HDFS、HBase、Hive、华为云Stack、华为云Stack GaussDB、GaussDB(DWS)或对象存储时，支持配置此参数。

    私钥密码

    用户自定义私钥密码，根据界面提示的复杂度要求输入。

    私钥密码用于对客户端软件包中的私钥文件进行加密，私钥文件用于客户端所在主机与OceanProtect鉴权。请妥善保管此私钥密码，后续安装过程中会使用。

    说明：
    • 当“操作系统”为“AIX”或“Solaris”时，输入的私钥密码请勿包含^`{_|}~特殊字符，否则会导致使用该软件包安装客户端时，安装失败。
    • 填写私钥密码后，OceanProtect将使用此处填写的私钥密码对客户端软件包中的私钥文件进行加密。
    • 如果在下载客户端软件包前，已参考《OceanProtect DataBackup 1.5.0-1.6.0 安全配置指南》导入客户端证书，此处可以填写与客户端证书相同的私钥密码，也可以自行定义私钥密码。

    确认私钥密码

    再次输入私钥密码。

  3. 单击“确定”。

    获取下载到的客户端软件包。

    在Windows Server使用Internet Explorer 11下载客户端软件包时，如果下载失败，可参考在Windows Server使用Internet Explorer 11下载客户端软件包失败进行处理。
