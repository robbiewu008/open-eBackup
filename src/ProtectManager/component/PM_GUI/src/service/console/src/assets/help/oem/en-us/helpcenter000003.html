Quick Start

This section describes functions of the product, helping you quickly find service operation entries to improve operation efficiency.

Table 1 describes the functions of the product 1.6.0. Some product models do not support ransomware protection or data anonymization. For details about the supported functions, see the product documentation on the technical support website or consult the product provider.

Table 1 Functions

Application Type

Function

Home

Helps you quickly learn the job status, managed resources, alarms, capacity usage, SLA compliance, data reduction, capacity trend, and historical system performance.

Protection

Overview allows you to view protected and unprotected resources as well as resource types.

Databases allows you to manage database applications that have been connected to the system and protect the managed database applications.

Big Data allows you to manage big data environments connected to the system and protect the managed big data environments.

Virtualization allows you to manage VMware and FusionCompute resources that have been connected to the system and protect the managed virtualization environments.

Container allows you to manage Kubernetes FlexVolume and Kubernetes CSI resources that have been connected to the system and protect the managed containers.

Cloud Platform allows you to manage Huawei Cloud Stack, OpenStack, Huawei Cloud Stack GaussDB, and GaussDB that have connected to the system and protect managed projects, resource sets, or cloud servers.

Applications allows you to manage Active Directory that has been connected to the system and protect managed Active Directory.

Files Systems allows you to manage storage devices, NAS file systems, NAS shares, common shares, object storage, filesets, and volumes that have been connected to the system and protect the managed NAS file systems, NAS shares, and common shares.

Clients allows you to manage ProtectAgent that has been connected to the system.

Protection Policies allows you to manage SLAs and rate limiting policies.

Explore

Copy Data allows you to manage all copies generated by protected resources.

Live Mount allows you to manage live mount of filesets, Oracle/MySQL/MariaDB databases, VMware VMs, NAS file systems, and NAS shares to reuse copies.

Restoration Drills allows you to perform restoration drills for databases.

Data Security

Data Anonymization allows you to identify and anonymize sensitive data in restored or mounted Oracle databases.

Ransomware Protection allows you to detect ransomware on the copies of VMware resources, NAS file systems, and NAS shares.

Air Gap controls connectivity of replication links.

Jobs

Jobs allows you to view details about running jobs and completed jobs.

Alarms and Events

Alarms and Events allows you to view uncleared and cleared alarms, operation logs, and run logs.

Reports

Reports allows you to view status of key resources and jobs in the system.

Performance

Performance allows you to configure performance monitoring and view the historical performance information (I/O response time, IOPS, and bandwidth) of nodes.

System

Infrastructure allows you to manage clusters, archive storage libraries, and backup storage unit groups.

Security allows you to manage users, roles, quotas, functions, security policies, certificates, Kerberos, data security, trust settings, LDAP service configurations, and SAML SSO configurations in the system.

License allows you to manage licenses in the system.

Log Management allows you to manage debug log levels and export logs.

Export Records allows you to export system logs, agent logs, detection reports, and copy files.

Settings allows you to manage the system data backup, email service, alarm settings, event dump, SNMP trap, SFTP service, and system time.

Backup Software Management allows you to manage eBackup.

Global search

Allows you to use in the upper right corner of the page to quickly search for copies and resources and quickly restore files in the copies.

The GUI may vary with versions and models. The information displayed on the GUI is only for reference and is subject to the actual situation.
