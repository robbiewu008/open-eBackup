Step 3: Adding a Tenant and Authorizing Resources

After registering Huawei Cloud Stack, you need to add the tenant where the project or resource set to be protected resides to the OceanProtect so that the project/resource set or EVS disk can be protected. After a tenant is added, you are advised to authorize the project or resource set of the tenant to a specified data protection administrator for further data protection operations.

Context

The system administrator of the OceanProtect has the permission to perform operations on all resources. You are advised to authorize a specified data protection administrator to manage resources by project or resource set.

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack.
  2. Click Huawei Cloud Stack where the resource to be protected resides. Click the Tenant tab on its right, and choose Add.
  3. Select the cloud platform tenant where the resource to be protected resides.

    If no tenant information is displayed or the displayed tenant information is incomplete, click Update Tenant Information to obtain all tenant information. If the tenant information is still not obtained, choose More > Resource Scan on the left of the page to scan the tenant information.

  4. Click Add in the VDC Manager area and enter the administrator username and password of the VDC where the to-be-protected project/resource set or EVS resides.

    After the tenant is successfully added, the system automatically discovers the projects/resource sets and cloud servers of the tenant and displays them on the Project/Resource Set and Cloud Server tab pages, respectively.

    If the VDC administrator to be added is not a first-level VDC administrator and you need to protect resources in lower-level VDCs of the VDC, you must grant the service management permissions of lower-level VDCs to the VDC administrator on ManageOne Operation Portal so that resources of the lower-level VDCs can be discovered.

  5. Authorize the protection permission of the tenant's resources to a specified data protection administrator

    • Perform the following operations for versions earlier than 1.6.0:
      1. Choose More > Authorize Resource on the right area of the tenant.
      2. Select the project or resource set to be authorized.
      3. Specify a data protection administrator.
      4. Click OK.

        After the authorization is successful, the owner of the project or resource set is the data protection administrator displayed on the Project/Resource Set page.

    • Perform the following operations for 1.6.0 and later versions:
      1. Choose System > Security > RBAC.
      2. On the Users tab page, locate the row of the data protection administrator to be authorized, and click Modify.
      3. On the Modify User page, click Next.
      4. Click Create Resource Set.
      5. Set the resource set name and choose Cloud Platforms > Huawei Cloud Stack, and select the corresponding tenant.
      6. Click OK.
      7. Select the resource set created in 5.e and click Next.
      8. Click Finish.
