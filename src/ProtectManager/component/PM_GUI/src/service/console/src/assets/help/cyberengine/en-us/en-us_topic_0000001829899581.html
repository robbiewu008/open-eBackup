Ransomware Detection Report Overview

After logging in to the OceanCyber 300 Data Security Appliance, you can view the ransomware detection report on the home page. Table 1 describes the areas in the ransomware detection report.

Table 1 Areas in the ransomware detection report

No.

Description

1

Displays storage device information, including the total number of storage devices, number of normal and abnormal storage devices, number of offline storage devices, storage device names, total number of file systems on each storage device, and number of abnormal file systems.

2

Displays the name, owning device, owning vStore, and latest detection time of each abnormal file system.

3

Displays the number and distribution of critical, major, and warning alarms.

4

Displays the number and proportion of file systems for which file interception has been enabled in the OceanCyber detection and Device detection scenarios.

5

Displays the number and proportion of file systems for which real-time detection has been enabled in the OceanCyber detection and Device detection scenarios.

6

Displays the number and proportion of file systems for which intelligent detection has been enabled in the OceanCyber detection and Device detection scenarios.

7

Displays jobs that are in the queuing, running, successful, failed, and stopped states, as well as job distribution.

8

Displays the names and statuses of the devices configured with Air Gap.

9

Displays the license capacity of the storage device.
