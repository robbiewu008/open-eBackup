Importing a Management Data Backup

This section describes how to import an original management data backup stored in a local directory into the system for restoring management data.

Constraints

Management data backup in a version cannot be imported to another.

Procedure

  1. Choose System > Settings > Management Data Backup.
  2. Click Import Backup.
  3. Select a file and click OK.
