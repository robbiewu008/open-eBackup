Disabling a Driver

This section describes how to disable a driver.

Prerequisites

The running status of the driver is Enabled.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Driver tab.
  4. Locate the row where the driver to be disabled resides, and choose More > Disable Driver.
