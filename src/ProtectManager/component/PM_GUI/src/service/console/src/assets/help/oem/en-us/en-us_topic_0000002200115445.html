Canceling Authorization of a Local Cluster Data Protection Administrator

This section describes how to cancel authorization of a local cluster data protection administrator. After authorization is canceled, the user cannot manage the managed cluster.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  1. In the multi-domain cluster area, choose More > Cancel Authorization in the row that contains the desired managed cluster.
  2. Select the desired local cluster data protection administrator and click OK.
  3. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
