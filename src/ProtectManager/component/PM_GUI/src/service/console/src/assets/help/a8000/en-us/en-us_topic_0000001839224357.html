Configuring the Capacity Alarm Threshold of the Local Storage

The OceanProtect allows you to modify the capacity alarm threshold of the local storage. After the modification, the system reports an alarm when the used capacity reaches the new threshold.

Procedure

1.5.0 and earlier versions

  1. Choose System > Infrastructure > Cluster Management.
  2. In the Local Cluster Nodes area of the Backup Cluster tab page, click the node name.
  3. In the Basic Info area of the Node detail dialog box that is displayed, click on the right of Capacity Alarm Threshold.
  4. Modify the capacity alarm threshold of the local storage.

    Set the upper threshold of the used capacity of the local storage. When the used capacity reaches the threshold, an alarm is reported.

  5. Click Save.

1.6.0 and later versions

  1. Choose System > Infrastructure > Backup Storage Unit Groups.
  2. Click the Backup Storage Units tab, locate the row that contains the backup storage unit whose alarm threshold you want to modify, and choose More > Modify.
  3. In the Modify dialog box that is displayed, modify the alarm threshold of the backup storage unit.
  4. Click OK.
