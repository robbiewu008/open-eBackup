Handling False Detection

Context

If an uninfected snapshot is detected as Infected, you can correct the snapshot status through false detection handling.

Prerequisites

Only snapshots in Infected status support false detection handling.

Procedure

  1. On the top menu bar, choose Data Security > Snapshot Data.
  2. In the row that contains a snapshot for which you want to handle it as a false report, choose More > Handle False Detection.
  3. In the dialog box that is displayed, confirm the information and click OK.

    You can also click the number of ransomware detection snapshots to go to the Snapshot Data page. In the row that contains the desired ransomware detection snapshot, choose More > Handle False Detection.
