A Backup Job Fails Because the System Writer Is Not Found, with an Error of Error Code 8078001D Displayed in the Error Details

Symptom

A backup job fails, with the message "Failed to execute the Active Directory backup subjob due to a system error. System error code: 8078001D; error description: The system writer is not found in the backup." displayed.

Possible Causes

The backup fails because the system writer cannot be found. The possible cause is that the user permissions on files in the %windir%\winsxs\temp\PendingRenames or %windir%\winsxs\filemaps directory on the production host are incorrect.

%windir% is an environment variable. By default, it refers to C:\Windows.

Procedure

For details, see System writer is not found in the backup.
