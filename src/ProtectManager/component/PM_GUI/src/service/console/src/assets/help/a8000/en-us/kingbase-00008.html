Backing Up a Kingbase Instance

This section describes how to configure backup. For 1.6.0 and later versions, IntelliMate (smart assistant) provides step-by-step help to streamline user operations. Specifically, log in to the OceanProtect WebUI and click in the upper part of the page. On the page that is displayed, click Learn How to Perform Backup Operations, select the type of the resource to be protected, and click Next to complete the backup configuration according to the backup configuration guide.
