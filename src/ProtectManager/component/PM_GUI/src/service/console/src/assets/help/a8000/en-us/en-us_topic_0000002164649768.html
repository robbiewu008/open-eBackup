Step 1: Registering an Exchange Single-Node System or Availability Group

After the client is installed on the host where the Exchange database resides, the OceanProtect automatically scans the Exchange database and user mailboxes. The OceanProtect uses the scanned databases and mailboxes as objects to be protected.

The method of registering an Exchange single-node system or availability group is the same as that in backing up an Exchange single-node system or availability group, or that of registering a single-node system or availability group in a database.

For details about the registration procedure, see Step 1: Registering an Exchange Single-Node System or Availability Group.
