查看对象存储副本信息

参考本节查看对象存储副本的详细信息。

操作步骤

  1. 选择“数据利用 > 副本数据 > 文件系统 > 对象存储”。
  2. 选择副本浏览方式浏览副本信息。

    • 复制副本仅支持在复制到的目标集群查看。
    • 部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。
    • 选择“资源”页签：以资源维度查看对象存储副本信息。当对象存储中的数据丢失，您需要恢复该对象存储时，建议从该入口进入查找相应时间点副本。

      参数说明如表1所示。

      表1 副本参数说明

      参数

      说明

      资源

      名称

      对象集合名称。

      位置

      所属对象存储。

      状态

      对象集合是否已被删除。

      • 是，则状态为不存在。
      • 否，则状态为存在。

      副本

      数量

      对象集合在当前集群的备份副本数量、复制副本数量或归档副本数量。

      复制副本SLA

      复制副本关联的SLA。

      如果已为该资源产生的复制副本配置SLA，则此处显示SLA名称。

      复制副本保护状态
      复制副本当前是否被保护。
      • 未保护：复制副本未关联SLA，或者复制副本已关联SLA但已禁用保护。
      • 已保护：复制副本已关联SLA，且已激活保护。
      • 创建中：复制副本已关联SLA，正在创建保护。仅1.5.0版本存在该状态。
    • 选择“副本”页签：以副本维度查看所有副本信息。当需要基于某个指定的副本进行恢复时，建议从该入口进入查找相应副本。

      参数说明如表2所示。

      表2 副本参数说明

      参数

      说明

      副本

      备份副本生成时间

      备份副本的生成时间。

      状态

      副本的状态。

      • 正常：副本可用。
      • 无效：副本无效。
      • 删除中：副本处于正在删除过程中。
      • 恢复中：副本处于恢复或即时恢复过程中。

      位置

      副本所在的位置。

      • 当副本的“生成方式”为“备份”、“复制”、“级联复制”、“反向复制”时，副本存储在对应的备份存储单元，此处显示为对应备份存储单元的名称。
      • 当副本的“生成方式”为“对象存储归档”或“导入”时，副本存储在对象存储的桶上，此处显示为桶名称。
      • 当副本的“生成方式”为“磁带归档”时，副本存储在磁带库，此处显示介质集名称。

      存储介质状态
      副本所在的存储介质的状态。
      • 在线：则支持对该副本进行管理。
      • 离线：则仅支持对副本进行删除。
      说明：

      当备份存储单元的“健康状态”为“正常”或“降级”，且“运行状态”为“预拷贝”、“重构”、“在线”、“正在均衡”或“降级”时，存储介质状态均显示为“在线”。

      生成方式
      副本的生成方式。
      • 复制：由复制任务产生的副本。
      • 备份：由备份任务产生的副本。
      • 对象存储归档：由归档任务（归档存储为对象存储）产生的副本。
      • 磁带归档：由归档任务（归档存储为磁带库）产生的副本。
      • 级联复制：由级联复制任务产生的副本。
      • 反向复制：由反向复制任务产生的副本。

      副本生成时间

      副本生成的时间。单击副本生成时间，可以查看副本的详细信息。

      副本类型

      生成副本的备份类型。

      包括：全量、增量、永久增量（合成全量）

      副本过期时间
      副本的过期时间。如果副本为永久有效，则显示为“--”。
      说明：

      副本实际过期时间以“副本过期时间”与“WORM过期时间”中的最晚时间为准。

      WORM

      说明：

      仅OceanProtect X系列备份一体机和OceanProtect E1000支持此参数。
      副本是否已设置为WORM副本。
      • ：未设置。副本未设置为WORM副本，即副本为普通副本。
      • ：已设置。副本已设置为WORM副本，WORM副本不支持删除和缩短保留时间。
      • ：设置中。副本正在设置为WORM副本。处于该状态的副本不支持删除和修改保留时间。
      • ：设置失败。副本设置为WORM副本失败。设置失败的副本不支持删除和缩短保留时间。
      • ：已过期。WORM副本已过有效期。处于该状态的副本支持删除。

      WORM过期时间

      说明：

      仅OceanProtect X系列备份一体机和OceanProtect E1000支持此参数。
      当副本为WORM副本时，则显示WORM过期的具体时间。否则，显示为“--”。
      说明：

      当WORM过期时间早于WORM文件系统法规时钟时，即为过了WORM有效期。WORM文件系统法规时钟可单击副本生成时间查看。如果WORM文件系统法规时钟与系统时间不一致，请等待WORM文件系统法规时钟到达当前设置的过期时间后，再查看WORM状态。

      ACL备份

      副本是否已设置ACL备份。

      • 未备份：副本未开启ACL备份。
      • 已备份：副本已开启ACL备份。

      副本

      索引

      副本生成后，系统将为文件集副本中的所有文件及目录建立索引，便于快速查找相应文件进行恢复。包括以下索引状态：

      • 已索引：系统已为副本建立索引。
      • 索引中：副本生成后，系统正在为副本建立索引。
      • 未索引：SLA中未开启索引功能。
      • 索引删除中：正在删除索引。
      • 索引失败：系统为副本创建索引失败。

      资源

      名称

      对象集合名称。

      位置

      所属对象存储。

      状态

      对象集合是否已被删除。

      • 是，则状态为不存在。
      • 否，则状态为存在。

      标签

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
