Loading Tapes

This section describes how to load a tape in a tape library from a card slot to a driver. After the tape is loaded to the drive, data can be read and written into the tape.

Prerequisites

A tape resides in the tape library and is not loaded to any driver.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Tape tab.
  4. In the row where the tape to be loaded resides, choose More > Load.

    You can also select multiple tapes and click Load to batch load tapes to the driver.

  5. In the window that is displayed, select a driver and click OK.

    After the tape is loaded, if the tape location displayed in the tape list is a slot, the possible cause is that the tape library is scanned automatically or manually during the loading. In this case, wait for 10 minutes and try again.
