Step 1: Enabling the Snapshot Function for the HDFS Directory

Before performing protection, enable the snapshot function for the directory where the HDFS files to be protected reside.

Procedure

  1. Use PuTTY to log in to any NameNode in the big data cluster.
  2. Run the following command to enable the snapshot function for the directory:

    source Installation directory/bigdata_env
hdfs dfsadmin -allowSnapshot directory_name

    directory_name indicates the parent directory of the fileset to be protected. For details about how to manage snapshots, see Managing Snapshots.
