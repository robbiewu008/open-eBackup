Step 6: Adding a Domain

After OpenStack is registered, the system automatically scans all domains. This section describes how to add the domains to be protected to the management page.

Procedure

  1. Choose Protection > Cloud Platforms > OpenStack.
  2. Click the Domain tab and click Add.

    1. Select a domain name to be added.
    2. Enter the username and password of the domain administrator in Domain Administrator and Password.
      That is, the username and password of the domain administrator created in Step 4: Creating a Domain Administrator.

      For Default and heat_stack_domain domains, set the domain administrator to the user created in Step 3: Creating a User for Interconnection.

    3. Click Test.

  3. Click OK.
