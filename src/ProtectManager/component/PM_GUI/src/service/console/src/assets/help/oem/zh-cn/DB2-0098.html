查询DB2信息

操作步骤

  1. 选择“保护 > 数据库 > DB2”。
  2. 在“集群”页签，查询DB2集群信息。

    1. 该页面展示已注册至本产品的DB2集群信息。

      相关参数说明如表1所示。

      部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

      表1 DB2集群信息

      参数

      说明

      名称

      DB2集群名称。

      状态

      DB2集群的状态。

      • 在线：当本产品与DB2集群所有节点连接正常时，“状态”为“在线”。
      • 离线：当本产品与DB2集群所有节点连接异常时，“状态”为“离线”。

      类型

      集群类型。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
      所属用户
      说明：

      仅1.5.0版本存在此参数。

      该主机或集群所属的数据保护管理员。

      当该主机或集群已授权给数据保护管理员时，将显示数据保护管理员的名称。如果未进行授权，则显示“--”。
    2. 单击集群名称，查看集群基本信息和与该集群相关的数据库信息。

  3. 在“实例”页签，查询DB2实例信息。

    1. 该页面展示DB2实例的详细信息。

      相关参数说明如表2所示。

      部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

      表2 DB2实例参数说明

      参数

      说明

      名称

      实例名称。

      状态

      实例状态。

      所属主机/集群

      实例所属的主机/集群。

      版本

      实例版本。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
    1. 单击名称，查看基本信息、数据库。
      • 基本信息

        显示实例的名称、所属主机/集群、版本及节点相关信息。

      • 数据库

        显示数据库的名称、SLA、SLA遵从度及保护状态。

  4. 在“数据库”页签，查询DB2数据库信息。

    1. 该页面展示DB2数据库的详细信息。

      相关参数说明如表3所示。

      部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

      表3 DB2数据库参数说明

      参数

      说明

      名称

      数据库名称。

      所属主机/集群

      数据库所属的主机/集群。

      所属实例

      数据库所属的实例。

      SLA

      当DB2数据库已关联SLA时，则显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      DB2数据库当前是否被保护。

      • 未保护：当前资源未关联SLA，或者已关联SLA但已禁用保护。
      • 已保护：当前资源已关联SLA，且已激活保护。
      • 创建中：当前资源已关联SLA，正在创建保护。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
    2. 单击名称，查看总览信息、副本信息以及相关任务。
      • 总览

        显示SLA信息、任务执行时间、副本总数等。

      • 副本数据

        您可以依次按照年、月、天查找相应时间的副本。

        时间下方显示，表示该时间存在副本。

      • 任务

        展示数据库相关的所有任务，包括备份、恢复、复制等任务。可选择展示执行中的任务，或历史任务。

  5. 在“表空间集”页签，查看DB2表空间集信息。

    1. 该页面展示DB2表空间集的详细信息。
