Importing the Security Certificate to the Terminal Browser

This section uses the Windows OS and Google Chrome 99.0.4844.84 as examples to explain how to install the certificate.

Prerequisites

You have prepared a new security certificate, that is, CA certificate, for replacement.

Procedure

  1. Open Google Chrome, click , and choose Settings.
  2. Choose Security and Privacy > Security.
  3. Click Manage certificates.
  4. In the Certificates dialog box that is displayed, select the Trusted Root Certification Authorities tab and click Import.
  5. In the Certificate Import Wizard dialog box that is displayed, read the prompt information and click Next.
  6. Click Browse, select the prepared CA certificate, and click Next.
  7. Select Place all certificates in the following store and click Browse. In the Select Certificate Store dialog box that is displayed, select Trusted Root Certification Authorities and click OK. The Certificate Import Wizard dialog box is displayed.
  8. Click Next. Then click Finish.
  9. In the Security Warning dialog box that is displayed, click Yes to continue importing the certificate.
  10. After the certificate is imported, close the browser, open it again, and enter https://OceanCyber management IP address:25080 in the address box. Press Enter to go to the OceanCyber login page.
