Backing Up an SAP on Oracle Database

Before creating a data protection job, ensure that the SAP ERP and client have been installed by referring to Configuring the Host Where the Client Is Installed and Installing the Client.

This section describes how to complete backup configuration. IntelliMate (smart assistant) provides step-by-step help to streamline user operations. Specifically, log in to the OceanProtect WebUI and click in the upper part of the page. On the IntelliMate page that is displayed, click Learn How to Perform Backup Operations, select the type of the resource to be protected, and click Next to complete the backup configuration according to the backup configuration guide.

  • Step 1: Registering an SAP on Oracle Database
  • Step 2: (Optional) Creating a Rate Limiting Policy
  • Step 3: (Optional) Enabling Backup Link Encryption (Applicable to OceanProtect X Series Backup Appliances and OceanProtect E1000 (with the OceanProtect Used as Backup Storage))
  • Step 4: Creating a Backup SLA
  • Step 5: Performing Backup
