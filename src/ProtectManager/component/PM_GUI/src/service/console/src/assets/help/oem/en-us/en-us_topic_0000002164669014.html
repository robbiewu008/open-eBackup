Step 1: Registering a GoldenDB Cluster

Prerequisites

Before registering a GoldenDB cluster, ensure that each host in the cluster has been registered.

Procedure

  1. Choose Protection > Databases > GoldenDB.
  2. Click the Cluster tab.
  3. Click Register to register the GoldenDB cluster.
  4. Customize a cluster name and click Add to add a management node.
  5. Specify Hosts and enter the username of the GoldenDB management node.

    Select a management node in the cluster for Hosts. Obtain the management node information from Table 1.

  6. Click OK.
