步骤2：获取存储资源CA证书（适用于存储层快照备份）

对于1.6.0及后续版本，注册Oracle数据库添加存储资源时，可以导入存储资源的CA证书，提升设备之间访问的安全性，否则系统将无法验证其所访问设备的信息，安全性存在风险。该操作为可选操作。

您可以向管理员获取，或者自行下载CA证书。

自行下载存储资源CA证书

  • 方式一：
    1. 登录PKI CA证书下载界面。
    2. 单击“二级CA证书”。
    3. 在搜索框中输入“Huawei IT Product CA”，单击“查询”。
    4. 在查询结果区域中，单击“Huawei IT Product CA”证书所在行的，下载证书并保存在本地。

      证书文件格式必须为.pem格式，如果保存的证书文件格式不符合要求，请手动修改证书文件后缀为.pem。

  • 方式二：
    不同浏览器提示方式不同，此处以Google Chrome 120为例进行说明。
    1. 在Chrome浏览器中，输入存储设备DeviceManager的URL地址，按回车。
    2. 单击URL输入栏左侧的“不安全”，单击“证书无效”。

    3. 在弹出的对话框中，选择“详细信息”页签，单击“导出”。
    4. 根据向导提示，导出证书并重命名证书文件为XXX.pem。

      XXX为实际的证书名，证书格式必须为.pem格式。
