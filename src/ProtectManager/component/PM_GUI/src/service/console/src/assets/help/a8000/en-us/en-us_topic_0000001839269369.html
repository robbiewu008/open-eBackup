Installation Process

A host running the ProtectAgent software is called a ProtectAgent host, which is a data protection agent host ("agent host" for short). An agent host is used for process control and data transmission during the process of backing up and restoring data. Before resource protection, prepare a host for installing ProtectAgent and install the software. After the installation is executed, perform configurations.
