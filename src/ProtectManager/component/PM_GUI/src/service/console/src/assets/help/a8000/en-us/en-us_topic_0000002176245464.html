Registering a Database (Applicable to the Single-Instance Deployment Mode)

This section describes how to register a single instance when the protected database is deployed in a single instance.

Procedure

  1. Choose Protection > Databases > DB2.
  2. Click the Instance tab.
  3. Click Register to register the target DB2 database.
  4. Set Type to Single instance.
  5. Configure the database instance and its authentication information.

    Table 1 describes the related parameters.

    Table 1 Registration information of a single DB2 instance

    Parameter

    Description

    Name

    Instance name.

    Hosts

    Database host to be registered.

    Username

    Username of the database administrator.

    Password

    Password of the database administrator.

  6. Click OK.
