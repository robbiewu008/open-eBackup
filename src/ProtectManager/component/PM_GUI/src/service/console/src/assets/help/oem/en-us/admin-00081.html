Viewing Local Storage Authentication Information

This section describes operations to view local storage authentication information in the Node detail dialog box.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. In the Local Cluster Nodes area of the Backup Cluster tab page, click the node name.
  3. In the displayed Node detail dialog box, view the authentication information.

    • Login authentication
      • Username: .

        By default, the maximum number of sessions per user is not limited. Do not change the value unless necessary.

      • Password: Login authentication user password.
      • Status: User authentication status. If the authentication status is Exception, modify authentication information.
    • Service authentication
      • Username: .

        By default, the maximum number of sessions per user is not limited. Do not change the value unless necessary.

      • Password: Service authentication user password.
      • Status: User authentication status. If the authentication status is Exception, modify authentication information.
