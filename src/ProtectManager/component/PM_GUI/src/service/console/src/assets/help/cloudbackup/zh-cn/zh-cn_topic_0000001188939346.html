步骤1：添加备份存储

将备份存储接入系统，建立备份存储与系统之间的逻辑连接，用于设备之间的数据传输。

前提条件

  • 已准备用于存放备份数据的对象存储平台。
  • 已获取对象存储平台的访问地址、AK/SK。
  • 已在对象存储平台创建存放备份数据的桶。

    为确保数据的安全性，建议指定的桶仅为文件系统备份使用。否则，将存在归档数据被非法获取或被篡改的风险。

  • 对象存储的数据桶已禁用多版本控制。

操作步骤

  1. 选择“系统 > 基础设施 > 备份存储”。
  2. 单击“添加备份存储”。
  3. 配置备份存储参数。

    相关参数说明如表1。

    表1 备份存储参数说明

    参数

    说明

    名称

    用户自定义备份存储的名称。

    类型

    备份存储的类型，当前仅支持S3协议的对象存储。

    对象存储产品

    接入系统的对象存储产品。

    支持FusionStorage OBS、OceanStor Pacific、华为云 OBS以及AWS S3。

    Endpoint

    备份存储的访问地址。

    地址格式：备份存储的域名或业务面IP地址。

    说明：

    以华为云OBS对象存储为例，通过OBS管理控制台查看访问域名的方法如下：

    1. 登录华为OBS对象存储服务网站（https://www.huaweicloud.com/product/obs.html），进入OBS管理控制台。
    2. 在对象存储服务的桶列表中，单击已创建的作为文件系统备份对象存储的桶，系统显示该桶的“概览”信息。
    3. 在“基本信息”区域，复制该桶所在区域的域名，即“Endpoint”。

    协议

    系统与备份存储之间的数据传输的协议。

    • 当使用“HTTPS”协议时，请导入有效的证书以便系统验证备份存储的信息。请提前向备份存储的管理员获取证书。

      导入证书的具体操作请参见导入证书章节。

    • 当协议设置为HTTP时，系统将无法验证备份存储环境信息，将存在安全风险。建议您选择安全性高的HTTPS协议。

    AK

    访问对象存储的接入证书。

    说明：

    以华为云OBS对象存储为例，通过OBS管理控制台创建访问密钥（AK和SK）的方法如下：

    1. 登录华为OBS对象存储服务网站（https://www.huaweicloud.com/product/obs.html），进入OBS管理控制台。
    2. 鼠标放置在页面右上角的用户名上，在弹出的下拉菜单中，选择“我的凭证”，进入“我的凭证”页面。
    3. 单击“访问密钥”。系统显示“访问密钥”的页面详情。
    4. 单击“新增访问密钥”，按照界面提示输入对该密钥的描述信息，单击“确定”。系统提示创建成功。
    5. 单击“立即下载”，浏览器会自动生成名为“credentials.csv”的密钥文件。
    6. 打开下载下来的“credentials.csv”文件即可查看访问密钥（密钥包含了“User Name”、“Access Key Id”和“Secret Access Key”）。“Access Key Id”即对应AK，“Secret Access Key”即对应SK。

    SK

    访问对象存储的安全证书。

    数据桶

    存放备份数据的桶名。

    说明：

    以华为云OBS对象存储为例，通过OBS管理控制台查看桶名的方法如下：

    1. 登录华为OBS对象存储服务网站（https://www.huaweicloud.com/product/obs.html），进入OBS管理控制台。
    2. 在对象存储服务的桶列表中，浏览并手动复制桶名称。

    索引桶

    存放索引数据的桶名。与数据桶名称相同，无需设置。

    使用代理服务器

    当系统使用代理服务器访问备份存储时，请配置代理服务器的地址及用户名、密码。

    说明：
    • 请联系企业数据中心的网络管理员获取代理的信息。
    • 当选择开启“使用代理服务器”时，界面会显示“地址”、“用户名”和“密码”参数。

    容量告警阈值

    设置桶使用容量的最大阈值，当使用容量达到设定的阈值时上报告警。

    当选择开启“容量告警阈值”时，界面会显示“告警阈值”参数。

  4. 单击“确定”。
