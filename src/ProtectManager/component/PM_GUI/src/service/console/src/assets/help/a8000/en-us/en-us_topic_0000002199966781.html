The Backup Job Failed and The Error Details Show that the Client Failed to Execute the Mount Operation

Symptom

The VM backup job failed to be executed. The error details show that the client failed to execute the mount operation.

Possible Cause

This problem is caused by the kernel issue of Red Hat 7.6.

Troubleshooting

  1. Use PuTTY to log in to the host where the client is to be installed as user root.
  2. Run the following commands to check whether the OS kernel version of the host where the client is installed is 3.10.0-957.5.1.el7 or later:

    cat /etc/redhat-release

    uname -r

    • If yes, contact technical support.
    • If not, go to 3.

  3. Download the Red Hat kernel package of 3.10.0-957.5.1.el7 or a later version and upload the downloaded kernel package to the host where the client is installed.
  4. Go to the directory where the installation package is stored and run the following command to install the kernel package:

    rpm -ivh Kernel package name

    Example:

    rpm -ivh kernel-3.10.0-957.5.1.el7.x86_64.rpm

  5. After restarting the OS, run the following command to check whether the current kernel version is the installed one:

    uname -r
