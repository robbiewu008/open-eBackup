Step 2: Registering the Alibaba Cloud Organization

Before backing up and restoring Alibaba Cloud, you need to register the Alibaba Cloud organization with the product.

Procedure

  1. Choose Protection > Cloud Platforms > Alibaba Cloud.
  2. Click Register to register the Alibaba Cloud information.

    For details about the Alibaba Cloud registration information, see Table 1.
    Table 1 Alibaba Cloud registration information

    Parameter

    Description

    Name

    Custom name of the cloud platform to be registered.

    AccessKey ID

    Access key ID and password of the organization to be registered. For details about how to obtain the value, see Preparations for Backup.

    AccessKey Secret

    Organization ID

    ID of the organization to be registered. For details about how to obtain the value, see Preparations for Backup.

    Clients

    If no client is selected, the system selects one or more clients from those configured to perform restoration.

    Resource Scanning Interval

    Interval for the system to automatically scan resources, in hours.

    Certificate verification

    After Certificate Verification is enabled, import the certificate. Certificate verification helps enhance resource security.

  3. (Optional) In the Advanced area, set Proxy Server.
  4. Click OK.
