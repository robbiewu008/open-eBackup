Checking the Driver

Drivers are used to read and write tapes. This section describes how to check the driver of a tape library.

Precautions

  • Ensure that at least one tape exists in the tape library and the tape is not in the import/export slot. Otherwise, the driver cannot be identified.
  • Unloading tapes from virtual tape libraries and canceling allocation of tape libraries or drives are high-risk operations, which will cause tapes or drives of the product to be unavailable.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Driver tab.
  4. View the drivers in the driver list. Table 1 describes the related parameters.

    Table 1 Driver parameters

    Parameter

    Description

    Name

    Name of a driver, which is generated by the system. The naming convention is Driver + ID. IDs are sorted according to the scanning sequence of drivers.

    For example, Driver 0.

    Running Status

    Running status of a driver.

    • Enabled: The driver has been enabled.
    • Disabled: The driver has been disabled.

    Compression Status

    Compression function status of the driver.

    • Enable: The compression function is enabled for the driver.
    • Disable: The compression function is disabled for the driver.

    Tag of Loaded Tape

    Label of the tape loaded for the driver.

    Serial No.

    Serial number of a driver.
