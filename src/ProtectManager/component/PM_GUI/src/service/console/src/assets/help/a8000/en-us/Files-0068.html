Managing Common Shares

The OceanProtect allows you to perform operations on databases, such as modifying, removing, activating, and disabling protection.

You need to log in to the management page, choose Protection > File Systems > Common Shares, find the common share resource to be operated, and choose More to select an operation to be performed.

You can also select multiple common shares and manage common share protection in batches.

Table 1 describes the detailed operations.

Table 1 Operations related to common share protection

Operation

Description

Navigation Path

Protect

Scenario

This operation enables you to protect a common share resource.

In the row of the target common share resource, choose More > Protection.

Modify

Scenario

This operation enables you to modify the name and share protocol of a common share resource.

In the row of the target common share resource, choose More > Modify.

Delete

Scenario

You can delete a common share resource when you do not need to protect it.

Note

Common share resources that have been associated with SLAs cannot be deleted.

In the row of the target common share resource, choose More > Delete.

Modify Protection

Scenario

If you need to reselect an SLA associated with a common share resource, you can modify the protection plan. After the modification, the next backup will be executed based on the new protection plan.

Note

  • The modification does not affect the protection job that is being executed. The modified protection plan will apply to the next protection period.
  • If the protection policy for a resource in the associated SLA changes, you are advised to modify the current SLA on the Protection > Protection Policies > SLAs page, rather than associate the resource with another SLA through protection modification.

    After you associate the resource with another SLA by modifying protection, if the copy of the original SLA still exists, the license capacity may be deducted repeatedly.

In the row of the target common share resource, choose More > Modify Protection.

Remove Protection

Scenario

You can remove the protection plan if a common share resource no longer needs to be protected. After the removal, the system cannot execute protection jobs for the common share resource. To protect the resource again, you need to associate an SLA with the resource.

Note

  • When a protection job of the common share resource is in progress, protection cannot be removed.
  • After you remove protection and associate the resource with another SLA, if the copy of the original SLA still exists, the license capacity may be deducted repeatedly.

In the row of the target common share resource, choose More > Remove Protection.

Manual Backup

Scenario

This operation enables you to perform a backup immediately.

Note

Copies generated by manual backup are retained for the duration defined in the SLA.

In the row of the target common share resource, choose More > Manual Backup.

Activate Protection

Scenario

This operation enables you to re-activate a disabled protection plan of a common share resource. After the protection plan is activated, the system backs up the common share resource based on the protection plan.

Note

This operation can be performed only for a common share resource that has been associated with an SLA and is in the Unprotected state.

In the row of the target common share resource, choose More > Activate Protection.

Disable Protection

Scenario

This operation allows you to disable the protection plan when periodic protection on a common share resource is not needed. After the protection plan is disabled, the system does not execute the automatic backup job of the common share resource.

Note

Disabling protection does not affect ongoing backup jobs.

In the row of the target common share resource, choose More > Disable Protection.

Add Tag

Scenario

This operation enables you to quickly filter and manage resources.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.

Remove Tag

Scenario

This operation enables you to remove the tag of a resource when it is no longer needed.

Choose More > Remove Tag.
