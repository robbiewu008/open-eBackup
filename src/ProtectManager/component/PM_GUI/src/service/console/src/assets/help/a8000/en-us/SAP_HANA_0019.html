Backing Up the SAP HANA Database (SAP HANA from the Applications Path) (Only for 1.6.0 and Later Versions)

Before creating a data protection job, ensure that ProtectAgent has been installed. For details, see the OceanProtect DataBackup 1.5.0-1.6.0 ProtectAgent Installation Guide.

The SAP HANA database can be backed up in the file or Backint mode. You can select only one backup mode.
