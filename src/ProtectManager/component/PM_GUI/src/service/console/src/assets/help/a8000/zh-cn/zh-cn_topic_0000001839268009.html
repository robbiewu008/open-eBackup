步骤2：设置Redo日志模式

Redo日志保存可设置为Redo日志归档模式（ARCHIVELOG）或者Redo日志在线模式（NOARCHIVELOG）。

  • 归档模式（ARCHIVELOG）

    当Redo日志写满后，把该日志内容拷贝到归档日志进行统一的存储，避免下次写入的时候覆盖。

  • 在线模式（NOARCHIVELOG）

    当Redo日志写满后，直接切换日志，而不做任何的备份。下次轮到写入该文件时，文件的内容将会被覆盖。

操作步骤

  1. 连接到Gauss100 OLTP服务器。
  2. 通过执行以下SQL语句查询当前日志模式。

    SELECT LOG_MODE FROM DV_DATABASE;

  3. 如果2查询的日志归档模式需要修改，执行以下SQL语句修改Redo日志模式，此语句为修改Redo日志归档模式的示例。

    ALTER DATABASE ARCHIVELOG;

  4. 修改完成后执行2查询是否修改成功。
  • 数据库主备部署时，只能设置Redo日志为归档模式（ARCHIVELOG），不能设置Redo日志为在线模式（NOARCHIVELOG）。
  • Gauss100 OLTP默认为归档模式。
