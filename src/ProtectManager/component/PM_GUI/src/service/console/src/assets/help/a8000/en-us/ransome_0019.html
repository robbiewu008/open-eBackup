Managing Detection Models

You can view, activate, and delete ransomware detection models.

Choose Data Security > Ransomware Protection > Ransomware Protection and WORM Configurations and click Detection Model. On the displayed Detection Model page, perform management operations by referring to Table 1.

For 1.5.0, choose Explore > Ransomware Protection and perform management operations on the Overview page by referring to Table 1.

Table 1 Detection model management operations

Operation

Description

Navigation Path

Viewing a detection model

Scenario

Viewing the detection model enables you to know the basic information and activation status of the ransomware protection policy.

In the detection model list on the right, view the version, import time, signature information, description, and activation status.

Updating a detection model

Scenario

Common detection models are preset in the system. You can import the latest detection model (if any) of the storage system, or import or reactivate a detection model of an earlier version if it works better in your scenario.

Note

  • Obtain the latest detection model from Huawei technical support engineers.
  • A detection model is automatically activated after being imported. The previously activated one changes to the Inactivated state.
  • The import does not affect ongoing detection jobs. Ongoing detection jobs still use the earlier detection model. The new detection model will be used in the next job.

On the Detection Model page, click Import.

Activating a detection model

Scenario

To use a deactivated detection model to detect ransomware, activate it first.

Note

  • Only detection models in the Inactivated state can be activated.
  • Only one detection model in the Activated state exists in the system. The activated detection model is automatically deactivated after you activate or import a new detection model.
  • If a previously activated detection model is being used for ransomware detection, the newly activated or imported detection model enters in the Activated state but does not affect the ongoing detection job and will take effect in the new job.

In the row of the target detection model, choose More > Activate in the Operation column on the right.

Deleting a detection model

Scenario

You can delete detection models unnecessary for ransomware detection.

Note

Only detection models in the Inactivated state can be deleted.

In the row of the target detection model, choose More > Delete in the Operation column on the right.
