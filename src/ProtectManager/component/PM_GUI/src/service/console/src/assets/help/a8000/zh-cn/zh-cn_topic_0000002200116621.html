如何修改主机时区和时间

当待安装客户端的主机与OceanProtect设备时间不一致时，可参考本章节修改待安装客户端的主机的时区和时间。

Linux OS

以下操作以SUSE Linux Enterprise Server 12 SP4为例：

  1. 使用PuTTY工具以root账户登录待安装客户端的主机。
  2. 执行以下命令，修改待安装客户端的主机的时区。

    • 如需修改为UTC，请执行以下命令：
      ln -sf /user/share/zoneinfo/UTC /etc/localtime
    • 如需修改为CST，请执行以下命令：
      ln -sf /user/share/zoneinfo/Asia/Shanghai /etc/localtime

      命令中的Asia/Shanghai请根据待修改的时区替换。

  3. 执行以下命令，修改待安装客户端的主机的系统时间。

    date -s "20241125 20:22:35"

  4. 执行以下命令，使待安装客户端的主机的硬件时间与系统时间保持一致。

    hwclock --systohc

AIX OS

以下操作以SUSE Linux Enterprise Server 12 SP4为例：

  1. 使用PuTTY工具以root账户登录待安装客户端的主机。
  2. 执行以下命令，进入SMIT（System Management Interface Tool）。

    smit

  3. 系统回显类似如下信息时，通过方向键选择“System Environments”，并按“Enter”。

    Move cursor to desired item and press Enter.

  Software Installation and Maintenance
  Software License Management
  Manage Editions
  Devices
  System Storage Management (Physical & Logical Storage)
  Security & Users
  Communications Applications and Services
  Workload Partition Administration
  Print Spooling
  Advanced Accounting
  Problem Determination
  Performance & Resource Scheduling
  System Environments
  Processes & Subsystems
  Applications
  Installation Assistant
  Electronic Service Agent
  Cluster Systems Management
  Using SMIT (information only)

  4. 系统回显类似如下信息时，通过方向键选择“Change / Show Date and Time”，并按“Enter”。

    Move cursor to desired item and press Enter.

  Stop the System
  Aix Security Expert
  Assign the Console
  Change / Show Date and Time
  Manage Language Environment
  Change / Show Characteristics of Operating System
  Change / Show Number of Licensed Users
  Broadcast Message to all Users
  Manage System Logs
  Change / Show Characteristics of System Dump
  Change / Show Default Browser
  Web-based System Manager
  System Cryptographic Operational Mode
  Manage Remote Reboot Facility
  Manage System Hang Detection

  5. 系统回显类似如下信息时，选择对应选项修改时区或时间，并按“Enter”。

    Move cursor to desired item and press Enter.

  Change / Show Date & Time
  Change Time Zone Using System Defined Values
  Change Time Zone Using User Inputted Values

Windows OS

以下操作以Windows 10为例：

  1. 打开“控制面板”，单击“时钟和区域”。
  2. 选择“设置时间和日期”。
  3. 在“日期和时间”界面，单击“更改时区”，选择时区后单击“确定”。
  4. 在“日期和时间”界面，单击“更改日期和时间”，设置时间后单击“确定”。
  5. 单击“确定”，完成修改。
