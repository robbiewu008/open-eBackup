The XtraBackup Tool Fails to Run, Leading to the Backup Failure

Symptom

The backup fails, and the error message "Exec backup cmd failed." is displayed in the log.

Possible Causes

The version of the GCC library file in the host environment is outdated and the XtraBackup backup tool cannot be supported.

Troubleshooting

  1. Log in to the host where the client is installed as user root.
  2. Run the yum install devtoolset-x command to install the GCC library of a later version.

    For example, yum install devtoolset-8.

    In the devtoolset-x parameter, x indicates the GCC library version. Replace it based on site requirements.
