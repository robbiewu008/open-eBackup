Managing Protection Plans for File Systems

The OceanCyber 300 Data Security Appliance allows you to modify, remove, activate, or disable protection plans for file systems.

You need to log in to the OceanCyber 300 Data Security Appliance, choose Data Security > Ransomware Protection > Intelligent Detection > File Systems, and find the desired file system. Table 1 lists the operations.

Table 1 Operations on protection plans related to a file system

Job

Description

Navigation Path

Modify Protection

Scenario

You can modify the protection plan of a file system to associate it with another intelligent detection policy. After the modification, the next protection job will be executed based on the new protection plan.

Note

The modification does not affect the protection job that is being executed. The modified protection plan will take effect in the next protection period.

In the row that contains the desired file system, choose More > Modify Protection.

Remove Protection

Scenario

If a file system does not need to be protected, you can remove its protection. After the protection is removed, the system cannot execute protection jobs for the file system. To protect the file system again, associate it with an intelligent detection policy again.

Note

When a protection job of the file system is running, protection cannot be removed.

In the row that contains the desired file system, choose More > Remove Protection.

Disable Protection

Scenario

If you do not need to perform periodic protection for a file system, you can disable the protection plan of the file system. After this function is disabled, the system does not automatically execute the ransomware detection snapshot generation job for the file system.

Note

Disabling protection does not affect the running ransomware detection snapshot generation job.

In the row that contains the desired file system, choose More > Disable Protection.

Activate Protection

Scenario

After the protection plan of a file system is disabled, you can activate the protection plan to make it take effect again. After the protection plan is activated, the system generates a ransomware detection snapshot for the file system based on the protection plan.

Note

This operation can be performed only on a file system that has been associated with an intelligent detection policy and is in the Unprotected state.

In the row that contains the desired file system, choose More > Activate Protection.
