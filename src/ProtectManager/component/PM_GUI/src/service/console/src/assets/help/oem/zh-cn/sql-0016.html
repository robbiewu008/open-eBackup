步骤4：设置日志备份恢复模式

执行SQL Server日志备份前，需要将实例下数据库恢复模式设置为FULL，否则将会导致日志备份失败。

操作步骤

  1. 登录SQL Server数据库主机，以操作系统Windows 2019、数据库SQL Server 2019为例。
  2. 使用数据库管理员账户登录Microsoft SQL Server Management Studio管理界面。
  3. 选中对应数据库，右键“Properties”，进入Database Properties设置界面。
  4. 单击“Options”，设置“Recovery model”为FULL。
  5. 单击“确定”。
