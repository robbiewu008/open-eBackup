Preparing for the Backup

Before backing up Hyper-V VMs, prepare related information listed in Table 1.

Table 1 Preparing for the backup

Item

Mandatory or Not (Y/N)

How to Obtain

Domain account and password of the Hyper-V virtualization platform

Y
  • Obtain the information from the Hyper-V virtualization platform administrator.
  • When the registration type is SCVMM, obtain the domain account and password of the SCVMM server.
