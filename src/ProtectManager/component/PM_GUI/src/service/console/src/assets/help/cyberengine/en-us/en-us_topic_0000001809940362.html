Performing Real-Time Detection

The system performs real-time detection on the desired file systems according to the associated real-time detection policy.

Precautions

If the storage device is the OceanProtect Backup Storage and DataTurbo sharing is enabled for the file system, real-time detection cannot be enabled.

Procedure

  1. On the top menu bar, choose Data Security > Ransomware Protection > Real-Time Detection.
  2. Click File Systems.
  3. Choose More > Protect in the row that contains the desired file system.

    You can also select multiple file systems and choose More > Protect.

  4. Select an existing real-time detection policy or click Create to create one.
  5. Click OK.
