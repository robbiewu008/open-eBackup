Authorizing a Backup Storage Unit (Applicable to 1.6.0 and Later Versions)

If a user of the Data Protection Administrator or custom role is used to perform protection operations on resources, the backup storage unit where backup data is to be stored must be authorized to the user before a backup SLA is created. As such, the user can use the authorized backup storage unit when creating a backup SLA. For a system administrator, skip this section.

Procedure

  1. Choose System > Infrastructure > Backup Storage Unit Groups.
  2. Switch to the Backup Storage Units tab page and click the name of a backup storage unit.
  3. On the displayed Basic Info page, locate the row that contains the target user for authorization and click .

    You can also select multiple users for authorization and click Batch Authorize.
