Viewing Snapshot Data

You can view information about ransomware detection snapshot data on the OceanCyber WebUI.

You need to log in to the OceanCyber. In the navigation pane, choose Snapshot Management and Restoration. Table 1 describes the ransomware detection snapshot data.

  • Click to view snapshot data by resource.
    Table 1 Ransomware detection snapshot data information

    Parameter

    Description

    Resource Object

    Name

    Name of the file system to which the snapshot data belongs.

    NOTE:

    You can click the name of the file system data to view details about the file system snapshot.

    Storage Device

    Storage device to which the file system snapshot belongs.

    Owning vStore

    vStore to which the file system snapshot belongs.

    Snapshots

    Number of snapshots of a file system.

    NOTE:

    You can click the number of snapshots to view details about the snapshot data.

    Latest Detection Result

    Snapshot Generation Time

    Time when a file system snapshot is generated.

    Status

    Security status of the snapshot data, which can be Infected, Uninfected, Detecting, or Undetected.

    Generation Mode

    Mode in which a snapshot is generated. The value can be intelligent detection or real-time detection.

    Total File Size

    Size of the file system snapshot data.

    New Files

    Number of new files obtained by comparing the latest snapshot with the last uninfected snapshot.

    Modified Files

    Number of modified files obtained by comparing the latest snapshot with the last uninfected snapshot.

    Deleted Files

    Number of deleted files obtained by comparing the latest snapshot with the last uninfected snapshot.

    Suspicious Files

    Added or modified suspicious files detected in the file.
  • Click to view snapshot data by snapshot.
    Table 2 Ransomware detection snapshot data information

    Parameter

    Description

    Snapshot

    Snapshot Generation Time

    Time when a file system snapshot is generated.

    Status

    Status of the file system snapshot data.

    Generation Mode

    Mode in which a snapshot is generated. The value can be intelligent detection or real-time detection.

    Secure Snapshot

    After Uninfected Snapshot Lock is enabled in the detection policy for a snapshot generated by intelligent detection, the uninfected snapshot is converted to a secure snapshot. In addition, the snapshot expiration time is delayed and cannot be modified or deleted before the snapshot expires.

    Snapshot Expiration Time

    Time when a file system snapshot expires.

    Resources

    Name

    Security status of the snapshot data, which can be Infected, Uninfected, Detecting, or Undetected.

    Storage Device

    Storage device to which the file system snapshot belongs.

    Owning vStore

    Number of snapshots of a file system.
