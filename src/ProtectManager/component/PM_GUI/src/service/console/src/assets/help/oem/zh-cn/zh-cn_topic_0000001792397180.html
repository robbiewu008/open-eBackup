步骤2：注册Kingbase集群实例下的数据库

前提条件

注册Kingbase集群实例前，请确保集群中每个主机已进行注册。

具体操作请参见操作步骤。

操作步骤

  1. 创建集群。

    1. 在Kingbase页面选择“集群”页签。
    2. 单击“注册”。
    3. 在“注册”页面，自定义集群名称并选择类型和集群节点。
    4. 单击“确定”。

  2. 注册集群实例下的数据库。

    1. 在Kingbase页面选择“实例”页签。
    2. 单击“注册”。
    3. “类型”选择“集群实例”。
    4. 自定义实例名，选择集群。
    5. 单击“添加”，在弹出的添加对话框中添加集群实例下的所有节点。

      相关参数说明如表1所示。

      表1 Kingbase集群实例的节点信息

      参数

      说明

      主机

      选择集群下的主机节点。

      用户名

      运行数据库的操作系统用户名。

      数据库安装路径

      要注册的数据库安装路径。

      业务IP地址

      数据库服务接收TCP/IP连接的主机IP地址。

      数据库端口

      数据库服务侦听TCP/IP连接的端口号。

      数据库用户名

      数据库的管理员用户名称。

      数据库密码

      数据库管理员用户的密码。
    6. 单击“确定”。
