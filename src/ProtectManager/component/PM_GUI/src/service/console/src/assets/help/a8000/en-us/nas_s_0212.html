Step 2: (Optional) Creating a File Directory

Directories can be created in a file system as resources. Backup, replication, archiving, and file-level restoration can be performed for these file directories.

Prerequisites

A file system has been added.

Procedure

  1. Log in to the OceanProtect WebUI by referring to Logging In to the OceanProtect WebUI.
  2. Choose Protection > File Systems > NDMP.
  3. Click the File Directory tab and click Create.

    Configure parameters as required. Table 1 describes the parameters.

    Table 1 Parameters for creating a file directory

    Parameter

    Description

    Name

    User-defined device name.

    Storage Device

    Select a created storage device. To create a storage device, refer to Step 1: Adding a Storage Device.

    File System

    Select a file system.

    Directory

    Enter the full path of the directory, for example, /opt/b.
