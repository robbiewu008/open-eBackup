Setting Functions (Applicable to 1.5.0)

This section describes how to set the functions of a data protection administrator.

Procedure

  1. Choose System > Security > Quotas and Functions.
  2. Select the desired user and choose More > Set Functions.
  3. Select functions to be supported, including Backup, Replicate, Archive, and Restore.
  4. Click OK.
