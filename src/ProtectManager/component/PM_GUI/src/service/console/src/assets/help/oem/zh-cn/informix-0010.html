步骤3：注册Informix/GBase 8s实例

操作步骤

注册Informix/GBase 8s单实例

  1. 选择“保护 > 数据库 > Informix/GBase 8s”。
  2. 选择“实例”页签。
  3. 单击“注册”，注册Informix/GBase 8s。
  4. “类型”选择“单实例”。
  5. 配置数据库实例及其认证信息。

    相关参数说明如表1所示。

    表1 Informix/GBase 8s单实例注册信息

    参数

    说明

    名称

    实例名称。

    集群

    选择已创建的集群（注册实例所在的主机）。

    Sqlhost文件路径

    输入Sqlhost文件绝对路径。

    Onconfig文件路径

    输入Onconfig文件绝对路径。

  6. 单击“确定”。

注册Informix/GBase 8s集群实例

  1. 在Informix/GBase 8s页面选择“实例”页签。
  2. 单击“注册”。
  3. “类型”选择“集群实例”。
  4. 自定义实例名。
  5. 单击“添加”，在弹出的添加对话框中添加集群实例下的所有节点。

    相关参数说明如表2所示。

    表2 Informix/GBase 8s集群注册信息

    参数

    说明

    名称

    实例名称。

    集群

    选择已创建的集群。

    Sqlhost文件路径

    输入Sqlhost文件绝对路径。

    Onconfig文件路径

    输入Onconfig文件绝对路径。

  6. 单击“确定”。
