Step 5: Creating a Remote Device Administrator

This section describes how to create a remote device administrator in the target cluster to be replicated. This administrator is used to verify the peer cluster during the replication.

Procedure

  1. Log in to the management page of the replication target cluster.
  2. Choose System > Security > Users and Roles.
  3. Click Create.

    Table 1 describes the related parameters.

    Table 1 User parameters

    Parameter

    Description

    User Type

    Select Local user.

    Name

    Name of a user, which can be customized.

    Login Authentication Mode

    Login mode for the user. The value can be Login password or Login password+Email One-time password.

    If you have high security requirements, you are advised to select Login password+Email One-time password.

    Password

    Password of a user.

    Confirm Password

    Confirm password.

    Role

    Choose Remote Device Administrator.

    Description

    Description of a user.

  4. Click OK.
