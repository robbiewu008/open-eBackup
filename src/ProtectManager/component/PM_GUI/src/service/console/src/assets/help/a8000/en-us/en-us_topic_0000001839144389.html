Resetting the System Administrator Password

This section describes how to reset the password of a system administrator.

Precautions

The password retrieval email address can be set, viewed, modified, or deleted only by the system administrator.

Procedure

  1. On the Password Retrieval page, enter the system administrator username, email, and verification code, and click Submit.

    • The verification code is valid for 15 minutes.
    • You can obtain verification codes for a maximum of 30 times within 24 hours.

  2. On the password resetting page, reset the password and click OK.

    The password resetting page is valid for 15 minutes.

  3. Return to the login page, enter the username and new password of the system administrator, and click Log In.
