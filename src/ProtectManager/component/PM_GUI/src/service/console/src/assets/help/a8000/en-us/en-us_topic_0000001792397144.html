Managing Identification Rules

You can view, modify, delete, and clone identification rules.

Precautions

Preset identification rules cannot be modified or deleted.

If a custom identification rule has been associated with an anonymization policy, you need to remove the identification rule from the associated anonymization policy before modifying or deleting the identification rule.

Management Operations

Table 1 describes the operations for managing identification rules.

Table 1 Operations for managing identification rules

Task

Description

Navigation Path

Viewing identification rules

You can view the quantity and details of identification rules.

On the top navigation bar, choose Explore > Data Utilization Policies. Select Anonymization Policies in the left navigation pane and click the Identification Rules tab to view the name, rule source, expression, anonymization rule, and number of associated anonymization policies of an identification rule.

You can click the number of associated anonymization policies to view the related anonymization policies.

Modifying identification rules

You can modify the name, description, and associated anonymization rule of a custom identification rule.

In the Operation column on the right of the row containing the identification rule, choose More > Modify.

Deleting identification rules

You can delete a custom identification rule if it is no longer required.

In the Operation column on the right of the row containing the identification rule, choose More > Delete. You can delete multiple custom identification rules in batches.

Cloning identification rules

If an identification rule to be created is similar to an existing rule, you can clone and alter the existing rule to create one.

In the Operation column on the right of the row containing the identification rule, choose More > Clone.

If the identification rule to be cloned is a preset rule, click Clone.
