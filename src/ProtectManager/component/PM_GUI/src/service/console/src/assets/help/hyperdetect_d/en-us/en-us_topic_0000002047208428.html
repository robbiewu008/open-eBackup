Viewing Local Storage Information

This section describes how to view the basic information of a local storage.

Procedure

  1. Choose System > Infrastructure > Local Storage.
  2. In the Basic Info and Authentication Information areas on the Local Storage page, view the basic information and authentication information of the local storage.

    Table 1 describes the related parameters.

    Table 1 Local storage parameters

    Parameter

    Description

    Basic Info

    Version

    Current version of the system.

    ESN

    Current ESN of the system.

    An equipment serial number (ESN) is a character string that uniquely identifies a device.

    Model

    Product model.

    WWN

    Current WWN of the system.

    A World Wide Name (WWN) is a globally unique name that identifies a connection or connection set on a network. It is principally used in Fibre Channel and SAS.

    Total Disk Capacity

    Total disk capacity of the system.

    Login Authentication

    Username

    Local storage username used for login authentication. Use a user who has the super administrator rights.

    Password

    Login authentication user password.

    Status

    User authentication status. If the authentication status is Abnormal, modify authentication information.

    Service Authentication

    Username

    Local storage username used for service authentication. Use a user who has the super administrator rights.

    Password

    Service authentication user password.

    Status

    User authentication status. If the authentication status is Abnormal, modify authentication information.
