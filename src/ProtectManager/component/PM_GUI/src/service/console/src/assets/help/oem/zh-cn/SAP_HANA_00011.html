数据库名称与实列ID

必选
要保护的数据库名称，数据库名称可通过以下方式查询：
  1. 使用PuTTY登录SAP HANA数据库所在主机。
  2. 执行以下命令，进入SAP HANA数据库终端系统，以用户名s00adm为例。
    su - s00adm
hdbsql -i 00 -n localhost:30013 -d SYSTEMDB -u SYSTEM -p 用户安装数据库时自定义的密码;
  3. 执行以下命令，查询数据库名称与实例ID。
    SELECT * FROM M_DATABASES;

    回显类似如下，其中SYSTEMDB是数据库名称，S00是实例ID。

    DATABASE_NAME,DESCRIPTION,ACTIVE_STATUS,ACTIVE_STATUS_DETAILS,OS_USER,OS_GROUP,RESTART_MODE
"SYSTEMDB","SystemDB-S00-00","YES","","","","DEFAULT"
"S00","S00-00","YES","","","","DEFAULT"
