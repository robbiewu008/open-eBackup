Step 2: Registering a TiDB Database

Register the TiDB database before registering the TiDB table set.

Procedure

  1. Choose Protection > Databases > TiDB.
  2. Click Databases.
  3. Click Create to create a TiDB database.

    Table 1 describes the related parameters.

    Table 1 TiDB database parameters

    Parameter

    Description

    Name

    Name of a new TiDB database, which can be customized.

    Cluster

    TiDB cluster that has been registered with the OceanProtect.

    Databases

    Database in the TiDB cluster in the production environment.

  4. Click OK.
