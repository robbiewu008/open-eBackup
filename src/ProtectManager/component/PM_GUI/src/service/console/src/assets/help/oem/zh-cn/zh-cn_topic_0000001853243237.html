安装ProtectAgent

执行应用备份恢复前，需要先安装ProtectAgent。

安装了ProtectAgent软件的主机称为数据保护代理主机，简称代理主机。代理主机用于备份恢复过程中的流程控制和数据传输。对资源进行保护前需要准备用于安装ProtectAgent软件的主机，并为主机安装ProtectAgent。

在代理主机上安装ProtectAgent的流程如表1所示。

表1 ProtectAgent安装流程

任务

说明/操作指导

准备代理主机
  • 代理主机为Exchange主机。

    受保护的Exchange数据库所在的主机均需要部署ProtectAgent。

  • 代理主机的资源要求，请参见《ProtectAgent安装指南》中的“ProtectAgent安装前检查（ProtectAgent部署在生产环境）”章节。

安装ProtectAgent前的配置

安装代理前，请确保安装代理的主机已关闭防火墙。

安装ProtectAgent
  • 请参见《ProtectAgent安装指南》中的“安装ProtectAgent软件”章节。

对于Exchange Server，需要在代理主机安装微软最新的Visual C++ Redistributable for Visual Studio 2012运行库组件。该组件的官方获取路径请参考组件下载路径。
