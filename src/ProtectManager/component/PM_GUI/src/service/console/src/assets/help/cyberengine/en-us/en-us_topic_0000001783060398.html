File Name Extension Filtering Rule Disassociated From A File System

This section describes how to disassociate a file name extension filtering rule from a file system.

Procedure

  1. Choose Data Security > File Interception.
  2. Click the File Name Extension Filtering Rules tab.
  3. In the row that contains a file name extension filtering rule, choose MoreDisassociate from vStore.

    You can also select multiple file name extension filtering rules and choose More > Disassociate from vStore in the upper part.

  4. Select the file system to be disassociated and click OK.
  5. In the dialog box that is displayed, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
