Managing Backup Storage Units (Applicable to 1.5.0)

To be compatible with backup storage cluster capabilities of 1.3.0 and earlier versions, Backup Storage Clusters is renamed as Backup Storage Units and Backup Storage Library is renamed as Backup Storage Unit Groups in 1.5.0.

For existing configurations in earlier versions, the following operations are supported.

  • The existing configuration is retained and can be managed. (Backup Storage Clusters -> Backup Storage Units, Backup Storage Library -> Backup Storage Unit Groups)
  • The backup storage cluster is upgraded as a backup member node.
