Step 3: Creating GaussDB (DWS) Schema Sets/Table Sets

If you want to protect GaussDB (DWS) schema sets or table sets, create such sets by following the instructions provided in this section.

Precautions

You cannot create a schema set for GaussDB (DWS) whose kernel version is GaussDB A 8.0.0.

Procedure

  1. Choose Protection > Big Data > GaussDB (DWS).
  2. Click Schema Sets or Table Sets tab.
  3. Click Create and create a schema set or table set.

    1. Configure related parameters. Table 1 and Table 2 describe the parameters.
      Table 1 Schema set parameters

      Parameter

      Description

      Name

      Customize the name of the schema set.

      Cluster

      Select the GaussDB (DWS) cluster to which the object to be protected belongs.

      Databases

      Select the database to which the object to be protected belongs.

      Schemas

      On the Available Schemas page, select the schemas to be protected in the database.
      Table 2 Table set parameters

      Parameter

      Description

      Name

      Customize the table set name.

      Cluster

      Select the GaussDB (DWS) cluster to which the object to be protected belongs.

      Databases

      Select the database to which the object to be protected belongs.

      Table

      On Available Tables page, select the tables to be protected in the database.
    2. Click OK.
