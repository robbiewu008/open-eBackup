Creating an Air Gap Policy

This section describes how to create an Air Gap policy.

Procedure

  1. Choose Data Security > Air Gap.

    For 1.5.0, choose Explore > Air Gap.

  2. Click the Air Gap Policies tab.
  3. Click Create.
  4. Set the basic information about the Air Gap policy. Table 1 describes the related parameters.

    Table 1 Air Gap policy parameters

    Parameter

    Description

    Name

    Customized name of an Air Gap policy.

    Description

    Customized description of an Air Gap policy.

    Period

    Air Gap is executed By day or By Week.

    Interval
    • If Period is set to By day, Interval is set to Everyday by default.
    • If Period is set to By Week, Interval can be set to one or more days in a week.

    Replication Time Window

    Replication is allowed only within the replication time window.

    NOTE:
    • For 1.5.0, the replication time window cannot reach one calendar day.
    • A maximum of three replication time windows can be set. Different time windows cannot overlap each other.
    • Ensure that the replication time window set in the Air Gap policy covers the start time and end time of each replication job. Otherwise, replication jobs may fail after the Air Gap policy takes effect.

  5. Click OK.
