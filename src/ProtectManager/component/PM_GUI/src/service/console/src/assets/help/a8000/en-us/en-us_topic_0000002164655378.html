Viewing Volume Information

After a volume is created, you can view its details on the management page.

Procedure

  1. Choose Protection > File Systems > Volumes.
  2. View the volume information.

    Some parameters are not displayed by default. You can click next to Operation and select a parameter to display it on the current page.

    Table 1 describes the related parameters.
    Table 1 Volume information

    Parameter

    Description

    Name

    Volume name.

    Host Name

    Name of the host where a volume is located.

    IP Address

    IP address of the host where the volume is located.

    SLAs

    If an SLA is associated with the volume, the SLA name is displayed. Otherwise, -- is displayed.

    SLA Compliance
    Whether a backup job is successfully executed based on the backup interval or backup window set in the SLA.
    • Met: If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
    • Missed: If the backup time does not meet the backup time window or backup interval requirements, for example, if the backup interval set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup interval or backup window.

    Protection Status

    Whether the volume is protected.

    • Unprotected: The volume is not associated with an SLA, or the volume is associated with an SLA but protection is disabled.
    • Protected: The volume has been associated with an SLA and protection has been activated.
    • Creating: The volume has been associated with an SLA and protection is being created.

    Tag

    If a tag has been added for the resource, the tag name is displayed. Otherwise, -- is displayed.

  3. Click a volume name to view the volume overview, copy information, protection details, and related jobs.

    • Overview

      Displays the volume name, SLA information, job execution time, and total number of copies.

      You can also view the files and directories that are protected in the current volume.

    • Copy Data

      You can search for copies by year, month, and day.

      If is displayed below the time, copies exist at that time.

    • Jobs

      Displays all jobs related to the volume, including backup and restoration jobs.
