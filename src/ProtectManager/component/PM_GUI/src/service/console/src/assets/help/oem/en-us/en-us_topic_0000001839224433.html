Setting Quotas

This section describes how to set a backup quota or an archive quota.

Precautions

  • For non-SAML users, the quota is not limited by default. For SAML users, the default quota is 0, and the quota cannot be set to Unlimited.
  • For non-SAML users, the quota capacity must be greater than the used capacity.

Procedure

  1. Choose System > Security > Quotas and Functions.
  2. Set the backup quota.

    1. Select the desired user and choose More > Set Backup Quotas.
    2. Set the quota.
      • For SAML users:

        Set Backup Quota and click OK.

      • For non-SAML users:
        Select Unlimited or Capacity Quota. Click OK.
        • If you select Unlimited, there is no quota upper limit.
        • If you select Capacity Quota, you need to set the quota capacity. The quota capacity must be greater than the used quota.

  3. Set the archive quota.

    1. Select the desired user and choose More > Set Archive Quotas.
    2. Set the quota.
      • For SAML users:

        Set Archive Quota and click OK.

      • For non-SAML users:
        Select Unlimited or Capacity Quota. Click OK.
        • If you select Unlimited, there is no quota upper limit.
        • If you select Capacity Quota, you need to set the quota capacity. The quota capacity must be greater than the used quota.
