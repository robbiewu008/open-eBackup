Managing Huawei Cloud Stack GaussDB Projects

After Huawei Cloud Stack GaussDB is registered with the OceanProtect, you can perform operations on Huawei Cloud Stack GaussDB.

Choose Protection > Cloud Platforms > Huawei Cloud Stack GaussDB. On the Huawei Cloud Stack GaussDB page that is displayed, select the cloud platform to be operated in the left pane.

Table 1 describes the related operations.

Table 1 Operations related to Huawei Cloud Stack GaussDB projects

Operation

Description

Navigation Path

Grant permissions on Huawei Cloud Stack GaussDB projects.

NOTE:

Only version 1.5.0 supports this function.

Scenario

By default, only a system administrator has the permission to protect resources. However, a system administrator can grant this permission to a data protection administrator.

Note

The permission on a single resource can be granted to only one data protection administrator.

Choose More > Authorize Resource and select a data protection administrator to be authorized.

Reclaim Huawei Cloud Stack GaussDB projects.

NOTE:

Only version 1.5.0 supports this function.

Scenario

This operation enables you to reclaim resources authorized to a data protection administrator. The data protection administrator is not permitted to protect the resources that have been reclaimed.

Choose More > Reclaim Resource.

Scan Huawei Cloud Stack GaussDB.

Scenario

The OceanProtect automatically scans the Huawei Cloud Stack GaussDB environment every 5 minutes. If the projects in the registered Huawei Cloud Stack GaussDB environment change, the changes will be automatically updated to the OceanProtect. This operation enables you to immediately update changes to the OceanProtect.

Choose More > Resource Scan.

Test Huawei Cloud Stack GaussDB connectivity.

Scenario

Manual connectivity test of the registered Huawei Cloud Stack GaussDB environment is used to check the connectivity between the Huawei Cloud Stack GaussDB environment and the host where the client is installed as well as the production environment.

Choose More > Test Connectivity.

Modify Huawei Cloud Stack GaussDB projects.

Scenario

Perform this operation when you need to modify a Huawei Cloud Stack GaussDB project.

Choose More > Modify.

Delete Huawei Cloud Stack GaussDB projects.

Scenario

Perform this operation when instances in a Huawei Cloud Stack GaussDB project do not need to be protected.

Choose More > Delete.
Adding a tag
NOTE:

This function is available only in 1.6.0 and later versions.

Scenario

This operation enables you to quickly filter and manage resources.

Note
  • The system administrator can add a maximum of 10,000 tags to a resource.
  • A non-system administrator must have the permission of Production resource management to add tags to resources.
  • A non-system administrator can add a maximum of five tags to a resource.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.
Removing a tag
NOTE:

This function is available only in 1.6.0 and later versions.

Scenario

This operation enables you to remove the tag of a resource when it is no longer needed.

Choose More > Remove Tag.
