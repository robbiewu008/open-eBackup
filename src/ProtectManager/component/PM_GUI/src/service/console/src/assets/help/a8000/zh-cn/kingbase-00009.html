步骤2：注册Kingbase单实例下的数据库

操作步骤

  1. 选择“保护 > 数据库 > Kingbase”。
  2. 选择“实例”页签。
  3. 单击“注册”，注册Kingbase。
  4. “类型”选择“单实例”。
  5. 配置数据库实例及其认证信息。

    相关参数说明如表1所示。

    表1 Kingbase单实例注册信息

    参数

    说明

    名称

    自定义实例名称。

    主机

    选择要注册的数据库主机。

    用户名

    运行数据库的操作系统用户名。用户名查询方式请参考备份前准备。

    数据库安装路径

    要注册的数据库安装路径。数据库安装路径查询方式请参考备份前准备。

    业务IP地址

    数据库服务接收TCP/IP连接的主机IP地址。业务IP查询方式请参考备份前准备。

    数据库端口

    数据库服务侦听TCP/IP连接的端口号。

    数据库用户名

    数据库的管理员用户名称。

    数据库密码

    数据库管理员用户的密码。

  6. 单击“确定”。
