步骤1：开启DmAPService服务

Dameng在执行备份和恢复任务前，需要开启DmAPService服务，否则无法执行备份和恢复操作。

注意事项

  • 本节操作如无特殊说明均以Dameng 8版本为例进行，数据库版本不同操作可能存在差异，请以实际为准。
  • 对于以集群方式部署的Dameng数据库，如无特殊说明，均需登录集群实例所在的所有主机执行以下操作。

操作步骤

  1. 登录Dameng数据库主机。
  2. 执行命令查询是否开启DmAPService服务。

    systemctl status DmAPService
    • 如果“Active”状态为“active（running）”，则表示已开启DmAPService服务。
    • 如果“Active”状态为“inactive（dead）”，则表示未开启DmAPService服务，请执行systemctl start DmAPService命令，开启DmAPService服务。
