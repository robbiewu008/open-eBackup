Installing Mariabackup

Before performing a backup job for MariaDB, you need to manually install Mariabackup on all MariaDB nodes.

Procedure

  1. Log in to the host where the MariaDB instance is located and run the following command to check whether Mariabackup has been installed.

    mariabackup --version
    • If the version number is displayed in the command output, Mariabackup has been installed.
    • If no version number is displayed, Mariabackup has not been installed.

  2. Install Mariabackup.

    You can use the binary package or RPM package for installation.

    • Install Mariabackup using a binary package.
      1. Find the mariabackup command in the bin folder in the MariaDB installation directory.
      2. Run the following command to install Mariabackup:
        ln -sf Installation directory/bin/mariabackup/usr/bin/
      3. Perform 1 to check whether Mariabackup has been installed.
    • Install Mariabackup using an RPM package.
      1. Run the following command to find the directory where the RPM package is stored:
        find / -name 'MariaDB-backup*'
      2. Run the following command to install Mariabackup:
        rpm -ivh RPM package directory/RPM package name
      3. Perform 1 to check whether Mariabackup has been installed.
