How Do I Download Management Data from the SFTP Server (Applicable Only to 1.6.0 and Later Versions) (Applicable to OceanProtect X Series Backup Appliances)

This section describes how to download management data to be imported to the OceanProtect from the SFTP server.

Procedure

  1. Query the ESN of the OceanProtect node. For details, see Viewing the ESN of a Device.
  2. Use WinSCP to log in to the SFTP server as the SFTP user.
  3. In the backup path, go to the folder corresponding to the node ESN queried in 1 and download the management data to the local PC.

    To view the backup path, choose System > Settings > Management Data Backup on the OceanProtect WebUI.

  4. On the local PC, create a folder named after the node ESN and save the management data downloaded in 3 to the folder.
  5. On the local PC, compress the folder named after the node ESN in zip format.
