步骤2：创建文件集

执行文件集保护前请先创建文件集，文件集包括主机下的多个文件或目录。支持为单个主机创建文件集或通过模板批量为多个主机创建文件集。

操作步骤

  1. 选择“保护 > 文件系统 > 文件集”。

    对于1.5.0版本，请选择“保护 > 裸机 > 文件集”。

  2. 单击“创建”。
  3. 选择文件集创建模式：“自定义”或“模板”。

    • 选择文件集创建模式为“自定义”，为单个主机创建文件集。

      相关参数说明如表1所示。

      表1 创建文件集参数说明

      参数

      说明

      名称

      自定义文件集名称。

      选择主机

      选择要进行文件集保护的主机。

      选择文件

      选择要保护的文件。

      最多支持选择64个文件或目录。

      展开数据源后可通过搜索文件名称选择相应文件。

      说明：

      不支持备份ProtectAgent所在目录（例如Windows操作系统“C:\DataBackup”和非Windows操作系统“/opt/DataBackup”）。

      过滤规则

      文件

      选择要包含或排除的文件，支持过滤的文件类型，请参考支持过滤的文件类型。

      过滤文件时，按照如下规则配置：

      • 输入文件的完整路径。
      • 配置多个路径时，单击“保存”后再填写新的路径。
      • 路径中可包含通配符*

      目录

      选择要包含或排除的目录，单击“保存”。

      过滤目录时，按照如下规则配置：

      • 输入目录的完整路径。
      • 配置多个路径时，单击“保存”后再填写新的路径。
      • 路径中可包含通配符*
      设置多个过滤规则时，可以通过以下原则判断最终生效结果：
      • 仅存在包含条件时，多个包含条件之间是“并”的关系，即筛选出来的结果是所选条件的并集。
      • 仅存在排除条件时，多个排除条件之间是“并”的关系，即筛选出来的结果是所选条件的并集。
      • 既有包含又有排除条件时，需在包含条件中判定排除。
    • 选择文件集创建模式为“模板”，批量为多个主机创建文件集。

      通过“关联模板”方式批量创建的文件集，文件集命名规则为：模板名称_主机IP_时间戳后四位，如：fileset_192_168_10_10_8981。

      相关参数说明如表2所示。

      表2 通过模板创建文件集

      参数

      说明

      模板

      选择步骤1：（可选）创建文件集模板中创建的模板。

      主机

      选择要进行文件集保护的主机。

      单次最多支持选择64个主机。

      说明：

      请确保所选主机上存在模板中设置的文件路径，否则将无法备份。

      关联SLA

      为文件集关联SLA。

      • 您可以在此处为要保护的文件集关联SLA，创建文件集后，系统将周期性对文件集执行保护，文件集保护高级参数具体请参考4。
      • 您也可以在后续操作再为其关联SLA。

      创建SLA的具体参数说明可参考步骤5：创建备份SLA。

  4. 单击“确定”。
