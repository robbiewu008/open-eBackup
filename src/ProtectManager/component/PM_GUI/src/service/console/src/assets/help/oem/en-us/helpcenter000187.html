Identifying Tapes

This section describes how to identify tape types. If the tape type is RW, RW is displayed in the Tape Type column. Otherwise, Unknown is displayed.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Tape tab.
  4. In the row where a tape resides, click More > Identify.

    You can also select multiple tapes and click Identify to batch identify tape types.
