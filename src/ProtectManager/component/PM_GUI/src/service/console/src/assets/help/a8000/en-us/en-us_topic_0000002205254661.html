Step 5: Creating a Remote Device Administrator (Applicable to 1.6.0 and Later Versions)

This section describes how to create a remote device administrator in the secondary cluster for replication. This administrator is used to verify the secondary cluster during the replication.

Procedure

  1. Log in to the OceanProtect management page of the secondary cluster for replication.
  2. Choose System > Security > RBAC.
  3. On the Users tab page, click Create.
  4. On the Basic User Information page that is displayed, set basic information by referring to Table 1 and click Next.

    Table 1 Basic information parameters

    Parameter

    Description

    User Type

    Select Local user.

    Name

    Name of a user, which can be customized.

    Login Authentication Mode

    Select Login password.

    Password

    Password of a user.

    Confirm Password

    Confirm password.

    Role

    In the Name column, select Remote Device Administrator.

    Description

    Enter information about a user.

  5. On the Overview page that is displayed, confirm the user information and click Finish.
