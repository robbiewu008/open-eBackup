Installing Mariabackup

Before performing a backup job for MariaDB, you need to manually install Mariabackup on all MariaDB nodes.

Procedure

  1. Log in to the host where the MariaDB instance is located and run the mariabackup --version command to check whether Mariabackup has been installed and environment variables have been set.

    • If the version number is displayed, Mariabackup has been installed and environment variables have been set.
    • If no version number is displayed, install Mariabackup and set environment variables.
      • Installing Mariabackup using a binary package
        1. Find the mariabackup command in the bin folder in the MariaDB installation directory.
        2. Run the ln -s full path of the mariabackup command /usr/bin/ command to set environment variables.
      • Installing Mariabackup using an RPM package
        1. Run the find / -name 'MariaDB-backup*' command to find the RPM package corresponding to Mariabackup.
        2. Run the rpm -ivh RPM package of mariabackup command to install Mariabackup and automatically set environment variables.
