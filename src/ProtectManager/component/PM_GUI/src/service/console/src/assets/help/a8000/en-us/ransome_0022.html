Managing Ransomware Detection Copies

This section describes how to delete ransomware detection copies that are unavailable or no longer needed.

Precautions

  • Before deleting a ransomware detection copy, ensure that the copy is no longer used for restoration.
  • A copy whose ransomware detection status is Detecting cannot be deleted.
  • If the last uninfected copy of a resource is deleted, the accuracy of subsequent ransomware detection for copies will be affected. Exercise caution.

Deleting a Copy (Applicable to 1.5.0)

  1. Choose Explore > Ransomware Protection.
  2. Select VMware/NAS File Systems/NAS Shares/Filesets.
  3. Click the number of copies of a resource.
  4. In the window that is displayed on the right, choose More > Delete.
  5. In the displayed dialog box shown as follows, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.

    When you delete the last uninfected copy of the current resource, the following dialog box is displayed. Confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.

Deleting a Copy (Applicable to 1.6.0 and Later Versions)

  1. Choose Data Security > Ransomware Protection > Detection Overview.
  2. On the Detection Overview page, click Select Resource Type to select the corresponding resource type.

    Resource types include VMware VM, NAS File Systems, NAS Shares, Filesets, CNware VM, Huawei Cloud Stack, FusionCompute VM, OpenStack Cloud Server, Hyper-V VM, and FusionOne Compute VM.

  3. Click the number of copies of a resource.
  4. In the window that is displayed on the right, choose More > Delete.
  5. In the displayed dialog box shown as follows, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.

    When you delete the last uninfected copy of the current resource, the following dialog box is displayed. Confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
