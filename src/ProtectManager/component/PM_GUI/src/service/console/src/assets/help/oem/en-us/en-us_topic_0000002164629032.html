Viewing Reports

Procedure

  1. Select Reports.
  2. Table 1 describes the parameters.

    Table 1 Report parameters

    Parameter

    Description

    Name

    Name of the report.

    Report Status

    Status of the report.

    [Value range]

    Generating, Generation Failed, and Generated

    Report Type

    Type of the report.

    • Storage space report: Collects the total capacity, used capacity, and unused capacity of the product.
    • Backup job report: Collects backup job details, including the job ID, job type, subtype, status, and other job execution information.
    • Restoration job report: Collects restoration job details, including the job ID, job type, status, and other job execution information.
    • Restoration drill job report: Collects restoration drill job details, including the restoration drill plan, job type, status, and other job execution information.
    • Host resource usage report: Collects host resource usage information, such as the host ID, host name, CPU usage, memory usage, and status.
    • Resource usage report: Collects resource usage information, such as the resource ID, resource name, resource type, protection status, and connection status.
    • Tape usage report: Collects tape usage information, such as the tape tag, tape status, write status, used capacity, and product model.
    NOTE:
    • 1.6.0 and later versions support the creation of Host resource usage report, Resource usage report, and Tape usage report.
    • For 1.6.0 and later versions, Backup job report is displayed as Protection job report. Protection job report contains details about backup, archive, and replication jobs.

    Data Source

    Local cluster node or external cluster node that has been connected to the product.

    Statistical Range

    Time range within which reports are collected. If the report type is Storage space report, -- is displayed.

    Created At

    Time when the report is created.
