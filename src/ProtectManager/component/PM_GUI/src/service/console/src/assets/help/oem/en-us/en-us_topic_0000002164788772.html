About the product

The product helps easily configure and manage data protection services.

Operating System and Browser Compatibility

Before using the product, check whether the management terminal meets requirements.

The product is compatible with various operating systems (OSs) and browsers. However, partially compatible browsers may fail to provide ease-of-use experience.

If the display effect of the product is not user-friendly, adjust the zoom ratio of the browser to obtain the optimal display effect.

Table 1 describes the OceanProtect WebUI.

Table 1 WebUI elements of the product

No.

Name

Description

1

Cluster switchover area

The product supports unified management of multiple clusters. In the cluster switchover area, you can select all clusters, a management cluster, or a managed cluster.

  • If you select all clusters, the statistics information of all clusters is displayed. By default, other pages show the local cluster information. You can switch to any management cluster or managed cluster.
  • If you select a management cluster, information about the management cluster is displayed on all pages.
  • If you select a managed cluster, information about the managed cluster is displayed on all pages.

2

Navigation area

Shows functional modules of the product.

3

Global search area

Displays the search scope and content supported by the product.

4

Alarm and job statistics area

The alarm statistics area displays the number of alarms by severity and helps you learn about the running status of the system.

The job statistics area displays all executed jobs and helps you learn whether the jobs are executed successfully.

5

Language selection area

A window is displayed for you to set the required operation language (Chinese or English).

6

Logout area

The logout area provides the logout and appearance buttons as well as the button for changing the password.

Available appearance modes are Light, Dark, and Use the OS appearance.

NOTE:

Only 1.6.0 and later versions support appearance mode setting.

7

Help and technical support

Provides links to the online help and technical support websites.

8

Function pane

Displays the window for the current operation.
