查看华为云Stack副本信息

本节介绍华为云Stack副本的详细信息。

操作步骤

  1. 选择“数据利用 > 副本数据 > 云平台 > 华为云Stack”。
  2. 浏览副本信息。

    复制副本仅支持在复制到的目标集群查看。

    • 选择“资源”页签：以云服务器维度查看资源和副本信息。当云服务器故障，您需要恢复该云服务器时，建议从该入口进入查找相应时间点副本。

      相关参数说明如表1所示。

      表1 副本参数说明

      参数

      说明

      资源

      资源ID

      资源的ID。

      名称

      云服务器名称。

      单击名称，可以查看云服务器的副本信息。

      您可以依次按照年、月、天查找相应时间的副本。

      时间下方显示，表示该时间存在副本。

      位置

      云服务器所在位置。位置格式：华为云Stack平台/租户/区域/项目（资源集）/云服务器

      状态

      云服务器是否注册在当前管理界面。

      • 是，则状态为存在。
      • 否，则状态为不存在。

      副本

      数量

      基于该云服务器产生的所有副本数量。其中，复制副本数量仅在目标集群显示。

      复制副本SLA

      复制副本关联的归档SLA。

      如果已为该资源产生的复制副本配置归档SLA，则此处显示SLA名称。

      复制副本保护状态
      复制副本当前是否被保护。
      • 未保护：复制副本未关联SLA，或者复制副本已关联SLA但已禁用保护。
      • 已保护：复制副本已关联SLA，且已激活保护。
      • 创建中：复制副本已关联SLA，正在创建保护。仅1.5.0及之前版本存在该状态。
    • 选择“副本”页签：以副本维度查看所有副本信息。当需要基于某个指定的副本进行原位置或新位置恢复时，建议从该入口进入查找相应副本。

      相关参数说明如表2所示。

      表2 副本参数说明

      参数

      说明

      副本

      备份时间

      备份副本的生成时间。

      副本时间

      副本生成的时间。

      单击“副本时间”，可以查看副本过期时间等信息，过期时间显示“--”表示副本永久保留。

      状态
      副本的状态。
      • 正常：副本可用。
      • 无效：副本无效。
      • 删除中：副本处于正在删除过程中。
      • 恢复中：副本处于恢复过程中。
      • 校验中：副本处于完整性校验中。
      • 删除失败：副本删除失败。

      位置

      副本所在的位置。

      • 对于1.5.0版本，当副本的“生成方式”为“备份”、“复制”、“级联复制”、“反向复制”时，副本存储在本地集群，此处显示为“Local”。
      • 对于1.6.0及后续版本，当副本的“生成方式”为“备份”、“复制”、“级联复制”、“反向复制”时，副本存储在对应的备份存储单元，此处显示为对应备份存储单元的名称。
      • 当副本的“生成方式”为“对象存储归档”或“导入”时，副本存储在对象存储的桶上，此处显示为桶名称。
      • 当副本的“生成方式”为“磁带归档”时，副本存储在磁带库，此处显示介质集名称。

      生成方式

      副本的生成方式。

      • 复制：由复制任务产生的副本。
      • 备份：由备份任务产生的副本。
      • 对象存储归档：由对象存储归档任务产生的副本。
      • 磁带归档：由磁带归档任务产生的副本。
      • 级联复制：由级联复制任务产生的副本。
      • 反向复制：由反向复制任务产生的副本。

      副本类型

      生成副本的备份类型。包括：全量、永久增量（合成全量）。
      过期时间
      说明：

      仅1.5.0SPC26及后续版本存在此参数。

      副本的过期时间。如果副本为永久有效，则显示为“--”。

      WORM
      副本是否已设置为WORM副本。
      • ：未设置。副本未设置为WORM副本，即副本为普通副本。
      • ：已设置。副本已设置为WORM副本，WORM副本不支持删除和缩短保留时间。
      • ：设置中。副本正在设置为WORM副本。处于该状态的副本不支持删除和修改保留时间。
      • ：设置失败。副本设置为WORM副本失败。设置失败的副本不支持删除和缩短保留时间。
      • ：已过期。仅1.6.0及后续版本存在此状态，WORM副本已过有效期。处于该状态的副本支持删除。
      WORM过期时间
      说明：

      仅1.6.0及后续版本存在此参数。
      当副本为WORM副本时，则显示WORM过期的具体时间。否则，显示为“--”。
      说明：

      WORM过期时间以WORM文件系统法规时钟为基准。如果WORM文件系统法规时钟与系统时间不一致，请等待WORM文件系统法规时钟到达当前设置的过期时间后，再查看WORM状态。

      副本校验状态
      副本的校验状态。
      • --：表示当前副本未校验。
      • 可用：表示当前副本已经校验且数据正确。
      • 无效：表示当前副本校验任务失败或者校验出副本数据不一致。

      副本

      索引
      当开启了自动索引或手动创建了索引，系统将为副本中的所有文件建立索引，便于快速查找相应文件进行恢复。包括以下索引状态：
      • 已索引：系统已为副本建立索引。
      • 索引中：副本生成后，系统正在为副本建立索引。
      • 未索引：未开启自动索引或未手动创建索引。
      • 索引删除中：正在删除索引。
      • 索引失败：建立索引失败。

      资源

      名称

      云服务器名称。

      位置

      云服务器所在位置。

      状态

      云服务器是否注册在当前管理界面。

      • 是，则状态为存在。
      • 否，则状态为不存在。
      WORM过期时间
      说明：

      仅1.6.0及后续版本存在此参数。
      当副本为WORM副本时，则显示WORM过期的具体时间。否则，显示为“--”。
      说明：

      WORM过期时间以WORM文件系统法规时钟为基准。如果WORM文件系统法规时钟与系统时间不一致，请等待WORM文件系统法规时钟到达当前设置的过期时间后，再查看WORM状态。
