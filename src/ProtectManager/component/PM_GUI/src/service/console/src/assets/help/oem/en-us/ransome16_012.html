Viewing Detection Details of All Resources

This section describes how to view the detection details of all resources.

Procedure

  1. Choose Data Security > Ransomware Protection > Detection Overview.
  2. On the Detection Overview page, view the number of copies in different detection states, copy detection result, and detection details of all resources.

    • Checking the number of copies in different detection states

    • Viewing the copy detection result

      You can view the detection results of the last week, last month, or last half year.

    • Viewing detection details of all resources

      You can view the resource information, including Name, Location, Associated Policy, Copies, Preparing, Detecting, Uninfected, Infected, Exception, and Undetected.
