邮件级恢复

当需要对已备份的邮箱进行邮件级恢复时，可以参考本节使用邮箱备份副本恢复邮件至原邮箱。

前提条件

  • 恢复前，请确保原邮箱存在，且邮箱所在数据库已挂载。
  • 执行恢复前，请确保Exchange Server服务和相关的依赖服务正常运行。对应服务可参考Exchange 服务器上的 Exchange 服务概述。
  • 存在可用的邮箱备份副本。

注意事项

邮件恢复前，删除了文件夹及文件夹下的邮件后，需要同时在Exchange邮箱中的“已删除邮件”中删除对应的文件夹和邮件，否则会导致恢复数据不完整。

操作步骤

  1. 选择“数据利用 > 副本数据 > 应用 > Exchange”。
  2. 支持以邮箱资源维度或以副本维度查找副本，本节以邮箱资源维度为例进行说明。

    在“资源”页签，根据邮箱名称查找待恢复邮件所在的邮箱，并单击邮箱名称。

  3. 在“副本数据”页签，依次选择年、月、天找到副本。

    当时间上显示，即表示该月、该天存在副本。

  4. 在Exchange邮箱类型的副本所在行，单击“更多 > 邮件级恢复”。
  5. 选择覆盖规则，替换或忽略已存在的邮件。
  6. 选择需要恢复的邮件。可选择单个或多个邮件进行恢复。
  7. 单击“确定”。

后续操作

恢复后，需要在生产端更新用户邮箱状态，并创建新的Active Directory域用户与恢复后的邮箱建立连接，以验证恢复后的用户邮箱数据。具体操作请参考验证恢复后的用户邮箱数据（适用于Microsoft Exchange Server 2010）或验证恢复后的用户邮箱数据（适用于Microsoft Exchange Server 2013及后续版本）。
