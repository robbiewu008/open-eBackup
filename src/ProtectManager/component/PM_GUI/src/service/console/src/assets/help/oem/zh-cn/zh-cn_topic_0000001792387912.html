步骤1：检查并安装VMware Tools

执行应用一致性备份时，需要在受保护虚拟机或目标虚拟机安装VMware Tools。

背景信息

当存在以下场景时需要为VMware虚拟机安装VMware Tools：

  • 如果需要执行应用一致性备份，那么要为受保护的VMware虚拟机安装VMware Tools。

操作步骤

  1. 使用管理员账号登录vCenter Server或ESXi主机。
  2. 展开左侧导航栏，查看所有已创建的虚拟机。
  3. 在虚拟机列表中，找到需要安装VMware Tools的虚拟机，单击该虚拟机的名称。
  4. 在弹出的虚拟机详情页面，查看VMware Tools是否已安装。如果未安装，请参考安装VMware Tools进行安装。

    • 如果是vCenter Server，请在客户机操作系统区域中查看。
    • 如果是ESXi主机，请在一般信息区域中查看。
