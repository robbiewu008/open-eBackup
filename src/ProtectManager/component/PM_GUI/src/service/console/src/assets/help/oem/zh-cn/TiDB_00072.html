在生产环境开启或关闭日志备份开关

操作步骤

  1. 登录TiUP节点。
  2. 执行以下命令开启或关闭日志备份开关（“172.0.0.0:80”为PD节点的IP和端口，“local:///test/bkpbinlog0000/”为存放日志备份文件的目录绝对路径，用户请根据实际情况输入）。

    • 请确保在TiUP、TiKV节点上（如果TiDB集群下存在TiFlash节点，也包括TiFlash节点），TiDB集群具有存放日志备份文件的目录路径的读写权限。
    • 打开日志备份开关前，需确保存放日志备份文件的目录在TiUP、TiKV（如果TiDB集群下存在TiFlash节点，TiFlash节点的此目录绝对路径也需要一致）节点的绝对路径一致且该目录为空。
    • 开启日志备份开关。
      tiup br log start --task-name=pitr --pd 172.0.0.0:80 --storage "local:///test/bkpbinlog0000/"
    • 关闭日志备份开关。
      tiup br log stop --task-name=pitr --pd 172.0.0.0:80

  3. 执行以下命令查询日志备份开关当前状态（172.0.0.0:80为PD节点的IP和端口）。

    tiup br log status --task-name=pitr --pd 172.0.0.0:80
