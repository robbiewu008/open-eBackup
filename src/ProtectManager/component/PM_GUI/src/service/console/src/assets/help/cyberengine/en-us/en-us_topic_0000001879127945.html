Managing the Password Retrieval Email Address

This section describes how to set the password retrieval email address. If the system administrator password is lost, an email containing the verification code can be sent by the OceanCyber 300 Data Security Appliance to the preset email address.

Precautions

This operation requires personal data such as the email address. You must take adequate measures under the governing laws and privacy policies to ensure that user data is fully protected.

Procedure

Setting the sender information

  1. Choose System > Settings > Mail Service.
  2. In the Sender Settings area, click Modify and set sender parameters. Table 1 describes related parameters.

    Table 1 Sender parameters

    Parameter

    Description

    SMTP Server

    IP address or domain name of the SMTP server for sending notification emails containing verification codes.

    The IP address of the SMTP server can be an IPv4 or IPv6 address.

    SMTP Port

    SMTP server port number.

    The value is an integer from 1 to 65535.

    Sender Email Address

    Email address for sending notification emails containing verification codes.

    Testing Email Address

    Email address used to test the communication between the email sender server and the email server.

    Encryption Mode

    Encryption mode used by the email server to communicate with the system. The possible values include Not encrypted, SSL/TLS, and STARTTLS.

    • Not encrypted: Data is not encrypted during transmission. If data transfer is not encrypted, there are security risks. You are advised to select SSL/TLS or STARTTLS to improve data transfer security.
    • SSL/TLS: SSL and TLS are different security protocols used for ensuring data security and integrity during network communication. After the two protocols are enabled, the system automatically selects the encryption mode according to the email server. When selecting SSL/TLS, you need to set the SSL port number to the port number corresponding to the SMTP encryption mode.
    • STARTTLS: If TLS encryption is implemented after STARTTLS is executed, the communication data before STARTTLS is executed will not be encrypted. If STARTTLS is selected, change the SMTP port number to the port number corresponding to the SMTP service encryption mode.

    The encryption mode configured on the system must be the same as that configured on the SMTP server.

    NOTE:

    If the encryption mode is set to SSL/TLS or STARTTLS, you need to import the CA certificate of the Email type in System > Security > Certificates. Otherwise, alarm notifications fail to be sent. For details about how to import a certificate, see Importing a Certificate in the Huawei OceanCyber 300 1.2.0 Security Configuration Guide.

    SSL Port

    SMTP server's SSL port number.

    This parameter is available only if Encryption Mode is set to SSL/TLS.

    SMTP Server Authentication

    Indicates whether the SMTP server authenticates a sender's identity.

    Username

    Username for accessing the SMTP server.

    This parameter is available only if SMTP Server Authentication is enabled.

    Password

    Password for accessing the SMTP server.

    This parameter is available only if SMTP Server Authentication is enabled.

    Proxy Server

    A proxy server needs to be configured if the system cannot communicate with the SMTP server.

    IP Address

    IP address of the proxy server.

    This parameter is available only if Proxy Server is enabled.

    Port

    Port ID used by the proxy server.

    This parameter is available only if Proxy Server is enabled.

    The value is an integer from 1 to 65535.

  3. After the parameters are specified, you can click Test to check whether the email sender server and SMTP server can properly communicate with each other.
  4. Click Save.
