Step 1: Checking and Installing VMware Tools

Before performing application-consistent backup, you must install VMware Tools on the protected VM or target VM.

Context

You need to install VMware Tools for VMware VMs in the following scenarios:

  • If you need to perform application-consistent backup, ensure that VMware Tools has been installed on the protected VMware VMs.

Procedure

  1. Log in to the vCenter Server or ESXi host as an administrator.
  2. Expand the navigation tree on the left to view all created VMs.
  3. In the VM list, locate the VM where VMware Tools needs to be installed and click its name.
  4. On the VM details page that is displayed, check whether VMware Tools has been installed. If no, install VMware Tools by referring to Installing VMware Tools.

    • If the vCenter Server is used, check the installation status in the Guest OS area.
    • If the ESXi host is used, check the installation status in the General Information area.
