Specifying an External Cluster as a Managed Cluster

This section describes how to specify a cluster as a managed cluster. After a cluster is specified as a managed cluster, you can only view information about the local cluster after login.

Prerequisites

The external cluster has been set as a management cluster.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  1. In the multi-domain cluster area, choose More > Set as Managed Cluster in the row that contains the desired external cluster.
  2. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
