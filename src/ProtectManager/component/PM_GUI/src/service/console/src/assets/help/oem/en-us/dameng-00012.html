Step 1: Enabling the DmAPService

Before performing backup and restoration jobs for Dameng, DmAPService must be enabled. Otherwise, backup and restoration operations cannot be performed.

Precautions

  • Unless otherwise specified, the operations in this section use Dameng 8 as an example. The operations may vary according to the database version.
  • If the Dameng database is deployed in a cluster, you need to log in to all hosts where the cluster instances reside and perform the following operations unless otherwise specified.

Procedure

  1. Log in to the Dameng database host.
  2. Run the following command to check whether the DmAPService service is enabled:

    systemctl status DmAPService
    • If the value of Active is active(running), the DmAPService service has been enabled.
    • If the value of Active is inactive(dead), the DmAPService service is disabled. Run the systemctl start DmAPService command to enable the DmAPService service.
