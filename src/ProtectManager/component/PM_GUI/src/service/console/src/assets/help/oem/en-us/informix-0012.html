Step 2: Registering an Informix/GBase 8s Cluster

Prerequisites

Before registering an Informix/GBase 8s cluster, ensure that each host in the cluster has been registered.

Procedure

  1. Choose Protection > Databases > Informix/GBase 8s.
  2. Click the Clusters tab.
  3. Click Register.
  4. On the Register page, customize the cluster name and select a host.
  5. (Optional) Enable log backup and enter the log backup path.

    • After the log backup function is enabled, the system can restore the database based on a specified time point.
    • Ensure that the entered path exists and the user has read/write permissions on the Informix/GBase 8s database.
    • The log path cannot be a built-in directory of the OS, such as /etc, /bin, /usr, or /dev.
    • Informix: It is recommended that you use the built-in path /home/informix/log as the log backup path.
    • GBase 8s: It is recommended that you use the built-in path /home/gbasedbt/log as the log backup path.

  6. Click OK.
