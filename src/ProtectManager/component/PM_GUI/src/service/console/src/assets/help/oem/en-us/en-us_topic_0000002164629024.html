Deleting a Role

The product allows you to delete existing role information.

Precautions

Built-in roles and roles associated with users cannot be deleted.

Procedure

  1. Choose System > Security > RBAC > Roles.
  2. (Optional) Click next to Name to search for a role.
  3. In the row of the target role, choose More > Delete.

    You can delete roles in batches. Specifically, select the target roles and click Delete.

  4. In the dialog box that is displayed, select I have read and understand the consequences associated with performing this operation and click OK.
