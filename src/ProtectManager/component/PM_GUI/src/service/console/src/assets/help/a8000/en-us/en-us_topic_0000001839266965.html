What Can I Do to Add User Permissions on vCenter?

VMware user permissions must be added on vCenter. Otherwise, VDDK initialization will fail. The following part uses VMware 7.0 as an example. The operations for other versions are similar.

  1. Access the address for logging in to the vCenter Server using a browser.

    Login address: https://vCenter Server IP address

  1. Right-click the target vCenter and select Add Permission from the shortcut menu.
  2. Specify Domain, User/Group, and Rule, and select Propagate to children.
  3. Click OK.
