Overview

This section describes how to view ransomware detection information.

Procedure

  1. Choose Explore > Ransomware Protection > Overview.
  2. The following information is displayed on the overview page:

    • File Interception area in the upper part: Number of vStores for which ransomware file interception is enabled or disabled.
    • Real-time Ransomware Detection area in the upper part: Number of vStores for which real-time ransomware detection is enabled or disabled.
    • Intelligent Ransomware Detection area in the upper part: Number of vStores for which intelligent ransomware detection is enabled or disabled.
    • Real-time Ransomware Detection Alarms area in the lower part: Real-time ransomware detection alarm information of the vStore.
    • Intelligent Ransomware Detection area: Whether intelligent ransomware detection is configured for the vStore's file systems.
