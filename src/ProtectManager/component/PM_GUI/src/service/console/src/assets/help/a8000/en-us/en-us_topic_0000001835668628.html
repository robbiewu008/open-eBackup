Creating a Backup Storage Device

The external backup storage is added as a backup storage device to the OceanProtect for storing backup data.

Prerequisites

The OceanProtect and the backup storage device to be added are running properly and can communicate with each other.

Precautions

  • Only the local storage corresponding to the OceanProtect can be added as a backup storage device.
  • The backup storage device cannot be used as the managed cluster at the same time.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. Select Backup Storage Devices.
  4. Click Create to create a backup storage device.

    Table 1 describes the related parameters.
    Table 1 Parameters for adding a backup storage device

    Parameter

    Description

    Name

    User-defined name of the backup storage device.

    Device Type

    Select OceanProtect at present.

    IP Address

    Management IP address of the external backup storage.

    Port

    Management port of the external backup storage.

    The default port ID is 8088.

    Username

    Username and password of the super administrator of the external backup storage.

    Password

  5. Click OK.
