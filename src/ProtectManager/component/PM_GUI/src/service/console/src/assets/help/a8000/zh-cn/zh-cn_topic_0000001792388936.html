步骤3：注册GaussDB T数据库

当GaussDB T数据库所在主机安装ProtectAgent后，OceanProtect会自动发现GaussDB T数据库主机，您需要注册主机上的数据库实例，以对其进行保护或将其作为恢复目标。

操作步骤

  1. 选择“保护 > 数据库 > GaussDB T”。
  2. 单击“注册”，注册新的GaussDB T。
  3. 配置数据库信息。

    在“类型”中选择“单机”或“集群”。

    单机类型相关参数说明如表1所示，集群类型相关参数说明如表2所示。

    表1 GaussDB T单机类型参数信息

    参数

    说明

    名称

    要注册的GaussDB T数据库名称。

    用户名

    运行GaussDB T单机的操作系统用户名。

    部署形态

    数据库的部署形态，单机类型只能选择“单机部署”。

    节点

    选择要注册的节点。只能选择单个节点。

    认证方式

    OS认证/数据库认证。

    系统数据库密码

    数据库的系统用户密码。

    当“认证方式”选择“数据库认证”时，才需要配置该参数。
    表2 GaussDB T集群类型参数信息

    参数

    说明

    名称

    要注册的GaussDB T数据库名称。

    用户名

    运行GaussDB T集群的操作系统用户名。

    部署形态

    单机部署/主备部署。

    节点

    选择要注册的节点。

  4. 单击“确定”。
