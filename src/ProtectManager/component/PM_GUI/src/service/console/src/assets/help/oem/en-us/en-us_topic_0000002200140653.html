VMware Resource Links Are Not Restored After the Management Port Fault Recovery

Symptom

VMware resource links are not restored after the management port fault recovery. The "Failed to get virtual machine information" error message is displayed.

Possible Causes

The system does not automatically restore VMware resource links after the management port fault recovery.

Troubleshooting

  1. Choose Protect > Virtualization > VMware.
  2. On the left of the VMware page, select the vCenter Server or ESXi host to be operated.
  3. Choose More > Resource Scan.

    After the resource scanning is successful, the VMware links are restored.
