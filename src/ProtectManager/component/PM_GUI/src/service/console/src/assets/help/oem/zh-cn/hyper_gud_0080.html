查看Hyper-V副本信息

本节介绍Hyper-V副本的详细信息。

操作步骤

  1. 选择“数据利用 > 副本数据 > 虚拟化 > Hyper-V”。
  2. 浏览副本信息。

    • 复制副本仅支持在复制到的目标集群查看。
    • 部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。
    • 选择“资源”页签：以虚拟机维度查看虚拟机的副本信息。当虚拟机故障，您需要恢复该虚拟机时，建议从该入口进入查找相应时间点副本。

      相关参数说明如表1所示。

      表1 副本参数说明

      参数

      说明

      资源

      名称

      虚拟机名称。

      位置

      虚拟机所在计算位置。

      状态

      虚拟机是否注册在本产品。

      • 是，则状态为存在。
      • 否，则状态为不存在。

      副本

      数量

      基于该虚拟机产生的所有副本数量。其中，复制副本数量仅在目标集群显示。

      复制副本SLA

      复制副本关联的SLA。

      如果已为该资源产生的复制副本配置SLA，则此处显示SLA名称。

      复制副本保护状态
      复制副本当前是否被保护。
      • 未保护：复制副本未关联SLA，或者复制副本已关联SLA但已禁用保护。
      • 已保护：复制副本已关联SLA，且已激活保护。
    • 选择“副本”页签：以副本维度查看所有副本信息。当需要基于某个指定的副本进行原机或异机恢复时，建议从该入口进入查找相应副本。

      相关参数说明如表2所示。

      表2 副本参数说明

      参数

      说明

      副本

      备份副本生成时间

      备份副本的生成时间。

      状态
      副本的状态。
      • 正常：副本可用。
      • 无效：副本无效。
      • 删除中：副本处于正在删除过程中。
      • 恢复中：副本处于恢复或即时恢复过程中。
      • 挂载中：副本处于正在挂载过程中。
      • 卸载中：副本处于正在销毁过程中。
      • 已挂载：挂载完成后生成的副本状态。
      • 删除失败：副本删除失败。

      位置

      副本所在的位置。

      • 当副本的“生成方式”为“备份”、“复制”、“级联复制”、“反向复制”时，副本存储在对应的备份存储单元，此处显示为对应备份存储单元的名称。
      • 当副本的“生成方式”为“磁带归档”时，副本存储在磁带库，此处显示介质集名称。

      存储介质状态
      副本所在的存储介质的状态。
      • 在线：则支持对该副本进行管理。
      • 离线：则仅支持对副本进行删除。
      说明：

      当备份存储单元的“健康状态”为“正常”或“降级”，且“运行状态”为“预拷贝”、“重构”、“在线”、“正在均衡”或“降级”时，存储介质状态均显示为“在线”。

      生成方式
      副本的生成方式。
      • 复制：由复制任务产生的副本。
      • 备份：由备份任务产生的副本。
      • 对象存储归档：由对象存储归档任务产生的副本。
      • 磁带归档：由磁带归档任务产生的副本。
      • 级联复制：由级联复制任务产生的副本。
      • 反向复制：由反向复制任务产生的副本。

      副本生成时间

      副本生成的时间。单击副本生成时间，可以查看副本的详细信息。

      副本类型

      生成副本的备份类型。包括：全量、永久增量（合成全量）。

      副本过期时间
      副本的过期时间。如果副本为永久有效，则显示为“--”。
      说明：

      副本实际过期时间以“副本过期时间”与“WORM过期时间”中的最晚时间为准。

      WORM
      副本是否已设置为WORM副本。
      • ：未设置。副本未设置为WORM副本，即副本为普通副本。
      • ：已设置。副本已设置为WORM副本，WORM副本不支持删除和缩短保留时间。
      • ：设置中。副本正在设置为WORM副本。处于该状态的副本不支持删除和修改保留时间。
      • ：设置失败。副本设置为WORM副本失败。设置失败的副本不支持删除和缩短保留时间。
      • ：已过期。WORM副本已过有效期。处于该状态的副本支持删除。

      WORM过期时间
      当副本为WORM副本时，则显示WORM过期的具体时间。否则，显示为“--”。
      说明：

      当WORM过期时间早于WORM文件系统法规时钟时，即为过了WORM有效期。WORM文件系统法规时钟可单击副本生成时间查看。如果WORM文件系统法规时钟与系统时间不一致，请等待WORM文件系统法规时钟到达当前设置的过期时间后，再查看WORM状态。

      副本

      文件级挂载状态
      副本的文件级挂载状态。
      • 已挂载：已成功创建文件级挂载点。
      • 未挂载：未创建文件级挂载点。
      • 挂载中：正在创建文件级挂载点。
      • 挂载失败：创建文件级挂载点失败。
      • 卸载中：正在删除文件级挂载点。
      • 卸载失败：删除文件级挂载点失败。

      副本

      索引
      当SLA中开启了自动索引或手动创建了索引，系统将为副本中的所有文件建立索引，便于快速查找相应文件进行恢复。包括以下索引状态：
      • 已索引：系统已为副本建立索引。
      • 索引中：副本生成后，系统正在为副本建立索引。
      • 未索引：SLA中未开启自动索引或未手动创建索引。
      • 索引删除中：正在删除索引。
      • 索引失败：建立索引失败。

      资源

      名称

      虚拟机名称。

      位置

      虚拟机所在计算位置。

      状态

      虚拟机是否注册在本产品。

      • 是，则状态为存在。
      • 否，则状态为不存在。

      标签

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
