Restoring Active Directory Objects in the Scenario with Primary and Secondary Domain Controllers

This section describes how to perform object-level restoration for Active Directory to restore Active Directory objects (including user accounts, computer accounts, and more) to the original location in the scenario with primary and secondary domain controllers.

Backup copies, replication copies (only reverse replication copies can be used for restoration to the original location), and archive copies can be used for restoration. Data can be restored to the original location.

Prerequisites

  • Before restoration, ensure that the remaining space of the data directory at the target location for restoration is greater than the size of the copy used for restoration before reduction. Otherwise, restoration will fail.
  • Before restoration, ensure that Backup Data is System state + Object for the copy to be used for restoration.
  • Active Directory object-level restoration applies to scenarios where objects, including users, user groups, and organizational units (OUs), or their attributes are deleted by mistake.
  • Before restoration, ensure that the Active Directory Recycle Bin has been enabled.

Precautions

After the restoration, data cannot be synchronized from the primary domain controller to the secondary domain controller immediately due to the time difference in data replication between the primary and secondary domain controllers.

Procedure

  1. Choose Protection > Applications > Active Directory.
  2. You can search for copies by Active Directory resource or copy. This section describes how to search for copies by resource.

    Locate the resource based on the name of the resource where the object to be restored resides and click the resource name.

  3. Click the Copy Data tab.
  4. Select the year, month, and day in sequence to find the copy.

    If is displayed below a month or date, copies exist in the month or on the day.

  5. In the row of the copy to be used for restoration, choose More > Object-Level Restoration to configure restoration parameters.

    Table 1 describes the related parameters.
    Table 1 Parameters for restoring an Active Directory object

    Parameter

    Description

    Restore To

    Select Original location.

    Location

    By default, the restoration location is displayed.

    In the Select Object area, you can select the desired object directly or select it by entering the object name in the search box.

  6. Click OK.
