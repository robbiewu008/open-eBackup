停止任务

本产品支持停止备份、复制和归档三种类型的任务。

操作步骤

  1. 选择“任务”。

    对于1.5.0版本，请选择“监控 > 任务”“任务”。

  2. 单击“执行中任务”页签，进入任务列表。
  3. 可选：在任务列表右上方选择任务的开始时间和SLA策略名称，对任务进行筛选。或单击“任务类型”、“对象类型”、“任务状态”旁的，对任务进行筛选。或单击“对象”、“任务ID”旁的，对任务进行搜索。或单击“开始时间”旁的，对任务进行排序。
  4. 勾选一个或多个任务，单击“停止”。
  5. 在弹出的对话框中，单击“确定”。

    正常停止任务后，任务状态应变更为“已停止”，且任务详情显示任务停止成功。实际可能由于任务时序，出现如下例外情况：

    • 停止任务后，任务状态变更为“已停止”，但是任务详情显示任务执行成功，可能是您执行停止任务时，副本已生成，实际任务执行成功。
    • 停止任务后，任务状态变更为“失败”，可能是您执行停止任务时，副本数据已备份（复制）完成，或者您执行停止任务时，任务已经失败，导致任务终止失败。
