Deleting an Internal Communication Network

This section describes how to delete the internal communication network of an active node.

Precautions

  • After a member node is added, the internal communication network cannot be deleted.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Cluster.
  3. In the row that contains the target active node, choose More > Delete Internal Communication Network.
  4. In the dialog box that is displayed, confirm the information and click OK.
