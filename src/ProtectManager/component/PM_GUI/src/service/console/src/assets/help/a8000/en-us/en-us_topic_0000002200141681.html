Step 1: Creating a Volume

Create a volume before performing volume protection.

Procedure

  1. Choose Protection > File Systems > Volumes.
  2. Click Create.
  3. In the Create dialog box that is displayed, create a volume for a single host.

    Table 1 describes the related parameters.

    Table 1 Parameters for creating a volume

    Parameter

    Description

    Name

    User-defined volume name.

    Client

    Select the client that you want to use for volume protection.

    OS Backup

    After this option is selected, you can enable or disable OS backup when creating or modifying volume protection. If you want to back up data of all system volumes, you must select all system volumes.

    Backup Type

    NOTE:

    This parameter is available only for the Windows OS.

    Select a backup type: Volume or Bare Metal.

    NOTE:

    Currently, only NTFS volumes can be backed up.

    Volume

    Select the volumes to be protected.

  4. Click OK.
