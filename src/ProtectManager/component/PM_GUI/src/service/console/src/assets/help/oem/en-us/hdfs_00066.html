Managing HDFS Clusters

This section describes how to modify or delete HDFS clusters registered with the product.

Related Operations

Log in to the management page, choose Protection > Big Data > HDFS, click the Cluster tab, and locate the desired cluster.

Table 1 describes the related operations.

Table 1 Cluster-related operations

Operation

Description

Navigation Path

Modifying cluster information

Scenario

This operation enables you to change the authentication mode, name, and configuration file of an HDFS cluster.

In the row where the cluster resides, choose More > Modify.

For details about HDFS cluster parameters, see Step 3: Registering an HDFS Cluster.

Deleting a cluster

Scenario

This operation enables you to delete an HDFS cluster if you no longer need to protect files in it or restore data to it.

Note

  • A cluster cannot be deleted if a backup or restoration job is being executed.
  • A cluster cannot be deleted if its filesets have been associated with an SLA.
  • Deletion of a cluster will delete all its filesets.

In the row where the cluster resides, choose More > Delete.
Adding a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to quickly filter and manage resources.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.
Removing a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to remove a tag of a resource when it is no longer needed.

Choose More > Remove Tag.
