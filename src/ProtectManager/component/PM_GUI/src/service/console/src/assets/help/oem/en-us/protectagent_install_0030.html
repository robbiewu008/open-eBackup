Downloading the ProtectAgent Software Package

If you want to manually install ProtectAgent on the agent host, first download the ProtectAgent software package by referring to this section.

Procedure

  1. Choose Protection > Clients > Clients.
  2. Click ProtectAgent Package Management and set related parameters.

    Table 1 describes the parameters.

    Table 1 Downloading the ProtectAgent software package

    Parameter

    Description

    Mode

    Choose Download.

    Agent Type
    Select an agent type based on the type of the protected resource.
    • VMware backup agent: Select this type when the protected resource is a VMware VM.
    • General backup agent: Select this type for protected resources other than VMware.
    • SAN Client agent: Select this type if the protected resource is DB2 and the AIX or Solaris host uses a Fibre Channel network.

    OS

    OS type of the agent host.
    Download Package Type
    NOTE:

    This parameter is available only in 1.6.0 and later versions.

    When OS is set to Linux, select the type of the installation package to be downloaded. The supported types are zip and tar.

    Application Type

    If Agent Type is set to General Backup Agent, select an application type that can be protected by the agent host.

    Private Key Password

    User-defined private key password. Enter a password according to the complexity requirements prompted.

    The private key password is used to encrypt the private key file in the ProtectAgent software package. The private key file is used for authentication between the agent host and product. Keep the private key password secure for future use during installation.

    NOTE:
    • If OS is set to AIX, HP-UX, or Solaris, the entered private key password cannot contain special characters ^`{_|}~. Otherwise, the installation will fail when the software package is used to install ProtectAgent.
    • After you enter the private key password, the product uses the password to encrypt the private key file in the ProtectAgent software package.
    • If you have imported the agent certificate by referring to the Security Configuration Guide before downloading the ProtectAgent software package, you can enter the private key password that is the same as that of the agent certificate or customize the private key password.

    Confirm Private Key Password

    Enter the private key password again.

  3. Click OK.

    Obtain the downloaded ProtectAgent software package.

    If you fail to download the ProtectAgent package using Internet Explorer 11 on the Windows Server, refer to What If I Failed to Download ProtectAgent Using Internet Explorer 11 on the Windows Server?.
