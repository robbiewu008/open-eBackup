Step 4: Creating an HDFS Fileset

Create a fileset to be protected in the HDFS cluster. A fileset is the collection of HDFS files and directories.

Procedure

  1. Choose Protection > Big Data > HDFS.
  2. On the Filesets tab page, click Create to create a fileset.

    Table 1 describes the related parameters.
    Table 1 Fileset parameters

    Parameter

    Description

    Name

    Customize a fileset name.

    Cluster

    Select the HDFS cluster to be protected.

    File to Select

    Choose the file or directory to be protected.

    Filter Rule

    File

    Filter the files to be included or excluded.

    The rules for filtering files are as follows:

    • Enter the full path of the file, for example, /opt/a.txt.
    • When multiple paths are to be configured, click Save and enter a new path.
    • The path can contain the wildcard character *.

    Directory

    Filter the directories to be included or excluded.

    The rules for filtering directories are as follows:

    • Enter the full path of the directory, for example, /opt/b.
    • When multiple paths are to be configured, click Save and enter a new path.
    • The path can contain the wildcard character *.

  3. Click OK.
