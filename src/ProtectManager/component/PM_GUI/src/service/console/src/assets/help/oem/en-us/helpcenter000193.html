Modifying a Media Set

This section describes how to modify a media set. The basic information and owning controller of a media set cannot be modified or changed.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Media Set.
  3. In the row where the media set to be modified resides, choose More > Modify.

    When you select tapes, the available tape libraries include the tape libraries of the original tapes in the media set and the current online tape libraries. The restrictions for selecting tapes are as follows:
