Step 2: Registering the CNware Virtualization Environment

Before performing CNware backup and restoration, you must register the CNware virtualization environment with the OceanProtect.

Context

After registering the CNware virtualization environment, you can view information about CNware VMs on the management page and protect or restore them.

Prerequisites

  • A storage pool has been added to the CNware cloud platform to be protected, and the protected target host and agent host use the same storage pool.

Procedure

  1. Choose Protection > Virtualization > CNware.
  2. Click Register, and register the CNware virtualization platform.

    Table 1 lists the CNware registration information.
    Table 1 CNware registration information

    Parameter

    Description

    Name

    User-defined name of the virtualization platform.

    IP Address/Domain Name

    Management IP address or domain name of the CNware platform.

    Port

    Port number used by the OceanProtect to communicate with the CNware platform.

    The default port number is 443.

    Username

    Username and password of the CNware platform to be registered.

    Password

    Agent Host

    Data protection agent registered with the OceanProtect.

    Resource Scanning Interval

    Interval for the system to automatically scan resources.

    Certificate Verification

    This function is enabled by default. When you register a CNware protected environment, the system verifies the information about the accessed devices. After this function is disabled, security risks may exist in the communication between the system and CNware. Exercise caution when performing this operation.

    Certificate

    After Certificate Verification is enabled, import the certificate obtained in Step 1: Obtaining the CNware Certificate.

    Revocation List

    After the CRL is imported, the system checks whether the client certificate has been revoked. If the certificate has been revoked, cluster registration fails.

  3. Click OK.

    After the registration is successful, information about the registered CNware virtualization environment is displayed on the left of the page. You can view clusters or hosts and their VM lists.
