Viewing the Job Progress

This section describes how to view job information so that you can know about the execution state and details of data jobs.

Procedure

  1. Choose Insight > Jobs.
  2. View the progress of Current Jobs or Historical Jobs.

    1. Click the Execution Jobs or Historical Jobs tab. The job list is displayed.
    2. (Optional) In the upper right corner of the job list, select the start time of the task to filter jobs. Or, click next to Task Type, Object Type, and Job Status to filter jobs. Or, click next to Object or Job ID to search for jobs. Or, you can click next to Start Time to sort jobs.
    3. Click Task Type to expand job details and view the protected object information and job execution details.
