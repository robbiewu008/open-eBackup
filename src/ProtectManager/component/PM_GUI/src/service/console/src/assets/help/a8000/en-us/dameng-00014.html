Step 3: Registering the Dameng Database

Registering a Single-Server Dameng Database

  1. Choose Protection > Databases > Dameng.
  2. Click Register to register the Dameng Single Server database.
  3. Set Type to Single Server.
  4. Configure database information.

    Table 1 describes the related parameters.

    Table 1 Registration information of a single-server Dameng

    Parameter

    Description

    Name

    Custom resource name.

    Hosts

    Database host to be registered.

    Port

    Listening port of the database. For details about how to query the port number, see Preparations for Backup.

    Authentication Method
    Database authentication method. The value can be OS authentication or Database authentication.
    • If the database authentication method is OS authentication, select OS authentication or Database authentication.
    • If the database authentication method is not OS authentication, select Database authentication.
    NOTE:

    For details about how to query the database authentication method, see Preparations for Backup.

    Database Username

    This parameter is displayed when Authentication Method is set to Database authentication.

    Username for logging in to the currently registered database.

    Database Password

    This parameter is displayed when Authentication Method is set to Database authentication.

    Password for logging in to the currently registered database.

  5. Click OK.

Registering the Dameng Cluster Database

  1. Choose Protection > Databases > Dameng.
  2. Click Register to register the Dameng Cluster database.
  3. Set Type to Active/standby MPP cluster.
  4. Customize a cluster name.
  5. Click Add to add an instance.

    Table 2 describes the related parameters.

    Table 2 Dameng cluster registration information

    Parameter

    Description

    Hosts

    Database host corresponding to the cluster instance.

    Port

    Listening port of the database instance. For details about how to query the port number, see Preparations for Backup.

    Authentication Method
    Database authentication method. The value can be OS authentication or Database authentication.
    • If the database authentication method is OS authentication, select OS authentication or Database authentication.
    • If the database authentication method is not OS authentication, select Database authentication.
    NOTE:

    For details about how to query the database authentication method, see Preparations for Backup.

  6. Click OK.
  7. Click OK.
