Managing Tenants

After a tenant is added to the product, you can perform operations on the tenant.

Select Protection > Cloud Platforms > Huawei Cloud Stack. On the Tenants tab page, find the tenant to be operated.

Table 1 describes the related operations.

Table 1 Tenant-related operations

Operation

Description

Navigation Path

Authorizing Huawei Cloud Stack resources

NOTE:

Only version 1.5.0 supports this function.

Scenario

By default, only the system administrator has the permission to protect resources. If you want to assign this permission to a data protection administrator, authorize the data protection administrator as the system administrator to protect resources.

Note

  • The protection permissions on a resource that has been associated with an SLA cannot be granted. Before permission granting, remove the protection.
  • The protection permissions on a single resource can be granted only to a single data protection administrator.

In the row of the target tenant, choose More > Authorize Resource and select the data protection administrator to be authorized.

Reclaiming Huawei Cloud Stack resources

NOTE:

Only version 1.5.0 supports this function.

Scenario

This operation enables you to reclaim resources authorized to the data protection administrator. The data protection administration is not permitted to protect the resources that have been reclaimed.

Note

The protection permissions on a resource that has been associated with an SLA cannot be reclaimed. Before reclamation, remove the protection.

In the row of the target tenant, choose More > Reclaim Resource.

Scanning resources of a tenant

Scenario

When resources of a tenant change, for example, projects/resource sets, ECSs, or EVS disks are added or deleted, you can perform this operation to manually scan resources.

In the row of the target tenant, choose More > Resource Scan.

Modifying a tenant

Scenario

This operation enables you to add a VDC administrator for a tenant, change the password of an existing VDC administrator, or delete an existing VDC administrator.

In the row of the target tenant, choose More > Modify.

On the Modify page, you can add, modify, and delete VDC administrators.

Deleting a tenant

Scenario

Perform this operation when resources of a tenant do not need to be protected.

In the row of the target tenant, choose More > Delete or select the tenants to be deleted and click Delete above the list.
Adding a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to quickly filter and manage resources.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.
Removing a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to remove the tag of a resource when it is no longer needed.

Choose More > Remove Tag.
