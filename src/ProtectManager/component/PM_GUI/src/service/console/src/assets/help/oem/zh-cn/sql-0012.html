备份SQL Server数据库

您可以根据本章节指引完成备份配置。对于1.6.0及后续版本，也支持通过智能助手提供伴随式教学和流程引导，在用户操作过程中及时提供指引，降低操作难度。您登录本产品管理界面后，单击界面上方的，在打开的“智能助手”页面，单击“了解如何进行备份操作”，选择待保护的资源类型后单击“下一步”，即可根据备份配置引导逐步完成本章节的备份配置。
