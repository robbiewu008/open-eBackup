Logging In to DeviceManager

DeviceManager is a piece of integrated storage management software developed by Huawei. On any management terminal connected to the storage system, you can use a browser to log in to DeviceManager as a local user using the management network port IP address of the controller of the storage system.

You can log in to DeviceManager of a storage device in either of the following ways:
  • Logging in to DeviceManager of the backup storage system using the management network port IP address of the storage system controller via a browser (applicable to all product models)
  • Switching from the OceanProtect WebUI to DeviceManager(applicable to OceanProtect X series backup appliances and the OceanProtect E6000 Appliance).

Logging in to DeviceManager of the Backup Storage System Using the Management Network Port IP Address of the Storage System Controller via a Browser (Applicable to All Product Models)

On any maintenance terminal connected to the storage system, you can use a browser to log in to DeviceManager of the backup storage as a local user using the management network port IP address of the controller of the storage system.

Before using DeviceManager, verify that the maintenance terminal meets the following requirements:
  • The OS and browser running on the maintenance terminal meet the compatibility requirements.

    DeviceManager supports multiple OSs and browsers. For details about the compatibility information, visit Compatibility Query.

  • The maintenance terminal communicates with the storage system properly.
Procedure
  1. Open a browser on the maintenance terminal.
  2. Enter https://Management IP address:8088 in the address box and press Enter.
    • Replace Management IP address with the actual IP address of the management network port on the controller enclosure of the storage system.
    • If a firewall is configured, you must enable port 8088 for the system to provide web services.
    • The browser may prompt that the website has a security certificate issue. If the IP address is correct, you can neglect the prompt and continue accessing the storage system.
  3. Optional: Click the language type in the upper right corner to switch the language.

    DeviceManager supports simplified Chinese and English.

  4. Select Local user or LDAP user in Authentication Mode, and enter the username and password in Username and Password. If Verification Code is displayed, enter the correct verification code.
    • To use a lightweight directory access protocol (LDAP) domain user account to log in to DeviceManager, you must first configure the LDAP domain server, and then set the LDAP server parameters and create an LDAP domain user account on the storage system.
    • The username and password are case-sensitive.
    • If the default password has been changed, use the new password to log in.
    • If Account Lockout is enabled (enabled by default) for an account in the security policy, the account will be locked if the number of consecutive incorrect password attempts (Lockout Threshold) reaches the upper limit (3 by default) within 5 minutes. (By default, the account is automatically unlocked after being locked for 5 minutes.)
    • To ensure security of a user account, you are advised to change the account password periodically. You can log in to DeviceManager, choose Settings > User and Security > Users and Roles > User Management, and change the login password.
  5. Click Log In.

    The DeviceManager WebUI is displayed.

    • When logging in to DeviceManager for the first time, you need to sign the EULA as a user with administrator permissions.

      Read the Huawei Enterprise End User License Agreement carefully, select I have read and agree to sign the Huawei Enterprise End User License Agreement, and click OK.

    • After the operation, click the current login user in the upper right corner of the page, and then click Log Out in the drop-down list box to log out of DeviceManager.

Switching from the OceanProtect WebUI to DeviceManager (applicable to OceanProtect X series backup appliances and the OceanProtect E6000 Appliance)

  • For OceanProtect X series backup appliances, perform the following operations:
    1. Choose System > Infrastructure > Cluster Management.
    2. On the Backup Clusters tab page, click a node name under the Local Cluster Nodes area.
    3. On the displayed Node Details page, click Open the device management platform to go to DeviceManager.
  • For the OceanProtect E6000 Appliance, perform the following operations:
    1. Choose System > Infrastructure > Cluster Management.
    2. On the Backup Clusters tab page, click View Cluster Details to go to the cluster details page. Click Open the device management platform to go to DeviceManager.
