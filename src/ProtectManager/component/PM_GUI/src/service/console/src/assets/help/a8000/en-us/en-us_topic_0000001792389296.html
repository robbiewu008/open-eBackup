Creating a Mount Update Policy

When creating a live mount, you can associate the live mount with a mount update policy. After the association, the mounted copies are updated based on the mount update policy.

Procedure

  1. Choose Explore > Data Utilization Policies > Mount Update Policies.
  2. Click Create to create a mount update policy.

    Table 1 describes the related parameters.

    Table 1 Parameters of mount update policies

    Parameter

    Description

    Name

    Name of a mount update policy, which can be customized.

    Copy Data

    Select the latest copy in a certain period to mount. If no copy is generated within a specified period, mount copies are not updated.

    • Last hour: The latest copy generated in the last hour is selected.

      For example, if the current time is 16:30, the latest copy generated from 15:00 to 16:00 is selected.

    • Last day: The latest copy generated in the last day is selected.

      For example, if the current date is January 30, the latest copy generated on January 29 is selected.

    • Last week: The latest copy generated in the last week is selected.

      For example, if the current week is the fifth week in 2021, the latest copy generated in the fourth week in 2021 is selected.

    • Last month: The latest copy generated in the last month is selected.

      For example, if the current month is December, the latest copy generated in November is selected.

    • Latest copy: The latest copy is selected.

    Schedule

    Interval for updating mount copies.

    • Update mount copies every xx hours, days, or weeks and set the time when mount copies are updated for the first time.

      If the preset update time is earlier than the current time, live mount is executed based on the scheduling plan.

    • Mount copies are updated immediately after a backup is complete.

    Keep For

    Retention period of unmounted copies after mount copies are updated.

    You can set the retention period to xx days, weeks, months, or years. You can also retain copies permanently or retain only the latest copy.

  3. Click OK.
