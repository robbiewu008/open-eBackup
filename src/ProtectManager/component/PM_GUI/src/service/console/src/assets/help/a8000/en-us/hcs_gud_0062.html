Managing Huawei Cloud Stack Registration Information

After Huawei Cloud Stack is registered with the OceanProtect, you can perform operations on Huawei Cloud Stack.

You can choose Protection > Cloud Platforms > Huawei Cloud Stack to go to the Huawei Cloud Stack page. Select the cloud platform to be operated in the left pane.

Table 1 describes the related operations.

Table 1 Operations related to Huawei Cloud Stack

Operation

Description

Navigation Path

Viewing Huawei Cloud Stack registration information

Scenario

This operation enables you to view Huawei Cloud Stack registration information, such as the tenant name, domain name, username, agent host, and storage resource information.

Choose More > Environment Info.

Modifying Huawei Cloud Stack registration information

Scenario

This operation enables you to change the name of Huawei Cloud Stack registration information. If the IP address, domain name, username/password, or certificate of registered Huawei Cloud Stack is changed, you need to perform this operation to synchronize the change.

Note

Modify the Huawei Cloud Stack registration information when no backup or restore job is running. Otherwise, the job will fail.

Choose More > Modify Registration.

Table 1 describes the parameters of Huawei Cloud Stack registration information.

Canceling Huawei Cloud Stack registration

Scenario

If you do not need to protect a Huawei Cloud Stack platform or use it as a restore target, you can perform this operation to remove the Huawei Cloud Stack platform.

Note

If a backup job is being executed, the registration cannot be canceled. You can perform deregistration after the job is complete.

Choose More > Unregister.

Scanning Huawei Cloud Stack

Scenario

The OceanProtect automatically scans the Huawei Cloud Stack environment every 24 hours. When changes of regions, projects/resource sets, or cloud servers in registered Huawei Cloud Stack are detected, the information is automatically updated to the OceanProtect after the scan. This operation enables you to immediately update the changed information to the OceanProtect.

Note

  • If you do not select to apply the SLA policy to the newly created cloud server when creating a periodic backup job, the system will not associate the newly discovered cloud server with an SLA after the scan.

Choose More > Resource Scan.

Testing Huawei Cloud Stack connectivity

Scenario

Manual connectivity test of the registered Huawei Cloud Stack environment is used to check the connectivity between the Huawei Cloud Stack environment and the agent host and between the Huawei Cloud Stack environment and the production environment.

Choose More > Test Connectivity.
