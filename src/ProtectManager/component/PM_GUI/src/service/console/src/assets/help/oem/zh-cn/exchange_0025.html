步骤1：注册Exchange单机/可用性组

当Exchange数据库所在主机安装ProtectAgent后，本产品会自动扫描Exchange数据库和用户邮箱，本产品以扫描出来的数据库和邮箱作为保护对象。

注册Exchange单机

  1. 选择“保护 > 应用 > Exchange”。
  2. 在“单机/可用性组”页签，单击“注册”。
  3. “类型”选择“单机”。
  4. 配置Exchange单机信息。

    相关参数说明如表1所示。

    表1 Exchange单机注册信息

    参数

    说明

    类型

    选择“单机”。

    名称

    自定义要注册的Exchange单机名称。

    主机

    选择要注册的主机，单机场景选择1个主机节点。

    用户名
    执行备份和恢复的Exchange主机用户名。输入格式如下（以用户名为Administrator，域名为EX为例）：
    • EX\Administrator
    • Administrator@ex.com

    密码

    执行备份和恢复的Exchange主机用户密码。

    并发数

    单机下的数据库或邮箱备份/恢复的并发数。并发数取值范围1-10。

  5. 单击“确定”。

注册Exchange可用性组

  1. 选择“保护 > 应用 > Exchange”。
  2. 在“单机/可用性组”页签，单击“注册”。
  3. “类型”选择“可用性组”。
  4. 配置Exchange可用性组信息。

    相关参数说明如表2所示。

    表2 Exchange可用性组信息

    参数

    说明

    类型

    选择“可用性组”。

    名称

    自定义要注册的Exchange可用性组名称。

    主机

    选择要注册的主机，可选择多个主机节点。

    用户名
    执行备份和恢复的Exchange主机用户名。输入格式如下（以用户名为Administrator，域名为EX为例）：
    • EX\Administrator
    • Administrator@ex.com

    密码

    执行备份和恢复的Exchange主机用户密码。

    并发数

    可用性组中的数据库或邮箱备份/恢复的并发数。并发数取值范围1-10。

  5. 单击“确定”。
