Modifying a Standby or Member Node

This section describes how to modify the information about the standby or member node.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Cluster.
  3. In the row that contains the standby or member node to be modified, choose More > Modify. You can modify the name, IP address, and port number of the standby or member node.
  4. Click OK.
