Specifying an External Cluster as a Management Cluster

The product allows you to specify an external cluster as the management cluster. After a management cluster is specified, the local cluster and the management cluster synchronize information about managed clusters in the form of jobs. You can log in to the management cluster to manage the managed clusters.

Prerequisites

  • The local cluster does not have other management clusters except local clusters.
  • The external cluster has been added as a managed cluster.
  • The CA certificate of the Server Certificate type of the local cluster has been added to the external cluster to be specified as the management cluster, and the type has been set to Backup/Multi-domain/Replication Cluster.

Precautions

A managed cluster can be specified as the management cluster of multiple clusters.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  1. In the Multi-domain Clusters area, choose More > Set as Management Cluster in the row that contains the desired external cluster.
  2. Enter information in the displayed dialog box. The following table describes related parameters.

    Table 1 Parameters for specifying a management cluster

    Parameter

    Description

    Name

    User-defined name of the local cluster displayed in the remote cluster.

    Username

    Username and password of the local cluster system administrator.
