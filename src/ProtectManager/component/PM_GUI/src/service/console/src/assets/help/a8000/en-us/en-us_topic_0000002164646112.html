Viewing SLA Information

Procedure

  1. Choose Protection > Protection Policies > SLAs.
  2. View SLA information.

    Table 1 describes the related parameters.

    Table 1 SLA parameters

    Parameter

    Description

    Name

    SLA name.

    SLAs can be sorted by name to facilitate locating.

    Application Type

    Application type associated with the SLA.

    The preset SLAs are Gold, Silver, and Bronze, and the associated application type is General.

    Protection Policy

    Protection policy created in the SLA.

    Associated Resources

    Number of resources associated with the SLA.

    You can click the number of resources to view resource details.
    NOTE:

    If Application Type is set to Replication Copy, resources associated with the SLA can be viewed only in the source cluster of remote replication.

    Archive Targets

    Number of archive policies created in the SLA.

    An archive storage system is specified in each archive policy.

    Replication Targets

    Number of replication policies created in the SLA.

    A target cluster for replication is specified in each replication policy.

    Created By

    User who creates the SLA. For preset SLAs, the value of this parameter is displayed as --.
