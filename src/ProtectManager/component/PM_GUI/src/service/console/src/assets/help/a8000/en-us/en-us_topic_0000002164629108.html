User Roles and Permissions

The OceanProtect supports report management. Users of different roles have different report management permissions. For details about user roles, see Overview of User Roles.

  • System administrators can create, view, download, send, and delete reports.
  • Data protection administrators can create reports as well as view, download, send, and delete reports created by themselves.
  • Auditors can only view reports.
  • Other roles cannot create, view, download, send, or delete reports.
