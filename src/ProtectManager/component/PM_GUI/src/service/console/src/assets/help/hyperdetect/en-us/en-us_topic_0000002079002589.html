Enabling Intelligent Ransomware Detection

The system detects known ransomware attack signatures to identify whether file systems or LUNs are infected by ransomware. If no ransomware infection is identified, the system analyzes and compares file system or LUN snapshot changes, and uses machine learning algorithms to further check whether the file systems or LUNs are infected by ransomware. This section describes how to enable intelligent ransomware detection.

Precautions

During intelligent ransomware detection, if the file system or LUN snapshot directory corresponding to the ransomware detection snapshot being checked is invisible, the system will create a clone file system or a clone LUN whose name starts with DEE-ANTI-CLONE based on the ransomware detection snapshot to generate metadata features of the snapshot. During the detection, do not delete or rename the clone file system or clone LUN whose name starts with this prefix. Otherwise, the detection may fail. After the intelligent ransomware detection, the system automatically deletes the clone file system or clone LUN whose name starts with this prefix.

Procedure

  1. On the navigation pane, choose Data Security > Ransomware Protection > Ransomware Detection Settings.
  2. On the NAS Ransomware Detection or SAN Ransomware Detection tab page, select the vStore or LUN for which you want to enable intelligent ransomware detection and choose More > Enable Intelligent Ransomware Detection. After intelligent ransomware detection is enabled, you need to create a ransomware detection snapshot by following the instructions in Creating a Ransomware Detection Snapshot. After the snapshot is generated, the system immediately performs ransomware detection on the snapshot.

    • You can also select multiple vStores or LUNs for which you want to enable intelligent ransomware detection and click Enable Intelligent Ransomware Detection above the list.
    • If intelligent ransomware detection is not required, locate the row that contains the vStore or LUN for which you want to disable intelligent ransomware detection, and choose More > Disable Intelligent Ransomware Detection. Alternatively, select multiple vStores or LUNs for which you want to disable intelligent ransomware detection and choose More > Disable Intelligent Ransomware Detection above the vStore list.
