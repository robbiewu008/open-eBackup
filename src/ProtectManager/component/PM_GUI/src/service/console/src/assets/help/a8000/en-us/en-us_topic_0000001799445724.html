Viewing Common Share Information

After creating a common share resource, you can view details about the share resource on the management page.

Procedure

  1. Choose Protection > File Systems > Common Shares.
  2. View the common share information.

    Table 1 describes the related parameters.
    Table 1 Common share information

    Parameter

    Description

    Name

    Name of a common share resource.

    SLAs

    If an SLA is associated with a common share resource, the SLA name is displayed. Otherwise, -- is displayed.

    SLA Compliance
    Whether a backup job is successfully executed based on the backup frequency or backup window set in the SLA.
    • If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
    • If the backup time does not meet the backup time window or backup frequency requirements, for example, if the backup frequency set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup frequency or backup window.

    Protection Status

    Whether a common share resource is protected.

    • Unprotected: The common share is not associated with an SLA, or the common share is associated with an SLA but protection is disabled.
    • Protected: The common share has been associated with an SLA and protection has been activated.
    • Creating: The common share has been associated with an SLA and protection is being created.

  3. Click the name of a common share resource to view its Overview, Share Information, Copy Data, and Jobs.

    • Overview

      Displays the common share name, SLA information, job execution time, and total number of copies.

    • Share Information

      Displays the share information of the current common share resource.

    • Copy Data

      You can search for copies by year, month, and day.

      If is displayed below the time, a copy at that time exists.

    • Jobs

      Displays all jobs related to the common share resource.
