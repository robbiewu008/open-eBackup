Viewing VMware Live Mount Information

After a live mount job is created, you can view the live mount information by referring to this section.

Procedure

  1. Choose Explore > Live mount > Application Type > VMware.

    Information about the mounted data is displayed. Table 1 describes the related parameters.

    Table 1 Live mount parameters

    Parameter

    Description

    Target Object

    Name

    Name of the new VM after live mount.

    Status

    Status of the new VM after live mount.

    Location

    Location of the new VM after live mount.

    Protected Objects

    Name

    Name of the source VM corresponding to the mount copy.

    Location

    Location of the source VM corresponding to the mount copy.

    Copy Data

    Copy Time

    Time when a copy is generated through live mount.

    Update Policy Name

    If an update policy has been set for a live mount job, the name of the update policy is displayed.

    Live Mount Status

    Status of a live mount job, which can be Activate or Disable.

    After a live mount job is created, its status is Activate by default. After mount update is disabled, its status changes to Disable.
