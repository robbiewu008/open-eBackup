Step 3: Registering an OceanBase Tenant Set

This section describes how to register a tenant set. If you need to protect one or more tenants in a cluster, you can register one or more tenants as a tenant set for protection.

Procedure

  1. Choose Protection > Databases > OceanBase.
  2. Click the Tenant Sets tab.
  3. Click Register to register the OceanBase tenant set.
  4. Configure tenant set information.

    Table 1 describes the related parameters.

    Table 1 Tenant set parameters

    Parameter

    Description

    Name

    Customize a tenant set name.

    Clusters

    Select the cluster to which the tenant set belongs.

    Tenant

    Select the tenant in the cluster.

  5. Click OK.
