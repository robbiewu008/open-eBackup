Managing Active Directory Domain Controllers

The product allows you to modify, remove, activate, or disable protection for Active Directory domain controllers.

You need to log in to the management page, choose Protection > Applications > Active Directory, and find the desired Active Directory domain controller.

Table 1 describes the related operations.

Table 1 Operations related to Active Directory protection

Operation

Description

Navigation Path

Modifying protection

Scenario

You can modify the protection plan to associate another SLA with the Active Directory domain controller. After the modification, the next backup will be executed based on the modified protection plan.

Notes

  • The modification does not affect the protection job that is in progress. The new protection plan is used from the next protection period.
  • If the SLA associated with a resource changes, you are advised to modify the current SLA by choosing Protection > Protection Policies > SLAs. You are advised not to associate the resource with another SLA by modifying the protection.

    When you associate the resource with another SLA by modifying protection, the license capacity may be deducted repeatedly if the copy of the original SLA still exists.

In the row of the target Active Directory domain controller, choose More > Modify Protection.

Removing protection

Scenario

You can remove the protection plan if an Active Directory domain controller no longer needs to be protected. After the removal, the system cannot execute protection jobs for the Active Directory domain controller.

Note

  • When a protection job of the Active Directory domain controller is running, the protection cannot be removed.
  • If the SLA associated with a resource changes, you are advised to modify the current SLA by choosing Protection > Protection Policies > SLAs. You are advised not to associate the resource with another SLA after removing the protection.

    After you remove protection and associate the resource with another SLA, the license capacity may be deducted repeatedly if the copy of the original SLA still exists.

In the row of the target Active Directory domain controller, choose Protect > Remove Protection.

Activating protection

Scenario

This operation enables you to activate a disabled protection plan for an Active Directory domain controller. After the protection plan is activated, the system backs up the Active Directory domain controller based on the protection plan.

Note

You can perform this operation only on an Active Directory domain controller that has been associated with an SLA and is in the Unprotected state.

In the row of the target Active Directory domain controller, choose More > Activate Protection.

Disabling protection

Scenario

This operation enables you to disable the protection plan of an Active Directory domain controller for which periodic protection is not required. After the protection plan is disabled, the system does not perform automatic backup jobs for the Active Directory domain controller.

Note

Disabling protection does not affect ongoing backup jobs.

In the row of the target Active Directory domain controller, choose More > Disable Protection.

Performing manual backup

Scenario

This operation enables you to perform a backup immediately.

Note

Copies generated by manual backup are retained for the duration defined in the SLA.

In the row of the target Active Directory domain controller, choose More > Manual Backup.

Adding a tag

Scenario

This operation enables you to quickly filter and manage resources.

Note
  • The system administrator can add a maximum of 10,000 tags to a resource.
  • A non-system administrator must have the permission of Production resource management to add tags to resources.
  • A non-system administrator can add a maximum of five tags to a resource.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.

Removing a tag

Scenario

This operation enables you to remove the tag of a resource when it is no longer needed.

Choose More > Remove Tag.

Note: The configuration for restoring Active Directory is similar to that in Restoration and is not described in this section.
