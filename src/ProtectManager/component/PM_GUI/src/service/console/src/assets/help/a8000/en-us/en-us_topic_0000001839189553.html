Managing openGauss Clusters

The OceanProtect allows you to delete, authorize, or reclaim clusters.

You need to log in to the management page, choose Protection > Databases > openGauss, click Cluster, and find the target cluster.

Table 1 describes the related operations.

Table 1 Cluster-related operations

Operation

Description

Navigation Path

Modifying

Scenario

This section describes how to modify the cluster name, type, and node information.

Choose More > Modify in the row where the cluster is located.

Deleting

Scenario

You can delete a host or cluster if its databases no longer require protection. After the host or cluster is deleted, its information disappears from the WebUI.

Note

A protected database or instance in a cluster cannot be deleted.

In the row where the target host or cluster resides, and choose More > Delete.

Authorizing resources

Scenario

By default, only the system administrator has the permission to protect resources. If you want to assign this permission to a data protection administrator, authorize the data protection administrator as the system administrator to protect resources.

Note

  • Users cannot be authorized to protect resources that have been associated with SLAs. You need to remove the protection of resources before performing authorization.
  • A single resource can be authorized only to a single data protection administrator.

Locate the row that contains the target host or cluster, choose More > Authorize Resource, and select the data protection administrator to be authorized.

Reclaiming resources

Scenario

This operation enables you to reclaim resources authorized to the data protection administrator. The data protection administration is not permitted to protect the resources that have been reclaimed.

Note

Users are not allowed to reclaim resources that have been associated with SLAs. You need to remove the protection of resources before reclaiming them.

In the row where the target host or cluster resides, choose More > Reclaim Resource.

Test Connectivity

Scenario

This operation enables you to test the connectivity between the OceanProtect and the cluster.

In the row of the target host or cluster, choose More > Test Connectivity.
