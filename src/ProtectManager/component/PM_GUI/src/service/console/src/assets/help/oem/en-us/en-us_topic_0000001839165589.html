Step 2: Setting Windows PowerShell Permissions

Ensure that Windows PowerShell has the execute script permission (RemoteSigned, Bypass, or Unrestricted). This operation is supported only in Windows scenarios.

Procedure

  1. Open Windows PowerShell and run the Get-ExecutionPolicy command to check whether Windows PowerShell has the RemoteSigned, Bypass, or Unrestricted permission to execute scripts.

    • If any of the preceding permissions exists, no further action is required.
    • If the preceding permissions do not exist, run the command in 2.

  2. Run the Set-ExecutionPolicy RemoteSigned command to grant the RemoteSigned permission.
