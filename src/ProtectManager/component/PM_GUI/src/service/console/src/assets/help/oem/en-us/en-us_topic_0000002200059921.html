Preparations for Backup

Before backing up TDSQL databases, prepare related information by following instructions in Table 1.

Table 1 Preparations for backup

Item

Mandatory or Not

How to Obtain

To Be Used In

Username and password of the administrator of the TDSQL CHITU management console

Mandatory

Contact the administrator of the TDSQL CHITU management console to obtain the username and password.

Step 1: Enabling the TDSQL Database Permission (Applicable to Non-distributed Instances)

Username and password for OSS interface authentication

Mandatory

Contact the database vendor to obtain the username and password for OSS interface authentication.

Step 3: Registering a TDSQL Database

tdsqlsys_repl User Password

Mandatory

Contact the database vendor to obtain the value, or run the select * from mysql.slave_master_info; command on the standby database node to obtain the value of User_password in the command output.

Viewing TDSQL node information

Optional
  1. Log in to the TDSQL CHITU management console.
  2. On the home page of the CHITU management console, click Dispatching and management system.
  3. View node information.
    • Click Zookeeper monitoring to view ZooKeeper node information.
    • Click Scheduler monitoring to view Scheduler node information.
    • Click Manager monitoring to view Manager node information.
    • Click OSS-service monitoring to view the information about OSS service nodes.
