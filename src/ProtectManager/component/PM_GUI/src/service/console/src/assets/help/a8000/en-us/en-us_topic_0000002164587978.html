Configuring the hbase.master.logcleaner.ttl Parameter on Different Big Data Platforms

Procedure

CDH as the big data platform

  1. Use a browser to log in to Cloudera Manager.
  2. Choose Cluster > HBase.
  3. On the cluster page, click Configuration, search for hbase.master.logcleaner.ttl in the search box, and press Enter.
  4. In the row that contains the parameter name, select the time for retaining the log file, and save the file.
  5. Restart the service.

MRS as the big data platform

  1. Log in to the MRS management page.
  2. On the HBase cluster page, click Configurations.
  3. On the Configurations page, enter hbase.master.logcleaner.ttl in the search box, enter the parameter value (unit: ms) in the row that contains the parameter name, and save the settings.
  4. Restart the service after changing the parameter value.

TDH as the big data platform

  1. Use a browser to log in to Transwarp Data Hub.
  2. Choose Dashboard > Cluster and click Hyperbase to go to the cluster page.
  3. Click Config.

    • If the hbase.master.logcleaner.ttl configuration item already exists, enter the parameter value (unit: ms) in the row that contains the configuration item and save the settings.
    • If the configuration item hbase.master.logcleaner.ttl does not exist, click Add Custom Property. Enter Config Name, Value, and Config File (the default value is hbase-site.xml), and click OK.
