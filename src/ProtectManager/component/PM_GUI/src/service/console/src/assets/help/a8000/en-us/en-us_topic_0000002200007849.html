Step 1: Adding a Storage Device

Before backing up or restoring an NDMP NAS file system, add the storage device where the NDMP NAS file system to be protected resides to the OceanProtect.

Context

The OceanProtect can protect NDMP-based NAS file systems.

Procedure

  1. Choose Protection > File Systems > Storage Devices.
  2. Click Add Device to add a storage device.
  3. Set the device type to NDMP Server.
  4. Configure parameters as required. Table 1 describes the related parameters.

    Table 1 Parameters for adding a device

    Parameter

    Description

    Device Name

    Name of a user-defined device.

    IP Address

    Management IP address of the storage device.

    Port

    Number of the port for communication between the OceanProtect and the management plane of a storage device.

    The default port number is 10000.

    Username

    Username and password of a storage device NDMP administrator. Ensure that the RESTful login mode is available for the user and the password has been initialized.

    NOTE:

    Use an administrator account. Otherwise, resources on the device may fail to be backed up.

    Password

    Clients

    By default, the built-in client is selected. If an external client exists, you can also select the external client.

    Number of Concurrent Tasks

    The default number of concurrent tasks is 8. The allowed value ranges from 1 to 64.

    NOTE:
    • The number of concurrent tasks cannot exceed the flow control limit provided by the corresponding storage vendor.
    • This parameter indicates the maximum number of concurrent backup or restoration jobs that can be performed at the same time on the current storage device.

  5. Click OK.

    When a storage device is added successfully, information about the added storage device is displayed on the Storage Device page.

    The OceanProtect automatically scans NDMP NAS file systems in the storage device and displays them on the Protection > File Systems > NDMP page. If a file system is added to the registered device, you can choose More > Resource Scan in the row that contains the device to update the file system resources.
