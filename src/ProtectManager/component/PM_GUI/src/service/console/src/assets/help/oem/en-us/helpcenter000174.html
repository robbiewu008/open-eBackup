Deleting an Archive Storage

The product allows you to delete archive storage. After an archive storage is deleted, it cannot be used to archive data.

Prerequisites

  • An archive storage has been added.
  • All archive copies in the archive storage have been deleted.

Precautions

An archive storage cannot be restored after being deleted.

Procedure

  1. On the Object Storage tab page, click More on the right of the target archive storage and select Delete.
  2. In the dialog box that is displayed, click OK.
