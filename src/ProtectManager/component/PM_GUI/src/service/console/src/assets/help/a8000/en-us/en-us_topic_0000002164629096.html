Modifying a Replication Cluster

The OceanProtect allows you to modify replication cluster information. If you have changed the IP address and port number of the replication cluster, you need to modify the replication cluster information on the local cluster accordingly to ensure that the local cluster is properly connected to the replication cluster.

Precautions

The role of a replication cluster cannot be changed.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click the Replication Clusters tab. The replication cluster page is displayed.
  1. In the row that contains the replication cluster to be modified, choose More > Modify.
  2. Modify replication cluster parameters.

    Table 1 describes the parameters.

  3. Click OK.
