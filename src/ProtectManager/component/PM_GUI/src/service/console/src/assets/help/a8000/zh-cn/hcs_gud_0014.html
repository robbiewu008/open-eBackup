步骤2：注册华为云Stack

执行华为云Stack备份恢复前，需要将华为云Stack注册至OceanProtect。

前提条件

  • 已为代理主机配置反向代理或免代理。
  • 已获取ManageOne运营面的运营管理员用户bss_admin及其对应的密码，该用户用于将华为云Stack接入OceanProtect。

    您也可以新创建类型为“系统”的华为云Stack运营管理员用户。华为云Stack 8.0.3及之前的版本，创建运营管理员的详细操作请参见《用户指南》，华为云Stack 8.1.0版本及之后的版本，创建运营管理员的详细操作请参见《资源发放指南》。

  • 已创建角色为“管理员”，登录方式为“RESTful”的存储设备用户，该用户用于将华为云Stack中的存储资源接入OceanProtect。创建用户的详细操作请参见存储设备对应版本的《管理员指南》。如果存储资源为OceanStor V5，还需要设置用户级别为“管理员”。
  • 使用iSCSI备份，请确保华为云Stack对接的存储设备已配置备份软件的备份网络平面可访问的iSCSI端口。如未配置，请参考存储侧对应版本的产品配置。

背景信息

注册华为云Stack后，可以在管理界面浏览华为云Stack的项目/资源集、云服务器的信息并对其进行保护，也可将其作为恢复目标。

操作步骤

  1. 选择“保护 > 云平台 > 华为云Stack”。
  2. 单击“注册”，注册华为云Stack信息。

    华为云Stack注册信息如表1所示。
    表1 华为云Stack注册信息

    参数

    说明

    名称

    用户自定义华为云Stack名称。

    IP地址

    API Gateway的浮动IP地址。获取方式如下：

    在华为云Stack部署工具导出的参数汇总文件《xxx_export_all_CN》的“工具生成的IP参数”页签，搜索“AGW-LB-Float-IP”，获取对应的参数值。

    域名

    华为云Stack的访问域名。获取方式如下：

    在华为云Stack部署工具导出的参数汇总文件《xxx_export_all_CN》的“基本参数”页签，搜索“external_global_domain_name”，获取对应的参数值。

    用户名

    访问ManageOne运营面的运营管理员的用户名和密码。

    密码

    代理主机

    选择可用于备份华为云Stack的代理主机。

    校验证书

    默认开启，注册华为云Stack时会对其证书进行校验，确保OceanProtect与外部设备交互的安全性。关闭后，系统与云平台通信可能存在安全风险，请谨慎操作。

    Cinder证书

    请导入步骤1：获取证书章节中获取的Cinder证书，导入后执行备份任务时会校验该证书的有效性。

  3. 在“存储资源”区域，单击“添加”。存储资源参数说明如表2所示。

    • 如果华为云Stack的生产存储为华为分布式块存储，且使用外置代理，无需添加存储资源。其他场景需要添加存储资源，否则会导致备份任务失败。
    • 当华为云Stack生产存储有多套存储资源时，需要将待保护对象所在的存储资源都添加到OceanProtect中。
    表2 存储资源信息

    参数

    说明

    类型

    存储资源的类型，包括：华为SAN存储和华为分布式块存储。

    管理IP地址
    • 华为SAN存储的管理IP地址获取方式：在华为云Stack部署工具导出的参数汇总文件《xxx_export_all_CN》的“1.2 基本参数”页签，搜索“业务存储REST URL”，获取对应参数值中的IP地址。
    • 华为分布式块存储的管理IP地址获取方式：在华为云Stack部署工具导出的参数汇总文件《xxx_export_all_CN》的“2.3 Portal”页签，搜索“华为分布式块存储 Business”，获取对应参数值中的IP地址。
      说明：

      如果待备份虚拟机的存储信息不在此汇总文件中，则请联系华为云Stack的环境管理员获取，并添加至华为云Stack的注册信息中。

    端口

    存储资源的访问端口，默认是8088。

    用户名

    登录存储资源DeviceManager界面的管理员或超级管理员的用户名和密码。

    密码

    校验证书

    默认开启，注册华为云Stack时会对存储资源的证书进行校验，确保OceanProtect与外部设备交互的安全性。关闭后，系统与存储设备通信可能存在安全风险，请谨慎操作。

    证书

    请导入步骤1：获取证书章节中获取的存储资源的CA证书。

    吊销列表

    吊销列表由证书颁发机构发布，导入证书吊销列表后，系统将校验客户端证书是否被吊销，如果证书已被吊销，则注册存储设备失败。

    VBS节点登录信息

    说明：

    仅1.6.0及后续版本“类型”选择“华为分布式块存储”时存在此参数。

    当需要使用内置代理对华为分布式块存储（8.1.3及后续版本）的卷进行增量备份时，请打开开关，并执行以下操作。

    1. 在当前界面配置VBS节点的用户名、IP、端口和密码。
      • “节点用户名”：请输入具有VBS节点登录权限的用户名，例如fsp。
      • “节点IP”：请输入VBS节点的IP。
      • “节点端口”：SSH服务端口，默认为22。
      • “节点密码”：请输入用户密码。
    2. 检查配置的节点用户是否有权限执行VBS命令。
      1. 使用PuTTY，以配置的节点用户登录配置的VBS节点。
      2. 执行以下命令，检查该用户是否有权限执行VBS命令。

        sudo /bin/vbs_cli

        sudo /opt/dsware/vbs/script/vbs/vbs_cmd.sh

        sudo /bin/sh

        如果能成功执行以上命令并无需输入密码，表示该用户有执行VBS命令权限，则结束操作。否则，请继续执行3。

    3. 请执行以下操作，为用户配置VBS命令执行权限。
      1. 执行su - root命令，输入root用户密码，切换到root用户。
      2. 执行vi /etc/sudoers命令，打开配置文件，并将以下内容增加至配置文件。
        fsp ALL=(root) NOPASSWD:/bin/vbs_cli
fsp ALL=(root) NOPASSWD:/opt/dsware/vbs/script/vbs/vbs_cmd.sh
fsp ALL=(root) NOPASSWD:/bin/sh
        说明：

        增加至配置文件内容中的fsp为1中配置的“节点用户名”，请根据实际替换。

  4. 单击“确定”。

    注册成功后，已注册的华为云Stack信息将在页面左侧显示。
