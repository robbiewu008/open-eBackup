Deleting a Backup Storage Unit Group

This section describes how to delete a backup storage unit group.

Precautions

  • A backup storage unit group associated with an SLA cannot be deleted.
  • If parallel datastore is enabled for a backup storage unit group and a copy exists, the backup storage unit group cannot be deleted.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. In the row that contains a backup storage unit group, choose More > Delete.
  4. In the Danger dialog box, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
