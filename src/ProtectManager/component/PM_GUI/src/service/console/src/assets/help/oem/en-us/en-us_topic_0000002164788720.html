Unlocking a User

The system administrator can unlock a user.

Prerequisites

  • Only the system administrator can unlock a user.
  • Locked Status of the user to be unlocked is Locked.

Procedure

  1. Choose System > Security > Users and Roles > Users.
  2. (Optional) Click next to Name to search for users.
  3. Click More on the right of a user to be unlocked and choose Unlock.
  4. In the dialog box that is displayed, enter the password of the current login user and click OK.
