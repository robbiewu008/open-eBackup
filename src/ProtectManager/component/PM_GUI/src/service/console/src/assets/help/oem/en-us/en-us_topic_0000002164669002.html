Step 2: Creating a GoldenDB Instance

Procedure

  1. Choose Protection > Databases > GoldenDB.
  2. Click the Instance tab.
  3. Click Create to create a GoldenDB Instance.
  4. Configure instance information.

    Table 1 describes the related parameters.

    Table 1 GoldenDB instance registration information

    Parameter

    Description

    Name

    User-defined instance name.

    Cluster

    A cluster that has been registered.

    Instance

    Instance in the cluster to be registered.

    NOTE:

    Only instances of the multi-shard tenant type are supported.

    Account Name

    Account name corresponding to the selected instance on the GoldenDB account management page.

    Password

    Password corresponding to the selected instance.

    Storage Configuration

    Cluster instance-level storage configuration file (ini format) configured on the GoldenDB management page.

    Data Nodes

    Host and username of the data node.

    GTM Nodes

    Host and username of the GTM node.

  5. Click OK.
