Viewing LUN Information

Prerequisites

LUN resources on the storage system have been discovered by ProtectManager.

Context

LUNs created on the storage system can be automatically discovered by ProtectManager and displayed on the Local LUNs tab page.

  • When ProtectManager scans for a LUN that is newly created on the storage system, it may take some time for ProtectManager to discover it.
  • LUNs discovered by ProtectManager cannot be deleted on ProtectManager.
  • Clone LUNs are not displayed on ProtectManager.

Procedure

  1. Choose Protection > Storage > Local Resources.
  2. Click the Local LUNs tab and view information about local LUNs.

    1. For details about a LUN, see Table 1.
      Table 1 Parameters related to a LUN

      Parameter

      Description

      Resource ID

      ID of a local LUN.

      Name

      Name of a local LUN.

      vStore

      vStore to which a LUN belongs. The default value is System_vStore.

      SLAs

      If a local LUN has been associated with an SLA, the SLA name is displayed. Otherwise, -- is displayed.

      SLA Compliance

      Indicates whether a backup job is successfully executed in compliance with the specified backup interval or backup time window.

      For example, if the backup time of a backup job (except the first backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met. If the backup interval set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed.

      Protection Status

      Indicates whether a local LUN is protected.

      • Unprotected: The local LUN is not associated with an SLA, or it is associated with an SLA but protection is disabled.
      • Protected: The local LUN is associated with an SLA and protection is activated.
      • Creating: The local LUN is associated with an SLA and protection is being created.
    2. Click the name of a local LUN to view its overview, copy data, and related jobs.
      • Overview of the local LUN

        Displays the name, SLA information, job execution time, and total number of copies of the local LUN.

      • Snapshot data of the local LUN

        Allows you to search for copies by year, month, and day.

        If is displayed below the time, a copy at that time exists.

      • Jobs related to the local LUN

        Displays all jobs related to the local LUN, including ransomware detection snapshot backup and restoration jobs.
