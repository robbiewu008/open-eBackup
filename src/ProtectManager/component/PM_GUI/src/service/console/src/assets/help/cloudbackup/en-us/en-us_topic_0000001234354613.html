Performing a Manual Backup

If you want to perform a backup immediately, you can manually perform the backup.

Context

  • During a manual backup, the parameters (except Automatic Retry) defined in the SLA, such as Rate Limiting Policies, are applied to the manual backup job.
  • Copies generated by manual backup are retained for the duration defined in the SLA.

Procedure

  1. Choose Protection > Storage > Local File System.
  2. In the local file system list, select the file systems that you want to protect and choose More > Manual Backup.

    You can also select multiple local file systems for batch manual backup.

  3. Click OK.
