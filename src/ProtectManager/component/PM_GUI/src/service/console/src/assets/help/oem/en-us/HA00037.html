Viewing a Backup Storage Unit

After a member node is added, the system automatically adds the backup storage corresponding to the member node as a backup storage unit.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. Click Backup Storage Units to view information about the backup storage unit. Table 1 describes the related parameters.

    Table 1 Backup storage unit parameters

    Parameter

    Description

    Name

    Name of a backup storage unit. For a backup storage unit that is automatically added, the default name is the name of the member node plus the suffix _auto_add.

    Status

    Status of a backup storage unit.

    • Online: The backup storage unit is properly connected to the active node.
    • Offline: The backup storage unit is disconnected from the active node.
    • Upgrading: The backup storage unit is being upgraded to a member node.

    Belongs to Local Cluster

    Whether the backup storage unit belongs to the local cluster.

    • Yes: backup storage unit corresponding to the member node
    • No: manually added backup storage unit

    IP

    Management IP address of the backup storage unit.

    Port

    Port number of the backup storage unit.
