生产存储扩容后，备份任务失败如何处理

当VMware虚拟机所在的数据存储对应的LUN进行了扩容，您需要参考以下步骤在客户端所在主机上重新扫描该LUN，否则备份任务会失败。

操作步骤

  1. 使用PuTTY工具以root账户登录待安装客户端的主机。
  2. 执行以下命令，扫描LUN。

    for host in $(ls -1d /sys/class/scsi_host/*); do echo "- - -" > ${host}/scan ; done

  3. 执行以下命令确认LUN的容量为扩容后的容量。

    fdisk -l

  4. 重启客户端所在主机。

    如果3查询到LUN容量与实际不符时，才需要执行该步骤，重启后再执行fdisk -l查看。
