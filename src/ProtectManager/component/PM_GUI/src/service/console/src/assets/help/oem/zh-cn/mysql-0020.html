步骤6：（可选）创建限速策略

如果需要限制保护任务的最大带宽以保证关键业务的性能，请参考本节配置限速策略。

操作步骤

  1. 选择“保护 > 保护策略 > 限速策略”。
  2. 单击“创建”。
  3. 配置限速策略参数。

    相关参数说明如表1。
    表1 限速策略参数说明

    参数

    说明

    名称

    自定义限速策略名称。

    最大带宽

    保护任务平均每秒最大传输的数据量。

    默认值：1024MB/s。

    取值范围：10MB/s~5120MB/s

    描述

    描述限速策略信息。

  4. 单击“确定”。
