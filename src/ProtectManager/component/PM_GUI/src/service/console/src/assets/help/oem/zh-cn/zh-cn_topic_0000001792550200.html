步骤3：注册Dameng数据库

注册Dameng单机数据库

  1. 选择“保护 > 数据库 > Dameng”。
  2. 单击“注册”，注册Dameng单机数据库。
  3. “类型”选择“单机”。
  4. 配置数据库信息。

    相关参数说明如表1所示。

    表1 Dameng单机注册信息

    参数

    说明

    名称

    自定义资源名称。

    主机

    选择要注册的数据库主机。

    端口

    注册的单机数据库侦听的端口。

    认证方式

    支持选择“OS认证”和“数据库认证”。

    • OS认证：基于操作系统的认证。
    • 数据库认证：通过数据库管理员用户名和数据库管理员密码进行认证。

    请登录数据库查询数据库认证方式，并根据查询结果确认此处如何选择。查询方式如下：

    1. 登录Dameng数据库。
    2. 执行select name, value from v$parameter where name = 'ENABLE_LOCAL_OSAUTH';命令，查看VALUE参数项取值。
      • 如果取值为1则表示支持OS认证，此处认证方式可以选择“OS认证”或“数据库认证”。
      • 如果取值为0则表示不支持OS认证，此处认证方式请选择“数据库认证”。

    数据库用户名

    当前注册的数据库登录用户名。

    数据库密码

    当前注册的数据库登录密码。

  5. 单击“确定”。

注册Dameng集群数据库

  1. 选择“保护 > 数据库 > Dameng”。
  2. 单击“注册”，注册Dameng集群数据库。
  3. “类型”选择“MPP主备集群”。

    1.5.0版本类型请选择“MPP集群”。

  4. 自定义集群名称。
  5. 单击“添加”添加集群所有实例。
  6. 单击“确定”。
