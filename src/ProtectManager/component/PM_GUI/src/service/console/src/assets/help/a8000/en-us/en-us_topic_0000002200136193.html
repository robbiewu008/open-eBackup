VSS Snapshots Cannot Be Created During Backup

Symptom

A full, incremental, or differential backup of an Exchange database fails, and a message indicating VSS snapshot creation failed is displayed in the task details on the OceanProtect page.

Possible Cause

If the plug-in cannot create VSS volume shadow snapshots, it may indicate that outdated volume shadow copies interfere with the VSS subsystem's snapshot creation. For example, outdated volume shadow copies cause insufficient space.

Troubleshooting

It is recommended that you use the Vssadmin and DiskShadow tools to maintain the VSS subsystem to create snapshots and resolve specific problems. You can use these utilities to list shadow copies stored by VSS, delete shadow copies, and manage space allocated in a specific volume to store shadow copies.
