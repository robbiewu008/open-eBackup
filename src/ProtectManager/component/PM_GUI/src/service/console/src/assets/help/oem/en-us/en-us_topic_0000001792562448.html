Step 3: Registering the Database of a Single PostgreSQL Instance

Prerequisites

Before registering an instance, check whether the owner of the bin/lib/share file in the database installation path is root. If the owner is root, set enable_root to 1 in the /opt/DataBackup/ProtectClient/Plugins/GeneralDBPlugin/bin/applications/postgresql/conf/switch.conf configuration file.

Procedure

  1. Choose Protection > Databases > PostgreSQL.
  2. Click the Instance tab.
  3. Click Register to register a PostgreSQL database.
  4. Set Type to Single instance.
  5. Configure the database instance and its authentication information.

    Table 1 describes the related parameters.

    Table 1 Registration information of a single PostgreSQL instance

    Parameter

    Description

    Name

    Username for installing the database.

    Hosts

    Select the database host to be registered.

    Username

    OS username of the database.

    Database Installation Path

    Path for installing the database to be registered.

    Service Plane IP Address

    IP address of the host for the database service to receive TCP/IP connections.

    Database Port

    Port number used by the database service to listen on TCP/IP connections.

    Database Username

    Name of the database administrator.

    Database Password

    Password of the database administrator.

  6. Click OK.
