Installing the Client

Before application backup or restoration, you must install the client. The client (referred to as the ProtectAgent software in 1.5.0) is used to connect the OceanProtect and the production environment. It obtains the production end data and transfers the data to the backup storage of the OceanProtect during backup. It is also used for process control and data transmission during backup and restoration.

Context

The client can be deployed only on production hosts. It must be deployed on all hosts where SAP on Oracle databases to be protected reside.

Precautions

  • Before installing the client, ensure that environment variables have been configured based on Configuring Environment Variables. If you configure environment variables after installing the client, you need to restart the client after configuring the environment variables.
  • Install the client as an administrator.
