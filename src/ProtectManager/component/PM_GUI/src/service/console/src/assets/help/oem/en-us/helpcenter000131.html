Locking Out a User

The system administrator can lock out a user so that the user can no longer log in to the device. A login user can no longer log in after being locked out.

Precautions

  • Only the system administrator can lock a user. All users except the current user and built-in users can be locked.
  • Locked Status of the user to be locked is Unlocked.

Procedure

  1. Choose System > Security > RBAC > Users.

    For 1.5.0, choose System > Security > Users and Roles > Users.

  2. (Optional) Click next to Role to filter users. You can also click next to Name to search for a user.
  3. Click More on the right of a user to be locked and choose Lock.
  4. In the dialog box that is displayed, click OK.
