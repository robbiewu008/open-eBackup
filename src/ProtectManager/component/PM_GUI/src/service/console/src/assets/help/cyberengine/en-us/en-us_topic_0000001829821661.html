Managing Alarms and Events

Alarms and events record the faults and events that occur during the running of storage devices. When a storage device is faulty, maintenance personnel can locate and rectify the fault by viewing information of alarms and events.

Context

Maintenance personnel can export the desired event information by specifying the severity and occurrence time.

Internet Explorer on Windows is used as an example here. If another browser is used, perform related operations based on site requirements.

Precautions

Exported events are saved in a *.xls file. Do not change the content of the file.

Procedure

  1. Choose Insight > Alarms and Events.

    The Alarms and Events page is displayed.

  2. View alarms.

    Click the Alarms tab to view alarms. Alarms of the Critical, Major, and Warning severities are displayed.

  3. View and export events.

    1. Click the Events tab. In the list, select events that you want to export.
      • The Events tab page includes Critical, Major, Warning, and Info events. Alarms on the Alarms tab page are also included in Events.
      • You can export all or specified entries on the Events tab page.
      • To export storage device alarms or events of a specific severity, set the filter criteria before exporting them.
    2. Click .

      The Save As dialog box is displayed. Select a save path and click Save.
