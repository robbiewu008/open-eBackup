Performance Indicator Introduction

This section introduces performance indicators so that you can know about the running state of the system by checking the performance indicators.

The following describes major performance indicators:

  • I/O response time: Time required to process a request within a statistical period, including the waiting time.
  • Input/output operations per second (IOPS): IOPS is a performance indicator that indicates the number of I/Os that can be processed per second. IOPS is used to measure I/O processing capabilities of storage systems.
  • Bandwidth: Bandwidth indicates the amount of data that can be processed per second, and the basic unit is MB/s or GB/s. It measures the storage system throughput.
