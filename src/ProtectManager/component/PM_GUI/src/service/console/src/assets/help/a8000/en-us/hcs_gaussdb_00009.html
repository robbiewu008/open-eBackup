Preparing for the Backup

Before the backup, prepare information such as certificates, accounts, and IP addresses by performing operations described in Table 1.

Table 1 Preparing for the Backup

Item

Mandatory or Not

How to Obtain

To Be Used In

Address of the Huawei Cloud Stack GaussDB management plane

Yes
  1. On the Basic_Parameters sheet of xxx_export_all_EN exported by the Huawei Cloud Stack deployment tool, search for region_id and external_global_domain_name to obtain their values.
  2. On the Portal sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, obtain the URL, username, and password for logging in to Service OM.
  3. On Service OM, click VM.
  4. Obtain any IP address of DBS-GaussCore on the VM page.
  5. Log in to the target host using the IP address through PuTTY.
  6. Run the ping iam-apigateway-proxy.{region_id}.{external_global_domain_name} command to obtain the management plane IP address.

  7. Check the IAM version.
    1. On the Portal sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, obtain the COP website, and the username and password for logging in to COP.
    2. Choose CloudCMDB > Service Mgmt, and search for IAM in the service tree. If IAM exists, IAM 2.0 is used. If IAM does not exist, IAM 1.0 is used.
  8. Obtain the management plane address.
    • IAM 1.0: https://XX.XX.XX.XX:26335 indicates the management plane address.
    • IAM 2.0: https://XX.XX.XX.XX:443 indicates the management plane address.

Step 1: Registering a Huawei Cloud Stack GaussDB Project

Address of the Huawei Cloud Stack GaussDB service plane

Yes

On the Tool_generated_IP_Params sheet of the xxx_export_all_EN file exported by the Huawei Cloud Stack deployment tool, search for AGW-LB-Float-IP to obtain the corresponding parameter value XX.XX.XX.XX. https://XX.XX.XX.XX indicates the service plane address.

IAM user and password

Yes

IAM user created on Huawei Cloud. For details about how to create an IAM user, see Create User Groups and Assign Permissions and Create IAM Users and Log In.

Resource set name and resource set ID

Yes
  1. Log in to ManageOne Operation Portal.
  2. Click the username in the upper right corner and click My Settings. The Name and ID columns in the resource set list present the required resource set names and IDs.

XBSA configuration path

Yes

Path to the xbsa.conf file configured during backUpAgent installation.

For details about how to install backUpAgent, see "Installing backUpAgent" in the OceanProtect Appliance 1.5.0-1.6.0 ProtectAgent Installation Guide.

Certificate

No

The certificate is required only when XBSA SSL Certificate Management is enabled. For details about how to generate and configure a certificate, see "Configuring Certificate Files for Mutual SSL Authentication" in the OceanProtect Appliance 1.5.0-1.6.0 ProtectAgent Installation Guide
