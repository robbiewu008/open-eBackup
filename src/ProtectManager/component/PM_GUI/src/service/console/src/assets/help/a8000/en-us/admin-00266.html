Deleting an SFTP User

If an SFTP user is no longer needed to transfer and store data and the user data can be deleted, you can remove the user.

Prerequisites

The SFTP service has been enabled.

Precautions

  • Before deleting an SFTP user, ensure that the user is not using the SFTP service to transfer files.
  • If a user is deleted, the user data will be deleted and the SFTP service becomes unavailable.

Procedure

  1. Choose System > Settings.
  2. In the navigation tree, click SFTP Service.
  3. Click More on the right of a user to be deleted and choose Delete.
  4. In the dialog box that is displayed, click OK.
