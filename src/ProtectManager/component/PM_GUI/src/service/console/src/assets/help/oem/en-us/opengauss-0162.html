Managing openGauss/CMDB Clusters

The product allows you to delete, authorize, or reclaim clusters.

You need to log in to the management page, choose Protection > Databases > openGauss, click the Cluster tab, and find the target cluster.

Table 1 describes related operations.

Table 1 Cluster-related operations

Operation

Description

Navigation Path

Modifying

Scenario

This section describes how to modify the cluster name, type, and node information.

Choose More > Modify in the row where the cluster is located.

Deleting

Scenario

You can delete a host or cluster if its databases no longer require protection. After the host or cluster is deleted, its information disappears from the WebUI.

Note

A protected database or instance in a cluster cannot be deleted.

In the row where the target host or cluster resides, choose More > Delete.

Authorizing resources

Scenario

By default, only the system administrator has the permission to protect resources. If you want to assign this permission to a data protection administrator, authorize the data protection administrator as the system administrator to protect resources.

Note

  • Users cannot be authorized to protect resources that have been associated with SLAs. You need to remove the protection of resources before performing authorization.
  • A single resource can be authorized only to a single data protection administrator.

Locate the row that contains the target host or cluster, choose More > Authorize Resource, and select the data protection administrator to be authorized.

Reclaiming resources

Scenario

This operation enables you to reclaim resources authorized to the data protection administrator. The data protection administration is not permitted to protect the resources that have been reclaimed.

Note

Users are not allowed to reclaim resources that have been associated with SLAs. You need to remove the protection of resources before reclaiming them.

In the row where the target host or cluster resides, choose More > Reclaim Resource.

Test Connectivity

Scenario

This operation enables you to test the connectivity between the product and the cluster.

In the row of the target host or cluster, choose More > Test Connectivity.

Resource Scan

This operation enables you to immediately update changes to the product.

In the row of the target host or cluster, choose More > Resource Scan.
Authorizing resources
NOTE:

Only version 1.5.0 supports this function.

Scenario

By default, only a system administrator has the permission to protect resources. However, a system administrator can grant this permission to a data protection administrator.

Note

  • Permissions on resources that have been associated with SLAs cannot be granted. Remove protection before authorization.
  • The permission on a single resource can be granted to only one data protection administrator.

In the row of the target host or cluster, choose More > Authorize Resource, and select a data protection administrator to grant resource permissions.
Reclaiming resources
NOTE:

Only version 1.5.0 supports this function.

Scenario

This operation enables you to reclaim resources authorized to a data protection administrator. The data protection administration is not permitted to protect the resources that have been reclaimed.

Note

Resources that have been associated with SLAs cannot be reclaimed. You need to remove the protection for resources before reclaiming them.

In the row of the target host or cluster, choose More > Reclaim Resource.
Adding a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to quickly filter and manage resources.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.
Removing a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to remove the tag of a resource when it is no longer needed.

Choose More > Remove Tag.
