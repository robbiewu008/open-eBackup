Step 7: (Optional) Creating a Cloud Server Group

A cloud server group is a group consisting of one or more cloud servers in a project. You can use a cloud server group to protect data of multiple specified cloud servers.

Procedure

  1. Choose Protection > Cloud Platforms > OpenStack.
  2. In the resource pane on the left of the OpenStack page, select the project where cloud servers for which a cloud server group is to be created reside, and click the Cloud Server Groups tab.

  3. Click Create and set a custom cloud server group name.
  4. On the left of the Select Resource area, select cloud servers for which you want to create a cloud server group. Then, the selected cloud servers are automatically added to the area of the selected cloud servers on the right. You can click to remove a cloud server from the area of the selected cloud servers, or click Delete All to remove all the selected cloud servers.

  5. After selecting the cloud servers for creation, click OK.
