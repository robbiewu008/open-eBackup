Managing Live Mount of a NAS File System

You can perform management operations related to live mount of a NAS file system copy.

You need to log in to the OceanProtect WebUI, and choose Explore > Live Mount > Application Type > NAS File Systems.

Table 1 describes the related operations.

Table 1 Operations related to live mount

Operation

Description

Navigation Path

Modifying mount information

Scenario

If parameters such as Bandwidth (MB/s), Normalized IOPS (8 KB), and Normalized Latency (8 KB) of a mount copy are not configured when a live mount job is created, this operation enables you to configure the parameters or modify the configured parameters.

Choose More > Modify in the row where a live mount job is located.

Unmounting a mount copy

Scenario

This operation enables you to unmount a mounted copy from the target client.

Note

  • During unmounting, you need to determine whether to retain the copy generated by live mount. If you choose to retain the mounted copy, the mounted copy is retained permanently. Otherwise, if you delete the mount copy, the file system where the mounted copy resides is also deleted.
  • After unmounting, the mounted NAS share will be deleted.
  • Forcibly delete after destruction failure

Choose More > Destroy in the row where a live mount job is located.

Viewing mount details

Scenario

This operation enables you to view the details of a live mount.

Choose More > View in the row where a live mount job is located.

View details about resources and copy data.
