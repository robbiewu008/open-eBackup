Preparing for the Backup

Before backup, prepare information such as the certificate, account, and Keystone V3 address by following instructions described in Table 1.

Table 1 Preparations for backup

Item

Mandatory or Not (Y/N)

How to Obtain

To Be Used In

Keystone V3 address

Y

Keystone V3 address used for registering OpenStack on the product. For details about how to obtain the IP address, see Step 1: Obtaining the Keystone V3 Address.

Step 5: Registering OpenStack

Username

Y

User for interconnecting OpenStack with the product. Create a user by referring to Step 3: Creating a User for Interconnection.

Password

Y

Certificate

N

When registering an OpenStack protected environment, you are advised to import the CA certificate of the protected environment. Otherwise, the system cannot verify the information about the accessed devices, resulting in security risks.

For details about how to obtain the certificate, see Step 2: Obtaining the Certificate.

Certificate revocation list

N

Issued by the certificate authority.

If a certificate is revoked due to security risks, import it into the revocation list.

Domain administrator

Y

Create a domain administrator for each protected domain (except Default and heat_stack_domain) on FusionSphere OpenStack. Default and heat_stack_domain are the default domains of OpenStack. You do not need to create a domain administrator for them.

For details, see Step 4: Creating a Domain Administrator.
