Step 5: Executing Backup

Before performing backup, you need to associate the resources to be protected with a specified SLA policy. The system protects the resources based on the SLA policy and periodically performs backup jobs based on the SLA policy. You can perform a backup job immediately through manual backup.

Precautions

Copies generated by manual backup are retained for the period defined in the SLA.

Procedure

  1. Choose Protection > Big Data > MongoDB.
  2. Select the instance to be protected and click Protect.

    You can also select multiple instances for batch protection.

  3. (Optional) Configure the percentage of the space required for creating a snapshot logical volume to that of the original logical volume in a backup job in the Advanced area.
  4. Select an SLA.

    You can also click Create to create an SLA.

  5. Set Percentage of Space for Created Logic Volume in the Advanced area.

    Percentage of the space required for creating a snapshot logical volume to that of the original logical volume in a backup job. Ensure that the percentage of free space in the original logical volume is greater than the parameter value.

  6. Click OK.

    If the current system time is later than the start time of the first backup set in the SLA, you can choose to perform the backup immediately or wait for the next scheduling based on the backup policy set in the SLA after the backup SLA is associated with the resources.

  7. Execute manual backup.

    If you want to execute a backup job immediately, perform manual backup through the following operations. Otherwise, skip this step.

    1. Choose More > Manual Backup in the row where the instance to be backed up is located.
    2. Set the name of the copy generated through manual backup.
    3. If this parameter is left unspecified, the system sets the name to backup_Timestamp by default.
    4. Select a protection policy, which can be Full Backup or Log Backup.
      1. Log backup is not supported for a single MongoDB instance whose oplog is not enabled.
    5. Click OK.
