步骤2：开启数据库本地归档

Dameng在执行备份任务前，需要开启Dameng数据库本地归档，否则会导致备份失败。

注意事项

  • 本节操作如无特殊说明均以Dameng 8版本为例进行，数据库版本不同操作可能存在差异，请以实际为准。
  • 对于以集群方式部署的Dameng数据库，如无特殊说明，均需登录集群实例所在的所有主机执行以下操作。

操作步骤

  1. 登录Dameng数据库主机。
  2. 执行以下命令查询dm.ini配置文件所在路径。

    ps -ef | grep dm.ini

  3. 进入配置文件所在路径，执行以下命令，打开并编辑配置文件。

    vim dm.ini

  4. 在配置文件的#configuration file区域，修改ARCH_INI配置项的值为1。

    ARCH_INI = 1

  5. 执行以下命令，查询dmarch.ini配置文件所在路径。

    find / -name dmarch.ini

  6. 进入配置文件所在路径，执行以下命令，打开并编辑配置文件。

    vim dmarch.ini

  7. 修改配置文件中的以下内容。

    [ARCHIVE_LOCAL1]
ARCH_TYPE = LOCAL 
ARCH_DEST = 归档文件路径
ARCH_FILE_SIZE = 128 
ARCH_SPACE_LIMIT = 0

    归档文件路径请根据实际路径替换。
