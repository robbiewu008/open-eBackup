Managing Storage Device Information

After a storage device is added to the product, you can perform operations on the device.

Choose Protection > File Systems > Storage Devices to go to the Storage Device page.

For 1.5.0, choose Protection > File Services > Storage Device.

Table 1 describes the operations.

Table 1 Operations on the storage device

Operation

Description

Navigation Path

Modifying storage device information

Scenario

When the IP address, port number, username, or password of an added storage device is changed, perform this operation to synchronize the change.

Note

  • Modify the storage device information when no backup or restoration job is in progress. Otherwise, the job fails.
  • You can only change the IP address to another management IP address of the same storage device.

Choose More > Modify.

Table 1 describes the storage device parameters.

Deleting a storage device

Scenario

If you do not need to protect NAS file systems or NAS shares on a storage device or use the device as the target of restoration, you can delete the storage device.

Note

  • If a backup or scanning job is in progress, the storage device cannot be deleted. Wait until the job is complete.
  • After a storage device is deleted, NAS file systems and NAS shares associated with the storage device are also deleted.

Choose More > Delete.

To delete storage devices in batches, select multiple storage devices and click Delete in the upper part of the page.

Rescanning storage devices

Scenario

The product automatically scans storage devices every hour. If NAS file systems or NAS shares are added, deleted, or modified on a registered storage device, data is updated to the product after the scanning. This operation immediately updates changed NAS file systems or NAS shares to the product. By default, the system automatically refreshes storage devices every hour.

Note

The product does not automatically associate newly discovered NAS file systems or NAS shares with SLAs.

Choose More > Resource Scan.
Adding a tag

NOTE:

Only 1.6.0 and later versions support this function.

This operation enables you to remove the tag of a resource when it is no longer needed.
