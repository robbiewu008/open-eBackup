Obtaining the kubeconfig Configuration File

Procedure

  1. Log in to the FusionCompute Web Client.
  2. Click Container Management and click to switch to the containerized application management page.
  3. On the K8s Clusters page, click the name of the cluster whose configuration file is to be obtained to switch to the cluster management page.
  4. Click Connections. On the displayed page, click download the kubeconfig file for the cluster to obtain the configuration file.

    kubeconfig can be downloaded only when the Kubernetes cluster is connected for the first time.
