Deleting an External Cluster

The OceanProtect allows you to delete external clusters. After an external cluster is deleted, the local cluster cannot communicate with the external cluster.

Precautions

  • An external cluster that has been set as the replication target cluster of the management cluster cannot be deleted.
  • The management cluster cannot be deleted.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  1. In the Multi-domain Clusters area, choose More > Delete in the row that contains the external cluster to be deleted.
  2. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
