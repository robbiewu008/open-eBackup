Enabling Real-Time Ransomware Detection

This section describes how to enable real-time ransomware detection.

Precautions

  • If LUNs for which you want to enable real-time ransomware detection are configured with the HyperMetro feature, enable real-time ransomware detection for the LUNs at both the primary and secondary ends.
  • Real-time ransomware detection can be enabled for a maximum of 1024 LUNs.

Procedure

  1. On the navigation pane, choose Explore > Ransomware Protection > Ransomware Detection.
  2. Click the SAN Ransomware Detection tab, select the LUN for which you want to enable real-time ransomware detection, and choose More > Enable Real-time Ransomware Detection.

    • You can also select multiple LUNs for which you want to enable real-time ransomware detection and click Enable Real-time Ransomware Detection above the list.
    • If real-time ransomware detection is not required, locate the row that contains the LUN for which you want to disable real-time ransomware detection and choose More > Disable Real-time Ransomware Detection. Alternatively, select multiple LUNs for which you want to disable real-time ransomware detection and choose More > Disable Real-time Ransomware Detection above the list.

  3. Confirm your operation as prompted.

    In the dialog box for confirmation that is displayed, you can determine whether to enable ransomware detection false alarm analysis and click OK.

    Ransomware detection false alarm analysis is enabled by default. After this function is enabled, the accuracy of ransomware detection is further improved, but the detection time is expected to increase by 30 minutes.

    • If you require high detection accuracy, you are advised to enable ransomware detection false alarm analysis.
    • If you require short detection duration, you are advised to disable ransomware detection false alarm analysis.
