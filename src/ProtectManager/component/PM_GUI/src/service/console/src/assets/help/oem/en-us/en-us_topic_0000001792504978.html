Marking a Tape as Empty

If you no longer need to use the data on a tape, you can mark the tape as empty. After a tape is marked as empty, the tape still contains data, but the write status of the tape changes to Unused. In this case, the tape can be overwritten.

Prerequisites

A tape resides in the tape library.

Precautions

Tapes whose write status is Unknown or Error cannot be marked as empty.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Tape tab.
  4. In the row where the tape to be marked as empty resides, choose More > Mark as Empty.
  5. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
