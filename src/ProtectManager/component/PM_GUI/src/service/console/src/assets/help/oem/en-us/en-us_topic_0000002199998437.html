Precautions for GDS Running on Other Nodes

Procedure

For external nodes where GDS is running and other GaussDB (DWS) nodes where GDS is not running and the client has been installed:

No further operation is required. For details about how to start GDS, see "Installing, Configuring, and Starting GDS" in the GaussDB (DWS) Product Documentation of the corresponding version.
