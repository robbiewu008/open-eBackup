Performing Periodic Backup

After associating the local file systems to be protected with a specified SLA, the storage system periodically backs up the file systems based on the SLA.

Procedure

  1. Choose Protection > Storage > Local File System.
  2. In the local file system list, select the file systems that you want to protect and choose More > Protect.

    You can also select multiple local file systems for batch protection.

  3. Select an SLA.

    You can also click Create to create an SLA. For details about how to create an SLA, see Step 3: Creating a Backup SLA.

  4. Click OK.

    If the current system time is later than the first backup start time of the SLA, you can perform manual backup immediately after the backup SLA is associated or wait for the next scheduling based on the backup policy set in the SLA.
