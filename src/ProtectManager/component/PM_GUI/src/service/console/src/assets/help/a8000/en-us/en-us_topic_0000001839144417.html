Viewing Cluster Information

This section describes operations to view the cluster list and cluster basic information.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click the Replication Clusters tab. The replication cluster page is displayed.
  3. (Optional) Click next to Role or Status to filter clusters. Alternatively, click next to Name or IP Address to search for a cluster.
  4. In the cluster list, view the basic information about the cluster.
