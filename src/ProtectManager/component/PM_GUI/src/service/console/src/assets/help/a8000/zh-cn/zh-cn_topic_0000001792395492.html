查看华为云Stack信息

当华为云Stack注册至OceanProtect后，您可以查看华为云Stack的租户、项目/资源集和弹性云服务器的信息。

操作步骤

  1. 选择“保护 > 云平台 > 华为云Stack”。
  2. 在“租户”页签浏览租户信息。

    租户的详细信息如表1所示。
    表1 租户信息

    参数

    说明

    名称

    租户名称。

    单击租户名称，可以查看租户下的VDC管理员信息。

    项目/资源集数量

    租户中的项目/资源集数量。

  3. 在“项目/资源集”页签浏览项目/资源集信息。

    项目/资源集的详细信息如表2所示。
    表2 项目/资源集信息

    参数

    说明

    名称

    项目/资源集名称。

    单击项目/资源集名称，可以查看项目/资源集的总览、任务。

    • 总览

      显示名称、SLA信息、任务执行时间、副本总数、弹性云服务器信息等。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    位置

    项目/资源集所在位置。

    SLA

    当项目/资源集已关联SLA时，则显示SLA名称，否则显示“--”。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态

    项目/资源集当前是否被保护。

    • 未保护：项目/资源集未关联SLA，或者项目/资源集已关联SLA但已禁用保护。
    • 已保护：项目/资源集已关联SLA，且已激活保护。
    • 创建中：项目/资源集已关联SLA，正在创建保护。

    所属用户

    将资源分配给某个数据保护管理员管理。当界面显示“--”时，表示还未将资源分配给数据保护管理员。

  4. 在“云服务器”页签浏览弹性云服务器信息。

    弹性云服务器的详细信息如表3所示。
    表3 弹性云服务器信息

    参数

    说明

    名称

    弹性云服务器名称。

    单击名称，可以查看弹性云服务器的总览、副本数据、任务信息。

    • 总览

      显示名称、SLA信息、任务执行时间、副本总数、云硬盘信息等。

    • 副本数据

      您可以依次按照年、月、天查找相应时间的副本。

      时间下方显示，表示该时间存在副本。单击日期进入副本数据页面。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    状态
    弹性云服务器状态。包括在线、离线、异常、软删除。
    • 在线：对应华为云Stack中的弹性云服务器处于运行中、正在关机。
    • 离线：对应华为云Stack中的弹性云服务器处于正在开机、关机、重装操作系统中、重装操作系统失败、切换操作系统中、切换操作系统失败。
    • 异常：对应华为云Stack中的弹性云服务器处于异常状态。
    • 软删除：对应华为云Stack中的弹性云服务器已被删除至回收站中。

    位置

    弹性云服务器所在位置。

    SLA

    弹性云服务器关联的SLA。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态

    弹性云服务器当前是否被保护。

    • 未保护：弹性云服务器未关联SLA，或者弹性云服务器已关联SLA但已禁用保护。
    • 已保护：弹性云服务器已关联SLA，且已激活保护。
    • 创建中：弹性云服务器已关联SLA，正在创建保护。
