查看VMware虚拟化环境信息

当VMware虚拟化环境注册至OceanProtect后，您可以在管理界面查看VMware集群、主机、虚拟机或虚拟机组的详细信息。

背景信息

不同版本的操作界面可能会有少许差异，操作时请以实际情况为准。

操作步骤

  1. 选择“保护 > 虚拟化 > VMware”。

    VMware虚拟化环境总览信息如图1所示。

    图1 VMware虚拟化环境总览信息

  2. 在界面左侧选中需要查看的VMware虚拟化环境，单击“更多 > 环境信息”，查看VMware虚拟化环境基本信息。
  3. 在“集群”页签浏览集群信息。

    集群的详细信息如表1所示。

    部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

    表1 集群信息

    参数

    说明

    名称

    集群名称。

    SLA

    当资源已关联SLA时，则显示SLA名称，否则显示“--”。

    保护状态
    当前资源是否被保护。
    • 未保护：当前资源未关联SLA，或者已关联SLA但已禁用保护。
    • 已保护：当前资源已关联SLA，且已激活保护。
    • 创建中：当前资源已关联SLA，正在创建保护。

    虚拟机数量

    当前集群下的虚拟机数量。

    单击虚拟机数量，界面展示该集群下的主机列表、虚拟机列表和虚拟设备节点。
    标签
    说明：

    仅1.6.0及后续版本存在此参数。

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。

  4. 在“主机”页签浏览ESXi主机信息。

    ESXi主机的详细信息如表2所示。

    部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

    表2 主机信息

    参数

    说明

    名称

    ESXi主机名称。

    单击主机名称，界面展示虚拟机列表和虚拟设备节点。

    当ESXi主机属于某集群时，此处不显示虚拟机列表。

    虚拟设备节点为虚拟磁盘所在的槽位号，表示方式为：磁盘接口类型(总线编号:槽位号)，如IDE(0:0)。

    位置

    ESXi主机所在路径。

    状态

    ESXi主机处于“在线”或“离线”状态。

    SLA

    当主机已关联SLA时，则显示SLA名称，否则显示“--”。

    保护状态

    主机当前是否被保护。

    • 未保护：主机未关联SLA，或者主机已关联SLA但已禁用保护。
    • 已保护：主机已关联SLA，且已激活保护。
    • 创建中：主机已关联SLA，正在创建保护。

    虚拟机数量

    ESXi主机上的虚拟机数量。

    如果当前主机是集群成员，则此处显示主机的虚拟机数量为0，需要在“集群”页签查看虚拟机数量。
    标签
    说明：

    仅1.6.0及后续版本存在此参数。

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。

  5. 在“虚拟机”页签浏览虚拟机信息。

    1. 虚拟机的详细信息如表3所示。

      部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

      表3 虚拟机信息

      参数

      说明

      名称

      虚拟机名称。

      位置

      虚拟机所在路径。

      状态

      虚拟机处于开机或关机状态。

      操作系统

      虚拟机所在主机的操作系统类型。

      标记

      虚拟机的标记。

      当您已在VMware虚拟化环境中为虚拟机设置标记，则此处显示虚拟机标记，否则显示“--”。

      您可以通过标记过滤虚拟机。

      SLA

      当虚拟机已关联SLA时，则此处显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      虚拟机当前是否被保护。

      • 未保护：虚拟机未关联SLA，或者虚拟机已关联SLA但已禁用保护。
      • 已保护：虚拟机已关联SLA，且已激活保护。
      • 创建中：虚拟机已关联SLA，正在创建保护。

      虚拟机组
      • 如果虚拟机通过“直接选择”方式加入虚拟机组，此处直接显示该虚拟机组名称。
      • 如果虚拟机通过“指定规则后由系统自动选择”方式加入虚拟机组，此处仅显示最新被保护且SLA与该虚拟机SLA一致的虚拟机组的名称；如果所有虚拟机组都未被保护，则显示“--”。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
    2. 单击虚拟机名称，浏览虚拟机总览信息、副本数据以及相关任务。
      • 虚拟机总览

        显示虚拟机的名称、SLA信息、任务执行时间、副本总数、硬盘信息等。

      • 虚拟机副本数据

        您可以依次按照年、月、天查找相应时间的副本。

        在“年”或“月”页签中，当出现时，表示对应时间存在副本。

      • 虚拟机相关任务

        虚拟机相关的所有任务展示，包括备份、恢复、即时挂载、归档等任务。

  6. 在“虚拟机组”页签浏览虚拟机组信息。

    仅1.6.0及后续版本支持此功能。

    虚拟机组的详细信息如表4所示。
