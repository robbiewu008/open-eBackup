Viewing Local Storage Authentication Information

This section describes how to view the local storage authentication information in the Authentication Information area.

Procedure

  1. Choose System > Infrastructure > Local Storage.
  2. In the Authentication Information area on the Local Storage page, view the local storage authentication information. Authentication information includes login authentication and service authentication. Login authentication is used for login to DeviceManager. Service authentication is used by the system to perform service operations related to data protection.

    • Login authentication
      • Username: indicates the username of the local storage used for login authentication. Use a user who has the super administrator permissions.

        By default, the maximum number of sessions per user is not limited. Do not change the value unless necessary.

      • Password: Login authentication user password.
      • Status: User authentication status. If the authentication status is Exception, modify authentication information.
    • Service authentication
      • Username: indicates the username of the local storage used for service authentication. Use a user who has the super administrator permissions.

        By default, the maximum number of sessions per user is not limited. Do not change the value unless necessary.

      • Password: Service authentication user password.
      • Status: User authentication status. If the authentication status is Exception, modify authentication information.
