创建报表

本产品提供了报表功能，您可以通过报表查阅系统中关键资源（如存储容量使用情况）状态及关键任务（如备份、恢复）的运行情况。本节介绍如何创建报表。

操作步骤

  1. 选择“报表”。
  2. 单击“创建”。参数说明如表1所示。

    表1 创建报表参数说明

    参数

    说明

    报表名称

    自定义报表的名称。

    报表类型

    创建报表的类型。

    • 存储空间报表：统计本产品的总容量、已使用容量及未使用容量。
    • 备份作业报表：统计备份任务的信息。
    • 恢复作业报表：统计恢复任务的信息。
    • 恢复演练作业报表：统计恢复演练任务的信息。
    • 主机资源使用报表：统计主机资源的使用信息。
    • 资源使用报表：统计资源的使用信息。
    • 磁带使用报表：统计磁带的使用信息。
    说明：
    • 1.6.0及后续版本支持创建“主机资源使用报表”、“资源使用报表”、“磁带使用报表”。
    • 1.6.0及后续版本“备份作业报表”展示为“保护作业报表”。
    • 1.6.0及后续版本“保护作业报表”包含备份、归档、复制任务信息，1.6.0之前版本“备份作业报表”仅包含备份任务信息。
    • 对于本产品，如果需要创建存储空间报表，请确保已在“性能”页面打开对应节点的性能监控开关。

    数据源

    选择本地存储或者已经接入本产品的外部集群。

    说明：

    对于1.6.0及后续版本，用户在创建保护作业报表、恢复作业报表、恢复演练作业报表和磁带使用报表时，如不设置“数据源”，生成的报表包含所有节点的所有任务信息；如设置“数据源”，生成的报表仅包含分发到所选节点的任务信息，但不包含分发失败的任务信息。

    统计范围

    统计某个时间范围内的数据。仅“备份作业报表”、“恢复作业报表”和“恢复演练作业报表”需要设置该参数。

    • 最近一周：统计最近一周的数据。
    • 最近一月：统计最近一月的数据。
    • 最近三月：统计最近三月的数据。
    • 自定义：自定义统计的开始时间和结束时间。
      说明：
      • 统计范围不能超过365天。
      • 1.6.0及后续版本“备份作业报表”展示为“保护作业报表”。

    统计频率

    报表的统计频率。仅“备份作业报表”、“恢复作业报表”和“恢复演练作业报表”需要设置该参数，系统将根据您选择的统计范围设置默认的统计频率：

    • 统计范围小于或等于10天，统计频率设置为1天；
    • 统计范围大于10天，且小于或等于30天，统计频率设置为5天；
    • 统计范围大于30天，且小于或等于90天，统计频率为10天或30天；
    • 统计范围大于90天，且小于或等于365天，统计频率设置为30天。
    说明：

    1.6.0及后续版本“备份作业报表”展示为“保护作业报表”。

    报表格式

    选择报表的格式。当前版本支持xls和pdf两种格式。
