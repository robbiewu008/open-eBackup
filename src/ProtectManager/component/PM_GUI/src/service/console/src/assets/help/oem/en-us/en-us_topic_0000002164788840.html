Modifying External Cluster Information

The product allows you to modify external cluster information. If you have changed the IP address and port number of the external cluster, you need to modify the external cluster information on the local cluster accordingly to ensure that the local cluster is properly connected to the external cluster.

Precautions

The role of an external cluster cannot be changed.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  1. In the Multi-domain Clusters, choose More > Modify in the row that contains the external cluster to be modified.
  2. Modify external cluster parameters.

    Table 1 describes the parameters.

  3. Click OK.
