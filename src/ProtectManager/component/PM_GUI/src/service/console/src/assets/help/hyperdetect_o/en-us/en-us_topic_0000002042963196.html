Scanning for File Systems

After a file name extension filtering rule is applied to a file system of a vStore, the system automatically performs file system scanning every hour. If a new file system is detected under the vStore, the file name extension filtering rule of the current vStore is synchronized to the new file system. After a file system is newly added, you can manually scan the file system and synchronize the file name extension filtering rules of the current vStore.

Procedure

  1. On the navigation pane, choose Data Security > Ransomware Protection > Ransomware Detection Settings.
  2. On the NAS Ransomware Detection tab page, click Scan.
  3. In the displayed dialog box, click OK.
