步骤9：（可选）登录iSCSI启动器

在以太网组网场景下，如果想要使用SAN传输模式或基于OceanStor Dorado LUN建立的Datastore的虚拟机磁盘需要使用存储层备份模式时，请参考本章节在代理主机上登录iSCSI启动器。

前提条件

代理主机已安装iSCSI启动器（iscsi-initiator-utils）。具体操作请参见《ProtectAgent安装指南》中的“安装iSCSI启动器”章节。

操作步骤

  1. 登录ESXi Datastore对应的生产存储设备，查询iSCSI逻辑端口，获取逻辑端口IP。

    1. 登录DeviceManager。
    2. 选择“服务 > 网络 > 逻辑端口”。
    3. 在信息显示区，查询“数据协议”为iSCSI的逻辑端口，记录逻辑端口IP。

  2. 在代理主机上，依次执行以下命令，登录iSCSI启动器（“IP”为1获取的逻辑端口IP）。

    iscsiadm -m discovery -t st -p IP
iscsiadm -m node -l -p IP

  3. 在代理主机上，执行iscsiadm -m session命令查看session信息，获取IQN。

  4. 在ESXi Datastore对应的生产存储设备上确认IQN是否在线。

    1. 登录DeviceManager。
    2. 选择“服务 > 块服务 > 主机组 > 启动器 > iSCSI”。
    3. 在“iSCSI”页签，查询3获取的IQN，当IQN状态为在线时，表示IQN状态为正常。
