备份任务失败，错误详情提示客户端执行挂载失败

现象描述

备份虚拟机任务执行失败，错误详情中提示客户端执行挂载失败。

可能原因

Redhat7.6版本内核问题导致。

处理步骤

  1. 使用PuTTY工具以root账户登录待安装客户端的主机。
  2. 执行以下命令，查看客户端所在主机操作系统内核版本是否为3.10.0-957.5.1.el7及以上。

    cat /etc/redhat-release

    uname -r

    • 是，联系技术支持协助处理。
    • 否，请继续执行3。

  3. 请自行下载3.10.0-957.5.1.el7及以上版本的Redhat内核包，并将下载到的内核包上传至客户端所在主机。
  4. 进入安装包所在目录后，执行以下命令安装内核包。

    rpm -ivh 内核包名

    示例：

    rpm -ivh kernel-3.10.0-957.5.1.el7.x86_64.rpm

  5. 重启操作系统后，再次执行以下命令查看内核版本是否为新安装的版本。

    uname -r
