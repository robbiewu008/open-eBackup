Global Search for Resources

This section describes how to perform a global search for resources.

Procedure

  1. Click on the right of the navigation bar. The Global Search page is displayed.

    • Search for resources by resource name.
      1. Select Resources in the search box.
      2. Enter a resource name and click . The resource details are displayed.
    • Perform advanced search.
      1. Click Advanced Search.
      2. Set search criteria on the Advanced Search page.
        Table 1 Advanced search for resources

        Parameter

        Description

        Keyword

        Select Resources and enter a resource name.

        Protection Status

        Filter resources based on the resource protection status.

        SLA Compliance

        Filter data based on whether the SLA is complied with.

        Resource location

        Enter the location of the resource.

        Note: Keyword is mandatory and other parameters are optional. The logic relationship between parameters is AND. That is, the search result is filtered based on all the selected conditions.

  2. The resource search result is displayed on the page.

    Table 2 describes the related parameters.

    Table 2 Resource parameters

    Parameter

    Description

    Name

    Resource name.

    You can click the name of a resource to view its protection information or select a copy for restoration.

    Type

    Resource type.

    Location

    Resource location.

    SLAs

    Whether a resource has been associated with an SLA.

    SLA Compliance

    SLA compliance indicates whether a backup job is successfully executed based on the specified backup interval or time window. For example, if the backup time of a backup job (except the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met. If the backup interval set in the SLA for a backup job is once every half an hour but the backup duration exceeds half an hour, SLA compliance is missed.

    Protection Status

    Resource protection status.

  3. Locate the row that contains the target resource, and click More to perform operations such as protection and modification on the resource.
