Manually Performing Backup

If you want to perform a backup immediately, you can manually perform the backup.

Context

Copies generated by manual backup are retained for the period defined in the SLA.

Procedure

  1. Choose Protection > Databases > OceanBase.
  2. Choose More > Manual Backup in the row where the target cluster or tenant set is located.
  3. Set the name of the copy generated during manual backup.

    If this parameter is left empty, the system sets the name to backup_Timestamp by default.

  4. Select a protection policy, which can be Full backup, Incremental backup, or Log Backup.

    Tenant sets only support full backup.

  5. Click OK.
