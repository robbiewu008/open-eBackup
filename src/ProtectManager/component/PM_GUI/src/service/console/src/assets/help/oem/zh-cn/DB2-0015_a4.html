注册数据库（适用于集群实例部署形态）

当要保护的数据库为集群实例部署时，请参考本节注册集群实例。

前提条件

注册DB2集群实例前，请确保DB2集群已注册。

操作步骤

  1. 在DB2页面选择“实例”页签。
  2. 单击“注册”。
  3. “类型”选择“集群实例”。
  4. 输入实例名，选择集群。
  5. 单击“添加”，在弹出的添加对话框中添加集群实例下的所有节点。

    相关参数说明如表1所示。

    表1 集群实例的节点信息

    参数

    说明

    主机

    选择集群下的主机节点。

    用户名

    数据库用户名。

    密码

    数据库用户密码。

  6. 单击“确定”。
