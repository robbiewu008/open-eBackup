Manually Executing Backup

If you want to perform a backup immediately, you can manually perform the backup.

Context

  • During manual backup, parameters (except Automatic Retry) defined in the SLA are applied to the manual backup job.
  • Copies generated by manual backup are retained for the duration defined in the SLA.

    If the protection policy selected during manual backup is not defined in the SLA, the generated backup copy is retained permanently. If full backup is not defined in the SLA but selected during manual backup, the copies generated by the manual full backup are retained permanently.

  • If a replication or archive policy has been defined in the SLA, the system will perform replication or archiving once based on the SLA when you perform manual backup.

Precautions

  • Ensure that the production storage has reserved snapshot space for backup. Reserved snapshot space = Data volume of the source LUN x Percentage of data changes in the backup period. By default, you are advised to reserve 20% of the source LUN capacity for creating snapshots.
  • An ECS can be backed up only when it is in the following states: Running, Starting, Stopping, Stopped, Reinstalling OS, Reinstalling OS failed, Changing OS, and OS Change failed.
  • The default number of concurrent backup jobs on each ProtectAgent is 20. You are advised not to change the number. Otherwise, disk scanning may fail, causing backup failures.
  • If ECSs or EVS disks on Huawei scale-out block storage are migrated to SAN storage, you must register SAN storage resources with the OceanProtect before backing up ECSs or EVS disks. For details about the registration, see 3 in Step 2: Registering Huawei Cloud Stack.
  • Before the first backup, complete resource scanning so that the latest ECS information can be displayed. For details, see "Managing an Agent Host" in the OceanProtect Appliance 1.5.0-1.6.0 ProtectAgent Installation GuideProtectAgent Installation Guide.
  • If a snapshot fails to be deleted due to a backup job exception, the system automatically deletes unnecessary snapshots when the next full backup job is performed.

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack.
  2. Click the Project/Resource Set or Cloud Server tab based on the type of the data to be backed up.

    When you select Project/Resource Set for backup, only ECSs with the same SLA as the Project/Resource Set are backed up. ECSs with different SLAs from the project/resource set will not be backed up.

  3. In the row that contains the project/resource set or ECS to be backed up, choose More > Manual Backup.

    • If the project/resource set or ECS to be protected is not displayed, choose More > Resource Scan in the left pane to update resource information.
    • You can select multiple resources to perform manual backup in batches. Select multiple resources and choose More > Manual Backup above the resources.

  4. Set the name of the copy generated through manual backup.

    If this parameter is left empty, the system sets the name to backup_Timestamp by default.

  5. Select a protection policy, which can be Full backup or Forever Incremental (Synthetic Full) Backup.
  6. Click OK.
