Step 4: Creating an Elasticsearch Backup Set

Create a backup set to be protected in the Elasticsearch cluster, that is, a collection of Elasticsearch indexes.

Precautions

When creating a backup set, you can select only Elasticsearch indexes that have not been added to other backup sets.

Procedure

  1. Choose Protection > Big Data > Elasticsearch.
  2. On the Backup Sets tab page, click Create to create a backup set.

    Table 1 describes related parameters.
    Table 1 Backup set parameters

    Parameter

    Description

    Name

    User-defined backup set name.

    Cluster

    Select the Elasticsearch cluster to be protected and determine whether to enable Automatic Elasticsearch Index Protection.

    • When Automatic Elasticsearch Index Protection is enabled, the system protects all indexes in the cluster, including new indexes.
    • When Automatic Elasticsearch Index Protection is disabled, you need to manually select the indexes to be protected.
      NOTE:

      If protected indexes exist in the cluster, Automatic Elasticsearch Index Protection cannot be enabled.

  3. Click OK.
