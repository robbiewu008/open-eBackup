Deleting a Backup Storage Unit

This section describes how to delete a backup storage unit that is no longer used.

Precautions

  • A backup storage unit that belongs to a local cluster cannot be deleted.
  • A backup storage unit that has been added to a backup storage unit group cannot be deleted.
  • A backup storage unit that contains copies cannot be deleted.
  • A backup storage unit that has been associated with an SLA cannot be deleted.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. Click Backup Storage Units.
  4. In the row that contains the backup storage unit to be deleted, choose More > Delete.
  5. In the Danger dialog box that is displayed, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
