Step 2: Registering the Database of the Kingbase Cluster Instance

Prerequisites

Before registering a Kingbase cluster instance, ensure that each host in the cluster has been registered.

For details, see Procedure.

Procedure

  1. Create a cluster.

    1. On the Kingbase page, click the Cluster tab.
    2. Click Register.
    3. On the Register page, customize the cluster name and select the type and cluster node.
    4. Click OK.

  2. Register the database of the cluster instance.

    1. On the Kingbase page, click the Instance tab.
    2. Click Register.
    3. Set Type to Cluster instance.
    4. Customize an instance name and select a cluster.
    5. Click Add. In the displayed Add dialog box, add all nodes of the cluster instance.

      Table 1 describes the related parameters.

      Table 1 Node information of a Kingbase cluster instance

      Parameter

      Description

      Hosts

      Host node in the cluster.

      Username

      OS username for running the database.

      Database Installation Path

      Path for installing the database to be registered.

      Service Plane IP Address

      IP address of the host for the database service to receive TCP/IP connections.

      Database Port

      Port number used by the database service to listen on TCP/IP connections.

      Database Username

      Name of the database administrator.

      Database Password

      Password of the database administrator.
    6. Click OK.
