Managing Logs

By viewing debug logs, you can understand the error information and prompt information that are triggered when you operate or design a template.

Procedure

  1. Choose System > Log Management.
  2. In the Set Log area, select the node to be modified, click the Log Level drop-down list, and select the log level for the export.

    You can export the debug logs at the selected level (and above).

    You can set log levels in batches. After selecting the levels of logs to be exported, click OK to complete the setting.

  3. In the Log Export area, select the node, controller, and software module whose logs are to be exported from the Node, Controller, and Software Module Name drop-down lists, respectively.

    The exported logs include system debug logs and system running parameters.

  4. Click Export to export debug logs.
