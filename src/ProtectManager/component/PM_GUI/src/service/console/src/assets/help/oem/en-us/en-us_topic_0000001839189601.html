Step1: Enabling the DmAPService

Before performing backup and restoration jobs for Dameng, DmAPService must be enabled. Otherwise, backup and restoration operations cannot be performed.

Procedure

  1. Log in to the Dameng database host.
  2. Run the systemctl status DmAPService command to check whether the DmAPService is enabled.

    If the value of Active is active(running), the DmAPService has been enabled.

    If the value of Active is inactive(dead), the DmAPService is disabled. Run the systemctl start DmAPService command to enable the DmAPService.
