Managing Real-Time Ransomware Detection Whitelists

You can delete a real-time ransomware detection whitelist or disassociate vStores from whitelists.

On the navigation pane, choose Data Security > Ransomware Protection > Real-Time Ransomware Detection > Real-Time Detection Whitelist. Perform operations by following the instructions in Table 1.

Table 1 Operations related to real-time detection whitelists

Operation

Description

Navigation Path

Deleting a ransomware detection whitelist

Context

This operation enables you to delete a real-time ransomware detection whitelist.

Choose More > Delete.

Disassociating from a vStore

Context

This operation enables you to disassociate a real-time ransomware detection whitelist from a vStore.

Choose More > Disassociate from vStore.

Associating with a vStore

Context

This operation enables you to associate a real-time ransomware detection whitelist with a vStore.

Choose More > Associate with vStore.
