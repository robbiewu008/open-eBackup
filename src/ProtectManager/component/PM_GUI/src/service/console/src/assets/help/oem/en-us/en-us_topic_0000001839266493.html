Preparations for Backup

Before backing up FusionCompute VMs, prepare related information by following instructions in Table 1.

Table 1 Preparations for backup

Item

Mandatory or Not (Y/N)

How to Obtain

To Be Used In

Username and password for interconnecting with FusionCompute

Y

Log in to the FusionCompute WebUI and create an interconnection user.

For details, see Step 1: Create a FusionCompute Interconnection User.

Step 2: Registering the FusionCompute Virtualization Environment

VRM IP address

Y

Obtain the information from the FusionCompute environment administrator.

CA certificate

N

When registering a FusionCompute protected environment, you are advised to import the CA certificate of the protected environment. Otherwise, the system cannot verify the information about the accessed devices, resulting in security risks.

For details about how to obtain the certificate, see 2.a.

Certificate revocation list

N

It is issued by the certificate authority.

If a certificate is revoked due to security risks, import it into the revocation list.
