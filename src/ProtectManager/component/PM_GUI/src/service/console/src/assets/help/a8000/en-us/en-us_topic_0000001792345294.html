Viewing System Version Information

You can use the OceanProtect management page and CLI to query and familiarize yourself with system version information so that you can quickly determine matching software versions according to the system version during maintenance.

Obtaining Current System Version Information Using the Management Interface

You can use the management interface to view current system version information.

  1. Choose System > Infrastructure > Cluster Management.
  2. In the Local Cluster Nodes area of the Backup Cluster tab page, click the node name.
  3. In the Basic Info area of the displayed Node detail dialog box, view Version, that is, the current version of the system, as shown in Figure 1.

    Figure 1 Basic system information

Obtaining Current System Version Information Using the CLI

You can use the CLI to query the current system version.

  1. Use PuTTY to log in to the CLI as user admin.
  2. Run the show system general command to view the system version.

    admin:/>show system general

  System Name         : Huawei.Storage               
  Health Status       : Normal                       
  Running Status      : Normal                       
  Total Capacity      : 996.471TB                    
  SN                  : 2102354PBA13D9988001         
  Location            :                              
  Product Model       : OceanProtect X8000          
  Product Version     : 1.2.1RC1                     
  High Water Level(%) : 80                           
  Low Water Level(%)  : 20                           
  WWN                 : 210030fbb8c80ded             
  Time                : 2022-06-17/10:06:16 UTC+08:00 
  Patch Version       :                              
  Description         :

    In the preceding command output, the value of Product Version indicates current system version.
