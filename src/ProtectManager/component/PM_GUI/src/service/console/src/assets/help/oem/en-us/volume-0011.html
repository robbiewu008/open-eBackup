Step 1: Creating a Volume

Create a volume before performing volume protection.

Procedure

  2. Choose Protection > File Systems > Volumes.
  3. Click Create.
  4. In the Create dialog box that is displayed, create a volume for a single host.

    Table 1 describes the related parameters.

    Table 1 Parameters for creating a volume

    Parameter

    Description

    Name

    User-defined volume name.

    Select Host

    Select the host for which volume protection is to be performed.

    NOTE:

    Only one volume set can be created for a host. The hosts for which volume sets have been created are not displayed.

    OS Backup

    After this option is selected, you can enable or disable OS backup when creating or modifying volume protection. If you want to back up data of all system volumes, you must select all system volumes.

    Backup Type

    NOTE:

    This parameter is available only in 1.6.0.

    NOTE:

    This parameter is available only for the Windows OS.

    Select a backup type: Volume or Bare Metal.

    Volume

    Select the volumes to be protected.

  5. Click OK.
