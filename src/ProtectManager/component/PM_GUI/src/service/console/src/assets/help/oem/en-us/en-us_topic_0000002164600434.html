Step 3: (Optional) Creating a FusionOne Compute VM Group

You can create a VM group and add VMs to the VM group to protect VM data in batches.

Precautions

A maximum of 40 VMs can be added to a VM group.

Procedure

  1. Choose Protection > Virtualization > FusionOne Compute.
  1. Select the FusionOne Compute virtualization platform on the left of the page.
  2. In the right pane, click the VM Groups tab.
  3. Click Create.
  4. On the page that is displayed, set VM Group Name.
  5. Select the resources to be added to the VM group.

    Resources that have been protected or added to other VM groups cannot be selected.

  6. Click OK.
