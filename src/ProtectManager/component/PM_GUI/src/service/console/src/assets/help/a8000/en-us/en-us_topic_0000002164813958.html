CD-ROM Drive Cannot Be Accessed After a VM Is Restored

A CD-ROM drive cannot be accessed after the restoration if the backup source VM is a VM on vCenter, the VM contains the CD-ROM drive, and the restoration target is an ESXi host. You can resolve this issue using either of the following methods in the restore, instant restore, and live mount scenarios:

  • Connect the ESXi host to vCenter.
  • Reconfigure the CD/DVD drive parameters on the client.
    1. Use a browser to log in to the ESXi host.
    2. Select a restored VM and choose Actions > Edit settings.
    3. Modify the CD/DVD drive on the client.
