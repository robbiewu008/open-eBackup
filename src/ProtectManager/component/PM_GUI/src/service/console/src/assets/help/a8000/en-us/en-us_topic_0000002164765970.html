Step 3: Registering the FusionCompute Virtualization Environment

CA certificate

N

When registering a FusionCompute protected environment, you are advised to import the CA certificate of the protected environment. Otherwise, the system cannot verify the information about the accessed devices, resulting in security risks.

For details about how to obtain the certificate, see 2.a.

Certificate revocation list

N

It is issued by the certificate authority.

If a certificate is revoked due to security risks, import it into the revocation list.

VM driver (Tools)

Conditionally mandatory

If you want to back up data when the VM is started, ensure that Tools has been installed on the VM. Otherwise, backup cannot be performed. For details about how to install the tool, see the FusionCompute online help.
