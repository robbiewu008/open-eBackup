开启“gaussdb_feature_supportSyncBackupCrossSite_pcs-lite”白名单

跨项目/TPOPS的实例恢复前，需要开启“gaussdb_feature_supportSyncBackupCrossSite_pcs-lite”白名单，确保恢复功能正常。

操作步骤

  1. 使用SSH工具，通过节点的管理IP地址，使用root用户，登录TPOPS Server主节点。
  2. 在TPOPS节点执行以下命令，通过dbadmin用户登录TPOPS节点。

    su - dbadmin

  3. 执行以下命令，连接TPOPS元数据库。此处登录密码以默认密码“DBS_guass12#$”为例。

    source gauss_env_file
gsql -p 8635 -U core -W DBS_guass12#$ -q core -h 127.0.0.1

  4. 执行以下命令，查询白名单是否开启。

    select * from dbs_feature where id = 'gaussdb_feature_supportSyncBackupCrossSite_pcs-lite';

    如果回显“status”为“open”证明白名单已开启。否则，请执行5。

  5. 执行以下命令，开启白名单。

    update core.dbs_feature set status = 'open' where id = 'gaussdb_feature_supportSyncBackupCrossSite_pcs-lite';
