Checking the GaussDB Instance Version

Procedure

  1. Log in to the TPOPS management page.

    The URL for logging in to the TPOPS varies depending on the version. Obtain the management plane IP address, username, and password from the environment administrator.

    • https://TPOPS management plane IP address:8002/gaussdb, for example, https://192.168.10.2:8002/gaussdb
    • https://TPOPS management plane IP address:7070/APW, for example, https://192.168.10.3:7070/APW

  2. Optional: For the GaussDB management platform (TPOPS):

    1. Choose Instances.
    2. Check the DB Engine Version of the target instance.
