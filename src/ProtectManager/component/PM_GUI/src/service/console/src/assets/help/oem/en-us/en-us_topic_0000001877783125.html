Creating a Report

The product provides the report function, through which you can view the key resource status (such as storage capacity usage) and the key job (such as backup and recovery jobs) status. This section describes how to create a report.

Procedure

  1. Choose Insight > Reports.
  2. Click Create. Table 1 describes the parameters.

    Table 1 Parameters for creating a report

    Parameter

    Description

    Report Name

    User-defined name of the report.

    Report Type

    Type of the report to be created.

    • Storage Space Report: Collects the total capacity, used capacity, and unused capacity of product.
    • Backup Job Report: Collects backup job information.
    • Restoration Job Report: Collects restoration job information.

    Data Source

    Select local storage or an external cluster that has been connected to product.

    Statistical Range

    Time range within which data is collected. This parameter is displayed only when Report Type is Backup Job Report or Restoration Job Report.

    • Last week: Data in the recent week is collected.
    • Last month: Data in the recent month is collected.
    • Last three months: Data in the recent three months is collected.
    • Custom: The start time and end time for data collection are specified.
      NOTE:

      The time range cannot exceed 365 days.

    Statistical Interval

    Data collection interval of the report. The system sets a default interval based on the statistical range you select.

    • If the statistical range is shorter than or equal to 10 days, the statistical interval is set to 1 day.
    • If the statistical range is longer than 10 days but shorter than or equal to 30 days, the statistical interval is set to 5 days.
    • If the statistical range is longer than 30 days but shorter than or equal to 90 days, the statistical interval is set to 10 or 30 days.
    • If the statistical range is longer than 90 days but shorter than or equal to 365 days, the statistical interval is set to 30 days.

    This parameter is displayed only when Report Type is Backup Job Report or Restoration Job Report.

    Report Format

    Format of the report. In the current version, optional values are xls and pdf.
