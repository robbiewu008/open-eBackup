Overview

This section describes how to view ransomware detection information.

Procedure

  1. Choose Data Security > Ransomware Protection > Overview.
  2. The following information is displayed on the overview page:

    • Ransomware File Interception area in the upper part: number of vStores for which NAS ransomware protection is enabled/total number of vStores
    • Real-Time Ransomware Detection area in the upper part: number of vStores for which NAS ransomware protection is enabled/total number of vStores
    • Intelligent Ransomware Detection area in the upper part: number of vStores for which NAS ransomware protection is enabled/total number of vStores
    • Real-time Ransomware Detection Alarms area in the lower part: real-time ransomware detection alarm information
    • Intelligent Ransomware Detection area in the lower part: number of vStores for which NAS ransomware protection is enabled/number of vStores for which protection is not enabled
