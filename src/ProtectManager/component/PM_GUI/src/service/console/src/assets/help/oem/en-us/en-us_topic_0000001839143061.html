Step 3: Creating a User for Interconnection

Before backup, you must create an interconnection user to connect OpenStack to the product. This section uses FusionSphere OpenStack OM 22.1.0.SPH6 as an example.

Procedure

  1. Use a browser to log in to FusionSphere OpenStack OM.
  2. Choose System > User > User Management.
  3. Click Create User and set user parameters.

    Set Password Policy to Password never expires and Role to administrator.

  4. Click OK.
