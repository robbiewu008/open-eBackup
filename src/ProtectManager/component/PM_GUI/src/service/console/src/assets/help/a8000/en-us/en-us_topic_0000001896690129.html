Viewing the Restoration Drill Overview

This section describes how to view restoration drill details and create reports for restoration drill jobs.

Procedure

  1. Choose Explore > Restoration Drills.
  2. In the Overview area, view the number of drill executions, drill success rate, and average restoration duration. In the Abnormal Drill Results area, view the abnormal drills.
  3. Click Create Report in the upper right corner to create a report for a restoration drill job. For details, see "Managing Reports" in the OceanProtect Appliance 1.5.0-1.6.0 Administrator Guide.
