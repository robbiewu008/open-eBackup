Uploading the ProtectAgent Software Package

This section describes how to upload the ProtectAgent software package to the OceanProtect as a system administrator if the current version of the built-in ProtectAgent software package on the OceanProtect does not meet your requirements. If the current version meets your requirements, skip this section.

Context

  • During deployment of the OceanProtect, the system pre-installs a ProtectAgent software package. Choose Protection > Clients, and click ProtectAgent Package Management to view the ProtectAgent software version in the current system in Upload mode. If the version of the pre-installed ProtectAgent software package does not meet your requirements, upload a ProtectAgent software package of another version by referring to this section.
  • Only one ProtectAgent software package exists on the OceanProtect. After a new one is uploaded, the old one will be replaced.

Prerequisites

You have obtained the ProtectAgent software package DataProtect_xxx_client.zip xxx indicates the ProtectAgent version number.

  • For enterprise users: Click here.
  • For carrier users: Click here.

Procedure

  1. Choose Protection > Clients.
  2. Click ProtectAgent Package Management and set related parameters.

    Table 1 describes the parameters.

    Table 1 Parameters for uploading the ProtectAgent software package

    Parameter

    Description

    Mode

    Choose Upload.

    Proxy File

    Click and select the obtained ProtectAgent software package.

    NOTE:
    • Only software packages in .zip format can be uploaded.
    • The size of the file to be imported cannot exceed 2 GB for version 1.5.0.
    • The size of the file to be imported cannot exceed 3 GB for 1.6.0 and later versions.

  3. Click OK to upload the software package.
