Managing WORM Copies

This section describes how to prolong the retention period of WORM copies.

Precautions

WORM copies cannot be deleted, and the retention period of WORM copies cannot be shortened.

Prolonging the Retention Period of WORM Copies (Applicable to 1.5.0)

  1. Choose Explore > Ransomware Protection.
  2. In the navigation tree on the left, select a resource type for which WORM copies have been configured.
  3. Click to go to the copy page. WORM copies are displayed as in the WORM column.
  4. Locate the WORM copy whose retention period needs to be prolonged and choose More > Modify Retention Policy.
  5. In the window that is displayed on the right, select Retained permanently or Retained for specified duration. If Retained for specified duration is selected, the retention period can only be prolonged.

Prolonging the Retention Period of WORM Copies (Applicable to 1.6.0 and Later Versions)

  1. Choose Data Security > Ransomware Protection > Detection Overview.
  2. On the Detection Overview page, select a resource type for which a WORM copy has been configured.
  3. Click to go to the copy page. WORM copies are displayed as in the WORM column.
  4. Locate the WORM copy whose retention period needs to be prolonged and choose More > Modify Retention Policy.
  5. In the window that is displayed on the right, select Retained permanently or Retained for specified duration. If Retained for specified duration is selected, the retention period can only be prolonged.
