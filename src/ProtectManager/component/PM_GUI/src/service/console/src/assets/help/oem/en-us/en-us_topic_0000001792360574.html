Step 1: Registering Object Storage

Procedure

  1. Choose Protection > File Systems > Object Storage.
  2. Switch to the Object Storage tab page and click Register.
  3. Configure the basic information and agent service of the object storage.

    Table 1 describes the related parameters.

    Table 1 Object storage registration information

    Parameter

    Description

    Name

    User-defined name of the object storage.

    Type

    Object storage type, which can be OceanStor Pacific, Huawei Cloud Stack OBS, or Alibaba Cloud OSS.

    Endpoint

    Access address of the object storage.

    Address format: domain name or service plane IP address of the object storage

    Protocol

    Protocol used for data transmission between the product and object storage.

    • If HTTPS is selected, import a valid certificate so that the product can verify the object storage information. Obtain the certificate from the object storage administrator in advance.
      NOTE:

      You need to import a certificate only when Type is set to OceanStor Pacific. To obtain a certificate, perform the following steps:

      1. Log in to the OceanStor Pacific management page.
      2. Choose Settings > Certificate > Certificate Management.
      3. Click Object storage service security certificate to display the Certificate Details page.
      4. Click the CA Certificate tab to view the CA certificate and ensure that the value of CN is RootCA.
      5. Return to the Certificate page. Click the Built-In CA tab, select Service, and click Export CA Certificate to export the Service certificate.
    • If HTTP is selected, the product cannot verify the archive storage environment information, which poses security risks. You are advised to select HTTPS for security purposes.

    Certificate

    This parameter is displayed when HTTPS is used. Select an object storage certificate of the object storage type to verify the object storage environment information.

    NOTE:
    • You need to import a certificate only when Type is set to OceanStor Pacific.
    • When Type is set to Alibaba Cloud OSS and Protocol is set to HTTPS, certificate configuration is not supported. (Only unidirectional authentication is supported when HTTPS is used to access Alibaba Cloud object storage resources.)

    AK

    Access certificate for accessing the object storage.

    SK

    Security certificate for accessing the object storage.

    Agent Host

    External agent host selected for the object storage. If this parameter is not specified, the built-in agent is used by default.

    NOTE:

    Ensure that the maximum request body size allowed by the agent server is greater than 6 MB if the size of a single object to be backed up or restored does not exceed 58 GB, and greater than 5 GB if the size exceeds 58 GB.

    Server URL

    URL address of the agent server for connection between the product and the production end. This parameter is displayed only when the agent server is enabled. Currently, only the HTTP protocol is supported.

    Username

    User of the agent server. This parameter is displayed only when the agent server is enabled.

    Password

    User password of the agent server. This parameter is displayed only when the agent server is enabled.

  4. Click OK.
