Step 2: Registering a Single AntDB Instance

Procedure

  1. Choose Protection > Databases > AntDB.
  2. Click Register to register an AntDB database.
  3. Set Type to Single instance.
  4. Configure the database instance and its authentication information.

    Table 1 describes the related parameters.

    Table 1 Registration information of a single AntDB instance

    Parameter

    Description

    Name

    Name of the user used for installing the database, which is user-defined.

    Hosts

    Database host to be registered.

    Username

    OS username of the database. For details about how to obtain the value, see Preparing for Backup.

    Database Installation Path

    Installation path of the database to be registered. For details about how to obtain the value, see Preparing for Backup.

    Service Plane IP Address

    IP address of the host for the database service to receive TCP/IP connections.

    Database Port

    Port number used by the database service to listen on TCP/IP connections. For details about how to obtain the value, see Preparing for Backup.

    Database Username

    Contact the database administrator to obtain the username and password of the database administrator.

    Database Password

  5. Click OK.
