查看OceanBase环境信息

当OceanBase资源已注册至OceanProtect后，您可以在管理界面查看OceanBase集群的详细信息。

操作步骤

  1. 选择“保护 > 数据库 > OceanBase”。
  2. 在“集群”页签浏览集群信息。

    当集群已通过注册，集群信息会显示在该页面。

    详细信息如表1所示。
    表1 集群信息

    参数

    说明

    名称

    集群名称。

    单击集群名称，显示当前集群的OBClient主机和OBServer主机信息，及集群下租户和副本数据信息。

    状态
    集群的状态。
    • 集群中所有主机均在线且OceanBase环境正常时，集群状态为“在线”。
    • 只要集群中有主机离线，则集群状态为“离线”。

    版本

    集群版本。

    SLA

    当数据库已关联SLA时，则显示SLA名称，否则显示“--”。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态

    实例当前是否被保护。

    • 未保护：实例未关联SLA，或者实例已关联SLA但已禁用保护。
    • 已保护：实例已关联SLA，且已激活保护。
    • 创建中：实例已关联SLA，正在创建保护。

    所属用户

    该集群所属的数据保护管理员。

    当该集群已授权给数据保护管理员时，将显示数据保护管理员的名称。如果未进行授权，则显示“--”。

  3. 在“租户集”页签浏览租户集信息。

    该界面显示的是已注册的租户集。
    1. 租户集的详细信息如表2所示。
      表2 租户集信息

      参数

      说明

      名称

      租户集名称。

      状态

      租户集在当前集群状态。包括：在线、离线和部分在线。

      所属集群

      租户集所在的集群。

      SLA

      当数据库已关联SLA时，则显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      实例当前是否被保护。

      • 未保护：实例未关联SLA，或者实例已关联SLA但已禁用保护。
      • 已保护：实例已关联SLA，且已激活保护。
      • 创建中：实例已关联SLA，正在创建保护。
    2. 单击名称，查看总览信息、副本信息以及任务。
      • 总览

        显示SLA信息、任务执行时间、副本总数等。

      • 副本数据

        您可以依次按照年、月、天查找相应时间的副本。

        时间下方显示，表示该时间存在副本。

      • 任务

        展示执行中任务和历史任务。
