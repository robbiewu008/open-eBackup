Step 1: Registering the DB2 Database

Registering a Single DB2 Instance

  1. Choose Protection > Databases > DB2.
  2. Click the Instance tab.
  3. Click Register to register the DB2 database.
  4. Set Type to Single instance.
  5. Configure the database instance and its authentication information.

    Table 1 describes the related parameters.

    Table 1 Registration information of a single DB2 instance

    Parameter

    Description

    Name

    Instance name.

    Hosts

    Database host to be registered.

    Username

    Username of the database.

    Password

    User password of the database.

  6. Click OK.

Registering a DB2 Cluster Instance

  1. Create a cluster.

    1. On the DB2 page, click the Cluster tab.
    2. Click Register.
    3. On the Register page, customize the cluster name and select the type and cluster node.
      Table 2 describes the related parameters.
      Table 2 Parameters for registering a cluster

      Parameter

      Description

      Name

      User-defined cluster name.

      Type

      The value can be Database Partitioning Feature, powerHA, or HADR.

      Nodes

      Host where the DB2 instance resides.
    4. Click OK.

  2. Register a database of the cluster instance.

    1. On the DB2 page, click the Instance tab.
    2. Click Create.
    3. Set Type to Cluster instance.
    4. Customize an instance name and select a cluster.
    5. Click Add. In the displayed Add dialog box, add all nodes of the cluster instance.

      Table 3 describes the related parameters.

      Table 3 Node information of a cluster instance

      Parameter

      Description

      Hosts

      Host node in the cluster.

      Username

      Username of the database.

      Password

      User password of the database.
    6. Click OK.
