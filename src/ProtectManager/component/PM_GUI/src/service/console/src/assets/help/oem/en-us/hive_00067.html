Managing Backup Sets

The product allows you to modify, remove, activate, or disable protection for backup sets.

Related Operations

Log in to the management page, choose Protection > Big Data > Hive, click the Backup Sets tab, and locate the desired backup set.

Table 1 describes the related operations.

Table 1 Operations related to backup set protection

Operation

Description

Navigation Path

Modifying protection

Scenario

This operation enables you to modify a protection plan to associate another SLA with a backup set and modify the execution scripts before and after backup set protection. After the modification, the next backup will be executed based on the new protection plan.

Note

The modification does not affect the protection job that is being executed. The modified protection plan will take effect in the next protection period.

In the row of the target backup set, choose More > Modify Protection.

Removing protection

Scenario

This operation enables you to remove the protection of a backup set when it does not need to be protected. After the removal, the system cannot execute protection jobs for the backup set. To protect the backup set again, associate an SLA with it again.

Note

Protection cannot be removed when a protection job of the backup set is running.

In the row of the target backup set, choose More > Remove Protection.

Activating protection

Scenario

This operation enables you to re-activate a disabled protection plan of a backup set. After the protection plan is activated, the system backs up the backup set based on the protection plan.

In the row of the target backup set, choose More > Activate Protection.

Disabling protection

Scenario

This operation enables you to disable the protection plan of a backup set when its periodic protection is not needed. After the protection plan is disabled, the system does not execute the automatic backup job of the backup set.

Note

Disabling protection does not affect ongoing backup jobs.

In the row of the target backup set, choose More > Disable Protection.

Manual backup

Scenario

This operation enables you to perform a backup immediately.

Note

Copies generated by manual backup are retained for the duration defined in the SLA.

In the row of the target backup set, choose More > Manual Backup.

Modifying a backup set

Scenario

This operation enables you to modify the name of, and the selected directory and table in a backup set.

In the row of the target backup set, choose More > Modify.

Deleting a backup set

Scenario

This operation enables you to delete a backup set if you do not need to protect it.

Note

A backup set that has been associated with an SLA cannot be deleted.

In the row of the target backup set, choose More > Delete.
Adding a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to quickly filter and manage resources.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag page, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.
Removing a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to remove a tag of a resource when it is no longer needed.

Choose More > Remove Tag.

Note: The procedure for restoring Hive backup sets is similar to that in Restoration and is not described here.
