查看阿里云资源信息

当阿里云注册至本产品后，您可以在管理界面查看阿里云项目和云服务器信息。

操作步骤

  1. 选择“保护 > 云平台 > 阿里云”。
  2. 在“可用区”页签浏览项目信息。

    部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

    项目的详细信息如表1所示。
    表1 项目信息

    参数

    说明

    名称

    可用区名称。

    单击可用区名称，可以查看可用区的总览、任务。

    • 总览

      显示名称、SLA信息、云服务器信息等。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    位置

    可用区所在位置。

    SLA

    当可用区已关联SLA时，则显示SLA名称，否则显示“--”。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态

    可用区当前是否被保护。

    • 未保护：可用区未关联SLA，或者可用区已关联SLA但已禁用保护。
    • 已保护：可用区已关联SLA，且已激活保护。
    • 创建中：可用区已关联SLA，正在创建保护。

    标签

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。

  3. 在“资源集”页签浏览云服务器信息。

    部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

    云服务器的详细信息如表2所示。
    表2 资源集信息

    名称

    资源集名称。

    • 任务

  4. 在“云服务器”页签浏览云服务器信息。

    部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

    云服务器的详细信息如表3所示。
    表3 云服务器信息

    参数

    说明

    名称

    云服务器名称。

    单击名称，可以查看云服务器的总览、副本数据、任务信息。

    • 总览

      显示名称、SLA信息、任务执行时间、副本总数、磁盘信息等。

    • 副本数据

      您可以依次按照年、月、天查找相应时间的副本。

      时间下方显示，表示该时间存在副本。单击日期进入副本数据页面。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    位置

    云服务器所在位置。位置格式：域/项目/云服务器

    状态

    云服务器状态。包括在线、离线。

    SLA

    云服务器关联的SLA。

    标签

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。

    云服务器组

    云服务器已加入的云服务器组。已加入云服务器组的云服务器不再支持相关保护操作，如需直接保护云服务器，请从所在的云服务器组中移除该云服务器。
