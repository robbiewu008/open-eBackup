Creating a Privileged NetBackup User for EWS Access

To use the backup function of the mailbox, you need to add a role for the user who registers a single-node system or availability group.

Adding the ApplicationImpersonation Role to a User

  1. On Exchange Server, use a browser to log in to https://localhost/ecp/.
  2. Choose permissions. On the admin roles tab page, click to go to the new role group page.
  3. Customize the user group name and add the ApplicationImpersonation role.
  4. Add a member user for registering a single-node system or availability group.
