修改GaussDB(DWS)配置文件

当客户端安装到GaussDB(DWS)节点时，需要修改GaussDB(DWS)配置文件，否则备份失败。

操作步骤

  1. 使用PuTTY，登录GaussDB(DWS)集群的节点。
  2. 执行以下命令进入配置文件目录。

    cd /opt/DataBackup/ProtectClient/Plugins/GeneralDBPlugin/bin/
applications/dws/

  3. 执行vi dws.conf命令编辑配置文件，相关参数说明如表1所示。

    表1 GaussDB(DWS)配置文件参数

    参数

    说明

    deployType

    GaussDB(DWS)部署的形态。

    • 0：GaussDB(DWS)集群部署在服务器宿主机。
    • 1：GaussDB(DWS)集群部署在沙箱中。

    loginSandboxIp

    沙箱中GaussDB(DWS)部署使用的IP地址（仅适用于GaussDB(DWS)部署于沙箱中的场景）。

    说明：

    在部署客户端的CN节点上执行hostname -i 命令查询IP地址。

    sandboxPath

    沙箱在主机中存放的路径（仅适用于GaussDB(DWS)部署于沙箱中的场景）。

    authmethod

    客户端认证策略。

    • trust：不验证密码。
    • reject：拒绝访问。
    • md5/sha256/cert：客户端证书认证。
    • gss：kerberos认证。
