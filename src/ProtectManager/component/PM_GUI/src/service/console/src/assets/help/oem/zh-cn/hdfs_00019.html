步骤8：执行备份

执行备份前将需要保护的资源关联到指定SLA，系统将对这些资源按照SLA进行保护，并根据SLA周期性执行备份任务。如需立即执行一次备份任务时，可以通过手动备份实现。

注意事项

  • 手动备份时，SLA中定义的参数，如“自动索引”会应用到手动备份任务中。
  • 手动备份产生的副本保留时长按照SLA中定义的保留时长进行保留。

    如果手动备份时选择的保护策略未在SLA中定义，则产生的备份副本保留时长为永久保留。如SLA中未定义增量备份，而手动备份时选择的是增量备份，则本次手动增量备份产生的副本保留时长为永久保留。

  • 如果SLA中定义了复制策略或归档策略，则执行手动备份时，系统会按照SLA的定义执行一次复制或归档。

操作步骤

  1. 选择“保护 > 大数据 > HDFS”。
  2. 选择“文件集”页签后，选择需要保护的文件集，单击“保护”。

    您也可以勾选多个文件集进行批量保护。

  3. 选择SLA。

    您也可以单击“创建”，创建新的SLA。

    如果待保护的资源已设置WORM策略，为避免WORM策略冲突，请选择不包含WORM策略的SLA创建保护。

  4. 配置保护高级参数。

    相关参数说明如表1所示。

    表1 文件集保护高级参数说明

    参数

    说明
    备份副本自动索引
    说明：

    仅1.6.0及后续版本支持此参数。

    开启后支持选择副本中的单个或多个文件进行恢复，支持在管理界面上搜索副本中的文件、目录或链接。

    如果您未在此处开启自动索引功能，后续可在副本页面为副本创建索引。

    开启后不支持关闭。
    对象存储归档副本自动索引
    说明：

    仅1.6.0及后续版本支持此参数。

    开启后支持选择副本中的单个或多个文件进行恢复，支持在管理界面上搜索副本中的文件、目录或链接。当归档存储为磁带库时，不支持对归档副本进行索引。

    开启后不支持关闭。

  5. 单击“确定”。

    如果当前系统时间已晚于SLA首次备份的开始时间，可以在弹出的对话框中选择立即执行手动备份或者根据SLA中设置的备份策略周期性自动执行备份。

  6. 执行手动备份。

    当需要立即执行一次备份任务时，可以执行以下操作进行手动备份。否则，可跳过本步骤。

    1. 在需要备份的文件集所在行单击“更多 > 手动备份”。

      支持选择多个资源批量进行手动备份。勾选多个资源，在页面左上角选择“更多 > 手动备份”。

    2. 设置手动备份产生的副本名称。

      如果您不设置，系统默认设置副本名称为：backup_时间戳。

    3. 选择保护策略：全量备份、永久增量（合成全量）备份。

      对于1.6.0及后续版本，如果选择的保护策略与关联的SLA中配置的保护策略不一致，则WORM配置不生效。

    4. 单击“确定”。
