Managing Device Time (Applicable to Some Models)

On the node details page, view the device time of the current system.

Applicable to Some Models of This Product

  1. Choose System > Infrastructure > Cluster Management.
  2. On the Backup Clusters tab page, click a node name under the Local Cluster Nodes area.
  3. In the Node Details dialog box that is displayed, view the current device time.

    For details about how to change the device time, see Configuring the Device Time.
