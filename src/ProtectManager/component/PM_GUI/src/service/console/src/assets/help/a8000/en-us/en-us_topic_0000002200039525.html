Database account and password

Yes

Contact the database administrator to obtain a database account with the backup or restoration permission and its password.

To create an account, perform the following steps:

  1. Use PuTTY to log in to any database host in the cluster.
  2. Run the following command to log in to the postgres database:
    gsql -d postgres -p 5432
  3. Run the following command to create an account and set a password:
    CREATE USER Account name with PASSWORD 'Account password';
  4. Run the following command to grant permissions to the account:
    GRANT ALL PRIVILEGES to Account name;
