Viewing Information About Operation Restrictions on Infected Copies

This section describes how to view details about operation restrictions on infected copies.

Procedure

  1. Choose Data Security > Ransomware Protection > Ransomware Protection and WORM Configurations.
  2. Click Operation Restrictions on Infected Copies.
  3. On the Operation Restrictions on Infected Copies tab page, view the configured operation restrictions on infected copies. Table 1 describes the related parameters.

    Table 1 Parameters of operation restrictions on infected copies

    Parameter

    Description

    Name

    Name of the resource to which the operation restriction on infected copies applies.

    NOTE:

    Click the name of a resource to view its information, including Overview, Copy Data, and Jobs.

    Resource Type

    Type of the resource to which the operation restriction on infected copies applies.

    Resource types include VMware VM, NAS File System, NAS Share, Filesets, CNware VM, Huawei Cloud Stack, FusionCompute VM, OpenStack Cloud Server, Hyper-V VM, and FusionOne Compute VM.

    Location

    Resource location.

    Restricted Object Type

    Type of a restricted object. Available options are Backup copy and Replication copy.

    Forbidden Operation

    Operations that are forbidden on infected copies. Available options include Restoration to the original location, Restoration to a new location, File-level restoration, Live mount, Instant recovery, Download and export, and Archive.

    Created

    Time when the operation restriction on infected copies is configured.
