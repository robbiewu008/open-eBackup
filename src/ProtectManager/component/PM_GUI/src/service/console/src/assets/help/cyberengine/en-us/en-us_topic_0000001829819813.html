Viewing Detection Results

You can view the latest and historical results of detecting the ransomware detection snapshot data on the OceanCyber WebUI.

You need to log in to the OceanCyber. In the navigation pane, choose Snapshot Management and Restoration. Table 1 describes the result information about detecting the ransomware detection snapshot data.

Table 1 Result information about detecting the ransomware detection snapshot data

Job

Description

Navigation Path

Latest Detection Result

This operation enables you to view the latest detection result of a ransomware detection snapshot, including the trend chart of the number of added, modified, deleted, and suspicious files and the list of protected objects.

Choose More > Latest Detection Result on the right of the row that contains the desired snapshot data.

NOTE:

You can also click the number of ransomware detection snapshots to go to the Snapshot Data page. Then, in the row that contains the desired ransomware detection snapshot, choose More > Detection Result.

Historical Detection Result

This operation enables you to view the historical detection result of a ransomware detection snapshot.

Choose More > Historical Detection Result on the right of the row that contains the desired snapshot data.

If the copy detection result is Infected, a suspicious file exists in the VM copy. If the suspicious file is deleted from the VM, the copy detection result may still be Infected in the next copy detection. You are advised to use the last uninfected copy data of the VM for restoration and then generate new copy data for detection.
