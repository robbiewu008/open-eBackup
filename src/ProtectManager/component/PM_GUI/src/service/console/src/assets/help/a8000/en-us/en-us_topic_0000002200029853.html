Viewing Tapes

This section describes how to view tapes. Information such as the tape location and write status can be viewed.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Tape Library.
  3. Click the Name of a tape library. In the window that is displayed, click the Tape tab.
  4. View the tape information in the tape list. Table 1 describes the related parameters.

    Table 1 Tape parameters

    Parameter

    Description

    Tape Tag

    Tag of a tape.

    Owning Media Set

    Name of the media set to which a tape belongs.

    Location

    Location of a tape.

    • The location of a tape that is not available in the library is displayed as --.
    • If a tape is not loaded to the driver, its location is the slot where the tape resides.
    • If a tape is loaded to the driver, its location is the driver where the tape resides.

    Write Status

    Write status of a tape.

    • Unknown: The write status of the tape is not identified.
    • Free: The tape is not in use.
    • Not Full: The tape is not fully written.
    • Full: The tape is fully written.
    • Error: Data cannot be written into or read from the tape.

    Status

    Status of a tape.

    • In Library: The tape stays in a tape library.
    • Not In Library: The tape does not stay in a tape library.
    • Identifying: The tape is being identified.
    • Deleting: The tape is being deleted.
    • Loading: The tape is being loaded to a driver.
    • Uninstalling: The tape is being unloaded from a driver to a slot.
    • Erasing: The tape is being erased.

    Tape Type

    Type of a tape.

    • RW: The tape can be overwritten.
    • Unknown: The type of the tape is not identified.

    Last Write Time

    Last time when data was written into a tape.

    Used Capacity

    Used capacity of a tape. The used capacity refers to the size of data that has been written into the tape.
