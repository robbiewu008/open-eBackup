Preparing for the Backup

Before backing up the ClickHouse, prepare related information as follows:

  • Table 1 lists the information required.
  • Table 2 lists the information required when the authentication mode for communication between the ClickHouse cluster and product is Database authentication.
  • Table 3 lists the information required when the authentication mode for communication between the ClickHouse cluster and product is Kerberos Authentication.
Table 1 Preparing for the backup

Item

Mandatory

How to Obtain

Section

MRS cluster Client installation path

Yes

Obtain the item from the MRS cluster administrator.

Step 1: Registering a ClickHouse Cluster

ClickHouse service IP address

Yes

Obtain the item from the MRS cluster administrator.

.conf configuration file (for example, krb5.conf) used during Kerberos authentication

No

If Kerberos authentication is performed using a keytab file, you need to obtain the .keytab file.
