Viewing Storage Device Information

After a storage device is added to the OceanProtect, you can view the device details.

Procedure

  1. Choose Protection > File Systems > Storage Devices.

    For 1.5.0, choose Protection > File Services > Storage Device.

    Table 1 lists the storage device information.

    Table 1 Storage device information

    Parameter

    Description

    Name

    Storage device name.

    Status

    Storage device status.

    • Online
    • Offline

    Type

    Storage device type.

    • OceanStor Dorado V7
    • OceanStor V7
    • OceanStor Dorado 6.x
    • OceanStor 6.x
    • OceanProtect 1.3.0 and later versions
    • OceanStor V5
    • OceanStor Pacific
    • NetApp ONTAP

    IP Address

    Management IP address of the storage device.

    Port

    Number of the port for communication between the OceanProtect and the management plane of the storage device.

    WWN

    World Wide Name (WWN) of the storage device.

    Serial No.

    Storage device SN.
    Tag
    NOTE:

    This parameter is available only in 1.6.0 and later versions.

    If a tag has been added for the resource, the tag name is displayed. Otherwise, -- is displayed.
