Minimum Permissions for Users Performing VMware Virtualization Environment Registration

Table 1 lists minimum permissions for users performing virtualization environment registration for VMware resource protection. The following part uses VMware 7.0 as an example. For other versions, permission setting methods may be different, but the permission requirements are the same.

VMware user permissions must be added on vCenter. Otherwise, VDDK initialization will fail. For details, see What Can I Do to Add User Permissions on vCenter?.
