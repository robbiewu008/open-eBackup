Modifying a Backup Storage Device

This section describes how to modify the backup storage device information.

Precautions

A backup storage device that belongs to a local cluster cannot be modified.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. Select Backup Storage Devices.
  4. In the row that contains the backup storage device to be modified, choose More > Modify.
  5. After modifying the backup storage device information, click OK.
