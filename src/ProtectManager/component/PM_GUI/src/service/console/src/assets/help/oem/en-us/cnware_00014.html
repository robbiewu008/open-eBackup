Step 3: (Optional) Creating a CNware VM Group

A VM group is a group of specified CNware VMs. You can register a VM group to protect data of VMs in the group.

Procedure

  1. Choose Protection > Virtualization > CNware.
  1. Click the VM Groups tab and click Create.

    Table 1 describes the information required for creating a VM group.
    Table 1 VM group creation information

    Parameter

    Description

    VM Group Name

    Customize a VM group name.

    Select Resource

    Select one or more VMs to create a VM group. You can filter the VM resources to be protected by resource ID or VM name. Resources that have been protected or added to other VM groups cannot be selected.

    NOTE:

    A maximum of 40 VMs can be selected for a VM group.

  2. Click OK.
