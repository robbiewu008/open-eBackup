安装Percona XtraBackup工具依赖软件

MySQL/GreatSQL在执行备份任务前，需要在MySQL/GreatSQL所有节点手动安装Percona XtraBackup工具依赖软件。

注意事项

备份过程中执行DDL （Data Definition Language）操作会阻塞Percona XtraBackup工具的备份，待DDL 操作完成后，备份才能继续进行。

操作步骤

  1. 登录MySQL/GreatSQL实例所在的主机。
  2. 依次执行以下命令，安装Percona XtraBackup工具依赖软件。

    yum install -y perl-DBD-MySQL
yum install -y perl-Digest-MD5
yum install -y perl-DBI
yum install -y perl-Time-HiRes
yum install -y perl-IO-Socket-SSL
