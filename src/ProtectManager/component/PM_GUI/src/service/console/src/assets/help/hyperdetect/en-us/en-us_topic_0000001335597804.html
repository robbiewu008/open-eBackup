Configuring the Real-Time Ransomware Detection Whitelist

If you want to skip some directories or files during real-time ransomware detection, perform the operations in this section to configure the real-time ransomware detection whitelist.

Procedure

  1. On the navigation pane, choose Explore > Ransomware Protection > Real-Time Ransomware Detection.
  2. Click Real-Time Ransomware Detection Whitelist.
  3. Add a real-time ransomware detection whitelist.

    1. Click Create.
    2. The type can be Directory or File Name Extension. Select a type, enter the directory name or file name extension, and click OK.

  4. Associate a vStore. The real-time ransomware detection whitelist takes effect only for associated vStores. After a vStore is associated, the corresponding real-time ransomware detection whitelist takes effect when real-time ransomware detection is performed on the file system of the vStore.

    1. Click Associate with vStore.
    2. Select the vStore to be associated with the real-time ransomware detection whitelist and click OK.

    Alternatively, you can associate vStores after configuring the real-time ransomware detection whitelist.
