Step 3: Registering the Database of a Single Kingbase Instance

Procedure

  1. Choose Protection > Databases > Kingbase.
  2. Click the Instance tab.
  3. Click Register to register a Kingbase instance.
  4. Set Type to Single instance.
  5. Configure the database instance and its authentication information.

    Table 1 describes the related parameters.

    Table 1 Registration information of a single Kingbase instance

    Parameter

    Description

    Name

    Instance name, which can be customized.

    Hosts

    Select the database host to be registered.

    Username

    OS username for running the database. For details about how to query the username, see Preparing for the Backup.

    Database Installation Path

    Path for installing the database to be registered. For details about how to query the database installation path, see Preparing for the Backup.

    Service Plane IP Address

    IP address of the host used by the database service to receive TCP/IP connections. For details about how to query the service IP address, see Preparing for the Backup.

    Database Port

    Port number used by the database service to listen on TCP/IP connections. The default port number is 54321.

    Database Username

    Name of the database administrator.

    Database Password

    Password of the database administrator.

  6. Click OK.
