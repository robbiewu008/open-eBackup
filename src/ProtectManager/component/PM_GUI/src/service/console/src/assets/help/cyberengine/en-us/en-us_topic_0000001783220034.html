(Optional) Configuring the Network

This section describes how to associate a service NIC or VLAN with a vStore of a storage device. After a vStore is associated, when the system performs intelligent detection on file systems of the vStore, the service data of the file system is mounted from the NIC to the security appliance for detection. If you have configured multiple NICs and need to select different NICs for different tenants, perform the operations described in this section.

Precautions

  • A tenant can be associated with only one NIC or VLAN.
  • When associating a NIC with a vStore, the gateway and subnet mask must be reachable to the logical port of the vStore on the storage device. Otherwise, the file system detection job of the vStore on the storage device may fail.

Procedure

  1. Choose System > Settings > Network Settings.
  2. In the row that contains the NIC or VLAN to be associated with tenants, choose More > Associate vStore.
  3. The Associate vStore window is displayed on the right. Set Storage Device and vStore, and set Gateway and Subnet Mask.
  4. Click OK.

    To disassociate a NIC from a tenant, click the NIC name. In the dialog box that is displayed, disassociate the NIC from the tenant.
