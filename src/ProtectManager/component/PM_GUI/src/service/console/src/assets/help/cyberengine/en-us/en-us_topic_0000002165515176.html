Performing Real-Time Detection

The system performs real-time detection on the desired file systems according to the associated real-time detection policy.

Precautions

  • If the storage device is the OceanProtect Backup Storage and DataTurbo sharing is enabled for the file system, real-time detection cannot be enabled.
  • During real-time detection, you must enable protection for both the primary and secondary file systems in the HyperMetro domain to ensure that I/Os of the primary and secondary file systems can be detected.
  • Real-time detection applies to OceanProtect 1.3.0, OceanStor Dorado 6.1.6, and their later versions.

Procedure

  1. Choose Data Security > Real-Time Detection.
  2. Click File Systems.
  3. In the row of the target file system, choose More > Protect.

    You can also select multiple file systems and choose More > Protect in the upper part.

  4. Select an existing real-time detection policy or click Create to create one.
  5. Click OK.
