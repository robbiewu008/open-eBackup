步骤1：注册DB2数据库

注册DB2单实例

  1. 选择“保护 > 数据库 > DB2”。
  2. 选择“实例”页签。
  3. 单击“注册”，注册DB2。
  4. “类型”选择“单实例”。
  5. 配置数据库实例及其认证信息。

    相关参数说明如表1所示。

    表1 DB2单实例注册信息

    参数

    说明

    名称

    实例名称。

    主机

    选择要注册的数据库主机。

    用户名

    数据库用户名。

    密码

    数据库用户密码。

  6. 单击“确定”。

注册DB2集群实例

  1. 创建集群。

    1. 在DB2页面选择“集群”页签。
    2. 单击“注册”。
    3. 在“注册”页面，自定义集群名称并选择类型和集群节点。
      相关参数说明如表2。
      表2 注册集群参数说明

      参数

      说明

      名称

      自定义集群名称。

      类型

      集群类型可选择Database Partitioning Feature、powerHA和HADR类型。

      节点

      DB2实例所在的主机。
    4. 单击“确定”。

  2. 注册集群实例下的数据库。

    1. 在DB2页面选择“实例”页签。
    2. 单击“创建”。
    3. “类型”选择“集群实例”。
    4. 输入实例名，选择集群。
    5. 单击“添加”，在弹出的添加对话框中添加集群实例下的所有节点。

      相关参数说明如表3所示。

      表3 集群实例的节点信息

      参数

      说明

      主机

      选择集群下的主机节点。

      用户名

      数据库用户名。

      密码

      数据库用户密码。
    6. 单击“确定”。
