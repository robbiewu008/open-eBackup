Managing the Email Service

If the system administrator password is lost or one-time passwords are required, an email containing the verification code can be sent by the product to the preset email address.

Precautions

This operation requires personal data such as the email address. You must take adequate measures under the governing laws and privacy policies to ensure that user data is fully protected.

Procedure

Setting the sender information

  1. In the navigation pane, choose System > Settings > Mail Service.
  2. In the Sender Settings area, click Modify and set sender parameters. Table 1 describes the parameters.

    Table 1 Sender parameters

    Parameter

    Description

    SMTP Server

    IP address or domain name of the SMTP server for sending notification emails containing verification codes.

    The IP address of the SMTP server can be an IPv4 or IPv6 address.

    SMTP Port

    SMTP server port number.

    The value is an integer from 1 to 65535.

    Sender Email Address

    Email address for sending notification emails containing verification codes.

    Testing Email Address

    Email address used to test the communication between the email sender server and the email server.

    Encryption Mode

    Encryption mode used by the email server to communicate with the system. Possible values are Not encrypted, SSL/TLS, and STARTTLS.

    • Not encrypted: Data is not encrypted during transmission. If data transfer is not encrypted, there are security risks. You are advised to select SSL/TLS or STARTTLS to enhance data transfer security.
    • SSL/TLS: SSL and TLS are different security protocols used for ensuring data security and integrity during network communication. After the two protocols are enabled, the system automatically selects the encryption mode according to the email server. When selecting SSL/TLS, you need to set the SSL port number to the port number corresponding to the SMTP encryption mode.
    • STARTTLS: If TLS is used for encrypting data after executing STARTTLS, the previous communication data will not be encrypted. If the STARTTLS security protocol is selected, you need to change the SMTP port number to the port number corresponding to the SMTP encryption mode.

    The encryption mode configured on the system must be the same as that configured on the SMTP server.

    NOTE:
    • If you select SSL/TLS or STARTTLS for Encryption Mode, you are advised to import a CA certificate of the Email type on the System > Security > Certificates page and use the CA certificate for mailbox server authentication, preventing man-in-the-middle (MITM) attacks and improving communication security. For details about how to import a certificate, see .

    SSL Port

    SMTP server's SSL port number.

    This parameter is available only when Encryption Mode is set to SSL/TLS.

    SMTP Server Authentication

    Indicates whether the SMTP server authenticates a sender's identity.

    Username

    Username for accessing the SMTP server.

    This parameter is available only when SMTP Server Authentication is selected.

    Password

    Password for accessing the SMTP server.

    This parameter is available only when SMTP Server Authentication is selected.

    Proxy Server

    A proxy server needs to be configured if the system cannot communicate with the SMTP server.

    IP Address

    IP address of the proxy server.

    This parameter is available only when Proxy Server is selected.

    Port

    Port ID used by the proxy server.

    This parameter is available only when Proxy Server is selected.

    The value is an integer from 1 to 65535.

  3. After the parameters are specified, you can click Test to check whether the email sender server and SMTP server can properly communicate with each other.
  4. Click Save.
