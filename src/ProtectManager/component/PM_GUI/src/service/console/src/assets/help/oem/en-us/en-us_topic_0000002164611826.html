Step 2: Registering an Informix/GBase 8s Cluster

Prerequisites

Ensure that each host in the cluster has been registered.

Procedure

  1. Choose Protection > Databases > Informix/GBase 8s.
  2. Click the Cluster tab.
  3. Click Register.
  4. On the Register page, customize the cluster name and select a host.
  5. (Optional) Enable log backup and enter the log backup path. Pay attention to the usage of the database log space and clear the log space in a timely manner. If the space is used up, services will be affected. For details, see Services Unable to Run Properly Because the Database Log Space Is Used Up.

    • After the log backup function is enabled, the system can restore the database based on a specified time point.
    • Ensure that the entered path exists and the user has read/write permissions on the Informix/GBase 8s database.
    • The log path cannot be a built-in directory of the OS, such as /etc, /bin, /usr, or /dev.
    • You are advised to use the default log backup path.
      • For an Informix database, the default path is /home/informix/log.
      • For a GBase 8s database, the default path is /home/gbasedbt/log.

  6. Click OK.
