Viewing Live Mount Information About a NAS Share

This section describes how to view the live mount information after a live mount job is created.

Procedure

  1. Choose Explore > Live Mount > Application Type > NAS Shares.

    Information about the mounted data is displayed. Table 1 describes the related parameters.

    Table 1 Live mount parameters

    Parameter

    Description

    Target Object

    Name

    Name of a mounted NAS share.

    Status

    Status of a live mount job

    • Preparing: A live mount job is prepared.
    • Mounting: A NAS share is being mounted.
    • Mounting failed: A live mount job fails.
    • Unmounting: A mounted NAS share is being unmounted.
    • Available: A mounted NAS share is available.
    • Invalid: The copy is invalid.

    Share Protocol

    Protocol of the mounted NAS share, which can be CIFS or NFS.

    Share Name/Path

    Name or path of a mounted file system share

    Protected Objects

    Name

    Name of the source share corresponding to a mounted copy

    Location

    Location of the source share corresponding to a mounted copy.

    Copy Data

    Copy Time

    Time when a copy is generated through live mount

    Live Mount Status

    Status of a live mount job, which can be Activate or Disable. After a live mount job is created, the status is Activate by default.
