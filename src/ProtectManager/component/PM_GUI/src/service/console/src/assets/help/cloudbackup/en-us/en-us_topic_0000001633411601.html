Importing a CRL

The storage system allows you to import CRLs of external certificates. After a CRL is imported, communication with the device whose certificate has been revoked is interrupted.

Context

On the storage system, you can import an external certificate CRL to identify whether the external certificate is available.

Prerequisites

You have obtained the CRL to be imported.

Precautions

  • Only one CRL file can be imported for a certificate.
  • The size of a CRL file must be less than or equal to 5 KB.
  • The CRL of the ExternalComponent certificate cannot be imported.

Procedure

  1. Choose System > Security > Certificate.
  2. Click the external certificate of which the CRL is to be imported. In the window that is displayed on the right, click the Certificate Revocation List tab.
  3. Click Import.
  4. Click Please choose file and select the CRL to be imported.
  5. The Danger dialog box is displayed. Select I have read and understand the consequences associated with performing this operation. and click OK.
