Modifying a Role

The product allows you to modify information about existing roles.

Precautions

Information about built-in roles cannot be modified.

Procedure

  1. Choose System > Security > RBAC > Roles.
  2. (Optional) Click next to Name to search for a role.
  3. In the row of the target role, click Modify on the right.
  4. Modify the role information. Table 1 describes the parameters.

    Table 1 Basic role parameters

    Parameter

    Description

    Role Name

    User-defined role name.

    [Value range]

    The value contains 4 to 64 characters, including letters, digits, periods (.), hyphens (-), and underscores (_), and cannot start with a hyphen (-) or period (.).

    Role Description

    Description of the role to be created.

    [Value range]

    The description can be left blank or contain up to 255 characters.

    Select Permissions

    Assign function permissions to the current role based on Table 2 to restrict users' operation permissions on the system.

  5. Click OK.
