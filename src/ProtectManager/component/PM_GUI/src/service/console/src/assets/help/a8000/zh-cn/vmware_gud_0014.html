表1 备份前准备

准备项

是否必须

VMware Tools安装文档

可选

执行应用一致性备份或文件级恢复时，需要在受保护虚拟机或目标虚拟机安装VMware Tools。如果虚拟机未安装VMware Tools，需要为其安装。

具体操作请参见安装VMware Tools。

步骤1：检查并安装VMware Tools

应用一致性备份脚本

可选

执行应用一致性备份时，需要在受保护的虚拟机中配置应用一致性备份前置脚本和后置脚本。

具体操作请参见步骤3：配置应用一致性备份脚本。

步骤3：配置应用一致性备份脚本

VMware虚拟化平台管理IP

必选
请向VMware虚拟化平台管理员获取。
  • 当注册vCenter Server至OceanProtect时，请获取vCenter Server的管理IP地址。
  • 当注册ESXi主机至OceanProtect时，请获取ESXi主机的管理IP地址。

步骤5：注册VMware虚拟化环境

VMware虚拟化平台的用户名及其密码

必选

请向VMware虚拟化平台管理员获取。

  • 当注册的是vCenter Server时，请获取登录VMware vSphere web Client的用户名及其密码。
  • 当注册的是ESXi时，请获取登录ESXi主机的用户名及其密码。

CA证书

可选

注册VMware受保护环境时，建议导入受保护环境的CA证书，否则系统将无法验证其所访问设备的信息，安全性存在风险。

证书获取操作请参见步骤4：获取VMware证书。

证书吊销列表

可选

由证书颁发机构发布。

当证书因安全风险被吊销后，导入吊销列表。
