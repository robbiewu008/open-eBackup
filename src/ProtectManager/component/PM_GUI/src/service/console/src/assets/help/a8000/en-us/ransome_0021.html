Managing the Detection Mode

This section describes how to enable or disable dynamic detection.

Setting the Detection Mode (Applicable to 1.5.0)

  1. Choose Explore > Ransomware Protection.
  2. Click Overview.
  3. In the Settings area, click Modify.
  4. Select whether to enable dynamic detection.

    • Dynamic detection is enabled by default. Change behaviors of copy data are detected. If dynamic detection is disabled, static detection is performed based on known ransomware features of a single copy. You can log in to the technical support website and obtain the Readme file in the detection model package to query supported ransomware protection software.
    • For differences between static detection and dynamic detection, see Implementation Principles.
    • Enabling or disabling dynamic detection does not affect ongoing detection jobs.

  5. Click Save.

Setting the Detection Mode (Applicable to 1.6.0 and Later Versions)

  1. Choose Data Security > Ransomware Protection > Ransomware Protection and WORM Configurations.
  2. Click Detection Model. On the displayed Detection Model tab page, determine whether to enable Dynamic Detection.

    • Dynamic detection is enabled by default. Change behaviors of copy data are detected. If dynamic detection is disabled, static detection is performed based on known ransomware features of a single copy. You can log in to the technical support website and obtain the Readme file in the detection model package to query supported ransomware protection software.
    • For differences between static detection and dynamic detection, see Implementation Principles.
    • Enabling or disabling dynamic detection does not affect ongoing detection jobs.

  3. Click Save.
