(Optional) Configuring a Whitelist

This section enables you to skip certain specific directories or files during real-time ransomware detection by configuring a whitelist.

Procedure

  1. Choose Data Security > Real-Time Detection.
  2. Click Whitelist.
  3. Add a whitelist.

    1. Click Create.
    2. The type can be Directory or File Name Extension. Select a type, enter the directory name or file name extension, and click OK.

  4. Associate a policy. The whitelist takes effect only for associated policies.

    1. Choose More > Associated Policies.
    2. Select the policies to be associated with the whitelist and click OK.

    You can also select multiple whitelists and choose More > Associate Policy to associate policies in batches.
