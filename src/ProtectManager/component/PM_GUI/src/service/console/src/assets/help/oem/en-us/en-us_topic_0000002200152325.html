Step 2: Creating a ClickHouse Table Set

If you want to perform ClickHouse table set protection, create a table set first.

Procedure

  1. Choose Protection > Big Data > ClickHouse.
  2. Click Table Sets.
  3. Click Create to create a ClickHouse table set.

    1. Configure related parameters as described in Table 1.
      Table 1 ClickHouse table set parameters

      Parameter

      Description

      Name

      Customize a table set name.

      Cluster

      Select the ClickHouse cluster to be protected.

      Before selecting a ClickHouse cluster to be protected, ensure that the cluster has been registered with the product.

      Databases

      Select the database to be protected in the ClickHouse cluster.

      Table

      Click Please select. On the Select Table page that is displayed on the right, select the tables to be protected in the database.
    2. Click OK.

  4. Click OK.
