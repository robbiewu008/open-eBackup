Preparations for Backup

Before the backup, prepare information such as certificates, accounts, and IP addresses by performing operations described in Table 1.

Table 1 Preparations for backup

Item

Mandatory or Not

How to Obtain

To Be Used In

Address of the Huawei Cloud Stack GaussDB management plane

Yes
  1. On the Basic_Parameters sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for region_id and external_global_domain_name to obtain the corresponding parameter values.
  2. On the Tool_generated_IP_Params sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for AGW-LB-Float-IP to obtain the management plane IP address.
  3. Map the domain name.
    1. Log in to the host where the client is installed.
    2. Run the following command to open the /etc/hosts file:
      vi /etc/hosts
    3. Add the following information to the /etc/hosts file to map the domain name. xx.x.x.xx indicates the management plane IP address.
      xx.x.x.xx iam-apigateway-proxy.{region_id}.{external_global_domain_name}
  4. Obtain the management plane address https://iam-apigateway-proxy.{region_id}.{external_global_domain_name}:443.

Step 1: Registering a Huawei Cloud Stack GaussDB Project

Address of the Huawei Cloud Stack GaussDB service plane

Yes

On the Tool_generated_IP_Params sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for AGW-LB-Float-IP to obtain the corresponding parameter value XX.XX.XX.XX. https://XX.XX.XX.XX indicates the service plane address.

Resource space name and ID

Yes
  1. Log in to ManageOne Operation Portal.
  2. Click the username in the upper right corner and click My Settings. The Resource Space Name and ID columns in the resource space list present the required resource space names and IDs.

Cloud platform tenant and password

Yes

Name of the tenant to which the Huawei Cloud Stack GaussDB instance to be protected belongs. To obtain the value, perform the following steps:

  1. Log in to ManageOne Operation Portal and choose Organization > Resource Spaces.
  2. Search for the resource set information based on the resource space ID obtained in Preparations for Backup. VDC indicates the name of the VDC to which the cloud platform tenant belongs.
  3. Click Users. The user management page is displayed. Search for a user based on the value of VDC and select a user to obtain the username.

XBSA configuration path

Yes
  • Automatic push of the client: The XBSA configuration path is /opt/backupAgent/xbsa.conf.
  • Manual installation of the client: For details, see the xbsa.conf path configured in "Installing backUpAgent" in the OceanProtect DataBackup 1.5.0-1.6.0 Client Installation Guide.

Certificate

No

The certificate is required only when XBSA SSL Certificate Management is enabled. For details about how to generate and configure a certificate, see "Configuring Certificate Files for Mutual SSL Authentication" in the OceanProtect DataBackup 1.5.0-1.6.0 Client Installation Guide.
