Viewing a Media Set

This section describes how to view a media set.

Procedure

  1. Choose System > Infrastructure > Archive Storage Library.
  2. On the Tape tab page, select a node and click Media Set.
  3. View information about the media set in the media set list. Table 1 lists the relevant parameters.

    Table 1 Media set parameters

    Parameter

    Description

    Name

    Name of a media set, which can be customized.

    Tape Type

    Type of a tape.

    RW: Data can be read and written, which corresponds to the type of bound tapes.

    Status

    Status of a media set.

    • Online: The tape libraries in the media set are online.
    • Offline: The tape libraries in the media set are offline.
    • Partially Online: Some tape libraries in the media set are online and some are offline.

    Used Capacity

    Used capacity of a media set.

    You can also click the Name of a media set. In the window that is displayed on the right, view the basic information, selected tapes, alarm settings for insufficient available tapes, and retention policy of the media set.
