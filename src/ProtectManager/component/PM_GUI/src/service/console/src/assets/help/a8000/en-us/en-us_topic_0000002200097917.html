Step 2: Registering Huawei Cloud Stack

Before Huawei Cloud Stack backup and restoration, you must register Huawei Cloud Stack with the OceanProtect.

Prerequisites

  • A reverse proxy or agentless mode has been configured for the host where the client is installed.
  • You have obtained the password of operation administrator bss_admin for accessing ManageOne Operation Portal, which is used to connect Huawei Cloud Stack to the OceanProtect.

    If you need to use a custom user to connect Huawei Cloud Stack to the OceanProtect with the minimum permission, create a custom user by referring to "Security Management" in the Product Documentation of the corresponding Huawei Cloud Stack version. The procedure is as follows:

    1. Create a policy by referring to "Managing Policies". Select SC for Service and the vdc_readonly role for Permissions. This role has the permission to query information about users and resource spaces as well as self-service OM data in the tenant.
    2. Create an integrated account by referring to "Managing Integrated Accounts".
    3. Create a user group by referring to "Managing User Groups". Add the policy created in 1 and the integrated account in 2 to the user group. After the configuration is complete, you can use this user group to connect Huawei Cloud Stack to the OceanProtect.
  • You have created a storage device user whose role is Administrator and login mode is RESTful, which is used to connect storage resources of Huawei Cloud Stack to the OceanProtect. For details about how to create a user, see the Administrator Guide corresponding to the storage device version. If OceanStor V5 storage is used, set the user level to Administrator.
  • If iSCSI is used for backup, ensure that an iSCSI port that can be accessed by the backup network plane of the backup software has been configured for the storage device connected to Huawei Cloud Stack. If it is not configured, configure one by referring to the product documentation of the corresponding version on the storage side.

Context

After registering Huawei Cloud Stack, you can view and protect its projects, resource sets, and cloud servers on the management page, and use them as restoration targets.

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack.
  2. Click Register to register Huawei Cloud Stack.

    Table 1 describes Huawei Cloud Stack registration information.
    Table 1 Huawei Cloud Stack registration information

    Parameter

    Description

    Name

    User-defined name of Huawei Cloud Stack.

    IP Address

    Floating IP address of API Gateway. To obtain the IP address:

    On the Tool_generated_IP_Params sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for AGW-LB-Float-IP to obtain the corresponding IP address.

    Domain Name

    Access domain name of Huawei Cloud Stack. To obtain the domain name:

    On the Basic_Parameters sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for external_global_domain_name to obtain the corresponding parameter value.

    Username

    Username and password of an operation administrator for accessing ManageOne Operation Portal.

    Password

    Clients

    Select a client that can be used for Huawei Cloud Stack backup.

    Certificate Verification

    This function is enabled by default. The certificate of Huawei Cloud Stack will be verified during registration to ensure security of interactions between the OceanProtect and external devices. After this function is disabled, security risks may exist in the communication between the system and the cloud platform. Exercise caution when performing this operation.

    Cinder Certificate

    Import the Cinder certificate obtained in Step1: (Optional) Obtaining the Certificate. After the certificate is imported, the validity of the certificate will be verified during backup job execution.

  3. Click Add in the Storage Resources area. Table 2 describes storage resource parameters.

    • If the production storage of Huawei Cloud Stack is Huawei scale-out block storage and the client is deployed independently, you do not need to add storage resources. In other scenarios, you need to add storage resources. Otherwise, the backup job will fail.
    • If Huawei Cloud Stack production storage has multiple sets of storage resources, you need to add all the storage resources where the objects to be protected reside to the OceanProtect.
    Table 2 Storage resource parameters

    Parameter

    Description

    Type

    Type of a storage resource. The value can be Huawei SAN storage or Huawei scale-out block storage.

    Management IP Address
    • To obtain the management IP address of Huawei SAN storage: On the Basic_Parameters sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for Service storage REST URL to obtain the IP address in the corresponding parameter information.
    • To obtain the management IP address of Huawei scale-out block storage: On the Portal sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for Huawei Distributed Block Storage Business to obtain the IP address in the corresponding parameter information.
      NOTE:

      If the storage information of the VM to be backed up is not in the summary file, contact the Huawei Cloud Stack environment administrator to obtain the information and add it to the Huawei Cloud Stack registration information.

    Port

    Port for accessing storage resources. Port 8088 is used by default.

    Username

    Username and password of the administrator or super administrator for logging in to DeviceManager of the storage resource.

    Password

    Certificate Verification

    This function is enabled by default. The storage resource certificate will be verified during Huawei Cloud Stack registration to ensure security of interactions between the OceanProtect and external devices. After this function is disabled, security risks may exist in the communication between the system and the storage device. Exercise caution when performing this operation.

    Certificate

    Import the CA certificate of the storage resource obtained in Step1: (Optional) Obtaining the Certificate.

    Revocation List

    The CRL is issued by the certificate authority. After the CRL is imported, the system checks whether the client certificate has been revoked. If the certificate has been revoked, storage device registration fails.
    VBS Node Login Information
    NOTE:

    This parameter is available only when Type is set to Huawei scale-out block storage in 1.5.0SPC19 and later versions.
    Enable this function and perform the following operations if you need to use the built-in client to perform incremental backup for volumes of Huawei scale-out block storage (8.1.3 or later).
    1. On the displayed page, configure the IP address, port number, username, and password of the VBS node.
      • IP Address: Enter the IP address of the VBS node.
      • Port: Enter the SSH service port. The default value is 22.
      • Username: Enter a username that has the permission to log in to the VBS node, for example, fsp.
      • Password: Enter the user password.
    2. Check whether the configured user has the permission to run VBS commands.
      1. Use PuTTY to log in to the configured VBS node as the configured user.
      2. Run the following commands to check whether the user has the permission to run VBS commands:

        sudo /bin/vbs_cli

        sudo /opt/dsware/vbs/script/vbs/vbs_cmd.sh

        sudo /bin/sh

        If the preceding commands can be successfully executed without entering the password, the user has the permission to run VBS commands. No further action is required. Otherwise, go to 3.

    3. Perform the following operations to configure the permission to run VBS commands for the user:
      1. Run the su - root command and enter the password of user root to switch to user root.
      2. Run the vi /etc/sudoers command to open the configuration file and add the following content to the file:
        fsp ALL=(root) NOPASSWD:/bin/vbs_cli
fsp ALL=(root) NOPASSWD:/opt/dsware/vbs/script/vbs/vbs_cmd.sh
fsp ALL=(root) NOPASSWD:/bin/sh
        NOTE:

        Replace fsp with the username configured in 1.

  4. Click OK.

    After registration is successful, Huawei Cloud Stack information is displayed on the left of the page.
