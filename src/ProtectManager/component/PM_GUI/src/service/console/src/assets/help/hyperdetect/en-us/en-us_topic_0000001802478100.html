Managing Honeypot Detection

You can enable, set, and disable honeypot detection and view honeyfiles.

On the navigation pane, choose Explore > Ransomware Protection > Real-Time Ransomware Detection > Honeypot Detection. Perform operations by referring to Table 1.

Table 1 Managing honeypot detection

Operation

Description

Navigation Path

Enabling honeypot detection

Scenario

This operation enables you to enable honeypot detection.

Click Enable Honeypot Detection.

Disabling honeypot detection

Scenario

This operation enables you to disable honeypot detection.

Click Disable Honeypot Detection.

Setting honeypot detection

Scenario

This operation enables you to set the update frequency of honeyfiles.

Choose More > Set Honeypot Detection.

Viewing honeyfiles

Scenario

This operation enables you to view the path, quantity, and names of honeyfiles.

Choose More > View Honeyfile.
