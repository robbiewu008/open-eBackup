How Can I Add a User to the Administrators User Group?

This section describes how to add a non-system administrator to the Administrators user group if the non-system administrator is required to have the permissions of the Administrators user group.

Procedure (Applicable to Non-AD-Domain Scenarios)

  1. Log in to the host where the client is installed as a system administrator.
  2. Press Win+R to open the Run window.
  3. Enter lusrmgr.msc to go to the local user and group management page.
  4. Choose Groups > Administrators.
  5. Right-click Administrators and choose Add to Group from the shortcut menu.
  6. In the displayed dialog box, click Add.
  7. Enter the user name and click OK.

Procedure (Applicable to AD-Domain Scenarios)

  1. Log in to the host where the client is installed as a system administrator.
  2. Press Win+R to open the Run window.
  3. Enter cmd to access the command prompt window.
  4. Run the following command to add the user (for example, user1) to the Administrators user group:

    net localgroup administrators user1 /add
