Step 1: Registering a Huawei Cloud Stack GaussDB Project

Before Huawei Cloud Stack GaussDB backup and restoration, you must register Huawei Cloud Stack GaussDB with the product.

Prerequisites

  • You have registered with Huawei Cloud and created a cloud platform tenant on Huawei Cloud. This user is used to connect Huawei Cloud Stack GaussDB to the product.
  • A private line has been established for the Huawei Cloud Stack GaussDB project and the host where the client is installed. For details, contact technical support engineers of the service provider.
  • If you need to enable XBSA SSL Certificate Management, ensure that the SSL certificate has been obtained during client installation. For details, see "Configuring Certificate Files for Mutual SSL Authentication" in the Client Installation Guide.

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack GaussDB.
  2. Click Register to register Huawei Cloud Stack GaussDB.

    Table 1 describes Huawei Cloud Stack GaussDB registration information.
    Table 1 Huawei Cloud Stack GaussDB registration information

    Parameter

    Description

    Name

    User-defined name of Huawei Cloud Stack GaussDB.

    Management Plane URL

    To obtain the management plane URL, perform the following steps:

    1. On the Basic_Parameters sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for region_id and external_global_domain_name to obtain the corresponding parameter values.
    2. On the Tool_generated_IP_Params sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for AGW-LB-Float-IP to obtain the management plane IP address.
    3. Map the domain name.
      1. Log in to the host where the client is installed.
      2. Run the following command to open the /etc/hosts file:
        vi /etc/hosts
      3. Add the following information to the /etc/hosts file to map the domain name. xx.x.x.xx indicates the management plane IP address.
        xx.x.x.xx iam-apigateway-proxy.{region_id}.{external_global_domain_name}
    4. Obtain the management plane address https://iam-apigateway-proxy.{region_id}.{external_global_domain_name}:443.

    Service Plane URL

    On the Tool_generated_IP_Params sheet of parameter information summary file xxx_export_all_v2_EN.xlsx exported from HCC Turnkey during Huawei Cloud Stack software installation, search for AGW-LB-Float-IP to obtain the corresponding parameter value XX.XX.XX.XX. https://XX.XX.XX.XX indicates the service plane address.

    Cloud Platform Tenant Name

    Name of the tenant to which the Huawei Cloud Stack GaussDB instance to be protected belongs. Obtain the name as instructed in Preparations for Backup.

    Cloud Platform Tenant Password

    Password of the tenant to which the Huawei Cloud Stack GaussDB instance to be protected belongs. Obtain the name as instructed in Preparations for Backup.

    VDC Name of Cloud Platform Tenant

    Name of the account to which the cloud platform tenant belongs. For details, see Cloud Platform Tenant Name in Table 1.

    Resource Space Name

    Name of the resource space of the tenant. Obtain the value from Preparations for Backup.

    Resource Space ID

    ID of the resource space of the tenant. Obtain the value from Preparations for Backup.

    Clients

    Host where the client registered with the product is installed.

    XBSA Configuration Path

    Path of the xbsa.conf file that is configured during backupAgent installation.

    Automatic push of the client: The XBSA configuration path is /opt/backupAgent/xbsa.conf.

    Manual installation of the client: You can obtain the path by referring to the XBSA configuration path in Table 1.

    XBSA SSL Certificate Management

    This function is enabled by default. After this function is enabled, the certificate of Huawei Cloud Stack GaussDB will be verified during registration to ensure security of interactions between the product and external devices. After this function is disabled, security risks may exist in the communication between the system and the cloud platform. Exercise caution when performing this operation.

    Root Certificate

    Root certificate file with the file name extension .pem. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    User Certificate

    Client user certificate file with the file name extension .crt. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    User Private Key

    Client user private key file with the file name extension .key. For details about how to obtain the file, see the preparation item Certificate in Table 1. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

    Private Key Password

    Client private key password character string set when the certificate file is created. This parameter is mandatory only when XBSA SSL Certificate Management is selected.

  3. Click OK.
