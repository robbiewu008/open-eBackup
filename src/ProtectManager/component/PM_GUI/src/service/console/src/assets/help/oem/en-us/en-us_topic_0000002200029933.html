Deleting a Management Data Backup

This section describes operations to delete invalid or unnecessary management data backup copies for saving storage space.

Precautions

After a management data backup is deleted, the backup cannot be used for system restoration. Exercise caution when performing this operation.

Procedure

  1. In the navigation pane, choose System > Settings > Management Data Backup.
  2. In the row of the target management data backup, choose More > Delete in the Operation column.
  3. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
