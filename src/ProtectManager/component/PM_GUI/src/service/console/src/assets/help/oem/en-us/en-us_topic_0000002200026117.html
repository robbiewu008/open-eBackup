Step 1: Registering a User with the Nutanix Virtualization Platform and Adding User Permissions

The Nutanix user permissions need to be added on the Nutanix virtualization platform. Otherwise, the backup will fail due to insufficient permissions. This section uses Nutanix 6.5.5.7 LTS as an example.

Procedure

  1. Log in to the Nutanix platform as a user with the role management permission through a browser.
  2. Choose Settings > Users and Roles > Local User Management.
  3. On the Local User Management tab page, click New User to add a custom user.
  4. Set the related information based on the instructions on the page.
  5. You need to select at least the Cluster Admin role for Roles to add user permissions.

    The user's role with the minimum permission is Cluster Admin.

  6. Click Save.
