Creating a Resource Set

After a source set is created, you can perform resource set association to assign the manageable resource scope and bind different resource sets to different role permissions.

Procedure

  1. Choose System > Security > RBAC > Resource Sets.
  2. Click Create.
  3. Create a resource set.

    On the Create Resource Set page, set the parameters listed in Table 1.

    Table 1 Parameters for creating a resource set

    Parameter

    Description

    Resource Set Name

    Custom resource set name.

    Resource Set Description

    Custom resource set description.

    Select Resources

    In the Select Resources area, select resources of the corresponding resource type.

  4. click OK.
