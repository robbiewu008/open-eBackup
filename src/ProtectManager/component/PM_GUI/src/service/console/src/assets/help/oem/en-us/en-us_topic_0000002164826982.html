Step 3: Registering an AntDB Cluster Instance

Prerequisites

Before registering an AntDB cluster instance, ensure that each host in the cluster has been registered.

Procedure

  1. Register a cluster instance.

    1. On the AntDB page, click Register.
    2. Set Type to Cluster instance.
    3. Configure the database instance and its authentication information.

      Table 1 describes the related parameters.

      Table 1 Registration information of an AntDB cluster instance

      Parameter

      Description

      Name

      Username for installing the database.

      Username

      OS username of the database. For details about how to obtain the value, see Preparing for Backup.

      Database Username

      Username and password of the database administrator. For details about how to obtain the value, see Preparing for Backup.

      Database Password

      Database Stream Replication Username

      Username for streaming replication in the database. For details about how to obtain the value, see Preparing for Backup.

      Database Stream Replication Password
    4. Click Add. In the displayed Add dialog box, add all nodes of the cluster instance.

      Table 2 describes the related parameters.

      Table 2 Node information of a cluster instance

      Parameter

      Description

      Hosts

      Host node in the cluster.

      Database Installation Path

      Path for installing the database to be registered. For details about how to obtain the value, see Preparing for Backup.

      Full Path of Configuration File

      Full path of the xxx. configuration file. Example:

      /etc/adbhamgr/adbhamgr_antdbcluster.yaml

      For details about how to obtain the value, see Preparing for Backup.

      Service Plane IP Address

      IP address of the host for the database service to receive TCP/IP connections.

      Database Port

      Port number used by the database service to listen on TCP/IP connections. For details about how to obtain the value, see Preparing for Backup.
    5. Click OK.
