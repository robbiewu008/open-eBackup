修改用户

OceanProtect支持修改已有用户的用户信息。

注意事项

不允许修改内置账号的信息。

操作步骤

  1. 选择“系统 > 安全 > RBAC > 用户”。
  2. （可选）可以单击“名称”旁的，对用户进行搜索。
  3. 单击需要修改的用户所在行右侧的“修改”。
  4. 修改用户信息。参数说明如表1所示。

    表1 用户基本参数

    参数

    说明

    登录认证方式

    支持“登录密码”和“登录密码+邮件动态口令”两种方式。仅本地用户支持设置登录认证方式。

    说明：
    • 如果您对安全要求较高，请使用“登录密码+邮件动态口令”方式。使用该方式前，请确认已参考管理邮件服务章节完成动态口令接收邮箱配置。
    • 内置账号只支持“登录密码”方式认证。

    收件人邮箱地址

    填写用于接收邮件动态口令的邮箱地址。仅创建“本地用户”类型，且登录认证方式为“登录密码+邮件动态口令”时需要填写该参数。

    角色

    单击“添加”，选择系统提供的内置角色或者创建新角色并设置用户的权限范围。创建新角色具体操作请参考创建角色。

    默认角色

    展示创建当前用户时选择的默认角色。

    描述

    新创建的用户的描述信息。

    [取值范围]

    长度范围为0~255位。

    [示例]

    用户

    用户最大连接数

    新创建的用户同时登录系统最大数量限制，不设置则无限制。

    • ：开启。
    • ：关闭。

    默认状态为。

    [取值范围]

    数值范围为1~8个。

    默认值：5

    说明：

    当角色为“远端设备管理员”时，不显示该参数。

  5. 单击“下一步”。
  6. （可选）为角色关联资源集，通过关联资源集分配可管理的资源范围，并将不同的资源集与不同的角色权限绑定。

    1. 选择需要关联资源集的角色。
    2. 单击“创建资源集”，自定义资源集名称和描述，并选择需要关联的资源。
    3. 单击“确定”。

  7. 预览修改后的用户信息，单击“完成”，完成修改用户操作。
