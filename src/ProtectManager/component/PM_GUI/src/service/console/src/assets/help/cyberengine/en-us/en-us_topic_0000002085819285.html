Updating a File Name Extension Filtering Rule

Modify a file name extension filtering rule.

Procedure

  1. In the navigation pane, choose Data Security > File Interception.
  2. On the File Systems page, choose More > Update File Name Extension Filtering Rule.
  3. On the Update File Name Extension Filtering Rule page, select the required file name extensions.
  4. Click OK.
