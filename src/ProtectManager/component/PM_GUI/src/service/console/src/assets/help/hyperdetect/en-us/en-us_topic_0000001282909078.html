Enabling Intelligent Ransomware Detection

The system detects known ransomware attack features to identify whether file systems are attacked by ransomware. If no ransomware attack is identified, the system analyzes and compares file system snapshot changes, and uses machine learning algorithms to further check whether the file systems are infected by ransomware. This section describes how to enable intelligent ransomware detection.

Precautions

During intelligent ransomware detection, if the file system snapshot directory corresponding to the ransomware detection snapshot being detected is invisible, the system will create a clone file system whose name starts with DEE-ANTI-CLONE based on the ransomware detection snapshot. The clone file system is used to generate metadata features of the snapshot. During the detection, do not delete or rename the clone file system whose name starts with DEE-ANTI-CLONE. If the file is deleted or renamed, the detection may fail. After the intelligent ransomware detection, the system automatically deletes the clone file system whose name starts with DEE-ANTI-CLONE.

Procedure

  1. On the navigation pane, choose Explore > Ransomware Protection > Ransomware Detection.
  2. On the Detection Settings tab page, select the vStore for which you want to enable intelligent ransomware detection and choose More > Enable Intelligent Ransomware Detection. After intelligent ransomware detection is enabled, you need to create a ransomware detection snapshot by referring to Creating a Ransomware Detection Snapshot. After the snapshot is generated, the system immediately performs ransomware detection on the snapshot.

    • You can also select multiple vStores for which you want to enable intelligent ransomware detection and choose More > Enable Intelligent Ransomware Detection above the vStore list.
    • If intelligent ransomware detection is not required, choose More > Disable Intelligent Ransomware Detection. Alternatively, you can select multiple vStores for which you want to disable intelligent ransomware detection and choose More > Disable Intelligent Ransomware Detection above the vStore list.
