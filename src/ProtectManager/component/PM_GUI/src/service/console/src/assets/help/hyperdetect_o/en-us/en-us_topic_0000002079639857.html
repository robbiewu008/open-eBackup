Managing Honeypot Detection

You can enable, set, and disable honeypot detection and view honeyfiles.

On the navigation pane, choose Data Security > Ransomware Protection > Real-Time Ransomware Detection > Honeypot Detection. Perform operations by following the instructions in Table 1.

Table 1 Managing honeypot detection

Operation

Description

Navigation Path

Enabling honeypot detection

Context

This operation enables you to enable honeypot detection.

Click Enable Honeypot Detection above the list, or choose More > Enable Honeypot Detection in the specific row.

Disabling honeypot detection

Context

This operation enables you to disable honeypot detection.

Click Disable Honeypot Detection above the list, or choose More > Disable Honeypot Detection in the specific row.

Setting honeypot detection

Context

This operation enables you to set the update frequency of honeyfiles.

Choose More > Set Honeypot Detection in the specific row.

Viewing honeyfiles

Context

This operation enables you to view the path, quantity, and names of honeyfiles.

Choose More > View Honeyfile in the specific row.
