Step 5: Creating a Remote Device Administrator (Applicable to 1.5.0)

This section describes how to create a remote device administrator in the secondary cluster for replication. This administrator is used to verify the primary cluster during the replication.

Procedure

  1. Log in to the product management page of the secondary cluster for replication.
  2. Choose System > Security > Users and Roles.
  3. Click Create.

    Table 1 describes the related parameters.

    Table 1 User parameters

    Parameter

    Description

    User Type

    Select Local user.

    Name

    Name of a user, which can be customized.

    Login Authentication Mode

    Select Login password.

    Password

    Password of a user.

    Confirm Password

    Confirm password.

    Role

    Select Remote Device Administrator.

    Description

    Description of a user.

  4. Click OK.
