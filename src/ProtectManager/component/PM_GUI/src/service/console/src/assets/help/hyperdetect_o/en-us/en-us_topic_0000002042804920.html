Updating a File Name Extension Filtering Rule

This section describes how to update a file name extension filtering rule for a vStore.

Prerequisites

If you need to add file name extensions to be intercepted for a vStore, create a file name extension filtering rule first by following the instructions in Creating a File Name Extension Filtering Rule.

Procedure

  1. On the navigation pane, choose Data Security > Ransomware Protection > Ransomware Detection Settings.
  2. On the Ransomware Detection Settings tab page, locate the row that contains the desired vStore and choose More > Update File Name Extension Filtering Rule.

    The Update File Name Extension Filtering Rule dialog box is displayed.

    Select or deselect file name extension filtering rules in the list as required. All selected file name extension filtering rules are automatically associated with the current vStore.

    By default, all file name extension filtering rules that have been associated with the current vStore are selected.

  3. Click OK.
