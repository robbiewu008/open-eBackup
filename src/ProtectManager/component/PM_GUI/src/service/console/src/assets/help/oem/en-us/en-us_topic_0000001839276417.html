Querying HDFS Information

Procedure

  1. On the Cluster tab page, query the HDFS cluster information.

    1. This page displays information about registered HDFS clusters.

      Table 1 describes the related parameters.

      Table 1 HDFS cluster information

      Parameter

      Description

      Name

      HDFS cluster name.

      fs.defaultFS

      Address of the HDFS cluster.

      Status
      Status of the HDFS cluster.
      • Online: If the connection between the agent host and the HDFS cluster is normal, Status is Online.
      • Offline: If the connection between the agent host and the HDFS cluster is abnormal, Status is Offline.

      Authentication Mode

      Authentication mode for communication between the HDFS cluster and the product.

      Agent Host

      Agent host associated with the cluster.
    2. Click the cluster name to view the basic information of the filesets related to the cluster.

  2. On the Filesets tab page, query the HDFS fileset information.

    1. This page displays the details about the filesets.

      Table 2 describes the related parameters.

      Table 2 Fileset parameters

      Parameter

      Description

      Name

      Name of the fileset.

      Cluster

      Name of the protected HDFS cluster.

      SLAs

      If an SLA has been associated with the fileset, the SLA name is displayed. Otherwise, -- is displayed.

      SLA Compliance
      Whether a backup job is successfully executed based on the backup frequency or backup window set in the SLA.
      • If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
      • If the backup time does not meet the backup time window or backup frequency requirements, for example, if the backup frequency set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup frequency or backup window.

      Protection Status

      Whether the fileset is protected.

      • Unprotected: The fileset is not associated with an SLA, or the fileset is associated with an SLA but protection is disabled.
      • Protected: The fileset is associated with an SLA and protection is activated.
      • Creating: The fileset is associated with an SLA and protection is being created.
    1. Click the name and check the overview, copy information, and related jobs.
      • Overview

        Display the SLA information, job execution time, and total number of copies.

      • Copy data

        You can search for copies by year, month, and day.

        If is displayed below the time, a copy exists at the time.

      • Related jobs

        Displays backup and restoration jobs.
