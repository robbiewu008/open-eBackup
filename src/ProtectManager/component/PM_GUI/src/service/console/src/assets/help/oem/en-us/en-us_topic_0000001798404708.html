Step 1: Creating a Volume

Create a volume before performing volume protection.

Procedure

  1. Choose Protection > File Systems > Volume.
  2. Click Create.
  3. In the Create dialog box that is displayed, create a volume for a single host.

    Table 1 describes the related parameters.

    Table 1 Parameters for creating a volume

    Parameter

    Description

    Name

    User-defined volume name.

    Select Host

    Select the host for which volume protection is to be performed.

    NOTE:

    Only one volume set can be created for a host. The hosts for which volume sets have been created are not displayed.

    Volume

    Select the volumes to be protected.

  4. Click OK.
