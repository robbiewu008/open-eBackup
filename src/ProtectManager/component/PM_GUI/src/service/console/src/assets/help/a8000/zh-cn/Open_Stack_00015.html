步骤4：注册阿里云组织

执行阿里云备份恢复前，需要将阿里云组织注册至OceanProtect。

操作步骤

  1. 选择“保护 > 云平台 > 阿里云”。
  2. 单击“注册”，注册阿里云信息。

    阿里云注册信息如表1所示。
    表1 阿里云注册信息

    参数

    说明

    名称

    自定义注册的云平台名称。

    运营控制台Endpoint

    地域对应的接入地址（Endpoint）。

    示例值：ecs.example.com

    Organization ak

    待注册的组织的AccessKey ID和密码。获取方式参见备份前准备。

    Organization sk

    Organization ID

    待注册的组织ID，获取方式请参见备份前准备。

    Region ID

    环境Region，获取方式请参见备份前准备。

    代理主机

    当用户未选择时，系统会从注册资源时配置的代理主机中选择1到多个代理主机执行备份。

    资源扫描频率

    系统自动扫描资源的间隔时间，单位小时。

    使用校验证书

    开启“校验证书”后，导入获取的证书。使用校验证书可增加资源的安全性。

  3. （可选）可在高级配置中选择添加“代理服务器”。
  4. 单击“确定”。
