Modifying Authorization of a Local Cluster Data Protection Administrator

This section describes how to modify the authorization information after granting permissions to a local cluster data protection administrator.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  1. In the multi-domain cluster area, choose More > Modify Authorization in the row that contains the desired managed cluster.
  2. In the Data Protection Administrator area, select the desired local cluster data protection administrator.
  3. In the Authorization Information area, enter the username and password of the data protection administrator of the managed cluster.
  4. Click OK.
  5. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
