Managing NDMP NAS File System Protection

You can modify, remove, activate, or disable NDMP NAS file system protection.

Choose Protection > File Systems > NAS File Systems and find the desired NDMP NAS file system.

Table 1 describes the operations.

Table 1 Operations related to NDMP NAS file system protection

Operation

Description

Navigation Path

Modify Protection

Scenario

You can modify the protection plan of an NDMP NAS file system. After the modification, the next backup will be executed based on the new protection plan.

Notes

  • The modification does not affect the protection job that is in progress. The new protection plan is used from the next protection period.
  • If the SLA associated with a resource changes, you are advised to modify the current SLA by choosing Protection > Protection Policies > SLAs. You are advised not to associate the resource with another SLA by modifying the protection.

Locate the row that contains the target resource, and choose More > Modify Protection.

Remove Protection

Scenario

You can remove the protection plan if the SLA file system no longer needs to be protected. After the protection plan is removed, the system cannot execute protection jobs for the NDMP NAS file system.

Note

  • When a protection job of the NDMP NAS file system is in progress, protection cannot be removed.
  • If the SLA associated with a resource changes, you are advised to modify the current SLA by choosing Protection > Protection Policies > SLAs. You are advised not to associate the resource with another SLA after removing the protection.

Locate the row that contains the target resource, and choose More > Remove Protection.

Restore

Scenario

If a file in the NDMP NAS file system is lost or damaged, you can restore the file using its backup.

Locate the row that contains the target resource, and choose More > Restore.

Disable Protection

Scenario

If you do not need to perform periodic protection for the NDMP NAS file system, you can disable the protection plan of the NDMP NAS file system. After the protection plan is disabled, the system does not automatically back up the NDMP NAS file system.

Note

Disabling protection does not affect ongoing backup jobs.

Locate the row that contains the target resource, and choose More > Disable Protection.

Activate Protection

Scenario

After the protection plan of the NDMP NAS file systems is disabled, you can activate it again. After the protection plan is activated, the system backs up the NDMP NAS file system based on the protection plan.

Note

This operation can be performed only for a NAS file system that has been associated with an SLA and is in the Unprotected state.

Locate the row that contains the target resource, and choose More > Activate Protection.
