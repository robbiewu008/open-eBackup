Modifying a Backup Storage Unit

This section describes how to modify the name or alarm threshold of a backup storage unit.

Precautions

The name of an automatically added backup storage unit in a local cluster cannot be modified.

Procedure

  1. Choose System > Infrastructure.
  2. Click Backup Storage Unit Groups.
  3. Click Backup Storage Units.
  4. In the row that contains the backup storage unit to be modified, choose More > Modify.
  5. After modifying the name or alarm threshold of the backup storage unit, click OK.
