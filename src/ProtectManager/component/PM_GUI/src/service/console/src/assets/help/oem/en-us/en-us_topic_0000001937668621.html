Periodically Executing Backup

Associate the NDMP NAS file systems to be protected with a specified SLA. The system backs up these resources periodically based on the SLA.

Procedure

  1. Choose Protection > File Systems > NAS File Systems.
  2. In the row of the target NDMP NAS file system, choose More > Protect .
  3. Select the created SLA.

    You can also click Create to create an SLA.

  4. Select the NDMP share protocol.
  5. Select the agent host.

    By default, the built-in agent is selected. If an external agent exists, you can also select the external agent.

  6. Click OK.

    If the current system time is later than the first backup start time of the SLA, you can select to perform backup immediately after the backup SLA is associated or wait for the next scheduling based on the backup policy set in the SLA.
