Preparations for Backup

Before backing up SAP on Oracle databases, prepare related information by following instructions in Table 1.

Table 1 Preparations for backup

Item

How to Obtain

To Be Used In

SAP instance ID

SAPSID in Context

Step 1: Registering an SAP on Oracle Database

Oracle instance ID

DBSID in Context.

Username and password of the database

Name and password of the user created in Starting the Database and Creating a Database User.

Database status

Check whether the database is in the OPEN state.

  1. Go to the run directory created in Manually Creating a run Directory and Copying Files to the Directory. Example:
    C:\>cd usr\sap\EH7\SYS\exe\run
  2. Run the following command and check the value of Instance status (status). If the value is OPEN, the database is in the OPEN state.
    brspace -c force -l E -u userName/passWord -f dbshow -c dbstate
    userName and passWord indicate the name and password of the database user created in Starting the Database and Creating a Database User. Example:
    C:\usr\sap\EH7\SYS\exe\run>brspace -c force -l E -u user1/Test123 -f dbshow -c dbstate
