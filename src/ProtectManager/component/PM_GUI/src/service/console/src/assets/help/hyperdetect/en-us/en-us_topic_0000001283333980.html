Quick Start

This section describes the ProtectManager management functions, helping you quickly find service operation entries and improving operation efficiency.

Table 1 describes the ProtectManager functions.

Table 1 Functions of ProtectManager

Application Type

Function

Home

Helps you quickly learn the job status, managed resources, alarms, and historical performance of the system.

Protect

Storage allows you to manage and protect local file systems.

Protection Policies allows you to manage SLAs.

NOTE:

A service level agreement (SLA) is a set of configurable data protection policies that define rules for data protection. These policies are associated with an environment to implement protection.

Explore

Ransomware Protection allows you to manage ransomware detection configurations of file systems and view ransomware detection results.

Insight

Alarms and Events allows you to view uncleared alarms, cleared alarms, operation logs, and run logs.

Jobs allows you to view details about running jobs and completed jobs.

System

Infrastructure allows you to manage local storage.

Security allows you to manage users, roles, security policies, and certificates in the system.

Log Management allows you to manage debug log levels and export logs.

Export Records allows you to view system logs, detection reports, and copy files.

Settings allows you to manage alarm notifications, event dump, and email services.
