全局标签搜索

通过本节操作您可以通过标签对各类生产资源进行浏览或搜索。

背景信息

资源的全局搜索功能支持对接入OceanProtect的所有应用进行搜索。

操作步骤

  1. 单击页面上方的，进入“全局搜索”页面。

    1. 在搜索栏选择“标签”。
    2. 输入标签名称并单击，当前页面将展示添加了该标签的资源的详细信息。

  2. 在界面左侧导航树上可以根据需要对展示结果进行再次筛选。

    参数间是“或”的关系，即筛选出来的结果是所选条件的并集。

    表1 搜索参数说明

    参数

    说明

    资源类型

    选择要搜索的资源类型。

    保护状态
    资源当前是否被保护。
    • 未保护：资源未关联SLA，或者资源已关联SLA但已禁用保护。
    • 已保护：资源已关联SLA，且已激活保护。
    • 创建中：资源已关联SLA，正在创建保护。

    SLA遵从度

    根据SLA是否遵从进行过滤。

    资源位置

    输入资源的位置。
    完成搜索后页面右侧展示搜索结果。相关参数说明如表2所示。
    表2 参数说明

    参数

    说明

    名称

    资源名称。

    类型

    资源类型。

    位置

    资源所在位置。

    SLA

    当资源已关联SLA时，则显示SLA名称，否则显示“--”。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态
    当前资源是否被保护。
    • 未保护：当前资源未关联SLA，或者已关联SLA但已禁用保护。
    • 已保护：当前资源已关联SLA，且已激活保护。
    • 创建中：当前资源已关联SLA，正在创建保护。

    标签

    当前资源关联的标签名称。

  3. 在资源所在行，单击“更多”，可以对资源添加保护、修改保护等操作。
