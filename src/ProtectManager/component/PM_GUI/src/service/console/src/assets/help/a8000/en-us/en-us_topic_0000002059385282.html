Deleting a Resource Set

You can delete a resource set that is no longer needed.

Precautions

The default resource sets and default public resource set cannot be deleted.

Procedure

  1. Choose System > Security > RBAC > Resource Sets.
  2. (Optional) Click next to Resource Set to search for a resource set.
  3. In the row of the resource set to be deleted, choose More > Delete.

    You can delete resource sets in batches. Specifically, select the target resource sets and click Delete.

  4. In the dialog box that is displayed, confirm the information and click OK.
