管理告警和事件

操作步骤

管理告警

  1. 选择“告警和事件 > 告警”。

    对于1.5.0版本，请选择“监控 > 告警和事件 > 告警”。

  2. （可选）可以单击“级别”或“告警对象”旁的，或单击“发生时间”旁的，对告警进行筛选。或通过搜索节点名称，对告警进行搜索。
  3. 单击某条告警的“描述”，参考右侧显示的“修复建议”处理告警。
  4. （可选）清除告警。

    1. 在列表中选择需要清除的当前告警，单击“清除”。
    2. 在弹出的对话框中，单击“确定”。

管理事件

  1. 选择“告警和事件 > 事件”。

    对于1.5.0版本，请选择“监控 > 告警和事件 > 事件”。

  2. （可选）可以单击“级别”或“告警对象”旁的，或单击“发生时间”旁的，对事件进行筛选。或单击“节点”旁的，对事件进行搜索。
  3. 单击某条事件的“描述”，参考右侧显示的“修复建议”处理事件。
  4. （可选）可以在列表中选择需要导出的当前事件，单击，导出事件。
