创建备份存储单元

将外部备份存储作为备份存储单元添加到当前本产品，用于存放备份数据。

前提条件

本产品和要添加的备份存储单元正常运行，且通信正常。

注意事项

  • 仅支持添加本产品对应的本地存储为备份存储单元。
  • 备份存储单元不支持同时作为被管理集群。

操作步骤

  1. 选择“系统 > 基础设施 > 集群管理”。
  2. 单击“备份存储单元组”。
  3. 选择“备份存储单元”。
  4. 单击“创建”。创建备份存储单元。

    相关参数说明如表1所示。
    表1 添加备份存储单元参数说明

    参数

    说明

    名称

    用户自定义备份存储单元的名称。

    IP地址

    外部备份存储的管理IP地址。

    端口

    外部备份存储的管理端口。

    默认端口为8088。

    用户名

    外部备份存储超级管理员的用户名及其密码。

    密码

  5. 单击“确定”。
