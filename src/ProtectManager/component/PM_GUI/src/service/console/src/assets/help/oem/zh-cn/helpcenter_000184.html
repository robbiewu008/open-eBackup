管理日志

通过查看调试日志，用户可以了解在操作或设计模板过程中所触发的错误信息、提示信息等。

操作步骤

  1. 选择“系统 > 日志 > 日志管理”。
  2. 在“日志设置”区域，选择需要修改的节点，单击“日志级别”右侧的下拉列表，选择需要导出的日志级别。

    通过修改日志级别，您可以导出所选级别（及其以上级别）的系统调试日志。

    支持批量设置日志级别，选择需要导出的日志级别后，单击“确定”，完成导出日志级别的设置。

  3. 配置“信息导出”的参数。

    相关参数说明如表1所示。

    表1 “信息导出”的参数信息

    参数

    说明

    导出内容

    说明：

    1.6.0及后续版本支持选择“导出内容”参数。

    包含日志文件和配置文件，可多选。

    说明：

    导出日志包含系统调试日志和系统运行参数。

    节点

    选择需要导出信息的节点。

    控制器

    选择控制器。

    软件模块名称

    包含“基础设施”、“数据保护引擎”、“管理服务”，可多选。

    导出范围

    说明：

    1.6.0及后续版本支持选择“导出范围”参数。

    可选择“自定义时间范围”或“全部”，自定义时间取值范围：1~10000天。

    导出日志文件名称

    输入日志文件名称。

    导出配置文件名称

    输入配置文件名称。

  4. 单击“导出”，完成导出调试日志的操作。
