Viewing Huawei Cloud Stack information

After Huawei Cloud Stack is registered with the product, you can view information about cloud servers of Huawei Cloud Stack.

Procedure

  1. Choose Protection > Cloud Platforms > Huawei Cloud Stack.
  2. On the Tenant tab page, view the tenant information.

    For details about tenants, see Table 1.
    Table 1 Tenant information

    Parameter

    Description

    Name

    Tenant name.

    You can click a tenant name to view information about VDC administrators of the tenant.

    Project/Resource Set Quantity

    Number of projects/resource sets in a tenant.
    Tag
    NOTE:

    This parameter is available only in 1.6.0 and later versions.

    If a tag has been added for the resource, the tag name is displayed. Otherwise, -- is displayed.

  3. On the Project/Resource Set tab page, view the project/resource set information.

    Table 2 lists the details about the projects/resource sets.
    Table 2 Project/Resource set information

    Parameter

    Description

    Name

    Name of a project/resource set.

    Click the name of a project/resource set to view its overview and jobs.

    • Overview

      The name, SLA information, job execution time, total number of copies, and cloud server information are displayed.

    • Jobs

      All jobs, including backup and restoration jobs, are displayed.

    Location

    Location of a project/resource set.

    SLAs

    If an SLA is associated with a project/resource set, the SLA name is displayed. Otherwise, -- is displayed.

    SLA Compliance
    Whether a backup job is successfully executed based on the backup interval or backup window set in the SLA.
    • Met: If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
    • Missed: If the backup time does not meet the backup time window or backup interval requirements, for example, if the backup interval set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup interval or backup window.

    Protection Status

    Whether a project/resource set is protected.

    • Unprotected: The project/resource set is not associated with an SLA, or the project/resource set is associated with an SLA but protection is disabled.
    • Protected: The project/resource set has been associated with an SLA and protection has been activated.
    • Creating: The project/resource set is associated with an SLA and protection is being created.
    Owner
    NOTE:

    This parameter is available only in 1.5.0.

    Data protection administrator to which resources are allocated for management. If -- is displayed, the resource has not been allocated to the data protection administrator.
    Tag
    NOTE:

    This parameter is available only in 1.6.0 and later versions.

    If a tag has been added for the resource, the tag name is displayed. Otherwise, -- is displayed.

  4. View the cloud server information on the Cloud Server tab page.

    Table 3 describes the detailed information.
    Table 3 Cloud server information

    Parameter

    Description

    Name

    Cloud server name.

    Click the name of a cloud server to view its overview, copy data, and job information.

    • Overview

      The name, SLA information, job execution time, total number of copies, and EVS disk information are displayed.

    • Copy Data

      You can search for copies by year, month, or day.

      If is displayed below the time, a copy at that time exists. Click a date to go to the copy data page.

    • Jobs

      All jobs, including backup and restoration jobs, are displayed.

    Status
    Status of the cloud server.
    • Online: The cloud server in Huawei Cloud Stack is running or being stopped.
    • Offline: The cloud server in Huawei Cloud Stack is starting, stopped, reinstalling the OS, or changing the OS, or fails to reinstall the OS or switch the OS.
    • Exception: The cloud server in Huawei Cloud Stack is abnormal.
    • Soft deleted: The cloud server in Huawei Cloud Stack has been deleted and moved to the recycle bin.

    Location

    Cloud server location.

    SLAs

    SLA associated with a cloud server.

    SLA Compliance
    Whether a backup job is successfully executed based on the backup interval or backup window set in the SLA.
    • Met: If the backup time of a backup job (except for the first full backup) meets the backup time window or backup interval requirements and the backup job is executed successfully, SLA compliance is met.
    • Missed: If the backup time does not meet the backup time window or backup interval requirements, for example, if the backup interval set in the SLA for a backup job is once every half an hour but the backup time exceeds half an hour, SLA compliance is missed. In this case, you are advised to adjust the backup interval or backup window.

    Protection Status

    Whether a cloud server is protected.

    • Unprotected: The cloud server is not associated with an SLA, or the cloud server is associated with an SLA but protection is disabled.
    • Protected: The cloud server has been associated with an SLA and protection has been activated.
    • Creating: The cloud server is associated with an SLA and protection is being created.
    Tag
    NOTE:

    This parameter is available only in 1.6.0 and later versions.

    If a tag has been added for the resource, the tag name is displayed. Otherwise, -- is displayed.
