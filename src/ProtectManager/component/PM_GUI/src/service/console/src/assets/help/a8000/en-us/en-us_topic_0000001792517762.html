Step 3: Registering an Informix Instance

Procedure

Registering a single Informix instance

  1. Choose Protection > Databases > Informix.
  2. Click the Instance tab.
  3. Click Register to register an Informix instance.
  4. Set Type to Single instance.
  5. Configure the database instance and its authentication information.

    Table 1 describes the related parameters.

    Table 1 Registration information of a single Informix instance

    Parameter

    Description

    Name

    Instance name.

    Cluster

    Select a created cluster (host where the registered instance is located).

    Sqlhost file path

    Enter the absolute path of the Sqlhost file.

    Onconfig file path

    Enter the absolute path of the Onconfig file.

  6. Click OK.

Registering an Informix cluster instance

  1. On the Informix page, click the Instance tab.
  2. Click Register.
  3. Set Type to Cluster instance.
  4. Customize an instance name.
  5. Click Add. In the displayed dialog box, add all nodes of the cluster instance.

    Table 2 describes the related parameters.

    Table 2 Registration information of a single Informix instance

    Parameter

    Description

    Name

    Instance name.

    Cluster

    Select a created cluster.

    Sqlhost file path

    Enter the absolute path of the Sqlhost file.

    Onconfig file path

    Enter the absolute path of the Onconfig file.

  6. Click OK.
