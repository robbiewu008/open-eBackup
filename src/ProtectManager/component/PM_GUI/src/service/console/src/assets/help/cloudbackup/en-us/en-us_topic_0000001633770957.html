Resetting the User Password

This section describes operations to recover or reset the user password if a user has forgotten the password.

Precautions

  • Only system administrators can reset user passwords. The password of user sysadmin cannot be reset.
  • If the number of times for which a user enters the login password incorrectly reaches the upper limit, the user cannot use the password resetting function on that terminal.

Procedure

  1. Choose System > Security > Users and Roles > Users.
  2. (Optional) Click next to Role to filter users. You can also click next to Name to search for a user.
  3. Click More on the right of a user and choose Reset Password.
  4. Set Current Login Password, New Password, and Confirm Password.

    • To ensure account security, change the password upon the first login to the system.
    • To ensure account security, change the password regularly.

  5. Click OK.
