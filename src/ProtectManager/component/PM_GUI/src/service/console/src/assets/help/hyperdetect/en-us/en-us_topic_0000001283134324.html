Viewing SLA Information

Procedure

  1. Choose Protect > Protection Policies > SLA.
  2. View SLA information.

    Table 1 describes the related parameters.
    Table 1 SLA parameters

    Parameter

    Description

    Name

    Name of an SLA.

    Application Type

    Application type associated with an SLA, that is, Local File System.

    Protection Policy

    Protection policy created in the SLA, that is, Ransomware Detection Snapshot.

    Associated Resources

    Number of resources associated with an SLA.

    You can click the number of resources to view resource details.
