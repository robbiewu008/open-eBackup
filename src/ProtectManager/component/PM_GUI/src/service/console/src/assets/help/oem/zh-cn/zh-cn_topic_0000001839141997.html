查询GaussDB(DWS)信息

操作步骤

  1. 选择“保护 > 大数据 > GaussDB(DWS)”。
  2. 在“集群”页签，查询GaussDB(DWS)集群信息。

    1. 该页面展示已注册至本产品的GaussDB(DWS)集群信息。

      相关参数说明如表1所示。

      表1 GaussDB(DWS)集群信息

      参数

      说明

      名称

      GaussDB(DWS)集群名称。

      状态

      GaussDB(DWS)集群的状态。

      • 离线：当代理主机与GaussDB(DWS)集群连接异常时，“状态”为“离线”。
      • 正常：GaussDB(DWS)集群可用，且数据有冗余备份。所有进程都在运行，主备关系正常。
      • 不可用：GaussDB(DWS)集群不可用。
      • 降级：GaussDB(DWS)集群可用，但数据没有冗余备份。
      • 未启动：GaussDB(DWS)集群未启动。

      版本

      GaussDB(DWS)的版本号。

      SLA

      当资源已关联SLA时，则显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      GaussDB(DWS)集群当前是否被保护。

      • 未保护：GaussDB(DWS)集群未关联SLA，或者已关联SLA但已禁用保护。
      • 已保护：GaussDB(DWS)集群已关联SLA，且已激活保护。
      • 创建中：GaussDB(DWS)集群已关联SLA，正在创建保护。

      所属用户

      将资源分配给某个数据保护管理员管理。当界面显示“--”时，表示还未将资源分配给数据保护管理员。
    2. 单击集群名称，查看总览信息、副本信息以及相关任务。

  3. 在“Schema集”、“表集”页签，查看GaussDB(DWS)Schema集/表集信息。

    1. 该页面展示GaussDB(DWS)Schema集、表集的详细信息。

      相关参数说明如表2所示。

      表2 GaussDB(DWS)Schema集/表集参数说明

      参数

      说明

      名称

      GaussDB(DWS)Schema集/表集名称。

      所属集群

      GaussDB(DWS)Schema集/表集所属的集群。

      所属数据库

      GaussDB(DWS)Schema集/表集所属的数据库。

      SLA

      当GaussDB(DWS)Schema集/表集已关联SLA时，则显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      GaussDB(DWS)Schema集/表集当前是否被保护。

      • 未保护：GaussDB(DWS)Schema集/表集未关联SLA，或者已关联SLA但已禁用保护。
      • 已保护：GaussDB(DWS)Schema集/表集已关联SLA，且已激活保护。
      • 创建中：GaussDB(DWS)Schema集/表集已关联SLA，正在创建保护。
    2. 单击名称，查看总览信息、副本信息以及相关任务。
      • 总览

        显示SLA信息、任务执行时间、副本总数等。

      • 副本数据

        您可以依次按照年、月、天查找相应时间的副本。

        时间下方显示，表示该时间存在副本。

      • 相关任务

        展示备份、恢复等任务。
