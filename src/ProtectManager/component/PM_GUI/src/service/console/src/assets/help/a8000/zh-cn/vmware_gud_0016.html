步骤1：检查并安装VMware Tools

执行应用一致性备份或文件级恢复时，需要在受保护虚拟机或目标虚拟机安装VMware Tools。以下以vSphere Client 8.0版本为例进行说明。

背景信息

当存在以下场景时需要为VMware虚拟机安装VMware Tools：

  • 如果需要执行应用一致性备份，那么要为受保护的VMware虚拟机安装VMware Tools。
  • 如果需要执行文件级恢复，那么要为目标VMware虚拟机安装VMware Tools。

操作步骤

  1. 使用浏览器，登录vCenter Server或ESXi。

    登录地址：https://vCenter Server或ESXi的管理IP地址

  2. 展开左侧导航栏，查看所有已创建的虚拟机。
  3. 在虚拟机列表中，找到需要执行应用一致性备份或文件级恢复的虚拟机，单击该虚拟机的名称。
  4. 在虚拟机详情页面，查看是否已安装10.2及以上版本的VMware Tools。如果未安装或为10.2之前的版本，请参考VMware Tools文档安装或升级VMware Tools。

    • 如果登录vCenter Server，请在“虚拟机详细信息”区域中查看。
    • 如果登录ESXi，请在“一般信息”区域中查看。
