Modifying a User

The OceanProtect allows you to modify the information of existing users.

Precautions

User information cannot be modified by built-in accounts.

Procedure

  1. Choose System > Security > RBAC > Users.

    For 1.5.0, choose System > Security > Users and Roles > Users.

  2. (Optional) Click next to Role to filter users. You can also click next to Name to search for a user.
  3. Click More on the right of a user whose information needs to be modified and choose Modify.
  4. Modify the user information. Table 1 lists the parameters.

    Table 1 User parameters

    Parameter

    Description

    Login Authentication Mode

    Login authentication mode of a local user. Optional values are Login password and Login password+Email One-time password.

    Description

    Description of a user.

    [Value range]

    The description can be left blank or contain up to 255 characters.

    [Example]

    User

    Max. Number of User Connections

    Maximum number of concurrent logins to the system using a user. If this parameter is not set, the number is not limited.

    • : enabled
    • : disabled

    The default status is .

    [Value range]

    The value ranges from 1 to 8.

    Default value: 5

    NOTE:

    When Role is set to Remote Device Administrator, you do not need to set this parameter.

  5. Click OK.
