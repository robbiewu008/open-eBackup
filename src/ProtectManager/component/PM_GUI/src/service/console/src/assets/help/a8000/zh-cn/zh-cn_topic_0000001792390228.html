添加源端重删用户到sudoers（适用于Linux OS）

通过自动推送方式安装ProtectAgent时，“用户类型”配置为“超级管理员”，且需开启源端重删功能时，请在首次备份前，执行本节操作将源端重删用户加到sudoers中。

操作步骤

  1. 使用root账户登录代理主机。
  2. 执行以下命令查找sudoers配置文件的目录。

    whereis sudoers

  3. 执行以下命令修改sudoers配置文件权限。

    chmod u+w /etc/sudoers

  4. 备份源文件。

    mkdir /etc/bak_sudoers
cp /etc/sudoers /etc/bak_sudoers

  5. 执行以下命令打开/etc/sudoers配置文件。

    vim /etc/sudoers

  6. 检查配置文件中是否存在“Defaults requiretty”字段。如果存在，请执行以下操作：

    • 方式一：删除“Defaults requiretty”字段。
    • 方式二：添加源端重删用户到sudoers中。
      • 如果需进行VMware的源端重删，需要将VMware的DataTurbo用户VMDataTurbo添加到配置文件中。
        Defaults:VMDataTurbo !requiretty
      • 如果需进行其他应用的源端重删，需要将该应用对应的DataTurbo用户data_turbo_account添加到配置文件中。
        Defaults:data_turbo_account !requiretty

  7. 查看配置文件，确认是否修改成功。

    cat /etc/sudoers
    • 如果修改失败，请回退配置文件，确认回退成功后，重新执行5和6。
      cp /etc/backup_sudoers/sudoers /etc/sudoers
cat /etc/sudoers
    • 如果修改成功，请删除备份文件。
      rm -fr /etc/backup_sudoers
