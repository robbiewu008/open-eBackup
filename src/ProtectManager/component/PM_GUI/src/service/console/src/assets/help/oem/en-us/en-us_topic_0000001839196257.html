Adding Anonymization Rules

Common anonymization rules have been preset in the product. If the preset anonymization rules do not meet your requirements, you can customize some rules.

For details about the types of anonymization rules, see Anonymization Rule Types.

Procedure

  1. On the top navigation bar, choose Explore > Data Utilization Policies. In the left navigation pane, click Anonymization Policies.
  2. In the right Anonymization Policies pane, click the Anonymization Rules tab.
  3. Click Add, and set anonymization rules as prompted.

    Table 1 provides an example of an anonymization rule.

    Table 1 Anonymization rule example

    Anonymization Rule Type

    Parameter & Example Value

    Source Data

    Rule Example

    Rule Example Description

    Full-Mask

    Mask Character: *

    abcde

    ***

    Uses 3-digit asterisks (*) to anonymize all source data.

    Mask Character Length: 3

    Partial-Mask

    Mask Character: *

    abcdefg

    ***fg

    Uses 3-digit asterisks (*) to anonymize the first to fifth digits of source data.

    Mask Character Length: 3

    Mask Position: The start character is located at position 1 while the end character is located at position 5.

    Numeric-Range

    Random Number Range: The minimum number is 1 and the maximum number is 5.

    1234567

    4

    Replaces source data with a number ranging from 1 to 5.

    Fixed-Number

    Specified Value: 5

    1234567

    5

    Replaces source data with the number 5.

  4. Click OK.
