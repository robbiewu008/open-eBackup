Step 2: Creating a Periodic Archive Plan for Replication Copies

This section describes how to associate a resource with an SLA if you need to periodically archive replication copies generated by the resource.

Procedure

  1. Log in to the management page of the cluster where the replication copies reside.
  2. Choose Explore > Copy Data > Databases > TDSQL.
  3. Click the Resources tab.
  4. Locate the resource for which you want to archive replication copies and choose More > Protect.
  5. Select the created archive SLA.

    You can also click Create to create an SLA.

  6. Click OK.
