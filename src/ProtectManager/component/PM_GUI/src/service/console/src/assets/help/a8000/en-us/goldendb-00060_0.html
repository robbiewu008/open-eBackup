Managing Instances

The OceanProtect allows you to perform operations on databases, such as modifying, removing, activating, and disabling protection.

Log in to the management page, choose Protection > Databases > GoldenDB, click Instance, and find the target instance.

Table 1 describes the related operations.

Table 1 Operations related to instance management

Operation

Description

Navigation Path

Modifying protection

Scenario

This operation enables you to modify a protection plan to perform SLA association again for an instance or modify user-defined script information. After the modification, the next backup will be executed based on the new protection plan.

Note

  • The modification does not affect the protection job that is being executed. The modified protection plan will apply to the next protection period.
  • If the protection policy for a resource in the associated SLA changes, you are advised to modify the current SLA on the Protection > Protection Policies > SLAs page, rather than associate the resource with another SLA through protection modification.

    After you associate the resource with another SLA by modifying protection, if the copy of the original SLA still exists, the license capacity may be deducted repeatedly.

In the row of the target database, choose More > Modify Protection.

Removing protection

Scenario

This operation enables you to remove the protection plan if a database no longer needs to be protected. After the removal, the system cannot execute protection jobs for the database.

Note

  • When a protection job of the database is running, the protection cannot be removed.
  • If the protection policy for a resource in the associated SLA changes, you are advised to modify the associated SLA on the Protection > Protection Policies > SLAs page, rather than perform SLA re-association after protection removal.

    After you remove protection and perform SLA re-association, if the copy of the original SLA still exists, the license capacity may be deducted repeatedly.

In the row of the target instance, choose More > Remove Protection.

Activating protection

Scenario

This operation enables you to activate a disabled protection plan for an instance. After the protection plan is enabled, the system backs up the instance based on the protection plan.

Note

This operation can be performed only on instances that have been associated with an SLA and are in the Unprotected state.

In the row of the target instance, choose More > Activate Protection.

Disabling protection

Scenario

This operation enables you to disable the protection plan of an instance for which periodic protection is not required. After the protection plan is disabled, the system does not execute automatic backup jobs for the instance.

Note

Disabling protection does not affect ongoing backup jobs.

In the row of the target instance, choose More > Disable Protection.

Performing manual backup

Scenario

This operation enables you to perform a backup immediately.

Note

Copies generated by manual backup are retained for the duration defined in the SLA.

In the row of the target instance, choose More > Manual Backup.

Testing connectivity

Scenario

This operation tests the connectivity between the OceanProtect and the protected environment.

In the row of the target resource, choose More > Connectivity Test.

Modifying database information

Scenario

This operation enables you to modify information such as the database name and database authentication method.

In the row of the target database, choose More > Modify.

Deleting a database

Scenario

This operation enables you to delete a database that is no longer used.

Note

Before deleting a database, remove protection for the database.

In the row of the target database, choose More > Delete.
Adding a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to quickly filter and manage resources.
Choose More > Add Tag.
NOTE:
  • On the displayed Add Tag dialog box, you can select existing tags or click Create to create a tag.
  • To modify or delete a tag, click Go to Tag Management on the Add Tag page to go to the Tag Management page.
Removing a tag
NOTE:

Only 1.6.0 and later versions support this function.

Scenario

This operation enables you to remove the tag of a resource when it is no longer needed.

Choose More > Remove Tag.
