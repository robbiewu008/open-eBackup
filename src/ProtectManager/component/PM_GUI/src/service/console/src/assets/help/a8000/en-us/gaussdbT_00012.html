Step 3: Registering the GaussDB T Database

After ProtectAgent is installed on the host where the GaussDB T database resides, the OceanProtect automatically discovers the GaussDB T database host. You need to register the database instance on the host for protection or using it as the restoration target.

Procedure

  1. Choose Protection > Databases > GaussDB T.
  2. Click Register to register a new GaussDB T.
  3. Configure database information.

    In Type, select Single Server or Clusters.

    Table 1 describes the parameters related to the single-node system type. Table 2 describes the parameters related to the cluster type.

    Table 1 Parameters of the GaussDB T single-node system

    Parameter

    Description

    Name

    Name of the GaussDB T database to be registered.

    Username

    Username of the OS running the GaussDB T single-node system.

    Deployment Mode

    Deployment mode of the database. For the single-node system, only Standalone is available.

    Nodes

    Node to be registered. Only one node can be selected.

    Authentication Method

    OS authentication and database authentication.

    System database password

    Password of the database system.

    This parameter needs to be set only when the Authentication Method is set to Database authentication.
    Table 2 GaussDB T cluster type parameters

    Parameter

    Description

    Name

    Name of the GaussDB T database to be registered.

    Username

    Username of the OS running the GaussDB T cluster.

    Deployment Mode

    This parameter can be set to Standalone or Active/Standby.

    Nodes

    Node to be registered.

  4. Click OK.
