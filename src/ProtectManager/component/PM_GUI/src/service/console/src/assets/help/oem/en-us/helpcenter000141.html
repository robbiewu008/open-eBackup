Deleting the SAML SSO Configuration

If you do not need to use a SAML SSO configuration, delete it by following the instructions provided in this section.

Precautions

The SAML SSO configuration in the Activated state cannot be deleted.

Procedure

  1. Choose System > Security > SAML SSO Configuration.
  2. Locate the row that contains the target SAML SSO configuration, and click Delete.
  3. In the Danger dialog box that is displayed, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
