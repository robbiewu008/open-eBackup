查看客户端信息

当主机安装客户端后，您可以在管理界面查看客户端的详细信息。

注意事项

不同版本的界面可能存在差异，请以实际为准。

操作步骤

  1. 选择“保护 > 客户端 > 客户端”。
  2. 查看客户端信息。

    相关参数说明如所示表1。

    部分参数默认不展示，您可以单击“操作”列后的，勾选对应参数，即可在当前界面展示。

    表1 客户端信息

    参数

    说明

    资源ID

    客户端的ID。

    名称

    客户端的名称。

    IP地址

    客户端所在主机的IP地址。

    状态
    客户端的状态。
    • 在线：客户端所在主机处于开机状态且客户端正常运行时，“状态”为“在线”。
    • 离线：客户端所在主机处于关机状态或者主机上的客户端故障时，“状态”为“离线”。可参考客户端离线原因排查章节排查离线原因。
    • 更新排队中：客户端所在主机处于准备更新客户端中。
    • 更新中：客户端所在主机处于更新客户端中。
    • 修改应用类型中：客户端所在主机处于修改支持保护的资源类型中。

    CPU占用率

    客户端占用主机CPU的百分比。

    内存占用率

    客户端占用主机物理内存的百分比。

    操作系统

    客户端所在主机的操作系统类型。
    客户端类型
    说明：

    对于1.5.0版本，此参数名为“代理类型”

    客户端的类型。包括“VMware备份客户端”、“通用备份客户端”和“SAN Client”。

    当前版本

    主机上安装的客户端版本。
    所属用户
    说明：

    仅1.5.0版本存在此参数。

    客户端所属的数据保护管理员。

    当该客户端已授权给其他数据保护管理员时，将显示数据保护管理员的名称。如果未进行授权，则显示“--”。
    是否支持源端重删
    说明：

    本产品部分型号不支持此参数，详细支持信息请参考技术支持网站发布的产品文档或咨询产品提供商。
    部分主机操作系统支持源端重删功能。
    • 是：表示主机安装或升级客户端时，DataTurbo服务存在。
    • 否：表示主机安装或升级客户端时，DataTurbo服务不存在。
    说明：
    • 配置源端重删功能具体操作请参见对应版本的《源端重删用户指南》。
    • 开启源端重删功能后，请在首次备份前将源端重删用户加到sudoers中，具体操作请参见添加源端重删用户到sudoers（适用于Linux OS）。

    日志级别

    客户端日志的日志级别。包括“运行”、“警告”、“错误”、“调试”。

    共享

    说明：

    仅1.6.0及后续版本存在此参数。

    客户端的共享状态。

    安装目录

    客户端的安装目录。

    备注

    客户端的备注信息。
    标签
    说明：

    仅1.6.0及后续版本存在此参数。

    当客户端已添加标签时，则此处显示标签名称，否则显示“--”。
