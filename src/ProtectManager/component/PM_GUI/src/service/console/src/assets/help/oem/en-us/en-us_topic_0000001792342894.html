Using Cluster HA

After the cluster HA feature is configured, copies can be stored on the backup storage unit corresponding to the member node. You can also monitor the status of the standby node or the member node on the active node. For example, you can monitor the alarms, performance, and jobs of the member node on the active node, or generate reports by member node.

Backup

After the cluster HA feature is configured, you can specify the target location of backup copies in the SLA during resource backup.

  • If you select a backup storage unit, copies are stored to the storage unit. If the capacity of the storage unit is insufficient, the backup job will fail.
  • If you select a backup storage unit group, the system automatically selects a backup target storage unit based on the storage policy of the storage unit group.
  • If you do not specify the backup target location, the system automatically selects a backup target storage unit through intelligent balancing.

Replication

Two replication modes are supported: Cross-domain and Intra-domain. After the cluster HA feature is configured, you can select a backup storage unit group for intra-domain replication to implement automatic switchover of the target location of replication copies.

  • Cross-domain replication: Select a replication target cluster. The copy will be replicated from the local cluster to the target cluster.
  • Intra-domain replication: Select a storage unit group. When a backup copy is replicated for the first time, the system selects a node from the replication target end as the replication target location. In subsequent replication, the backup storage unit used for the first replication is preferentially selected.

Monitoring Alarms of Member Nodes

Choose Insight > Alarms and Events > Alarms. On the displayed page, select a node and view the alarm information.

Monitoring the Performance of Member Nodes

Choose Insight > Performance. On the displayed page, select a node and view the performance information.

Monitoring the Job of Member Nodes

Choose Insight > Jobs, click on the right of the displayed page, select Nodes to view the jobs of the active node and member nodes.

Generating a Report by Member Node

Choose Insight > Reports. On the displayed page, create a global report or create a report by member node.
