Managing the Detection Mode

This section describes how to enable or disable dynamic detection.

Procedure

  1. Choose Explore > Ransomware Protection.
  2. Click Overview.
  3. In the Settings area, click Modify.
  4. Select whether to enable dynamic detection.

    • Dynamic detection is enabled by default. Change behaviors of copy data are detected. If dynamic detection is disabled, static detection is performed based on known ransomware features of a single copy. You can log in to the technical support website and obtain the Readme file in the detection model package to query supported anti-ransomware software.
    • For differences between static detection and dynamic detection, see Implementation Principles.
    • Enabling or disabling dynamic detection does not affect ongoing detection jobs.

  5. Click Save.
