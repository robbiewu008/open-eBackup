Registering a Database (Applicable to the Cluster Instance Deployment Mode)

This section describes how to register a cluster instance when the protected database is deployed in a cluster instance.

Prerequisites

Before registering a DB2 cluster instance, ensure that the DB2 cluster has been registered.

Procedure

  1. On the DB2 page, click the Instance tab.
  2. Click Register.
  3. Set Type to Cluster instance.
  4. Customize an instance name and select a cluster.
  5. Click Add. In the dialog box that is displayed, add all nodes of the cluster instance.

    Table 1 describes the related parameters.

    Table 1 Node information of a cluster instance

    Parameter

    Description

    Hosts

    Host node in the cluster.

    Username

    Name of the database user.

    Password

    Password of the database user.

  6. Click OK.
