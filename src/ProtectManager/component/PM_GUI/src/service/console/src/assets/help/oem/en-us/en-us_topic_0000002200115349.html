Unlocking a User

The system administrator can unlock a user.

Prerequisites

  • Only the system administrator can unlock a user.
  • Lock Status of the user to be unlocked is Locked.

Procedure

  1. Choose System > Security > RBAC > Users.
  2. (Optional) Click next to Name to search for a user.
  3. In the row of the target user, choose More > Unlock on the right.
  4. In the dialog box that is displayed, enter the password of the current login user and click OK.
