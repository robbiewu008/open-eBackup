Changing the Password of an SFTP User

The product allows you to change the password of an SFTP user.

Prerequisites

The SFTP service has been enabled.

Procedure

  1. In the navigation pane, choose System > Settings > SFTP Service.
  2. Click More on the right of the user whose password needs to be changed and choose Change Password.
  3. Set Original Password, New Password, and Confirm Password.
  4. Click OK.
