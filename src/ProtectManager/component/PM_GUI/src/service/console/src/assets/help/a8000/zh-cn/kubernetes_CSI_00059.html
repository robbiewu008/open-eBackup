查看信息

当Kubernetes集群注册至OceanProtect后，您可以查看集群、命名空间和数据集信息。

操作步骤

  1. 选择“保护 > 容器 > Kubernetes CSI”。
  2. 在“集群”页签浏览集群信息。

    集群的详细信息如表1所示。
    表1 集群信息

    参数

    说明

    名称

    集群名称。

    单击名称，可以查看集群的基本信息和命名空间信息。

    版本

    Kubernetes集群版本号。

    状态

    集群状态，包含“在线”和“离线”。

    • 在线：当OceanProtect与Kubernetes集群连接正常时，“状态”为“在线”。
    • 离线：当OceanProtect与Kubernetes集群连接异常时，“状态”为“离线”。

    IP地址

    集群的管理IP地址。
    标签
    说明：

    仅1.6.0及后续版本存在此参数。

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
    所属用户
    说明：

    仅1.5.0版本存在此参数。

    集群资源分配给某个数据保护管理员管理。当界面显示“--”时，表示还未将资源分配给数据保护管理员。

  3. 在“命名空间”页签浏览命名空间信息。

    命名空间的详细信息如表2所示。
    表2 命名空间信息

    参数

    说明

    名称

    命名空间名称。

    单击名称，可以查看命名空间的总览、副本数据、任务信息。

    • 总览

      显示名称、SLA信息、任务执行时间、副本总数、数据集信息等。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    所属集群

    命名空间所在的集群名称。

    集群IP地址

    命名空间所在集群的管理IP地址。

    SLA

    当命名空间已关联SLA时，则显示SLA名称，否则显示“--”。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态
    命名空间当前是否被保护。
    • 未保护：命名空间未关联SLA，或者命名空间已关联SLA但已禁用保护。
    • 已保护：命名空间已关联SLA，且已激活保护。
    • 创建中：命名空间已关联SLA，正在创建保护。
    标签
    说明：

    仅1.6.0及后续版本存在此参数。

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。

  4. 在“数据集”页签浏览表信息。

    数据集的详细信息如表3所示。
    表3 数据集信息

    参数

    说明

    名称

    数据集名称。

    单击名称，可以查看数据集的总览、副本数据、任务信息。

    • 总览

      显示名称、SLA信息、任务执行时间、副本总数、卷信息等。

    • 副本数据

      您可以依次按照年、月、天查找相应时间的副本。

      时间下方显示，表示该时间存在副本。单击日期进入副本数据页面。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    所属命名空间

    数据集所在的命名空间名称。

    所属集群

    数据集所在的集群名称。

    SLA

    数据集关联的SLA。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态
    数据集当前是否被保护。
    • 未保护：数据集未关联SLA，或者数据集已关联SLA但已禁用保护。
    • 已保护：数据集已关联SLA，且已激活保护。
    • 创建中：数据集已关联SLA，正在创建保护。
    标签
    说明：

    仅1.6.0及后续版本存在此参数。

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
