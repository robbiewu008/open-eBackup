Modifying a Backup Storage Unit Group

This section describes how to modify information about a backup storage unit group.

Precautions

  • The name of a backup storage unit group cannot be changed.
  • If parallel data storage is enabled for a backup storage unit group and a copy exists, the backup storage units of the group cannot be deleted.
  • After a backup storage unit group for which parallel data storage is disabled is associated with an SLA policy of a non-DWS resource, parallel data storage cannot be enabled when the backup storage unit group is being modified.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Storage Unit Groups.
  3. In the row that contains a backup storage unit group, choose More > Modify.
  4. Click OK.
