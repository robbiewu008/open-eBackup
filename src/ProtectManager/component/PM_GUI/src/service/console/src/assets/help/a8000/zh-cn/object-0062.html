查看对象存储信息

创建对象存储后，您可以在管理界面查看对象存储的详细信息。

操作步骤

  1. 选择“保护 > 文件系统 > 对象存储”。
  2. 在“对象存储”页签浏览对象存储信息。

    该界面显示的是已注册的对象存储。

    1. 对象存储的详细信息如表1所示。
      表1 对象存储信息

      参数

      说明

      名称

      对象存储名称。

      状态

      对象存储状态。

      • 在线：当代理主机与对象存储连接正常时，“状态”为“在线”。
      • 离线：当代理主机与对象存储连接异常时，“状态”为“离线”。

      类型

      对象存储类型，包括：OceanStor Pacific、华为云Stack对象存储，阿里云对象存储。

      Endpoint

      对象存储的访问地址。

      地址格式：对象存储的域名或业务面IP地址。

      协议

      OceanProtect与对象存储之间的数据传输的协议。

      AK

      访问对象存储的接入证书。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
    2. 单击对象存储名称，浏览对象存储基本信息以及所属的对象集合。

  3. 在“对象集合”页签浏览对象集合信息。

    该界面显示的是已注册的对象集合。

    1. 对象集合的详细信息如表2所示。
      表2 对象集合信息

      参数

      说明

      名称

      对象集合名称。

      所属对象存储

      对象集合所属对象存储。

      所属对象存储类型

      对象集合所属对象存储类型。

      SLA

      当对象集合已关联SLA时，则显示SLA名称，否则显示“--”。

      SLA遵从度
      SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
      • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
      • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

      保护状态

      对象集合当前是否被保护。

      • 未保护：对象集合未关联SLA，或者对象集合已关联SLA但已禁用保护。
      • 已保护：对象集合已关联SLA，且已激活保护。
      • 创建中：对象集合已关联SLA，正在创建保护。
      标签
      说明：

      仅1.6.0及后续版本存在此参数。

      当资源已添加标签时，则此处显示标签名称。否则，显示“--”。
    2. 单击对象集合名称，浏览对象集合总览信息、副本信息以及任务。
      • 总览

        显示对象集合的名称、SLA信息、任务执行时间、副本总数等。

      • 副本数据

        您可以依次按照年、月、天查找相应时间的副本。

        时间下方显示，表示该时间存在副本。

      • 任务

        对象集合相关的所有任务展示，包括备份、恢复等任务。
