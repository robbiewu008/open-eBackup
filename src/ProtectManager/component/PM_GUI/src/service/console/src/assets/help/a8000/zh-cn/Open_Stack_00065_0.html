查看OpenStack信息

当OpenStack注册至OceanProtect后，您可以在管理界面查看OpenStack项目和云服务器信息。

操作步骤

  1. 选择“保护 > 云平台 > OpenStack”。
  2. 在“项目”页签浏览项目信息。

    项目的详细信息如表1所示。
    表1 项目信息

    参数

    说明

    名称

    项目名称。

    单击项目名称，可以查看项目的总览、任务。

    • 总览

      显示名称、SLA信息、云服务器信息等。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    位置

    项目所在位置。位置格式为：域/项目

    SLA

    当项目已关联SLA时，则显示SLA名称，否则显示“--”。

    SLA遵从度
    SLA遵从度指某次备份任务是否按设定的备份频率或备份窗口成功完成备份。
    • 遵从：如果某次备份任务（首次全量备份除外）的备份时间满足备份窗口或备份频率要求，且备份成功，则表示SLA遵从。
    • 不遵从：如果备份时间不满足备份窗口或备份频率要求，例如某次备份任务SLA设定的备份频率为每半小时执行一次，最终备份时间超过半小时，则表示SLA不遵从。建议调整备份频率或备份窗口。

    保护状态

    项目当前是否被保护。

    • 未保护：项目未关联SLA，或者项目已关联SLA但已禁用保护。
    • 已保护：项目已关联SLA，且已激活保护。
    • 创建中：项目已关联SLA，正在创建保护。
    标签
    说明：

    仅1.6.0及后续版本存在此参数。

    当资源已添加标签时，则此处显示标签名称。否则，显示“--”。

  3. 在“云服务器”页签浏览云服务器信息。

    云服务器的详细信息如表2所示。
    表2 云服务器信息

    参数

    说明

    名称

    云服务器名称。

    单击名称，可以查看云服务器的总览、副本数据、任务信息。

    • 总览

      显示名称、SLA信息、任务执行时间、副本总数、磁盘信息等。

    • 副本数据

      您可以依次按照年、月、天查找相应时间的副本。

      时间下方显示，表示该时间存在副本。单击日期进入副本数据页面。

    • 任务

      所有任务展示，包括备份、恢复等任务。

    状态

    云服务器状态。包括在线、离线、挂起。

    位置

    云服务器所在位置。位置格式：域/项目/云服务器

    SLA

    云服务器关联的SLA。
