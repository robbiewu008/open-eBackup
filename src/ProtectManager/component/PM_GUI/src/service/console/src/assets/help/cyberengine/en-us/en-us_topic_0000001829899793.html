Scanning for File Systems

After a file name extension filtering rule is applied to a file system, the system automatically performs file system scanning every hour. If a new file system is detected in the system, all preset file name extension filtering rules will be associated with the new file system. You can also manually scan files by referring to this section.

Procedure

  1. In the navigation pane, choose Data Security > File Interception.
  2. On the File Systems page, click Scan.
