Viewing SLA Information

Procedure

  1. Choose Protection > Protection Policies > SLAs.
  2. View SLA information.

    Table 1 describes the related parameters.
    Table 1 SLA parameters

    Parameter

    Description

    Name

    SLA name.

    Click the name of an SLA to view its basic information and associated resources.

    Application Type

    Application type associated with the SLA.

    Protection Policy

    Protection policy created in the SLA.

    Associated Resources

    Number of resources associated with the SLA.

    You can click the number of the resources to view details.

    Owner

    User who creates the SLA.
