Windows OS恢复完成后部分磁盘处于Foreign状态

当磁盘类型为动态磁盘（dynamic disk）时，磁盘是在WinPE系统下进行裸机恢复。恢复完成后操作系统会认为这是一个外部动态磁盘（foreign dynamic disk），导致Windows OS恢复完成后部分磁盘处于Foreign状态。

前提条件

已登录Windows OS。

处理步骤

  1. 以Windows Server 2016为例，按“Windows+R”打开“运行”对话框。
  2. 在“运行”对话框中输入diskmgmt.msc进入磁盘管理界面，如图图1所示。

    图1 磁盘管理

  3. 右键单击外部磁盘，选择“导入外部磁盘”。
  4. 在导入外部磁盘的对话框中，勾选磁盘组，单击“确定”。
