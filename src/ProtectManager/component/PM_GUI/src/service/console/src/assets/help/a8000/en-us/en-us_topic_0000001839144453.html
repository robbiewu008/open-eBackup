Deleting a Replication Cluster

Precautions

An external cluster that has been set as the replication target cluster of the management cluster cannot be deleted.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click the Replication Clusters tab. The replication cluster page is displayed.
  1. In the row that contains the external cluster to be deleted, choose More > Delete.
  2. Read the warning message carefully, select I have read and understand the consequences associated with performing this operation., and click OK.
