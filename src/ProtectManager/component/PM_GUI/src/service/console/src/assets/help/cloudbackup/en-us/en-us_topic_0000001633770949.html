Viewing Local Storage Information

This section describes how to view local storage information on the Local Storage page.

Procedure

  1. Choose System > Infrastructure > Local Storage.
  2. In the Basic Info and Available Capacity fields on the Local Storage tab page, view the basic information and available capacity of the local storage.

    Table 1 describes related parameters.

    Table 1 Local storage parameters

    Parameter

    Description

    Basic Info

    Version

    Current version of the system.

    ESN

    Current ESN of the system.

    An equipment serial number (ESN) is a character string that uniquely identifies a device.

    Model

    Product model.

    WWN

    Current WWN of the system.

    A World Wide Name (WWN) is a globally unique name that identifies a connection or connection set on a network. It is principally used in Fibre Channel and SAS.

    Total Disk Capacity

    Total disk capacity of the system.

    Available Capacity

    Total Capacity

    Usable capacity of a storage pool (Total physical capacity of disks â€“ Capacity consumed by RAID and metadata).

    Threshold

    When the percentage of the storage pool's used capacity to its total capacity reaches the alarm threshold, the system generates an alarm. A proper capacity alarm threshold helps you monitor the capacity usage of a storage pool. The value ranges from 1 to 95. The default value is 80.

    Usage

    Percentage of the used capacity to the total capacity of the storage pool.

    Used Capacity

    Used capacity of the storage pool.

    Unused Capacity

    Unused capacity of the storage pool.
