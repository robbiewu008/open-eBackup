What Should I Do If the Oracle Environment Variable Registry Is Deleted?

If the environment variable registry of the Oracle host is deleted by mistake, the host is displayed as offline on the management page, and database backup and restoration jobs fail.

Procedure

  1. Log in to the Oracle database host as the system administrator.
  2. Press Win+R to open the Run window.
  3. Enter regedit to open the registry and check whether the Oracle registration content is correct. If the content is incorrect, correct it.

    The value of ORACLE_HOME must be added to the value of the system variable Path. The value of Path is the value of ORACLE_HOME plus bin, which are separated by semicolons (;).

    Example:

    If the value of ORACLE_HOME is C:\app\oracle\product\12.1.0\dbhome_1, the value of the system variable Path must contain C:\app\oracle\product\12.1.0\dbhome_1\bin.
