修改本地存储认证信息

本产品支持修改本地存储认证信息。当您在DeviceManager修改了对接用户名的密码后，还需要在管理界面更新本地存储认证的密码。

操作步骤

  1. 选择“系统 > 基础设施 > 集群管理”。
  2. 在“备份集群”页签的“本地集群节点”区域，单击节点名称。
  3. 在弹出的“节点详情”对话框中的“登录认证”区域和“业务认证”区域，单击右侧的“修改”或“更新密码”。
  4. 修改本产品与本地存储间认证的用户名和密码。

    • 请确保当系统有两个控制器时，单用户最大会话数至少为2；当系统有四个控制器时，单用户最大会话数至少为4。
    • 请确保从当前页面登录DeviceManager的用户已注销。

  5. 单击“保存”。
