Checking the Exchange Writer Status

Ensure that Exchange Writer has been installed on the Exchange server and is in the Stable state. You can run the vssadmin list writers command to check the status.

C:\Users\Administrator>vssadmin list writers

If the following information is displayed, the status is normal.

C:\Users\Administrator>vssadmin list writers
