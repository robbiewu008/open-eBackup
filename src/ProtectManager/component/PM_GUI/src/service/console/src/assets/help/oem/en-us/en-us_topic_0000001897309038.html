Manually Executing Backup

If you want to perform a backup immediately, you can manually perform the backup.

Context

  • During the execution of a manual backup job, the parameters (except Automatic Retry) defined in the SLA, such as Rate Limiting Policies, will take effect.
  • Copies generated by manual backup are retained for the duration defined in the SLA.

Procedure

  1. Choose Protection > File Systems > NAS File Systems.
  2. In the row where a NAS file system to be protected resides, click More > Manual Backup .

    You can select multiple resources to perform manual backup in batches. Select multiple resources and choose More > Manual Backup in the upper left corner of the page.

  3. Set the name of the copy generated by the manual backup.

    If this parameter is left empty, the system sets the name to backup_Timestamp by default.

  4. Click OK.
