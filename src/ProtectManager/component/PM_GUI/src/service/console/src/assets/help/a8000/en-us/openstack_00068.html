Managing Domains

The OceanProtect allows you to modify and delete domains. You need to log in to the management page and choose Protection > Cloud Platforms > OpenStack. On the Domain tab page, locate the target domain.

Table 1 describes the related operations.

Table 1 Relevant operations of managing domains

Operation

Description

Navigation Path

Modifying a domain

Scenario

If the username or password of a domain administrator is changed, perform this operation to change the username or password accordingly.

In the row of the target domain, choose More > Modify.

Change the username and password of the domain administrator.

Deleting a domain

Scenario

If a cloud server in a domain no longer needs to be protected or the domain will not be used as the restoration target, perform this operation to delete the domain.

Note

The domain cannot be deleted if it contains protected cloud servers. Remove the protection first.

In the row of the target domain, choose More > Delete.
