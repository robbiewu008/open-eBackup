Deleting a Member Node

This section describes how to delete a member node that is no longer used.

Precautions

  • A standby node cannot be deleted. If you need to delete a standby node, remove the HA member and then delete the member node.
  • A member node in the Setting or Deleting state cannot be deleted.

Procedure

  1. Choose System > Infrastructure > Cluster Management.
  2. Click Backup Cluster.
  3. In the row that contains the member node to be deleted, choose More > Delete.
  4. In the Danger dialog box, confirm the information, select I have read and understand the consequences associated with performing this operation., and click OK.
